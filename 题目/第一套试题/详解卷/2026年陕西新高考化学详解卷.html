一、选择题（本题共15小题，每小题3分，共45分）
第1题 【答案】A
【解析】
A项正确：石墨烯是碳的一种单质，属于无机非金属材料。
B项错误：光导纤维的主要成分是二氧化硅，属于无机非金属材料，不是合成纤维。
C项错误：医用酒精的浓度为75%，95%的酒精会使细菌表面蛋白质迅速凝固，形成保护膜，反而不能杀死细菌。
D项错误：明矾溶于水生成氢氧化铝胶体，能吸附水中悬浮物，但不能杀菌消毒。
【考点】化学与生活
第2题 【答案】C
【解析】
A项错误：中子数 = 质量数 - 质子数 = 238 - 92 = 146。
B项错误：235U与238U的质子数相同，中子数不同，互为同位素。
C项正确：235U与238U的核外电子数相同，化学性质几乎相同。
D项错误：235U的相对原子质量约为235，单位不是g。
【考点】原子结构；同位素
第3题 【答案】B
【解析】
A项错误：NaOH溶液会与玻璃中的SiO2反应，不能用玻璃塞，应使用橡胶塞。
B项正确：浓硝酸见光易分解，应保存在棕色试剂瓶中。
C项错误：金属钠应保存在煤油中，不能保存在水中（钠会与水反应）。
D项错误：氢氟酸会与玻璃反应，应保存在塑料瓶中。
【考点】化学试剂的保存
第4题 【答案】D
【解析】
A项错误：标准状况下，水不是气体，不能用气体摩尔体积计算。
B项错误：NO2与N2O4存在平衡：2NO2 ⇌ N2O4，46g混合气体中含有的分子数小于NA。
C项错误：1mol Fe与足量Cl2反应生成FeCl3，转移3NA个电子。
D项正确：1mol C2H6中含有7mol共价键（1个C-C键，6个C-H键）。
【考点】阿伏加德罗常数
第5题 【答案】A
【解析】
A项正确：苯与液溴在FeBr3催化下发生取代反应生成溴苯。
B项错误：乙烯与溴水发生加成反应，使溴水褪色；苯与溴水不反应，但能萃取溴水中的溴。
C项错误：乙醇与钠反应比水与钠反应平缓，说明乙醇中羟基的活泼性比水中的弱。
D项错误：乙酸乙酯在碱性条件下水解生成乙酸盐和乙醇，不是可逆反应。
【考点】有机化学基础
第6题 【答案】C
【解析】
W、X、Y、Z分别为H、C、O、Na。
A项错误：原子半径：Na > C > O > H。
B项错误：H2CO3是弱酸，不是强酸。
C项正确：Na2O2中含有离子键和共价键。
D项错误：H、C、O可以形成多种化合物，如CH3COOH、HCOOCH3等。
【考点】元素周期律；物质结构

第9题 【答案】A

B项错误：SO2具有漂白性，能使品红溶液褪色，加热后又恢复红色。

D项错误：Fe与稀HNO3反应生成NO，不是H2。
【考点】元素化合物性质
第10题 【答案】C
【解析】
A项错误：加入BaCl2溶液产生白色沉淀，可能是Ag+、CO32-、SO32-等，不一定是SO42-。
B项错误：Fe3+与I-会发生氧化还原反应，不能大量共存。
C项正确：先加稀盐酸排除干扰，再加BaCl2溶液，产生白色沉淀，说明含有SO42-。
D项错误：NH4+的检验应加入NaOH溶液并加热，产生能使湿润的红色石蕊试纸变蓝的气体。
【考点】离子检验
第11题 【答案】B
【解析】
A项错误：反应①中N元素化合价降低，NH3是还原剂。
B项正确：反应②中NO与O2反应生成NO2，氧化剂与还原剂的物质的量之比为1:2。
C项错误：反应③中NO2与H2O反应生成HNO3和NO，氧化产物与还原产物的物质的量之比为2:1。
D项错误：反应④为复分解反应，不是氧化还原反应。
【考点】氧化还原反应
第12题 【答案】D
【解析】
A项错误：稀释醋酸溶液，c(H+)减小，但c(OH-)增大。
B项错误：加入CH3COONa固体，CH3COO-浓度增大，平衡逆向移动，c(H+)减小，pH增大。
C项错误：加入NaOH固体，中和H+，CH3COOH电离程度增大。
D项正确：升高温度，CH3COOH电离程度增大，Ka增大。
【考点】弱电解质的电离平衡

第15题 【答案】B
【解析】
A项错误：pH=7时，c(H+)=c(OH-)，由电荷守恒可知c(Na+)=c(CH3COO-)。

二、非选择题（本题共4小题，共55分）
第16题（14分）【答案】
【答案】
（1）三颈烧瓶（1分）；平衡气压，使液体顺利滴下（1分）
（2）2MnO4- + 16H+ + 10Cl- = 2Mn2+ + 5Cl2↑ + 8H2O（2分）
（3）饱和食盐水（1分）；除去Cl2中的HCl（1分）
（4）水浴加热（1分）；温度过低反应速率慢，温度过高H2O2分解（2分）
（5）过滤（1分）；洗涤（1分）；干燥（1分）
（6）79.2%（2分）
【解析】
本实验制备CaO2·8H2O的流程：CaCl2与NH3·H2O、H2O2反应生成CaO2·8H2O。
（1）仪器a为三颈烧瓶，恒压滴液漏斗的作用是平衡气压。
（2）装置A中高锰酸钾与浓盐酸反应制取氯气。
（3）装置B中盛放饱和食盐水，用于除去Cl2中的HCl。
（4）控制温度在0-5℃，采用水浴加热；温度过低反应速率慢，温度过高H2O2分解。
（5）从溶液中获得晶体的操作：过滤、洗涤、干燥。
（6）计算产率：理论产量为10.8g，实际产量为8.55g，产率为79.2%。
【考点】化学实验综合
第17题（14分）【答案】
【答案】
（1）增大接触面积，加快反应速率，提高浸出率（2分）
（2）SiO2（1分）；CaSO4（1分）
（3）将Fe2+氧化为Fe3+，便于后续除杂（2分）
（4）3.2 ≤ pH < 4.7（2分）；Fe(OH)3（1分）
（5）Mn2+ + 2HCO3- = MnCO3↓ + CO2↑ + H2O（2分）
（6）蒸发浓缩、冷却结晶、过滤（2分）；取最后一次洗涤液，加入BaCl2溶液，若无白色沉淀生成，则已洗净（1分）
【解析】
本题以软锰矿为原料制备硫酸锰的工艺流程。
（1）粉碎矿石可增大接触面积，加快反应速率。
（2）滤渣1为不溶于酸的SiO2和微溶的CaSO4。
（3）加入MnO2将Fe2+氧化为Fe3+，便于调节pH除去。
（4）调节pH使Fe3+完全沉淀而Mn2+不沉淀，pH范围为3.2 ≤ pH < 4.7。
（5）Mn2+与NH4HCO3反应生成MnCO3沉淀。
（6）从溶液中获得晶体的操作：蒸发浓缩、冷却结晶、过滤。
【考点】化工流程

【答案】
（1）①CO(g) + 2H2(g) ⇌ CH3OH(g) ΔH = -90.1 kJ·mol-1（2分）
②低温（1分）；该反应为放热反应，低温有利于平衡正向移动（1分）
（2）①0.05 mol·L-1·min-1（2分）
②升高温度（1分）；加入催化剂（1分）
（3）①正极（1分）；CO2 + 6H+ + 6e- = CH3OH + H2O（2分）
②3.36（2分）

（1）①根据盖斯定律，由反应I - 反应II得目标反应。
②该反应为放热反应且气体分子数减少，低温高压有利于提高CO转化率。
（2）①0-10min内，CO浓度变化为0.5mol/L，v(CO) = 0.05 mol·L-1·min-1。
②改变条件使反应速率加快但平衡不移动，可能是升高温度或加入催化剂。
（3）①CO2在正极得电子被还原为CH3OH。
