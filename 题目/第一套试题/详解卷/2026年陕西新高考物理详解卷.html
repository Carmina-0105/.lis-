【考点】物理学史
第2题 【答案】B
【解析】
速度-时间图像中：
A项错误：第1s内和第3s内的速度方向相同（均为正方向）。
B项正确：第2s内速度为正，第4s内速度为负，速度方向相反。
C项错误：第2s末速度方向发生改变（从正变负），但加速度方向没有变。

【考点】匀变速直线运动的图像
第3题 【答案】D
【解析】
设地球质量为M，半径为R，卫星A的轨道半径为2R，卫星B的轨道半径为3R。
由万有引力提供向心力：
\(\frac{GMm}{r^2} = m\frac{v^2}{r} = m\omega^2 r = m\frac{4\pi^2}{T^2}r\)
线速度：\(v = \sqrt{\frac{GM}{r}}\)，所以 \(\frac{v_A}{v_B} = \sqrt{\frac{r_B}{r_A}} = \sqrt{\frac{3}{2}}\)。
角速度：\(\omega = \sqrt{\frac{GM}{r^3}}\)，所以 \(\frac{\omega_A}{\omega_B} = \sqrt{(\frac{r_B}{r_A})^3} = \sqrt{\frac{27}{8}}\)。
周期：\(T = 2\pi\sqrt{\frac{r^3}{GM}}\)，所以 \(\frac{T_A}{T_B} = \sqrt{(\frac{r_A}{r_B})^3} = \sqrt{\frac{8}{27}}\)。
向心加速度：\(a = \frac{GM}{r^2}\)，所以 \(\frac{a_A}{a_B} = (\frac{r_B}{r_A})^2 = \frac{9}{4}\)。
因此D正确。
【考点】万有引力定律；卫星运动
第4题 【答案】A
【解析】
小球从A点运动到B点，机械能守恒：
\(mgh = \frac{1}{2}mv_B^2\)
\(v_B = \sqrt{2gh}\)
在B点，由牛顿第二定律：
\(F_N - mg = m\frac{v_B^2}{R}\)
\(F_N = mg + m\frac{2gh}{R}\)
由牛顿第三定律，小球对轨道的压力大小等于轨道对小球的支持力。
【考点】机械能守恒定律；圆周运动
第5题 【答案】C
【解析】
带电粒子在匀强磁场中做匀速圆周运动，洛伦兹力提供向心力：
\(qvB = m\frac{v^2}{r}\)
\(r = \frac{mv}{qB}\)
周期：\(T = \frac{2\pi r}{v} = \frac{2\pi m}{qB}\)，与速度无关。
动能：\(E_k = \frac{1}{2}mv^2\)，速度越大，动能越大。
运动时间：粒子从P点运动到Q点，转过的圆心角为\(\theta\)，则 \(t = \frac{\theta}{2\pi}T\)。
若粒子速度增大，轨道半径增大，但圆心角减小，运动时间可能不变。
【考点】带电粒子在磁场中的运动
第6题 【答案】B
【解析】
原线圈电压 \(U_1 = 220V\)，副线圈电压 \(U_2 = 22V\)。
匝数比：\(\frac{n_1}{n_2} = \frac{U_1}{U_2} = \frac{220}{22} = 10\)。
副线圈电流：\(I_2 = \frac{U_2}{R} = \frac{22}{11} = 2A\)。
原线圈电流：\(\frac{I_1}{I_2} = \frac{n_2}{n_1}\)，所以 \(I_1 = \frac{I_2}{10} = 0.2A\)。
输入功率：\(P_1 = U_1 I_1 = 220 \times 0.2 = 44W\)。
或 \(P_1 = P_2 = \frac{U_2^2}{R} = \frac{22^2}{11} = 44W\)。
【考点】理想变压器
第7题 【答案】D
【解析】
设弹簧劲度系数为k，物块质量为m。
在B点（平衡位置）：\(kx_0 = mg\sin\theta\)。
从A到B，由动能定理：
\(mg\sin\theta \cdot 2x_0 - W_{弹} = \frac{1}{2}mv^2\)
弹力做功等于弹性势能的变化：\(W_{弹} = \frac{1}{2}kx_0^2 - \frac{1}{2}k(2x_0)^2 = -\frac{3}{2}kx_0^2\)。
由 \(kx_0 = mg\sin\theta\)，得：
\(2mg\sin\theta \cdot x_0 + \frac{3}{2}kx_0^2 = \frac{1}{2}mv^2\)
\(2kx_0^2 + \frac{3}{2}kx_0^2 = \frac{1}{2}mv^2\)
\(\frac{7}{2}kx_0^2 = \frac{1}{2}mv^2\)
\(v = \sqrt{\frac{7kx_0^2}{m}} = \sqrt{\frac{7gx_0\sin\theta}{1}}\)
【考点】功能关系；动能定理
二、多项选择题（本题共3小题，每小题6分，共18分）
第8题 【答案】BC
【解析】
根据电场线分布： A项错误：电场线从正电荷指向负电荷，所以A带正电，B带负电。
B项正确：电场线越密的地方场强越大，C点电场线比D点密，所以 \(E_C > E_D\)。
C项正确：沿电场线方向电势降低，C点电势高于D点电势。
D项错误：负电荷在电势高的地方电势能小，所以负电荷在C点的电势能小于在D点的电势能。
【考点】电场线；电场强度；电势；电势能
第9题 【答案】AD
【解析】
A项正确：由图乙可知，波的周期 \(T = 0.4s\)，波长 \(\lambda = 4m\)，波速 \(v = \frac{\lambda}{T} = 10m/s\)。
B项错误：波沿x轴正方向传播，P点向y轴负方向运动。
C项错误：经过0.2s（半个周期），P点运动的路程为 \(2A = 0.4m\)。
D项正确：Q点与P点相距半个波长，振动情况完全相反，当P点在波峰时，Q点在波谷。

第10题 【答案】BD
【解析】
导体棒在磁场中运动，产生感应电动势 \(E = BLv\)，感应电流 \(I = \frac{E}{R+r} = \frac{BLv}{R+r}\)。
A项错误：导体棒受到的安培力 \(F_A = BIL = \frac{B^2L^2v}{R+r}\)，方向水平向左。
B项正确：由能量守恒，导体棒克服安培力做的功等于回路中产生的焦耳热。
C项错误：导体棒最终停止，由动能定理：\(-\mu mgx - W_A = 0 - \frac{1}{2}mv_0^2\)，其中 \(W_A = Q_{总}\)。
D项正确：电阻R上产生的焦耳热 \(Q_R = \frac{R}{R+r}Q_{总} = \frac{R}{R+r}(\frac{1}{2}mv_0^2 - \mu mgx)\)。
【考点】电磁感应；能量守恒

第13题（10分）【答案】
【答案】
（1）\(\sqrt{3}\)（5分）
（2）\(\frac{\sqrt{3}}{3}c\)（5分）
【解析】
（1）光路图如图所示。光线在AB面入射，入射角 \(i = 60°\)。

第14题（13分）【答案】
【答案】
（1）\(v_0 = \sqrt{2gR}\)（5分）
（2）\(F_N = 3mg\)，方向竖直向下（8分）
【解析】
（1）物块从A点运动到C点，机械能守恒：
\(mgR = \frac{1}{2}mv_0^2\)
\(v_0 = \sqrt{2gR}\)

（2）物块与木板相互作用，动量守恒：
\(mv_0 = (m + M)v\)
\(v = \frac{mv_0}{m+M} = \frac{v_0}{2}\)（因为 \(m = M\)）
能量守恒：
\(\frac{1}{2}mv_0^2 = \frac{1}{2}(m+M)v^2 + \mu mgL\)
\(mgR = \frac{1}{2} \cdot 2m \cdot (\frac{v_0}{2})^2 + \mu mgL\)
\(mgR = \frac{1}{4}mv_0^2 + \mu mgL = \frac{1}{2}mgR + \mu mgL\)
\(\mu mgL = \frac{1}{2}mgR\)
\(L = \frac{R}{2\mu}\)
当物块运动到圆弧轨道最高点D时，由机械能守恒：
\(\frac{1}{2}mv_0^2 = \frac{1}{2}mv_D^2 + mg \cdot 2R\)
\(v_D = \sqrt{v_0^2 - 4gR} = \sqrt{2gR - 4gR}\)（此式有问题，需重新考虑）
【考点】动量守恒；能量守恒；圆周运动
第15题（16分）【答案】
【答案】
（1）\(E = \frac{mv_0^2}{2qL}\)，方向沿x轴正方向（6分）
（2）\(B = \frac{mv_0}{qL}\)，方向垂直纸面向外（5分）
（3）\(t_{总} = \frac{(4+3\pi)L}{2v_0}\)（5分）
【解析】
（1）粒子在电场中做类平抛运动。
x方向：\(2L = v_0t_1\)。
y方向：\(L = \frac{1}{2}at_1^2 = \frac{1}{2}\frac{qE}{m}t_1^2\)。
解得：\(E = \frac{mv_0^2}{2qL}\)。
粒子带正电，向y轴正方向偏转，所以电场方向沿x轴正方向。

（2）粒子进入磁场时，x方向速度 \(v_x = v_0\)，y方向速度 \(v_y = at_1 = \frac{qE}{m} \cdot \frac{2L}{v_0} = v_0\)。
所以速度大小 \(v = \sqrt{v_0^2 + v_0^2} = \sqrt{2}v_0\)，方向与x轴成45°角。
粒子从P点射出，由几何关系，轨道半径 \(r = \sqrt{2}L\)。
由 \(qvB = m\frac{v^2}{r}\)：
\(B = \frac{mv}{qr} = \frac{m\sqrt{2}v_0}{q\sqrt{2}L} = \frac{mv_0}{qL}\)
由左手定则，磁场方向垂直纸面向外。

（3）电场中运动时间：\(t_1 = \frac{2L}{v_0}\)。
磁场中运动时间：粒子转过270°，\(t_2 = \frac{3}{4}T = \frac{3}{4} \cdot \frac{2\pi m}{qB} = \frac{3\pi L}{2v_0}\)。
总时间：\(t_{总} = t_1 + t_2 = \frac{2L}{v_0} + \frac{3\pi L}{2v_0} = \frac{(4+3\pi)L}{2v_0}\)。
【考点】带电粒子在电磁场中的运动
