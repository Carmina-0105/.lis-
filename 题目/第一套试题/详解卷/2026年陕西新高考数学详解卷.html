第2题 【答案】C
【解析】
已知 \(|z| = 2\)，即 \(|a + bi| = \sqrt{a^2 + b^2} = 2\)，所以 \(a^2 + b^2 = 4\)。
计算 \(|z - 1|^2\)：
\(z - 1 = (a-1) + bi\)
\(|z - 1|^2 = (a-1)^2 + b^2 = a^2 - 2a + 1 + b^2 = (a^2 + b^2) - 2a + 1 = 4 - 2a + 1 = 5 - 2a\)
当 \(a = -2\) 时，\(|z - 1|^2\) 取得最大值 \(5 - 2 \times (-2) = 9\)。
因此 \(|z - 1|\) 的最大值为 \(3\)。

因为 \(d \neq 0\)，所以 \(d = \frac{5}{3}\)。

【考点】函数的周期性
第6题 【答案】C
【解析】
抛物线 \(y^2 = 4x\) 的焦点为 \(F(1, 0)\)，准线为 \(x = -1\)。
设 \(A(x_1, y_1)\)，\(B(x_2, y_2)\)。
由抛物线定义：\(|AF| = x_1 + 1\)，\(|BF| = x_2 + 1\)。
已知 \(|AF| = 2|BF|\)，所以 \(x_1 + 1 = 2(x_2 + 1)\)，即 \(x_1 = 2x_2 + 1\)。
设直线 \(AB\) 的方程为 \(x = my + 1\)，代入 \(y^2 = 4x\)：
\(y^2 = 4(my + 1)\)
\(y^2 - 4my - 4 = 0\)
由韦达定理：\(y_1 + y_2 = 4m\)，\(y_1 y_2 = -4\)。
由 \(x_1 = 2x_2 + 1\) 和 \(y_1^2 = 4x_1\)，\(y_2^2 = 4x_2\)：
\(\frac{y_1^2}{4} = 2 \cdot \frac{y_2^2}{4} + 1\)
\(y_1^2 = 2y_2^2 + 4\)
结合 \(y_1 y_2 = -4\)，设 \(y_2 = t\)，则 \(y_1 = -\frac{4}{t}\)：
\(\frac{16}{t^2} = 2t^2 + 4\)
\(16 = 2t^4 + 4t^2\)
\(t^4 + 2t^2 - 8 = 0\)
\((t^2 + 4)(t^2 - 2) = 0\)
所以 \(t^2 = 2\)，\(t = \pm\sqrt{2}\)。
计算 \(|AB|\)：
\(|AB| = |AF| + |BF| = 3|BF| = 3(x_2 + 1) = 3(\frac{y_2^2}{4} + 1) = 3(\frac{2}{4} + 1) = 3 \times \frac{3}{2} = \frac{9}{2}\)
【考点】抛物线的性质；直线与抛物线的位置关系
第7题 【答案】A
【解析】
函数 \(f(x) = \sin(\omega x + \frac{\pi}{6})\) 在 \((0, \frac{\pi}{3})\) 上单调递增。
求导：\(f'(x) = \omega \cos(\omega x + \frac{\pi}{6})\)。
要使 \(f(x)\) 在 \((0, \frac{\pi}{3})\) 上单调递增，需要 \(f'(x) \geq 0\)。
即 \(\cos(\omega x + \frac{\pi}{6}) \geq 0\) 在 \((0, \frac{\pi}{3})\) 上恒成立。
当 \(x \in (0, \frac{\pi}{3})\) 时，\(\omega x + \frac{\pi}{6} \in (\frac{\pi}{6}, \frac{\omega\pi}{3} + \frac{\pi}{6})\)。
要使 \(\cos(\theta) \geq 0\)，需要 \(\theta \in [-\frac{\pi}{2}, \frac{\pi}{2}]\)。
因此：
\(\frac{\omega\pi}{3} + \frac{\pi}{6} \leq \frac{\pi}{2}\)
\(\frac{\omega\pi}{3} \leq \frac{\pi}{3}\)
\(\omega \leq 1\)
又因为 \(\omega > 0\)，所以 \(0 < \omega \leq 1\)。
最大值为 \(1\)。
【考点】三角函数的单调性
第8题 【答案】D
【解析】
设 \(f(x) = e^x - ax^2 - bx - c\)。
由 \(f(x) \geq 0\) 对所有 \(x \in \mathbb{R}\) 成立，且等号可以取到。
求导：\(f'(x) = e^x - 2ax - b\)，\(f''(x) = e^x - 2a\)。
若 \(a \leq 0\)，则 \(f''(x) > 0\)，\(f'(x)\) 单调递增，\(f(x)\) 先减后增或单调递增，最小值在边界取到，不可能对所有 \(x\) 都非负。
所以 \(a > 0\)。
令 \(f''(x) = 0\)，得 \(x = \ln(2a)\)。
当 \(x < \ln(2a)\) 时，\(f''(x) < 0\)；当 \(x > \ln(2a)\) 时，\(f''(x) > 0\)。
所以 \(f'(x)\) 在 \(x = \ln(2a)\) 处取最小值。
为使 \(f(x) \geq 0\) 恒成立，需要 \(f(x)\) 在某点取到最小值 \(0\)。
设最小值点为 \(x_0\)，则 \(f(x_0) = 0\) 且 \(f'(x_0) = 0\)。
由 \(f'(x_0) = 0\)：\(e^{x_0} = 2ax_0 + b\)。
由 \(f(x_0) = 0\)：\(e^{x_0} = ax_0^2 + bx_0 + c\)。
消去 \(b\)：\(b = e^{x_0} - 2ax_0\)。
代入：\(e^{x_0} = ax_0^2 + (e^{x_0} - 2ax_0)x_0 + c = ax_0^2 + e^{x_0}x_0 - 2ax_0^2 + c\)。
整理：\(e^{x_0}(1 - x_0) = -ax_0^2 + c\)。
为使 \(b - c\) 最小，取 \(x_0 = 0\)：
\(e^0 = c\)，即 \(c = 1\)
\(b = e^0 - 0 = 1\)
此时 \(b - c = 0\)。
验证：当 \(a = \frac{1}{2}\)，\(b = 1\)，\(c = 1\) 时，\(f(x) = e^x - \frac{1}{2}x^2 - x - 1\)。
\(f(0) = 0\)，\(f'(0) = 0\)，\(f''(x) = e^x - 1\)，当 \(x > 0\) 时 \(f''(x) > 0\)，当 \(x < 0\) 时 \(f''(x) < 0\)。
\(f'(x)\) 在 \(x = 0\) 处取最小值 \(0\)，所以 \(f(x) \geq 0\)。
因此 \(b - c\) 的最小值为 \(0\)。

\(b^2 = 3a^2\)

等等，重新检查：若 \(e = \frac{2\sqrt{3}}{3}\)：

\(c^2 = \frac{4 \times 3}{9}a^2 = \frac{4}{3}a^2 = a^2 + b^2\)
\(b^2 = \frac{1}{3}a^2\)
\(\frac{b}{a} = \frac{\sqrt{3}}{3}\)
渐近线方程为 \(y = \pm \frac{\sqrt{3}}{3}x\)。

（1）\(A = \frac{\pi}{3}\)（6分）
（2）周长为 \(3 + \sqrt{3}\)（7分）
【解析】
（1）由正弦定理：\(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R\)
已知 \(2b\cos A = a\cos C + c\cos A\)
由正弦定理代入：
\(2\sin B\cos A = \sin A\cos C + \sin C\cos A = \sin(A+C) = \sin B\)
因为 \(\sin B \neq 0\)，所以 \(2\cos A = 1\)，\(\cos A = \frac{1}{2}\)。
因为 \(A \in (0, \pi)\)，所以 \(A = \frac{\pi}{3}\)。

（2）已知 \(a = 2\)，\(A = \frac{\pi}{3}\)。
由余弦定理：\(a^2 = b^2 + c^2 - 2bc\cos A\)
\(4 = b^2 + c^2 - bc\)
由基本不等式：\(b^2 + c^2 \geq 2bc\)，所以 \(4 = b^2 + c^2 - bc \geq 2bc - bc = bc\)。
当 \(b = c\) 时取等号，此时 \(4 = b^2 + b^2 - b^2 = b^2\)，\(b = 2\)。
所以 \(b = c = 2\)。
周长为 \(a + b + c = 2 + 2 + 2 = 6\)。
等等，若 \(a = \sqrt{3}\)：
\(3 = b^2 + c^2 - bc\)
当 \(b = c\) 时，\(3 = b^2\)，\(b = \sqrt{3}\)
周长 \(= 3\sqrt{3}\)

（2）建立空间直角坐标系。
设 \(AB = 2\)，则 \(PO = CO = \sqrt{3}\)。
取 \(O\) 为原点，\(OB\) 为 \(x\) 轴，\(OC\) 为 \(y\) 轴，过 \(O\) 作垂直于平面 \(ABC\) 的直线为 \(z\) 轴。
则 \(A(-1, 0, 0)\)，\(B(1, 0, 0)\)，\(C(0, \sqrt{3}, 0)\)。
设 \(P(0, y, z)\)，由 \(PO = \sqrt{3}\) 得 \(y^2 + z^2 = 3\)。
平面 \(ABC\) 的法向量为 \(\vec{n_1} = (0, 0, 1)\)。
设平面 \(PAB\) 的法向量为 \(\vec{n_2}\)。
\(\overrightarrow{PA} = (-1, -y, -z)\)，\(\overrightarrow{PB} = (1, -y, -z)\)。
由 \(\vec{n_2} \cdot \overrightarrow{PA} = 0\) 和 \(\vec{n_2} \cdot \overrightarrow{PB} = 0\) 得 \(\vec{n_2} = (0, -z, y)\)。
二面角余弦值为 \(\frac{1}{3}\)：
\(|\cos\theta| = \frac{|\vec{n_1} \cdot \vec{n_2}|}{|\vec{n_1}||\vec{n_2}|} = \frac{|y|}{\sqrt{y^2+z^2}} = \frac{|y|}{\sqrt{3}} = \frac{1}{3}\)
\(|y| = \frac{\sqrt{3}}{3}\)，\(y^2 = \frac{1}{3}\)
\(z^2 = 3 - \frac{1}{3} = \frac{8}{3}\)，\(|z| = \frac{2\sqrt{6}}{3}\)
点 \(C\) 到平面 \(PAB\) 的距离：
\(d = \frac{|\overrightarrow{CA} \cdot \vec{n_2}|}{|\vec{n_2}|} = \frac{|( -1, -\sqrt{3}, 0) \cdot (0, -z, y)|}{\sqrt{3}} = \frac{|\sqrt{3}z|}{\sqrt{3}} = |z| = \frac{2\sqrt{6}}{3}\)

第19题（17分）【答案】
【答案】
（1）单调递减区间为 \((-\infty, 0)\)，单调递增区间为 \((0, +\infty)\)；极小值为 \(f(0) = 1\)，无极大值（5分）
（2）证明见解析（6分）
（3）\(a \leq 1\)（6分）
【解析】
（1）\(f(x) = e^x - x\)，\(f'(x) = e^x - 1\)。
令 \(f'(x) = 0\)，得 \(x = 0\)。
当 \(x < 0\) 时，\(f'(x) < 0\)，\(f(x)\) 单调递减。
当 \(x > 0\) 时，\(f'(x) > 0\)，\(f(x)\) 单调递增。
极小值为 \(f(0) = 1\)，无极大值。

（2）要证 \(e^x \geq x + 1\) 对所有 \(x \in \mathbb{R}\) 成立。
设 \(g(x) = e^x - x - 1\)，则 \(g'(x) = e^x - 1\)。
\(g(x)\) 在 \(x = 0\) 处取最小值 \(g(0) = 0\)。
所以 \(g(x) \geq 0\)，即 \(e^x \geq x + 1\)。

（3）\(f(x) = e^x - ax^2 - x - 1\)，\(f(0) = 0\)。
要使 \(f(x) \geq 0\) 对所有 \(x \geq 0\) 成立。
\(f'(x) = e^x - 2ax - 1\)，\(f'(0) = 0\)。
\(f''(x) = e^x - 2a\)。
若 \(a \leq \frac{1}{2}\)，则 \(f''(x) = e^x - 2a \geq 1 - 2a \geq 0\)。
所以 \(f'(x)\) 单调递增，\(f'(x) \geq f'(0) = 0\)。
所以 \(f(x)\) 单调递增，\(f(x) \geq f(0) = 0\)。
若 \(a > \frac{1}{2}\)，则存在 \(x_0 > 0\) 使 \(f''(x_0) = 0\)。
当 \(x \in (0, x_0)\) 时，\(f''(x) < 0\)，\(f'(x)\) 单调递减，\(f'(x) < f'(0) = 0\)。
所以 \(f(x)\) 单调递减，\(f(x) < f(0) = 0\)，不满足条件。
综上，\(a \leq \frac{1}{2}\)。
等等，若 \(f(x) = e^x - ax^2 - x - 1\)：
\(f'(x) = e^x - 2ax - 1\)，\(f'(0) = 0\)
\(f''(x) = e^x - 2a\)
若 \(a \leq \frac{1}{2}\)，\(f''(x) \geq 1 - 2a \geq 0\)，\(f'(x)\) 单调递增，\(f'(x) \geq 0\)。
所以 \(a \leq \frac{1}{2}\)。
【考点】利用导数研究函数的单调性、极值；不等式恒成立问题
