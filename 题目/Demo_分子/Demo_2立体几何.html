2025 陕西西安新高考立体几何 6 题（标准正等轴测视角）

1. 正方体 \(ABCD-A_1B_1C_1D_1\) 棱长为 2，求其外接球表面积。

方法一：几何法（高考标准答题模板）
答题步骤（满分5分）
步骤1（1分）：求正方体体对角线长
正方体棱长 \(a=2\)，由体对角线公式：
\(l = \sqrt{a^2 + a^2 + a^2} = \sqrt{3a^2} = 2\sqrt{3}\)
步骤2（1分）：确定外接球半径
正方体外接球直径等于体对角线长，即 \(2R = l = 2\sqrt{3}\)，得 \(R = \sqrt{3}\)
步骤3（2分）：计算球的表面积
代入球的表面积公式 \(S = 4\pi R^2\)：
\(S = 4\pi \times (\sqrt{3})^2 = 12\pi\)
步骤4（1分）：写出最终结论
正方体的外接球表面积为 \(12\pi\)
易错警示：混淆外接球（直径=体对角线）与内切球（直径=棱长）的半径关系
方法二：空间向量法
1. 以D为原点建立空间直角坐标系，体对角线向量 \(\overrightarrow{DB_1} = (2,2,2)\)
2. 向量模长 \(|\overrightarrow{DB_1}| = \sqrt{2^2 + 2^2 + 2^2} = 2\sqrt{3}\)，即体对角线长
3. 后续步骤同几何法，结果为 \(12\pi\)
【考查知识点】空间几何体的外接球模型、球的表面积公式、正方体的结构特征

2. 直三棱柱 \(ABC-A_1B_1C_1\)，\(\angle ACB=90^\circ\)，\(AC=BC=2\)，\(AA_1=3\)，求体积。

方法一：几何法（高考标准答题模板）
答题步骤（满分6分）
步骤1（2分）：求底面直角三角形面积
由 \(\angle ACB=90^\circ\)，\(AC=BC=2\)，得：
\(S_{\triangle ABC} = \frac{1}{2} \times AC \times BC = \frac{1}{2} \times 2 \times 2 = 2\)
步骤2（1分）：确定直棱柱的高
直三棱柱的高等于侧棱长，即 \(h = AA_1 = 3\)
步骤3（2分）：计算棱柱体积
代入直棱柱体积公式 \(V = S_{底} \times h\)：
\(V = 2 \times 3 = 6\)
步骤4（1分）：写出最终结论
该直三棱柱的体积为6
易错警示：误将斜棱柱体积公式用于直棱柱，额外乘以夹角正弦值
方法二：矩阵/混合积法
1. 以C为原点建系，向量 \(\overrightarrow{CA}=(2,0,0),\overrightarrow{CB}=(0,2,0),\overrightarrow{CC_1}=(0,0,3)\)
2. 混合积行列式：\(\begin{vmatrix}2&0&0\\0&2&0\\0&0&3\end{vmatrix} = 12\)
3. 棱柱体积 \(V = |混合积| = 12\)（平行六面体体积），直三棱柱体积为其一半，即 \(V=6\)
【考查知识点】直棱柱的体积公式、直角三角形面积计算、空间向量混合积的几何意义

3. 正方体棱长为 2，E 为 AB 中点，求异面直线 \(D_1E\) 与 \(BC_1\) 所成角的余弦值。

方法一：空间向量法（高考标准答题模板）
答题步骤（满分8分）
步骤1（2分）：建立空间直角坐标系
以D为原点，DA、DC、DD₁分别为x、y、z轴，得各点坐标：
\(D_1(0,0,2)\)，\(E(2,1,0)\)，\(B(2,2,0)\)，\(C_1(0,2,2)\)
步骤2（2分）：求方向向量
\(\overrightarrow{D_1E} = (2,1,-2)\)，\(\overrightarrow{BC_1} = (-2,0,2)\)
步骤3（2分）：计算向量点积与模长
\(\overrightarrow{D_1E} \cdot \overrightarrow{BC_1} = 2×(-2) + 1×0 + (-2)×2 = -8\)
\(|\overrightarrow{D_1E}| = \sqrt{2^2 + 1^2 + (-2)^2} = 3\)，\(|\overrightarrow{BC_1}| = \sqrt{(-2)^2 + 0^2 + 2^2} = 2\sqrt{2}\)
步骤4（2分）：求异面直线所成角的余弦值
设异面直线所成角为\(\theta\)（\(0 < \theta \leq 90^\circ\)），则：
\(\cos\theta = \frac{|\overrightarrow{D_1E} \cdot \overrightarrow{BC_1}|}{|\overrightarrow{D_1E}| \cdot |\overrightarrow{BC_1}|} = \frac{|-8|}{3×2\sqrt{2}} = \frac{2\sqrt{2}}{3}\)
易错警示：忽略异面直线所成角范围，未对点积取绝对值
方法二：几何平移法
1. 平移\(BC_1\)至\(AD_1\)，则\(\angle AD_1E\)为异面直线所成角
2. 计算\(AD_1=2\sqrt{2}\)，\(D_1E=3\)，\(AE=\sqrt{5}\)
3. 由余弦定理：\(\cos\angle AD_1E = \frac{(2\sqrt{2})^2 + 3^2 - (\sqrt{5})^2}{2×2\sqrt{2}×3} = \frac{2\sqrt{2}}{3}\)
【考查知识点】异面直线所成角、空间直角坐标系、向量的点积与模长计算

4. 四棱锥 \(P-ABCD\)，底面正方形，\(PA\perp\) 底面，\(PA=AB=2\)，求 \(PC\) 与底面所成角的正切值。

方法一：几何法（高考标准答题模板）
答题步骤（满分10分）
步骤1（2分）：确定线面角的平面角
由\(PA\perp\)底面\(ABCD\)，得\(PC\)在底面的射影为\(AC\)，故\(\angle PCA\)为\(PC\)与底面所成的角
步骤2（3分）：计算相关线段长度
底面为正方形，\(AB=2\)，则\(AC = \sqrt{AB^2 + BC^2} = 2\sqrt{2}\)
由\(PA\perp\)底面，得\(PA\perp AC\)，且\(PA=2\)
步骤3（3分）：在Rt△PAC中求正切值
\(\tan\angle PCA = \frac{PA}{AC} = \frac{2}{2\sqrt{2}} = \frac{\sqrt{2}}{2}\)
步骤4（2分）：写出最终结论
直线\(PC\)与底面\(ABCD\)所成角的正切值为\(\frac{\sqrt{2}}{2}\)
易错警示：混淆线面角与向量夹角的关系（线面角\(\theta\)满足\(\sin\theta=|\cos\varphi|\)，\(\varphi\)为方向向量与法向量夹角）
方法二：向量法
1. 以A为原点建系，\(\overrightarrow{PC}=(2,2,-2)\)，底面法向量\(\boxed{n}=(0,0,1)\)
2. 线面角\(\theta\)满足：\(\sin\theta = \frac{|\overrightarrow{PC} \cdot \boxed{n}|}{|\overrightarrow{PC}| \cdot |\boxed{n}|} = \frac{2}{2\sqrt{3}} = \frac{\sqrt{3}}{3}\)
3. 由\(\sin\theta = \frac{\sqrt{3}}{3}\)，得\(\cos\theta = \frac{\sqrt{6}}{3}\)，故\(\tan\theta = \frac{\sqrt{2}}{2}\)
【考查知识点】线面角的定义、空间垂直关系、直角三角形的边角关系、空间向量的应用

5. 直三棱柱 \(ABC-A_1B_1C_1\)，\(AB=AC=AA_1=2\)，\(\angle BAC=90^\circ\)，M 为 \(A_1C_1\) 中点，求平面 \(ABM\) 与底面 \(ABC\) 所成锐二面角的余弦值。

方法一：空间向量法（高考标准答题模板）
答题步骤（满分12分）
步骤1（2分）：建立空间直角坐标系
以A为原点，AB、AC、AA₁分别为x、y、z轴，得各点坐标：
\(A(0,0,0)\)，\(B(2,0,0)\)，\(M(0,1,2)\)
步骤2（3分）：求平面的法向量
① 底面ABC的法向量：\(\boxed{n_1} = (0,0,1)\)（垂直于xOy平面）
② 平面ABM的法向量：取\(\overrightarrow{AB}=(2,0,0)\)，\(\overrightarrow{AM}=(0,1,2)\)
设\(\boxed{n_2}=(x,y,z)\)，则\(\begin{cases}\boxed{n_2} \cdot \overrightarrow{AB} = 2x = 0\\\boxed{n_2} \cdot \overrightarrow{AM} = y + 2z = 0\end{cases}\)，令\(z=1\)，得\(\boxed{n_2}=(0,-2,1)\)
步骤3（4分）：计算法向量夹角的余弦值
\(\boxed{n_1} \cdot \boxed{n_2} = 0×0 + 0×(-2) + 1×1 = 1\)
\(|\boxed{n_1}| = 1\)，\(|\boxed{n_2}| = \sqrt{0^2 + (-2)^2 + 1^2} = \sqrt{5}\)
\(\cos<\boxed{n_1},\boxed{n_2}> = \frac{|\boxed{n_1} \cdot \boxed{n_2}|}{|\boxed{n_1}| \cdot |\boxed{n_2}|} = \frac{1}{\sqrt{5}} = \frac{\sqrt{5}}{5}\)
步骤4（3分）：确定二面角的余弦值
所求为锐二面角，故平面ABM与底面ABC所成锐二面角的余弦值为\(\frac{\sqrt{5}}{5}\)
易错警示：求法向量时方程组求解错误，或未判断二面角为锐角导致符号错误
方法二：几何法
1. 过M作MN⊥AC于N，连接BN，则∠MBN为二面角的平面角
2. 计算MN=2，BN=\(\sqrt{AB^2 + AN^2} = \sqrt{5}\)，BM=3
3. 由余弦定理：\(\cos\angle MBN = \frac{BN^2 + BM^2 - MN^2}{2×BN×BM} = \frac{\sqrt{5}}{5}\)
【考查知识点】二面角的求解、平面法向量的计算、空间直角坐标系的应用、向量的点积公式

6. 棱锥 \(P-ABC\)，\(PA\perp\) 面 \(ABC\)，\(AB\perp BC\)，\(PA=AB=BC=2\)，E 为 PB 中点，求证：\(AE\perp\) 平面 \(PBC\)；求 \(V_{E-ABC}\)。

方法一：几何法（高考标准答题模板）
答题步骤（满分12分）
第一问：证明AE⊥平面PBC（7分）
步骤1（2分）：证明AE⊥PB
∵ \(PA\perp\)平面ABC，\(AB\subset\)平面ABC，∴ \(PA\perp AB\)
又\(PA=AB=2\)，E为PB中点，∴ \(AE\perp PB\)（等腰直角三角形三线合一）
步骤2（3分）：证明AE⊥BC
∵ \(PA\perp\)平面ABC，\(BC\subset\)平面ABC，∴ \(PA\perp BC\)
又\(AB\perp BC\)，\(PA\cap AB=A\)，\(PA,AB\subset\)平面PAB，∴ \(BC\perp\)平面PAB
∵ \(AE\subset\)平面PAB，∴ \(BC\perp AE\)
步骤3（2分）：证明线面垂直
∵ \(AE\perp PB\)，\(AE\perp BC\)，\(PB\cap BC=B\)，\(PB,BC\subset\)平面PBC，∴ \(AE\perp\)平面PBC
第二问：求三棱锥体积（5分）
步骤1（2分）：确定高和底面积
E为PB中点，PA⊥平面ABC，∴ E到平面ABC的距离\(h = \frac{1}{2}PA = 1\)
底面\(\triangle ABC\)面积：\(S = \frac{1}{2}×AB×BC = \frac{1}{2}×2×2 = 2\)
步骤2（3分）：计算体积
三棱锥体积公式：\(V_{E-ABC} = \frac{1}{3}×S×h = \frac{1}{3}×2×1 = \frac{2}{3}\)
易错警示：证明线面垂直时遗漏“两条相交直线”的条件，体积计算时忘记乘以\(\frac{1}{3}\)
方法二：向量法
1. 以A为原点建系，\(\overrightarrow{AE}=(0,1,1),\overrightarrow{PB}=(0,2,-2),\overrightarrow{BC}=(-2,0,0)\)
2. 点积：\(\overrightarrow{AE}·\overrightarrow{PB}=0\)，\(\overrightarrow{AE}·\overrightarrow{BC}=0\)，故AE⊥PB，AE⊥BC
3. 由线面垂直判定定理得AE⊥平面PBC
4. 体积：向量混合积行列式值为4，\(V = \frac{1}{6}×4×\frac{1}{2} = \frac{2}{3}\)（中点减半）
【考查知识点】线面垂直的判定定理、三棱锥的体积公式、空间垂直关系的传递性、中点的性质
