第一章 平面向量

1.1 向量的概念与线性运算

💡 向量的定义：既有大小又有方向的量叫做向量。

向量的表示：
  • 几何表示：有向线段 $\overrightarrow{AB}$
  • 字母表示：$\vec{a}$，$\vec{b}$，$\vec{c}$ 等
  • 坐标表示：$\vec{a} = (x, y)$
向量的线性运算：

1. 加法：
• 三角形法则：$\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}$
• 平行四边形法则
• 坐标运算：$(x_1, y_1) + (x_2, y_2) = (x_1+x_2, y_1+y_2)$

2. 减法：
• $\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$
• 坐标运算：$(x_1, y_1) - (x_2, y_2) = (x_1-x_2, y_1-y_2)$

3. 数乘：
• $\lambda\vec{a}$ 是与 $\vec{a}$ 共线的向量
• $|\lambda\vec{a}| = |\lambda||\vec{a}|$
• 坐标运算：$\lambda(x, y) = (\lambda x, \lambda y)$
📝 例题1：已知 $\vec{a} = (2, 1)$，$\vec{b} = (-1, 3)$，求：
(1) $\vec{a} + \vec{b}$
(2) $\vec{a} - \vec{b}$
(3) $2\vec{a} + 3\vec{b}$

【分析思路】
利用向量坐标运算的线性性质直接计算

【规范解答】
(1) $\vec{a} + \vec{b} = (2, 1) + (-1, 3) = (2-1, 1+3) = (1, 4)$

(2) $\vec{a} - \vec{b} = (2, 1) - (-1, 3) = (2+1, 1-3) = (3, -2)$

(3) $2\vec{a} + 3\vec{b} = 2(2, 1) + 3(-1, 3) = (4, 2) + (-3, 9) = (1, 11)$

【方法总结】
• 向量坐标运算：对应分量相加减
• 数乘向量：每个分量都乘以该数
• 运算规律与实数类似

【易错提醒】
• 注意符号，特别是减法
• 数乘时每个分量都要乘
• 结果仍是向量，用坐标表示

1.2 平面向量基本定理与坐标表示

💡 平面向量基本定理：如果 $\vec{e_1}$，$\vec{e_2}$ 是同一平面内的两个不共线向量，那么对于这一平面内的任意向量 $\vec{a}$，有且只有一对实数 $\lambda_1$，$\lambda_2$，使 $\vec{a} = \lambda_1\vec{e_1} + \lambda_2\vec{e_2}$。

基底：不共线的向量 $\vec{e_1}$，$\vec{e_2}$ 叫做表示这一平面内所有向量的一组基底。
向量的坐标：
在平面直角坐标系中，取与x轴、y轴方向相同的两个单位向量 $\vec{i}$，$\vec{j}$ 作为基底。

若 $\vec{a} = x\vec{i} + y\vec{j}$，则 $(x, y)$ 叫做向量 $\vec{a}$ 的坐标，记作 $\vec{a} = (x, y)$。

重要结论：
• 若 $A(x_1, y_1)$，$B(x_2, y_2)$，则 $\overrightarrow{AB} = (x_2-x_1, y_2-y_1)$
• 中点坐标：$M(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2})$
• 重心坐标：$G(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3})$
📝 例题2：已知 $A(1, 2)$，$B(4, 6)$，$C(3, -2)$，求：
(1) 向量 $\overrightarrow{AB}$ 和 $\overrightarrow{AC}$
(2) AB的中点坐标
(3) $\triangle ABC$ 的重心坐标

【分析思路】
直接套用坐标公式计算

【规范解答】
(1) $\overrightarrow{AB} = (4-1, 6-2) = (3, 4)$
$\overrightarrow{AC} = (3-1, -2-2) = (2, -4)$

(2) AB的中点 $M = (\frac{1+4}{2}, \frac{2+6}{2}) = (\frac{5}{2}, 4)$

(3) 重心 $G = (\frac{1+4+3}{3}, \frac{2+6-2}{3}) = (\frac{8}{3}, 2)$

【方法总结】
• 向量坐标 = 终点坐标 - 起点坐标
• 中点坐标：两坐标平均值
• 重心坐标：三坐标平均值

【易错提醒】
• 注意向量的方向，终点减起点
• 中点和重心公式不要混淆
• 计算要细心

1.3 平面向量的数量积

💡 数量积定义：已知两个非零向量 $\vec{a}$ 与 $\vec{b}$，它们的夹角为 $\theta$，则数量 $|\vec{a}||\vec{b}|\cos\theta$ 叫做 $\vec{a}$ 与 $\vec{b}$ 的数量积（或内积），记作 $\vec{a} \cdot \vec{b}$。

$$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$$
数量积的性质：
1. $\vec{a} \cdot \vec{a} = |\vec{a}|^2$ 或 $|\vec{a}| = \sqrt{\vec{a} \cdot \vec{a}}$
2. $\vec{a} \perp \vec{b} \Leftrightarrow \vec{a} \cdot \vec{b} = 0$
3. $\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$

坐标运算：
若 $\vec{a} = (x_1, y_1)$，$\vec{b} = (x_2, y_2)$，则：
• $\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2$
• $|\vec{a}| = \sqrt{x_1^2 + y_1^2}$
• $\cos\theta = \frac{x_1x_2 + y_1y_2}{\sqrt{x_1^2+y_1^2}\sqrt{x_2^2+y_2^2}}$
📝 例题3：已知 $\vec{a} = (1, 2)$，$\vec{b} = (3, -1)$，求：
(1) $\vec{a} \cdot \vec{b}$
(2) $|\vec{a}|$ 和 $|\vec{b}|$
(3) 向量 $\vec{a}$ 与 $\vec{b}$ 的夹角

【分析思路】
套用数量积的坐标运算公式

【规范解答】
(1) $\vec{a} \cdot \vec{b} = 1 \times 3 + 2 \times (-1) = 3 - 2 = 1$

(2) $|\vec{a}| = \sqrt{1^2 + 2^2} = \sqrt{5}$
$|\vec{b}| = \sqrt{3^2 + (-1)^2} = \sqrt{10}$

(3) $\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|} = \frac{1}{\sqrt{5}\sqrt{10}} = \frac{1}{\sqrt{50}} = \frac{1}{5\sqrt{2}} = \frac{\sqrt{2}}{10}$

$\theta = \arccos\frac{\sqrt{2}}{10}$

【方法总结】
• 数量积坐标运算：对应坐标相乘再相加
• 模长：各分量平方和开方
• 夹角：用数量积公式反解

【易错提醒】
• 数量积结果是数，不是向量
• 计算模长时不要漏掉开方
• 夹角范围是 $[0, \pi]$

1.4 向量的应用

向量在几何中的应用：

1. 证明平行：$\vec{a} \parallel \vec{b} \Leftrightarrow \vec{a} = \lambda\vec{b} \Leftrightarrow x_1y_2 - x_2y_1 = 0$

2. 证明垂直：$\vec{a} \perp \vec{b} \Leftrightarrow \vec{a} \cdot \vec{b} = 0 \Leftrightarrow x_1x_2 + y_1y_2 = 0$

3. 求夹角：$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$

4. 求距离：$|AB| = |\overrightarrow{AB}| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
📝 例题4：证明以 $A(1, 2)$，$B(4, 6)$，$C(7, 2)$ 为顶点的三角形是等腰三角形

【分析思路】
计算三边的长度，看是否有两边相等

【规范解答】
$\overrightarrow{AB} = (3, 4)$，$|AB| = \sqrt{9+16} = 5$

$\overrightarrow{BC} = (3, -4)$，$|BC| = \sqrt{9+16} = 5$

$\overrightarrow{AC} = (6, 0)$，$|AC| = 6$

∵ $|AB| = |BC| = 5$

∴ $\triangle ABC$ 是等腰三角形

【方法总结】
• 用向量证明几何性质：
- 等腰：两边长度相等
- 直角：两向量数量积为0
- 平行：向量共线
• 步骤：求向量 → 计算 → 判断

【易错提醒】
• 计算距离用模长公式
• 注意向量的方向
• 证明要严谨

第二章 复数

2.1 复数的概念

💡 复数的定义：形如 $a + bi$（$a, b \in \mathbb{R}$）的数叫做复数，其中 $i$ 叫做虚数单位，满足 $i^2 = -1$。

复数的分类：
  • $a$ 叫做实部，$b$ 叫做虚部
  • 当 $b = 0$ 时，$z = a$ 是实数
  • 当 $b \neq 0$ 时，$z = a + bi$ 是虚数
  • 当 $a = 0$ 且 $b \neq 0$ 时，$z = bi$ 是纯虚数
复数相等：$a + bi = c + di \Leftrightarrow a = c$ 且 $b = d$

共轭复数：实部相等，虚部互为相反数的两个复数
$z = a + bi$ 的共轭复数记作 $\bar{z} = a - bi$

复数的模：$|z| = |a + bi| = \sqrt{a^2 + b^2}$
📝 例题5：已知复数 $z = (m^2 - 5m + 6) + (m^2 - 3m)i$，求：
(1) 当m为何值时，z是实数？
(2) 当m为何值时，z是纯虚数？

【分析思路】
根据复数分类的定义，实部和虚部分别满足的条件

【规范解答】
(1) z是实数，则虚部为0：
$m^2 - 3m = 0$
$m(m-3) = 0$
$m = 0$ 或 $m = 3$

(2) z是纯虚数，则实部为0且虚部不为0：
$m^2 - 5m + 6 = 0$ 且 $m^2 - 3m \neq 0$

由 $m^2 - 5m + 6 = 0$：
$(m-2)(m-3) = 0$
$m = 2$ 或 $m = 3$

由 $m^2 - 3m \neq 0$：
$m \neq 0$ 且 $m \neq 3$

∴ $m = 2$

【方法总结】
• 实数：虚部 = 0
• 纯虚数：实部 = 0 且 虚部 ≠ 0
• 注意纯虚数要排除虚部为0的情况

【易错提醒】
• 纯虚数要求实部为0且虚部不为0
• 解方程后要检验
• 分类讨论要全面

2.2 复数的四则运算

复数的加减法：
$(a + bi) \pm (c + di) = (a \pm c) + (b \pm d)i$

复数的乘法：
$(a + bi)(c + di) = ac + adi + bci + bdi^2$
$= (ac - bd) + (ad + bc)i$

复数的除法：
$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{(c + di)(c - di)} = \frac{(ac + bd) + (bc - ad)i}{c^2 + d^2}$

重要性质：
• $z \cdot \bar{z} = |z|^2 = a^2 + b^2$
• $\overline{z_1 \pm z_2} = \bar{z_1} \pm \bar{z_2}$
• $\overline{z_1 \cdot z_2} = \bar{z_1} \cdot \bar{z_2}$
• $\overline{(\frac{z_1}{z_2})} = \frac{\bar{z_1}}{\bar{z_2}}$
📝 例题6：计算：
(1) $(3 + 2i) + (1 - 4i)$
(2) $(2 + i)(3 - 2i)$
(3) $\frac{1 + 2i}{3 - 4i}$

【分析思路】
按照复数运算法则计算，除法要乘以共轭复数

【规范解答】
(1) $(3 + 2i) + (1 - 4i) = (3+1) + (2-4)i = 4 - 2i$

(2) $(2 + i)(3 - 2i) = 6 - 4i + 3i - 2i^2$
$= 6 - i + 2 = 8 - i$

(3) $\frac{1 + 2i}{3 - 4i} = \frac{(1 + 2i)(3 + 4i)}{(3 - 4i)(3 + 4i)}$
$= \frac{3 + 4i + 6i + 8i^2}{9 + 16}$
$= \frac{3 + 10i - 8}{25}$
$= \frac{-5 + 10i}{25}$
$= -\frac{1}{5} + \frac{2}{5}i$

【方法总结】
• 加减：实部、虚部分别相加减
• 乘法：多项式乘法，$i^2 = -1$
• 除法：分子分母同乘分母的共轭复数

【易错提醒】
• 记住 $i^2 = -1$
• 除法一定要乘以共轭复数
• 计算要细心，特别是符号

2.3 复数的几何意义

💡 复平面：建立了直角坐标系来表示复数的平面叫做复平面，x轴叫做实轴，y轴叫做虚轴。

复数的几何表示：
  • 复数 $z = a + bi$ 对应复平面上的点 $Z(a, b)$
  • 复数 $z = a + bi$ 对应平面向量 $\overrightarrow{OZ} = (a, b)$
  • $|z| = |\overrightarrow{OZ}| = \sqrt{a^2 + b^2}$ 表示点Z到原点的距离
复数的三角形式：
$z = r(\cos\theta + i\sin\theta)$

其中 $r = |z| = \sqrt{a^2 + b^2}$，$\cos\theta = \frac{a}{r}$，$\sin\theta = \frac{b}{r}$

三角形式的运算：
• 乘法：$r_1(\cos\theta_1 + i\sin\theta_1) \cdot r_2(\cos\theta_2 + i\sin\theta_2) = r_1r_2[\cos(\theta_1+\theta_2) + i\sin(\theta_1+\theta_2)]$
• 除法：$\frac{r_1(\cos\theta_1 + i\sin\theta_1)}{r_2(\cos\theta_2 + i\sin\theta_2)} = \frac{r_1}{r_2}[\cos(\theta_1-\theta_2) + i\sin(\theta_1-\theta_2)]$
• 棣莫弗定理：$[r(\cos\theta + i\sin\theta)]^n = r^n(\cos n\theta + i\sin n\theta)$
📝 例题7：已知复数 $z = 1 + i$，求：
(1) z在复平面上对应的点
(2) |z|
(3) z的三角形式

【分析思路】
根据复数的几何意义和三角形式的定义求解

【规范解答】
(1) $z = 1 + i$ 对应复平面上的点 $(1, 1)$

(2) $|z| = \sqrt{1^2 + 1^2} = \sqrt{2}$

(3) $r = \sqrt{2}$，$\cos\theta = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$，$\sin\theta = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$

∴ $\theta = \frac{\pi}{4}$

三角形式：$z = \sqrt{2}(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4})$

【方法总结】
• 代数形式 → 三角形式：
① 求模 $r = \sqrt{a^2+b^2}$
② 求辐角 $\cos\theta = \frac{a}{r}$，$\sin\theta = \frac{b}{r}$
• 三角形式 → 代数形式：
$a = r\cos\theta$，$b = r\sin\theta$

【易错提醒】
• 注意辐角的主值范围 $[0, 2\pi)$
• 三角形式的实部是 $r\cos\theta$，不是 $r$
• 模一定是非负的
