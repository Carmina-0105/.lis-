第一章：集合与常用逻辑用语

1.1 集合运算

选择题 2023新课标I卷 基础
题目：已知集合$M = \{-2, -1, 0, 1, 2\}$，$N = \{x | x^2 - x - 6 \geq 0\}$，则$M \cap N =$（　　）

A. $\{-2, -1, 0, 1\}$　　B. $\{0, 1, 2\}$　　C. $\{-2\}$　　D. $\{2\}$
【命题意图】考查集合的交集运算和一元二次不等式的解法。
【规范解答】

解不等式$x^2 - x - 6 \geq 0$：

$(x-3)(x+2) \geq 0$

解得：$x \leq -2$ 或 $x \geq 3$

∴ $N = \{x | x \leq -2$ 或 $x \geq 3\}$

$M \cap N = \{-2\}$

答案：C
【方法总结】
• 解一元二次不等式：先求根，再根据开口方向确定解集
• 交集运算：找出同时属于两个集合的元素
• 注意：端点值是否包含
选择题 2022新课标I卷 基础
题目：若集合$M = \{x | \sqrt{x} < 4\}$，$N = \{x | 3x \geq 1\}$，则$M \cap N =$（　　）

A. $\{x | 0 \leq x < 2\}$　　B. $\{x | \frac{1}{3} \leq x < 2\}$　　C. $\{x | 3 \leq x < 16\}$　　D. $\{x | \frac{1}{3} \leq x < 16\}$
【命题意图】考查集合的交集运算和不等式的解法。
【规范解答】

解集合M：$\sqrt{x} < 4$

首先$x \geq 0$（根号内非负）
两边平方：$x < 16$

∴ $M = \{x | 0 \leq x < 16\}$

解集合N：$3x \geq 1$

$x \geq \frac{1}{3}$

∴ $N = \{x | x \geq \frac{1}{3}\}$

$M \cap N = \{x | \frac{1}{3} \leq x < 16\}$

答案：D
⚠️ 易错提醒
• 解$\sqrt{x} < 4$时，容易忘记$x \geq 0$这个隐含条件
• 注意不等式两边平方的条件

1.2 常用逻辑用语

选择题 2021新课标I卷 基础
题目：已知命题$p: \exists x \in R, \sin x < 1$；命题$q: \forall x \in R, e^{|x|} \geq 1$。则下列命题中为真命题的是（　　）

A. $p \land q$　　B. $\neg p \land q$　　C. $p \land \neg q$　　D. $\neg (p \lor q)$
【命题意图】考查全称量词、存在量词以及复合命题的真假判断。
【规范解答】

判断命题p的真假：

$p: \exists x \in R, \sin x < 1$

当$x = 0$时，$\sin 0 = 0 < 1$

∴ 命题p为真命题

判断命题q的真假：

$q: \forall x \in R, e^{|x|} \geq 1$

∵ $|x| \geq 0$，且$e^t$在$t \geq 0$时单调递增

∴ $e^{|x|} \geq e^0 = 1$

∴ 命题q为真命题

判断选项：

A. $p \land q$：真$\land$真 = 真 ✓
B. $\neg p \land q$：假$\land$真 = 假
C. $p \land \neg q$：真$\land$假 = 假
D. $\neg (p \lor q) = \neg p \land \neg q$：假$\land$假 = 假

答案：A
【方法总结】
• 存在量词命题：找到一个例子即可证明为真
• 全称量词命题：需要证明对所有情况都成立
• 复合命题真假判断：
- $p \land q$：全真才真
- $p \lor q$：有真则真
- $\neg p$：真假相反

第二章：函数与导数

2.1 函数性质

选择题 2023新课标I卷 中档
题目：设函数$f(x) = 2^{x(x-a)}$在区间$(0, 1)$单调递减，则a的取值范围是（　　）

A. $(-\infty, -2]$　　B. $[-2, 0)$　　C. $(0, 2]$　　D. $[2, +\infty)$
【命题意图】考查复合函数的单调性，以及指数函数和二次函数的性质。
【规范解答】

令$g(x) = x(x-a) = x^2 - ax$

则$f(x) = 2^{g(x)}$

∵ 底数$2 > 1$，指数函数单调递增

∴ $f(x)$的单调性与$g(x)$相同

$g(x) = x^2 - ax$是开口向上的抛物线，对称轴为$x = \frac{a}{2}$

要使$g(x)$在$(0, 1)$单调递减，需要对称轴在区间右侧：

$\frac{a}{2} \geq 1$

∴ $a \geq 2$

答案：D
【方法总结】
• 复合函数单调性：同增异减
• 指数函数：底数>1时单调递增，0<底数<1时单调递减
• 二次函数单调性：开口向上时，对称轴左侧递减，右侧递增
解答题 2024新课标I卷 较难
题目：已知函数$f(x) = e^x - ax^2 - x - 1$。
(1) 当$a = 1$时，讨论$f(x)$的单调性；
(2) 当$x \geq 0$时，$f(x) \geq 0$，求a的取值范围。
【命题意图】考查导数在研究函数单调性和最值中的应用，以及恒成立问题的处理。
【规范解答】

(1) 当a = 1时

$f(x) = e^x - x^2 - x - 1$
$f'(x) = e^x - 2x - 1$

令$g(x) = e^x - 2x - 1$，则$g'(x) = e^x - 2$

令$g'(x) = 0$，得$x = \ln 2$

当$x < \ln 2$时，$g'(x) < 0$，g(x)单调递减
当$x > \ln 2$时，$g'(x) > 0$，g(x)单调递增

$g(x)_{min} = g(\ln 2) = 2 - 2\ln 2 - 1 = 1 - \ln 4 < 0$

又$g(0) = 0$，$g(2) = e^2 - 5 > 0$

∴ g(x) = 0有两个根：$x_1 = 0$，$x_2 \in (\ln 2, 2)$

分析f'(x) = g(x)的符号：
• $x \in (-\infty, 0)$：f'(x) > 0，f(x)单调递增
• $x \in (0, x_2)$：f'(x) < 0，f(x)单调递减
• $x \in (x_2, +\infty)$：f'(x) > 0，f(x)单调递增

(2) 求a的取值范围

当$x = 0$时，$f(0) = 1 - 0 - 0 - 1 = 0$ ✓

当$x > 0$时，$f(x) \geq 0$即$e^x - ax^2 - x - 1 \geq 0$

$ax^2 \leq e^x - x - 1$

$a \leq \frac{e^x - x - 1}{x^2}$

设$h(x) = \frac{e^x - x - 1}{x^2}$（$x > 0$）

求导得：$h'(x) = \frac{(x-2)e^x + x + 2}{x^3}$

分析分子：设$\varphi(x) = (x-2)e^x + x + 2$

$\varphi'(x) = (x-1)e^x + 1$，$\varphi''(x) = xe^x > 0$（$x > 0$）

∴ $\varphi'(x)$在$(0, +\infty)$单调递增，$\varphi'(x) > \varphi'(0) = 0$

∴ $\varphi(x)$在$(0, +\infty)$单调递增，$\varphi(x) > \varphi(0) = 0$

∴ $h'(x) > 0$，h(x)在$(0, +\infty)$单调递增

$\lim_{x \to 0^+} h(x) = \lim_{x \to 0^+} \frac{e^x - x - 1}{x^2} = \lim_{x \to 0^+} \frac{e^x - 1}{2x} = \lim_{x \to 0^+} \frac{e^x}{2} = \frac{1}{2}$

∴ $h(x) > \frac{1}{2}$对所有$x > 0$成立

∴ $a \leq \frac{1}{2}$
【方法总结】
• 恒成立问题：分离参数→构造函数→求最值
• 多次求导：判断导数符号困难时，对导数再求导
• 极限计算：洛必达法则或等价无穷小
• 注意：$x = 0$需要单独验证

第三章：三角函数与解三角形

3.1 三角恒等变换

填空题 2023新课标I卷 中档
题目：已知$\sin(\alpha - \beta) = \frac{1}{3}$，$\cos\alpha \sin\beta = \frac{1}{6}$，则$\sin(\alpha + \beta) =$ ________。
【命题意图】考查两角和与差的正弦公式，以及三角恒等变换。
【规范解答】

由$\sin(\alpha - \beta) = \sin\alpha \cos\beta - \cos\alpha \sin\beta = \frac{1}{3}$

已知$\cos\alpha \sin\beta = \frac{1}{6}$

∴ $\sin\alpha \cos\beta = \frac{1}{3} + \frac{1}{6} = \frac{1}{2}$

$sin(\alpha + \beta) = \sin\alpha \cos\beta + \cos\alpha \sin\beta$

$= \frac{1}{2} + \frac{1}{6} = \frac{2}{3}$

答案：$\frac{2}{3}$
【方法总结】
• 两角和差公式：
$\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta$
$\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta$
• 观察已知和所求的关系，灵活变形

3.2 解三角形

解答题 2024新课标I卷 中档
题目：在△ABC中，内角A、B、C的对边分别为a、b、c，已知$b^2 = a^2 + c^2 - ac$。
(1) 求角B的大小；
(2) 若$a + c = 4$，求△ABC周长的最小值。
【命题意图】考查余弦定理的应用，以及利用基本不等式求最值。
【规范解答】

(1) 求角B

由余弦定理：$b^2 = a^2 + c^2 - 2ac\cos B$

与已知$b^2 = a^2 + c^2 - ac$对比：

$-2ac\cos B = -ac$

$2\cos B = 1$

$\cos B = \frac{1}{2}$

∵ $0 < B < \pi$

∴ $B = \frac{\pi}{3}$

(2) 求周长最小值

周长$L = a + b + c = 4 + b$

由余弦定理：

$b^2 = a^2 + c^2 - ac = (a+c)^2 - 3ac = 16 - 3ac$

由基本不等式：$ac \leq \left(\frac{a+c}{2}\right)^2 = 4$

当且仅当$a = c = 2$时取等号

∴ $b^2 = 16 - 3ac \geq 16 - 12 = 4$

∴ $b \geq 2$

∴ 周长$L = 4 + b \geq 6$

当$a = c = 2$时，$b = 2$，此时三角形为等边三角形

∴ 周长最小值为6
【方法总结】
• 余弦定理：$b^2 = a^2 + c^2 - 2ac\cos B$
• 基本不等式：$ac \leq \left(\frac{a+c}{2}\right)^2$
• 求最值：建立目标函数→利用不等式或导数求最值

第四章：数列

4.1 等差数列与等比数列

选择题 2023新课标I卷 基础
题目：记$S_n$为等差数列$\{a_n\}$的前n项和。若$a_2 + a_6 = 10$，$a_4 a_8 = 45$，则$S_5 =$（　　）

A. 25　　B. 22　　C. 20　　D. 15
【命题意图】考查等差数列的性质和前n项和公式。
【规范解答】

设等差数列首项为$a_1$，公差为d

由$a_2 + a_6 = 10$：

$(a_1 + d) + (a_1 + 5d) = 10$

$2a_1 + 6d = 10$

$a_1 + 3d = 5$ ... ①

由$a_4 a_8 = 45$：

$(a_1 + 3d)(a_1 + 7d) = 45$

由①：$a_1 + 3d = 5$，代入：

$5(a_1 + 7d) = 45$

$a_1 + 7d = 9$ ... ②

②-①：$4d = 4$，∴ $d = 1$

代入①：$a_1 = 5 - 3 = 2$

$S_5 = 5a_1 + \frac{5 \times 4}{2}d = 5 \times 2 + 10 \times 1 = 10 + 10 = 20$

答案：C
【方法总结】
• 等差数列通项：$a_n = a_1 + (n-1)d$
• 等差数列求和：$S_n = na_1 + \frac{n(n-1)}{2}d = \frac{n(a_1+a_n)}{2}$
• 等差中项：$a_2 + a_6 = 2a_4 = 10$，∴ $a_4 = 5$

第五章：立体几何

5.1 空间几何体

选择题 2023新课标I卷 中档
题目：已知正四棱台$ABCD-A_1B_1C_1D_1$中，$AB = 2$，$A_1B_1 = 1$，$AA_1 = \sqrt{2}$，则该四棱台的体积为 ________。
【命题意图】考查棱台的体积公式，以及空间几何体的计算。
【规范解答】

正四棱台的上底面边长$a = 1$，下底面边长$b = 2$

上底面积$S_1 = 1^2 = 1$
下底面积$S_2 = 2^2 = 4$

求高h：

正四棱台的高、斜高和上下底面边长的一半构成直角三角形

上下底面中心距在底面的投影：$\frac{2-1}{2} = \frac{1}{2}$

由勾股定理：$h^2 + \left(\frac{\sqrt{2}}{2}\right)^2 = (\sqrt{2})^2$

这里需要重新计算...

实际上，侧棱$AA_1 = \sqrt{2}$，上下底面中心距的水平距离为$\frac{\sqrt{2}}{2}$

$h^2 + \left(\frac{\sqrt{2}}{2}\right)^2 = (\sqrt{2})^2$

$h^2 + \frac{1}{2} = 2$

$h^2 = \frac{3}{2}$，$h = \sqrt{\frac{3}{2}} = \frac{\sqrt{6}}{2}$

体积：$V = \frac{1}{3}h(S_1 + S_2 + \sqrt{S_1S_2})$

$= \frac{1}{3} \cdot \frac{\sqrt{6}}{2} \cdot (1 + 4 + \sqrt{4})$

$= \frac{\sqrt{6}}{6} \cdot 7 = \frac{7\sqrt{6}}{6}$

答案：$\frac{7\sqrt{6}}{6}$
【方法总结】
• 棱台体积公式：$V = \frac{1}{3}h(S_1 + S_2 + \sqrt{S_1S_2})$
• 关键是求出棱台的高，通常利用勾股定理
• 正四棱台的高、侧棱和上下底面中心距构成直角三角形

第六章：解析几何

6.1 直线与圆

选择题 2023新课标I卷 基础
题目：过点$(0, -2)$与圆$x^2 + y^2 - 4x - 1 = 0$相切的两条直线的夹角为$\alpha$，则$\sin\alpha =$（　　）

A. 1　　B. $\frac{\sqrt{15}}{4}$　　C. $\frac{\sqrt{10}}{4}$　　D. $\frac{\sqrt{6}}{4}$
【命题意图】考查圆的方程、直线与圆的位置关系，以及三角函数的计算。
【规范解答】

将圆方程化为标准形式：

$x^2 - 4x + y^2 = 1$

$(x-2)^2 - 4 + y^2 = 1$

$(x-2)^2 + y^2 = 5$

圆心$C(2, 0)$，半径$r = \sqrt{5}$

点$P(0, -2)$到圆心的距离：

$|PC| = \sqrt{(2-0)^2 + (0+2)^2} = \sqrt{4 + 4} = \sqrt{8} = 2\sqrt{2}$

设切线与PC的夹角为$\theta$，则：

$\sin\theta = \frac{r}{|PC|} = \frac{\sqrt{5}}{2\sqrt{2}} = \frac{\sqrt{10}}{4}$

两条切线的夹角$\alpha = 2\theta$

$\sin\alpha = \sin(2\theta) = 2\sin\theta\cos\theta$

$\cos\theta = \sqrt{1 - \sin^2\theta} = \sqrt{1 - \frac{10}{16}} = \sqrt{\frac{6}{16}} = \frac{\sqrt{6}}{4}$

$\sin\alpha = 2 \cdot \frac{\sqrt{10}}{4} \cdot \frac{\sqrt{6}}{4} = \frac{2\sqrt{60}}{16} = \frac{2 \cdot 2\sqrt{15}}{16} = \frac{\sqrt{15}}{4}$

答案：B
【方法总结】
• 圆的标准方程：$(x-a)^2 + (y-b)^2 = r^2$
• 切线性质：圆心到切线的距离等于半径
• 切线夹角问题：利用直角三角形和三角函数
• 倍角公式：$\sin(2\theta) = 2\sin\theta\cos\theta$

6.2 圆锥曲线

解答题 2024新课标I卷 较难
题目：已知椭圆$C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$（$a > b > 0$）的右焦点为F，点$M(1, \frac{3}{2})$在C上，且$MF \perp x$轴。
(1) 求C的方程；
(2) 过点P(4, 0)的直线交C于A、B两点，求△ABF面积的最大值。
【命题意图】考查椭圆的标准方程、直线与椭圆的位置关系，以及最值问题。
【规范解答】

(1) 求椭圆方程

∵ $MF \perp x$轴，M的横坐标为1

∴ 右焦点$F(1, 0)$，即$c = 1$

点$M(1, \frac{3}{2})$在椭圆上：

$\frac{1}{a^2} + \frac{9}{4b^2} = 1$ ... ①

又$a^2 = b^2 + c^2 = b^2 + 1$ ... ②

由②代入①：

$\frac{1}{b^2+1} + \frac{9}{4b^2} = 1$

通分：$\frac{4b^2 + 9(b^2+1)}{4b^2(b^2+1)} = 1$

$4b^2 + 9b^2 + 9 = 4b^4 + 4b^2$

$4b^4 - 9b^2 - 9 = 0$

设$u = b^2$：$4u^2 - 9u - 9 = 0$

$(4u + 3)(u - 3) = 0$

$u = 3$（舍去负根）

∴ $b^2 = 3$，$a^2 = 4$

椭圆方程为$\frac{x^2}{4} + \frac{y^2}{3} = 1$

(2) 求面积最大值

设过P(4, 0)的直线为$x = my + 4$

联立椭圆方程：

$\begin{cases} x = my + 4 \\ \frac{x^2}{4} + \frac{y^2}{3} = 1 \end{cases}$

代入：$\frac{(my+4)^2}{4} + \frac{y^2}{3} = 1$

$3(my+4)^2 + 4y^2 = 12$

$(3m^2+4)y^2 + 24my + 36 = 0$

设$A(x_1, y_1)$，$B(x_2, y_2)$，则：

$y_1 + y_2 = -\frac{24m}{3m^2+4}$，$y_1y_2 = \frac{36}{3m^2+4}$

$|y_1 - y_2| = \sqrt{(y_1+y_2)^2 - 4y_1y_2} = \frac{12\sqrt{m^2-4}}{3m^2+4}$

需要$m^2 > 4$才能保证有两个交点

△ABF的面积：

$S = \frac{1}{2}|PF| \cdot |y_1 - y_2| = \frac{1}{2} \cdot 3 \cdot \frac{12\sqrt{m^2-4}}{3m^2+4}$

$= \frac{18\sqrt{m^2-4}}{3m^2+4}$

设$t = \sqrt{m^2-4}$（$t > 0$），则$m^2 = t^2 + 4$

$S = \frac{18t}{3(t^2+4)+4} = \frac{18t}{3t^2+16}$

求最大值：令$f(t) = \frac{18t}{3t^2+16}$

$f'(t) = \frac{18(3t^2+16) - 18t \cdot 6t}{(3t^2+16)^2} = \frac{18(16-3t^2)}{(3t^2+16)^2}$

令$f'(t) = 0$，得$t^2 = \frac{16}{3}$，$t = \frac{4}{\sqrt{3}} = \frac{4\sqrt{3}}{3}$

最大值为：

$S_{max} = \frac{18 \cdot \frac{4\sqrt{3}}{3}}{3 \cdot \frac{16}{3} + 16} = \frac{24\sqrt{3}}{16 + 16} = \frac{24\sqrt{3}}{32} = \frac{3\sqrt{3}}{4}$

△ABF面积的最大值为$\frac{3\sqrt{3}}{4}$
【方法总结】
• 椭圆标准方程：利用已知条件建立方程组
• 直线与椭圆相交：联立方程，利用韦达定理
• 面积计算：$S = \frac{1}{2} \times$底$\times$高
• 最值问题：构造函数，利用导数或基本不等式

第七章：概率与统计

7.1 概率计算

选择题 2023新课标I卷 中档
题目：某学校开设了4门体育类选修课和4门艺术类选修课，学生需从这8门课中选修2门或3门课，并且每类选修课至少选修1门，则不同的选课方案共有（　　）种

A. 64　　B. 80　　C. 96　　D. 104
【命题意图】考查分类计数原理和组合数的计算。
【规范解答】

情况一：选修2门课

每类至少1门，则体育1门、艺术1门

$C_4^1 \times C_4^1 = 4 \times 4 = 16$种

情况二：选修3门课

每类至少1门，有两种子情况：

① 体育2门、艺术1门：$C_4^2 \times C_4^1 = 6 \times 4 = 24$种

② 体育1门、艺术2门：$C_4^1 \times C_4^2 = 4 \times 6 = 24$种

共$24 + 24 = 48$种

总计：$16 + 48 = 64$种

答案：A
【方法总结】
• 分类计数：先分类，再分步，用加法原理汇总
• 限制条件"每类至少1门"：分类讨论
• 组合数计算：$C_n^k = \frac{n!}{k!(n-k)!}$

7.2 统计与数据分析

解答题 2024新课标I卷 中档
题目：某工厂为了提高生产效率，对生产设备进行了技术改造。现从改造前和改造后生产的大量产品中各随机抽取了100件产品作为样本，测量其质量指标值，得到如下数据：

质量指标值区间 [95,100) [100,105) [105,110) [110,115) [115,120]
改造前频数 10 30 40 15 5
改造后频数 5 20 45 20 10

(1) 估计改造前和改造后产品质量指标值的平均数（同一组中的数据用该组区间的中点值作代表）；
(2) 根据上述数据，能否有95%的把握认为技术改造提高了产品质量？

附：$\chi^2 = \frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}$，$P(\chi^2 \geq 3.841) = 0.05$
【命题意图】考查平均数的估计、独立性检验，以及数据分析能力。
【规范解答】

(1) 估计平均数

各区间中点值：97.5, 102.5, 107.5, 112.5, 117.5

改造前：

$\bar{x}_{前} = \frac{1}{100}(10 \times 97.5 + 30 \times 102.5 + 40 \times 107.5 + 15 \times 112.5 + 5 \times 117.5)$

$= \frac{1}{100}(975 + 3075 + 4300 + 1687.5 + 587.5)$

$= \frac{10625}{100} = 106.25$

改造后：

$\bar{x}_{后} = \frac{1}{100}(5 \times 97.5 + 20 \times 102.5 + 45 \times 107.5 + 20 \times 112.5 + 10 \times 117.5)$

$= \frac{1}{100}(487.5 + 2050 + 4837.5 + 2250 + 1175)$

$= \frac{10800}{100} = 108$

(2) 独立性检验

将数据分为"优质品"（指标值≥110）和"非优质品"（指标值<110）：

优质品 非优质品 合计
改造前 20 80 100
改造后 30 70 100
合计 50 150 200

计算$\chi^2$：

$a = 20, b = 80, c = 30, d = 70, n = 200$

$\chi^2 = \frac{200 \times (20 \times 70 - 80 \times 30)^2}{100 \times 100 \times 50 \times 150}$

$= \frac{200 \times (1400 - 2400)^2}{75000000}$

$= \frac{200 \times 1000000}{75000000}$

$= \frac{200000000}{75000000} = \frac{8}{3} \approx 2.667$

∵ $\chi^2 = 2.667 < 3.841$

∴ 没有95%的把握认为技术改造提高了产品质量
【方法总结】
• 平均数估计：用组中值代表该组数据
• 独立性检验步骤：
① 提出假设
② 计算$\chi^2$统计量
③ 比较临界值
④ 得出结论
• 注意：$\chi^2$越大，越有理由拒绝原假设
