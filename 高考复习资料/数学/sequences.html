第一章 等差数列

1.1 等差数列的基本概念与公式

一、等差数列定义

定义：如果一个数列从第二项起，每一项与它的前一项的差等于同一个常数，这个数列就叫做等差数列。

即：$a_{n+1} - a_n = d$（d为常数，称为公差）
等差数列核心公式：

1. 通项公式：
$a_n = a_1 + (n-1)d$
$a_n = a_m + (n-m)d$（推广式）

2. 等差中项：
若a, A, b成等差数列，则 $A = \frac{a+b}{2}$

3. 前n项和公式：
$S_n = \frac{n(a_1+a_n)}{2}$
$S_n = na_1 + \frac{n(n-1)}{2}d$
$S_n = \frac{d}{2}n^2 + (a_1-\frac{d}{2})n$（关于n的二次函数）

4. 重要性质：
• 若$m+n=p+q$，则$a_m+a_n=a_p+a_q$
• $S_n, S_{2n}-S_n, S_{3n}-S_{2n}$成等差数列
• 等差数列中，$a_n = S_n - S_{n-1}$（n≥2）
📝 例题1（原创）：已知等差数列{an}中，$a_3 = 5$，$a_7 = 13$，求通项公式和前n项和公式。

【命题意图】考查等差数列基本量的计算，以及方程思想。
【分析思路】
已知两项，可以列方程组求出$a_1$和d，再写通项公式。

【规范解答】
设首项为$a_1$，公差为d

由$a_3 = 5$：$a_1 + 2d = 5$ ... ①
由$a_7 = 13$：$a_1 + 6d = 13$ ... ②

②-①：$4d = 8$，∴ $d = 2$

代入①：$a_1 + 4 = 5$，∴ $a_1 = 1$

通项公式：$a_n = 1 + (n-1)×2 = 2n - 1$

前n项和：$S_n = \frac{n(1+2n-1)}{2} = \frac{n·2n}{2} = n^2$

或：$S_n = n×1 + \frac{n(n-1)}{2}×2 = n + n(n-1) = n^2$
【方法总结】
• 等差数列基本量：$a_1, d, n, a_n, S_n$——知三求二
• 列方程组是通用方法
• 也可以用性质：$d = \frac{a_7-a_3}{7-3} = \frac{13-5}{4} = 2$

二、等差数列的性质应用

📝 例题2（原创）：等差数列{an}中，$a_3 + a_8 + a_{13} = 12$，求$a_8$和$S_{15}$。

【命题意图】考查等差数列性质：若$m+n=2p$，则$a_m+a_n=2a_p$。
【分析思路】
观察下标：3+13=16=2×8，所以$a_3+a_{13}=2a_8$。

【规范解答】
∵ 3 + 13 = 16 = 2×8
∴ $a_3 + a_{13} = 2a_8$

∴ $a_3 + a_8 + a_{13} = 2a_8 + a_8 = 3a_8 = 12$

∴ $a_8 = 4$

求$S_{15}$：
$S_{15} = \frac{15(a_1+a_{15})}{2} = \frac{15×2a_8}{2} = 15a_8 = 15×4 = $ 60

（注：$a_1+a_{15}=2a_8$因为1+15=16=2×8）
【方法总结】
• 性质：若$m+n=p+q$，则$a_m+a_n=a_p+a_q$
• 特别地，若$m+n=2p$，则$a_m+a_n=2a_p$
• $S_{2n-1} = (2n-1)a_n$（奇数项和=项数×中间项）
• 看到等差数列求和问题，先想能否用性质简化
📝 例题3（2024新课标风格）：等差数列{an}的前n项和为$S_n$，若$S_3 = 3$，$S_6 = 15$，求$S_9$。

【命题意图】考查等差数列片段和性质：$S_n, S_{2n}-S_n, S_{3n}-S_{2n}$成等差数列。
【方法一：用性质】
由性质：$S_3, S_6-S_3, S_9-S_6$成等差数列

即：$3, 15-3, S_9-15$成等差数列
即：$3, 12, S_9-15$成等差数列

∴ $2×12 = 3 + (S_9-15)$
$24 = S_9 - 12$
$S_9 = 36$

【方法二：基本量法】
$S_3 = 3a_1 + 3d = 3$ ... ①
$S_6 = 6a_1 + 15d = 15$ ... ②

由①：$a_1 + d = 1$
由②：$2a_1 + 5d = 5$

解得：$d = 1, a_1 = 0$

$S_9 = 9a_1 + 36d = 0 + 36 = $ 36
【方法总结】
• 片段和性质：$S_n, S_{2n}-S_n, S_{3n}-S_{2n}$成等差数列，公差为$n^2d$
• 这个性质可以大大简化计算
• 基本量法是通用方法，但计算量较大

三、等差数列的最值问题

📝 例题4（原创）：等差数列{an}中，$a_1 = 13$，$S_3 = S_{11}$，求$S_n$的最大值。

【命题意图】考查等差数列前n项和的二次函数性质，以及最值问题。
【分析思路】
$S_n$是关于n的二次函数，$S_3=S_{11}$说明对称轴在n=7处。

【规范解答】
设公差为d

$S_3 = 3×13 + \frac{3×2}{2}d = 39 + 3d$
$S_{11} = 11×13 + \frac{11×10}{2}d = 143 + 55d$

由$S_3 = S_{11}$：
$39 + 3d = 143 + 55d$
$52d = -104$
$d = -2$

∴ $a_n = 13 + (n-1)×(-2) = 15 - 2n$

令$a_n ≥ 0$：$15 - 2n ≥ 0$，$n ≤ 7.5$

∴ 当n=7时，$S_n$取得最大值：
$S_7 = \frac{7(13+a_7)}{2} = \frac{7(13+1)}{2} = \frac{7×14}{2} = $ 49

或用公式：$S_7 = 7×13 + \frac{7×6}{2}×(-2) = 91 - 42 = 49$
【方法总结】
• $S_n$是开口向下的抛物线（d<0时），有最大值
• 求$S_n$最大值的两种方法：
① 找最后一个正项：令$a_n≥0$，求最大n
② 用二次函数：对称轴处取最值
• $S_3=S_{11}$说明对称轴为$n=(3+11)/2=7$

第二章 等比数列

2.1 等比数列的基本概念与公式

一、等比数列定义

定义：如果一个数列从第二项起，每一项与它的前一项的比等于同一个常数，这个数列就叫做等比数列。

即：$\frac{a_{n+1}}{a_n} = q$（q为常数，称为公比，$q ≠ 0$）
等比数列核心公式：

1. 通项公式：
$a_n = a_1 · q^{n-1}$
$a_n = a_m · q^{n-m}$（推广式）

2. 等比中项：
若a, G, b成等比数列，则 $G^2 = ab$，即 $G = ±\sqrt{ab}$
（注意：等比中项有两个，互为相反数）

3. 前n项和公式：
当$q = 1$时：$S_n = na_1$
当$q ≠ 1$时：$S_n = \frac{a_1(1-q^n)}{1-q} = \frac{a_1-a_nq}{1-q}$

4. 重要性质：
• 若$m+n=p+q$，则$a_m·a_n=a_p·a_q$
• $S_n, S_{2n}-S_n, S_{3n}-S_{2n}$成等比数列（$q≠-1$时）
• 等比数列中，偶数项同号，奇数项同号
【易错提醒】
⚠️ 等比数列中不能有0！$a_n ≠ 0$，$q ≠ 0$
⚠️ 求等比中项时，注意有两个值$±\sqrt{ab}$
⚠️ 用求和公式时，先判断$q=1$还是$q≠1$
⚠️ 等比数列片段和性质要求$q≠-1$
📝 例题5（原创）：等比数列{an}中，$a_2 = 6$，$a_5 = 48$，求通项公式和前n项和公式。

【规范解答】
设首项为$a_1$，公比为q

由$a_2 = 6$：$a_1q = 6$ ... ①
由$a_5 = 48$：$a_1q^4 = 48$ ... ②

②÷①：$q^3 = 8$，∴ $q = 2$

代入①：$a_1 = 3$

通项公式：$a_n = 3×2^{n-1}$

前n项和：$S_n = \frac{3(1-2^n)}{1-2} = 3(2^n-1) = 3·2^n - 3$
【方法总结】
• 等比数列基本量：$a_1, q, n, a_n, S_n$——知三求二
• 已知两项求公比：$q^{n-m} = \frac{a_n}{a_m}$
• 注意：等比数列中所有项同号（当q>0时）

二、等比数列的性质应用

📝 例题6（原创）：等比数列{an}的各项均为正数，且$a_5·a_6 + a_4·a_7 = 18$，求$\log_3a_1 + \log_3a_2 + ... + \log_3a_{10}$。

【命题意图】考查等比数列性质：若$m+n=p+q$，则$a_m·a_n=a_p·a_q$，以及对数运算。
【分析思路】
注意到5+6=4+7=11，所以$a_5·a_6 = a_4·a_7$。

【规范解答】
∵ 5+6 = 4+7 = 11
∴ $a_5·a_6 = a_4·a_7$

由$a_5·a_6 + a_4·a_7 = 18$：
$2a_5·a_6 = 18$
$a_5·a_6 = 9$

注意到1+10=2+9=3+8=4+7=5+6=11
∴ $a_1·a_{10} = a_2·a_9 = a_3·a_8 = a_4·a_7 = a_5·a_6 = 9$

$a_1·a_2·...·a_{10} = (a_1·a_{10})(a_2·a_9)(a_3·a_8)(a_4·a_7)(a_5·a_6) = 9^5 = 3^{10}$

∴ $\log_3a_1 + \log_3a_2 + ... + \log_3a_{10}$
$= \log_3(a_1·a_2·...·a_{10})$
$= \log_3(3^{10})$
$= $ 10
【方法总结】
• 性质：若$m+n=p+q$，则$a_m·a_n=a_p·a_q$
• 特别地，若$m+n=2p$，则$a_m·a_n=a_p^2$
• 对数求和：$\log a_1 + \log a_2 + ... = \log(a_1·a_2·...)$
• 等比数列中，距离两端等远的两项积相等

第三章 数列求和

3.1 常见数列求和方法

一、公式法

基本公式：

1. 等差数列：$S_n = \frac{n(a_1+a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$

2. 等比数列：$S_n = \frac{a_1(1-q^n)}{1-q}$（$q≠1$）

3. 自然数和：$1+2+3+...+n = \frac{n(n+1)}{2}$

4. 自然数平方和：$1^2+2^2+...+n^2 = \frac{n(n+1)(2n+1)}{6}$

5. 自然数立方和：$1^3+2^3+...+n^3 = [\frac{n(n+1)}{2}]^2$

二、错位相减法

适用：等差×等比型数列，即$c_n = a_n·b_n$，其中{an}是等差数列，{bn}是等比数列

步骤：
1. 写出$S_n = c_1 + c_2 + ... + c_n$
2. 两边乘公比q：$qS_n = qc_1 + qc_2 + ... + qc_n$
3. 两式相减（错位相减）
4. 化简求解
📝 例题7（原创）：已知数列{an}的通项公式为$a_n = (2n-1)·2^n$，求前n项和$S_n$。

【命题意图】考查错位相减法，这是高考解答题常考题型。
【规范解答】

$S_n = 1·2^1 + 3·2^2 + 5·2^3 + ... + (2n-1)·2^n$ ... ①

两边乘2：
$2S_n = 1·2^2 + 3·2^3 + 5·2^4 + ... + (2n-1)·2^{n+1}$ ... ②

①-②：
$-S_n = 1·2^1 + (3-1)·2^2 + (5-3)·2^3 + ... + [(2n-1)-(2n-3)]·2^n - (2n-1)·2^{n+1}$

$-S_n = 2 + 2·2^2 + 2·2^3 + ... + 2·2^n - (2n-1)·2^{n+1}$

$-S_n = 2 + 2(2^2 + 2^3 + ... + 2^n) - (2n-1)·2^{n+1}$

括号内是等比数列求和：$2^2 + 2^3 + ... + 2^n = \frac{4(2^{n-1}-1)}{2-1} = 2^{n+1} - 4$

$-S_n = 2 + 2(2^{n+1} - 4) - (2n-1)·2^{n+1}$
$= 2 + 2^{n+2} - 8 - (2n-1)·2^{n+1}$
$= 2^{n+2} - 6 - (2n-1)·2^{n+1}$
$= 2·2^{n+1} - 6 - (2n-1)·2^{n+1}$
$= [2-(2n-1)]·2^{n+1} - 6$
$= (3-2n)·2^{n+1} - 6$

∴ $S_n = (2n-3)·2^{n+1} + 6$
【方法总结】
• 错位相减法适用于"等差×等比"型数列
• 关键步骤：写出$S_n$和$qS_n$，错位相减
• 注意对齐：$c_k$与$qc_{k-1}$对齐
• 最后化简时要仔细，容易出错
【易错提醒】
⚠️ 相减时注意符号！最后一项是减去的
⚠️ 等比数列求和时，注意项数
⚠️ 最后化简要彻底

三、裂项相消法

适用：通项可拆成两项之差的形式

常见裂项公式：
1. $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$

2. $\frac{1}{n(n+k)} = \frac{1}{k}(\frac{1}{n} - \frac{1}{n+k})$

3. $\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}(\frac{1}{2n-1} - \frac{1}{2n+1})$

4. $\frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$
📝 例题8（原创）：已知$a_n = \frac{1}{n(n+2)}$，求数列{an}的前n项和$S_n$。

【规范解答】

$a_n = \frac{1}{n(n+2)} = \frac{1}{2}(\frac{1}{n} - \frac{1}{n+2})$

$S_n = \frac{1}{2}[(1-\frac{1}{3}) + (\frac{1}{2}-\frac{1}{4}) + (\frac{1}{3}-\frac{1}{5}) + ... + (\frac{1}{n}-\frac{1}{n+2})]$

观察抵消规律：
-1/3与+1/3抵消
-1/4与后面的+1/4抵消
...

剩余：$1 + \frac{1}{2} - \frac{1}{n+1} - \frac{1}{n+2}$

$S_n = \frac{1}{2}[\frac{3}{2} - \frac{1}{n+1} - \frac{1}{n+2}]$
$= \frac{3}{4} - \frac{1}{2(n+1)} - \frac{1}{2(n+2)}$
$= $ $\frac{3}{4} - \frac{2n+3}{2(n+1)(n+2)}$
【方法总结】
• 裂项相消法关键是找到合适的裂项方式
• 注意系数：$\frac{1}{n(n+k)} = \frac{1}{k}(\frac{1}{n} - \frac{1}{n+k})$
• 写几项观察抵消规律，确定剩余哪些项
• 一般剩余：前面正的几项 + 后面负的几项

第四章 递推数列

4.1 递推数列求通项

一、累加法

适用：$a_{n+1} - a_n = f(n)$型递推式

方法：
$a_n = a_1 + (a_2-a_1) + (a_3-a_2) + ... + (a_n-a_{n-1})$
$= a_1 + f(1) + f(2) + ... + f(n-1)$
📝 例题9（原创）：数列{an}中，$a_1 = 1$，$a_{n+1} = a_n + 2n$，求通项公式$a_n$。

【规范解答】

由$a_{n+1} - a_n = 2n$

当n≥2时：
$a_n = a_1 + (a_2-a_1) + (a_3-a_2) + ... + (a_n-a_{n-1})$
$= 1 + 2×1 + 2×2 + ... + 2(n-1)$
$= 1 + 2[1 + 2 + ... + (n-1)]$
$= 1 + 2·\frac{(n-1)n}{2}$
$= 1 + n(n-1)$
$= n^2 - n + 1$

验证n=1：$a_1 = 1-1+1 = 1$ ✓

∴ $a_n = n^2 - n + 1$

二、累乘法

适用：$\frac{a_{n+1}}{a_n} = f(n)$型递推式（$a_n ≠ 0$）

方法：
$a_n = a_1 · \frac{a_2}{a_1} · \frac{a_3}{a_2} · ... · \frac{a_n}{a_{n-1}}$
$= a_1 · f(1) · f(2) · ... · f(n-1)$
📝 例题10（原创）：数列{an}中，$a_1 = 1$，$a_{n+1} = \frac{n}{n+1}·a_n$，求通项公式$a_n$。

【规范解答】

由$\frac{a_{n+1}}{a_n} = \frac{n}{n+1}$

当n≥2时：
$a_n = a_1 · \frac{a_2}{a_1} · \frac{a_3}{a_2} · ... · \frac{a_n}{a_{n-1}}$
$= 1 · \frac{1}{2} · \frac{2}{3} · \frac{3}{4} · ... · \frac{n-1}{n}$

观察约分规律：分子分母交叉约分
$= \frac{1}{n}$

验证n=1：$a_1 = 1/1 = 1$ ✓

∴ $a_n = \frac{1}{n}$
【方法总结】
• 累乘法适用于比值型递推式
• 关键是观察约分规律
• 最后一定要验证n=1是否成立

三、构造法

适用：$a_{n+1} = pa_n + q$型递推式（p,q为常数，p≠1）

方法：设$a_{n+1} + λ = p(a_n + λ)$，解出λ，构造等比数列

由$a_{n+1} + λ = pa_n + q + λ = p(a_n + \frac{q+λ}{p})$
令$λ = \frac{q+λ}{p}$，解得$λ = \frac{q}{p-1}$
📝 例题11（原创）：数列{an}中，$a_1 = 1$，$a_{n+1} = 2a_n + 1$，求通项公式$a_n$。

【命题意图】考查构造法求递推数列通项，这是高考常考题型。
【规范解答】

设$a_{n+1} + λ = 2(a_n + λ)$
则$a_{n+1} = 2a_n + λ$

与$a_{n+1} = 2a_n + 1$对比，得$λ = 1$

∴ $a_{n+1} + 1 = 2(a_n + 1)$

令$b_n = a_n + 1$，则$b_{n+1} = 2b_n$，且$b_1 = a_1 + 1 = 2$

∴ {bn}是首项为2，公比为2的等比数列

$b_n = 2×2^{n-1} = 2^n$

∴ $a_n = b_n - 1 = 2^n - 1$

验证：$a_1 = 2^1-1 = 1$ ✓
【方法总结】
• $a_{n+1} = pa_n + q$型：构造$a_{n+1}+λ=p(a_n+λ)$
• 解出$λ = \frac{q}{p-1}$
• {an+λ}是等比数列，公比为p
• 也可以用待定系数法直接求
