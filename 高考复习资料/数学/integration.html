第一章：函数与方程思想

1.1 函数思想的核心应用

核心概念：函数思想是用运动和变化的观点，分析和研究数学中的数量关系，通过建立函数关系或构造函数，运用函数的图像和性质去分析问题、转化问题，从而使问题获得解决。
函数思想的主要应用场景

1. 数列问题函数化
• 等差数列：$a_n = a_1 + (n-1)d = dn + (a_1-d)$，是关于n的一次函数
• 等差数列求和：$S_n = \frac{d}{2}n^2 + (a_1-\frac{d}{2})n$，是关于n的二次函数
• 等比数列：$a_n = a_1 \cdot q^{n-1}$，是指数型函数

2. 不等式问题函数化
• 恒成立问题：转化为求函数最值
• 参数范围问题：分离参数，构造函数

3. 方程问题函数化
• 方程根的问题：转化为函数零点问题
• 利用函数图像研究方程根的个数和分布
📝 例题1（函数与数列融合）：已知等差数列{an}的前n项和为$S_n$，若$S_m = S_n$（$m \neq n$），求$S_{m+n}$。

【命题意图】考查等差数列求和公式的函数特征，以及二次函数对称性的应用。
【规范解答】

设等差数列首项为$a_1$，公差为d

$S_n = na_1 + \frac{n(n-1)}{2}d = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$

这是关于n的二次函数（当$d \neq 0$时）

由$S_m = S_n$（$m \neq n$），根据二次函数的对称性：

对称轴为$n = \frac{m+n}{2}$

又二次函数$S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$的对称轴为：
$n = -\frac{a_1 - \frac{d}{2}}{2 \cdot \frac{d}{2}} = -\frac{a_1 - \frac{d}{2}}{d} = \frac{d - 2a_1}{2d}$

∴ $\frac{m+n}{2} = \frac{d - 2a_1}{2d}$

即$m+n = \frac{d-2a_1}{d} = 1 - \frac{2a_1}{d}$

$S_{m+n} = \frac{d}{2}(m+n)^2 + (a_1 - \frac{d}{2})(m+n)$

设$t = m+n$，则：
$S_t = \frac{d}{2}t^2 + (a_1 - \frac{d}{2})t = \frac{d}{2}t(t + \frac{2a_1-d}{d}) = \frac{d}{2}t(t - t) = $ 0
【方法总结】
• 等差数列求和公式是关于n的二次函数
• 利用二次函数对称性：若$f(m) = f(n)$，则对称轴为$x = \frac{m+n}{2}$
• 关键发现：$S_{m+n} = 0$与具体数值无关，是定值
📝 例题2（函数与不等式融合）：已知函数$f(x) = \ln x - ax + 1$，若$f(x) \leq 0$对所有$x > 0$恒成立，求实数a的取值范围。

【命题意图】考查恒成立问题的函数化处理，以及导数在求最值中的应用。
【规范解答】

方法一：分离参数法

$f(x) \leq 0$即$\ln x - ax + 1 \leq 0$

当$x > 0$时，$ax \geq \ln x + 1$

$a \geq \frac{\ln x + 1}{x}$对所有$x > 0$恒成立

设$g(x) = \frac{\ln x + 1}{x}$，只需$a \geq g(x)_{max}$

求导：$g'(x) = \frac{\frac{1}{x} \cdot x - (\ln x + 1) \cdot 1}{x^2} = \frac{1 - \ln x - 1}{x^2} = \frac{-\ln x}{x^2}$

令$g'(x) = 0$，得$\ln x = 0$，即$x = 1$

当$0 < x < 1$时，$g'(x) > 0$，g(x)单调递增
当$x > 1$时，$g'(x) < 0$，g(x)单调递减

∴ $g(x)_{max} = g(1) = \frac{\ln 1 + 1}{1} = 1$

∴ $a \geq 1$

方法二：直接求最值法

$f'(x) = \frac{1}{x} - a = \frac{1-ax}{x}$

当$a \leq 0$时，$f'(x) > 0$，f(x)在$(0, +\infty)$单调递增
又$f(1) = 1 - a > 0$，不满足$f(x) \leq 0$恒成立

当$a > 0$时，令$f'(x) = 0$，得$x = \frac{1}{a}$

f(x)在$(0, \frac{1}{a})$单调递增，在$(\frac{1}{a}, +\infty)$单调递减

$f(x)_{max} = f(\frac{1}{a}) = \ln\frac{1}{a} - a \cdot \frac{1}{a} + 1 = -\ln a - 1 + 1 = -\ln a$

要使$f(x) \leq 0$恒成立，需$f(x)_{max} \leq 0$
即$-\ln a \leq 0$，$\ln a \geq 0$，$a \geq 1$
【方法总结】
• 恒成立问题两种思路：
① 分离参数：$a \geq f(x)$恒成立→$a \geq f(x)_{max}$
② 直接求最值：求$f(x)_{max} \leq 0$
• 分离参数法适用于参数容易分离的情况
• 直接求最值法适用于含参讨论更简便的情况

1.2 方程思想的核心应用

核心概念：方程思想是通过设未知数、建立方程或方程组，通过解方程来解决问题的思想方法。它是解决各类数学问题的基本工具。
方程思想的主要应用场景

1. 解析几何中的方程思想
• 直线与圆锥曲线联立方程组
• 利用韦达定理建立关系
• 设而不求、整体代换

2. 函数零点与方程根
• 函数$y = f(x)$的零点⇔方程$f(x) = 0$的根
• 函数$y = f(x) - g(x)$的零点⇔方程$f(x) = g(x)$的根⇔两函数图像交点

3. 数列中的方程思想
• 基本量法：用$a_1$和d（或q）表示所有量
• 建立方程组求解基本量
📝 例题3（方程思想在解析几何中的应用）：已知椭圆$C: \frac{x^2}{4} + \frac{y^2}{3} = 1$，过点$P(1, 1)$的直线l交椭圆于A、B两点，且P为AB的中点，求直线l的方程。

【命题意图】考查点差法的应用，体现设而不求、整体代换的方程思想。
【规范解答】

方法一：点差法

设$A(x_1, y_1)$，$B(x_2, y_2)$

∵ A、B在椭圆上：
$\frac{x_1^2}{4} + \frac{y_1^2}{3} = 1$ ... ①
$\frac{x_2^2}{4} + \frac{y_2^2}{3} = 1$ ... ②

①-②：
$\frac{x_1^2 - x_2^2}{4} + \frac{y_1^2 - y_2^2}{3} = 0$

$\frac{(x_1+x_2)(x_1-x_2)}{4} + \frac{(y_1+y_2)(y_1-y_2)}{3} = 0$

∵ P(1,1)是AB中点
∴ $x_1 + x_2 = 2$，$y_1 + y_2 = 2$

代入：
$\frac{2(x_1-x_2)}{4} + \frac{2(y_1-y_2)}{3} = 0$

$\frac{x_1-x_2}{2} + \frac{2(y_1-y_2)}{3} = 0$

斜率$k = \frac{y_1-y_2}{x_1-x_2} = -\frac{3}{4}$

直线方程：$y - 1 = -\frac{3}{4}(x - 1)$

即$3x + 4y - 7 = 0$

方法二：联立方程组

设直线l：$y - 1 = k(x - 1)$，即$y = kx + (1-k)$

联立椭圆方程：
$\begin{cases} y = kx + (1-k) \\ \frac{x^2}{4} + \frac{y^2}{3} = 1 \end{cases}$

代入得：$3x^2 + 4[kx + (1-k)]^2 = 12$

展开整理（过程略），利用中点坐标公式$x_1 + x_2 = 2$

由韦达定理可求得$k = -\frac{3}{4}$

（点差法更简洁，计算量更小）
【方法总结】
• 点差法适用条件：已知弦中点坐标，求弦所在直线方程
• 点差法步骤：
① 设弦两端点，代入曲线方程
② 两式相减，因式分解
③ 利用中点坐标，求斜率
• 核心思想：设而不求、整体代换

第二章：数形结合思想

2.1 以形助数

核心概念：以形助数是利用图形的直观性来理解和解决代数问题。通过画图，可以将抽象的代数关系转化为直观的几何关系，从而找到解题思路。
以形助数的典型应用

1. 函数图像的应用
• 利用函数图像判断零点个数
• 利用图像比较函数值大小
• 利用图像分析函数性质（单调性、奇偶性等）

2. 几何意义的应用
• $\sqrt{(x-a)^2 + (y-b)^2}$表示点$(x,y)$到点$(a,b)$的距离
• $\frac{y-b}{x-a}$表示点$(x,y)$与点$(a,b)$连线的斜率
• 利用几何意义求代数式的最值

3. 向量的几何应用
• 利用向量加减法的几何意义
• 利用数量积的几何意义（投影）
📝 例题4（几何意义求最值）：求函数$y = \sqrt{x^2 + 2x + 2} + \sqrt{x^2 - 4x + 8}$的最小值。

【命题意图】考查代数式的几何意义转化，以及利用几何性质求最值。
【规范解答】

将函数变形：

$y = \sqrt{(x+1)^2 + 1} + \sqrt{(x-2)^2 + 4}$

$= \sqrt{(x+1)^2 + (0-1)^2} + \sqrt{(x-2)^2 + (0-2)^2}$

几何意义：
设点$P(x, 0)$在x轴上，点$A(-1, 1)$，点$B(2, 2)$

则$y = |PA| + |PB|$

问题转化为：在x轴上找一点P，使P到A、B两点的距离之和最小

利用对称性：作A关于x轴的对称点$A'(-1, -1)$

则$|PA| = |PA'|$

∴ $y = |PA'| + |PB| \geq |A'B|$（三角形两边之和大于第三边）

当P在A'B与x轴交点时取等号

$|A'B| = \sqrt{(2-(-1))^2 + (2-(-1))^2} = \sqrt{9 + 9} = \sqrt{18} = 3\sqrt{2}$

∴ 最小值为$3\sqrt{2}$
【方法总结】
• 距离公式的识别：$\sqrt{(x-a)^2 + (y-b)^2}$
• 常见变形：将代数式配成完全平方
• 对称法求最短路径：作对称点，化折线为直线
• 关键步骤：识别几何意义→构造几何图形→利用几何性质
📝 例题5（函数零点问题）：讨论方程$|x^2 - 4x + 3| = a$的实根个数。

【命题意图】考查利用函数图像分析方程根的个数，体现数形结合思想。
【规范解答】

设$f(x) = |x^2 - 4x + 3| = |(x-1)(x-3)|$

先画$y = x^2 - 4x + 3 = (x-1)(x-3)$的图像：
• 开口向上，与x轴交于(1,0)和(3,0)
• 顶点为$(2, -1)$

再画$f(x) = |x^2 - 4x + 3|$的图像：
• 将x轴下方的部分翻折到上方
• 顶点$(2, -1)$变为$(2, 1)$

画水平线$y = a$，观察交点个数：

分析：
• 当$a < 0$时：0个实根（无交点）
• 当$a = 0$时：2个实根（x=1, x=3）
• 当$0 < a < 1$时：4个实根（4个交点）
• 当$a = 1$时：3个实根（顶点处相切）
• 当$a > 1$时：2个实根（2个交点）

结论：
a的取值 a < 0 a = 0 0 < a < 1 a = 1 a > 1
实根个数 0 2 4 3 2
【方法总结】
• 方程根的个数 = 两函数图像交点个数
• 步骤：分离函数→画图像→观察交点
• 含绝对值的函数：先画内部函数，再翻折
• 关键：找准临界点（顶点、零点）

核心概念：以数解形是利用代数方法来研究几何问题。通过建立坐标系，将几何问题转化为代数问题，利用代数运算来解决。

📝 例题6（向量法解几何问题）：在△ABC中，AB=3，AC=4，∠BAC=60°，点D是BC的中点，求AD的长度。

【命题意图】考查向量法在几何问题中的应用，以及向量数量积的运算。
【规范解答】

方法一：向量法

∵ D是BC中点

∴ $\overrightarrow{AD} = \frac{1}{2}(\overrightarrow{AB} + \overrightarrow{AC})$

$|\overrightarrow{AD}|^2 = \frac{1}{4}(\overrightarrow{AB} + \overrightarrow{AC})^2$

$= \frac{1}{4}(|\overrightarrow{AB}|^2 + 2\overrightarrow{AB} \cdot \overrightarrow{AC} + |\overrightarrow{AC}|^2)$

$= \frac{1}{4}(9 + 2 \times 3 \times 4 \times \cos 60° + 16)$

$= \frac{1}{4}(9 + 12 + 16)$

$= \frac{37}{4}$

∴ $|AD| = \sqrt{\frac{37}{4}} = $ $\frac{\sqrt{37}}{2}$

方法二：余弦定理+中线公式

先用余弦定理求BC：
$BC^2 = AB^2 + AC^2 - 2AB \cdot AC \cdot \cos 60°$
$= 9 + 16 - 2 \times 3 \times 4 \times \frac{1}{2}$
$= 25 - 12 = 13$

再用中线公式：
$AD^2 = \frac{2AB^2 + 2AC^2 - BC^2}{4}$
$= \frac{2 \times 9 + 2 \times 16 - 13}{4}$
$= \frac{18 + 32 - 13}{4} = \frac{37}{4}$

∴ $AD = \frac{\sqrt{37}}{2}$
【方法总结】
• 向量法求中线：$\overrightarrow{AD} = \frac{1}{2}(\overrightarrow{AB} + \overrightarrow{AC})$
• 关键公式：$|\vec{a}|^2 = \vec{a}^2$
• 数量积：$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$
• 中线长公式：$m_a^2 = \frac{2b^2 + 2c^2 - a^2}{4}$

第三章：分类讨论与整合思想

3.1 分类讨论的原则与方法

核心概念：分类讨论是当问题所给的对象不能进行统一研究时，需要对研究对象按某个标准分类，然后对每一类分别研究，得出每一类的结论，最后综合各类结果得到整个问题的解答。
分类讨论的常见类型

1. 由概念引起的分类
• 绝对值：$|a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases}$
• 等比数列求和：$q = 1$和$q \neq 1$
• 直线的斜率：斜率存在与不存在

2. 由运算引起的分类
• 不等式两边同乘（除）数：正数、负数
• 对数运算：底数$0 < a < 1$和$a > 1$

3. 由参数引起的分类
• 含参方程、不等式的讨论
• 二次项系数为0和不为0
• 判别式$\Delta > 0$、$\Delta = 0$、$\Delta < 0$
分类讨论的原则
  1. 不重不漏：各类之间互斥，且覆盖所有情况
  2. 标准统一：每次分类只能用一个标准
  3. 层次清晰：多级分类要有明确的层次
  4. 及时整合：分类后要综合各类结论
📝 例题7（含参不等式分类讨论）：解关于x的不等式$ax^2 - (a+1)x + 1 < 0$。

【命题意图】考查含参二次不等式的分类讨论，体现分类讨论思想的系统应用。
【规范解答】

原不等式：$ax^2 - (a+1)x + 1 < 0$

因式分解：$(ax - 1)(x - 1) < 0$

第一层分类：按a是否为0

情况一：a = 0

不等式变为：$-x + 1 < 0$，即$x > 1$

情况二：a ≠ 0

方程$(ax-1)(x-1) = 0$的根为$x_1 = \frac{1}{a}$，$x_2 = 1$

第二层分类：按a的正负

(1) a > 0

抛物线开口向上，不等式<0取两根之间

再按两根大小分类：

• 当$\frac{1}{a} < 1$即$a > 1$时：$\frac{1}{a} < x < 1$

• 当$\frac{1}{a} = 1$即$a = 1$时：
不等式变为$(x-1)^2 < 0$，无解

• 当$\frac{1}{a} > 1$即$0 < a < 1$时：$1 < x < \frac{1}{a}$

(2) a < 0

抛物线开口向下，不等式<0取两根之外

此时$\frac{1}{a} < 0 < 1$，所以$\frac{1}{a} < 1$

解集为：$x < \frac{1}{a}$或$x > 1$

【综合结论】

a的范围 解集
a < 0 $(-\infty, \frac{1}{a}) \cup (1, +\infty)$
a = 0 $(1, +\infty)$
0 < a < 1 $(1, \frac{1}{a})$
a = 1 $\varnothing$
a > 1 $(\frac{1}{a}, 1)$
【方法总结】
• 含参二次不等式讨论层次：
① 二次项系数是否为0
② 二次项系数正负（开口方向）
③ 两根大小关系
• 关键：找到临界值（使根相等、使系数为0的值）
• 建议画数轴辅助分析

3.2 整合思想的应用

核心概念：整合思想是在分类讨论后，将各类结论进行综合，或者将分散的条件、方法进行整合，形成完整的解题方案。
整合思想的主要形式

1. 分类后的整合
• 将各类情况的结论汇总
• 用表格、集合等形式呈现完整结论

2. 条件的整合
• 将分散的条件集中使用
• 寻找条件之间的联系

3. 方法的整合
• 多种方法联合使用
• 不同知识模块的综合应用

📝 例题8（正难则反）：从1,2,3,...,10这10个数中任取3个不同的数，求这3个数中至少有一个偶数的取法种数。

【命题意图】考查"正难则反"的转化策略，以及对立事件的应用。
【规范解答】

方法一：直接法（分类讨论）

10个数中有5个偶数(2,4,6,8,10)，5个奇数(1,3,5,7,9)

"至少一个偶数"包括：
• 1个偶数2个奇数：$C_5^1 \times C_5^2 = 5 \times 10 = 50$
• 2个偶数1个奇数：$C_5^2 \times C_5^1 = 10 \times 5 = 50$
• 3个偶数0个奇数：$C_5^3 = 10$

共：$50 + 50 + 10 = 110$种

方法二：间接法（推荐）

"至少一个偶数"的对立事件是"全是奇数"

总取法：$C_{10}^3 = 120$

全是奇数的取法：$C_5^3 = 10$

∴ 至少一个偶数的取法：
$C_{10}^3 - C_5^3 = 120 - 10 = $ 110种

显然间接法更简洁！
【方法总结】
• "至少"问题优先考虑间接法
• 公式：$N(至少一个) = N(总数) - N(全没有)$
• 正面情况复杂（需要分类）时，反面往往简单
• 关键词识别："至少"、"至多"、"不都"、"不全"
📝 例题9（换元转化）：求函数$f(x) = 4^x - 3 \cdot 2^x + 2$的值域。

【命题意图】考查换元法将指数函数转化为二次函数，体现复杂向简单的转化。
【规范解答】

令$t = 2^x$，则$t > 0$

$f(x) = (2^x)^2 - 3 \cdot 2^x + 2 = t^2 - 3t + 2$

设$g(t) = t^2 - 3t + 2$（$t > 0$）

配方：$g(t) = (t - \frac{3}{2})^2 - \frac{9}{4} + 2 = (t - \frac{3}{2})^2 - \frac{1}{4}$

∵ $t > 0$，抛物线开口向上，对称轴$t = \frac{3}{2}$

当$t = \frac{3}{2}$时，$g(t)_{min} = -\frac{1}{4}$

当$t \to +\infty$时，$g(t) \to +\infty$

∴ 值域为$[-\frac{1}{4}, +\infty)$
【方法总结】
• 指数型函数换元：令$t = a^x$（$t > 0$）
• 对数型函数换元：令$t = \log_a x$
• 三角函数换元：令$t = \sin x$或$t = \cos x$（注意范围）
• 换元后一定要注意新变量的取值范围

第五章：特殊与一般思想

5.1 特殊化方法

核心概念：特殊化是从一般到特殊的思维过程。通过考察特殊情形，发现规律、验证猜想、简化计算，从而解决一般问题。
特殊化的常见应用

1. 特殊值法
• 选择题、填空题中，用特殊值代替一般变量
• 常用特殊值：0, 1, -1, 边界值等

2. 特殊位置法
• 几何问题中取特殊位置（中点、端点等）
• 动点问题中取特殊点

3. 特殊情形法
• 先解决简单情形，再推广到一般
• 归纳猜想
📝 例题10（特殊值法）：设函数$f(x)$是定义在R上的奇函数，且$f(x+2) = f(x)$，当$0 \leq x \leq 1$时，$f(x) = x$，求$f(7.5)$。

【命题意图】考查函数周期性和奇偶性的综合应用，可用特殊值思路简化。
【规范解答】

已知条件：
• 奇函数：$f(-x) = -f(x)$，$f(0) = 0$
• 周期为2：$f(x+2) = f(x)$
• $0 \leq x \leq 1$时，$f(x) = x$

求$f(7.5)$：

利用周期性：
$f(7.5) = f(7.5 - 2 \times 4) = f(7.5 - 8) = f(-0.5)$

利用奇函数性质：
$f(-0.5) = -f(0.5)$

∵ $0 \leq 0.5 \leq 1$

∴ $f(0.5) = 0.5$

∴ $f(7.5) = -0.5 = $ $-\frac{1}{2}$
【方法总结】
• 周期性：$f(x+T) = f(x)$，可将大数化小
• 奇函数：$f(-x) = -f(x)$，负变正
• 综合策略：周期性化小→奇偶性变号→代入已知表达式
• 步骤：$f(7.5) \to f(-0.5) \to -f(0.5) \to -0.5$

5.2 一般化方法

核心概念：一般化是从特殊到一般的思维过程。通过将特殊问题推广到一般情形，发现普遍规律，获得更深刻的认识。
一般化的主要形式

1. 从具体到抽象
• 将具体数字推广为一般变量
• 发现一般规律

2. 从特殊到普遍
• 归纳猜想一般结论
• 数学归纳法证明

3. 类比推广
• 平面到空间的推广
• 低维到高维的推广

第六章：有限与无限思想

6.1 极限思想

核心概念：极限思想是通过有限认识无限、通过近似认识精确、通过量变认识质变的数学思想。它是微积分的核心思想，也是解决许多数学问题的重要工具。
极限思想的应用

1. 导数的定义
$f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}$

2. 渐近线
• 水平渐近线：$\lim_{x \to \infty} f(x) = a$
• 垂直渐近线：$\lim_{x \to x_0} f(x) = \infty$

3. 无穷等比数列求和
$S = \frac{a_1}{1-q}$（$|q| < 1$）

4. 定积分的定义
$\int_a^b f(x)dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(\xi_i)\Delta x$
📝 例题11（极限思想应用）：求曲线$y = \frac{1}{x}$（$x > 0$）与直线$x = 1$、$x = n$（$n > 1$）及x轴围成的区域面积，并求当$n \to \infty$时的极限。

【命题意图】考查定积分的几何意义，以及极限思想在面积计算中的应用。
【规范解答】

面积$S = \int_1^n \frac{1}{x} dx = [\ln x]_1^n = \ln n - \ln 1 = \ln n$

当$n \to \infty$时：

$\lim_{n \to \infty} S = \lim_{n \to \infty} \ln n = $ $+\infty$

这说明虽然曲线$y = \frac{1}{x}$无限接近x轴，但围成的区域面积却是无限的（广义积分发散）。

补充：有趣的是，将这个区域绕x轴旋转形成的旋转体体积却是有限的：

$V = \pi \int_1^{\infty} (\frac{1}{x})^2 dx = \pi [-\frac{1}{x}]_1^{\infty} = \pi(0 - (-1)) = \pi$

这就是著名的"加百列号角"悖论：有限的体积，无限的表面积。
【方法总结】
• 定积分求面积：$S = \int_a^b f(x)dx$
• 反常积分：上限为无穷时取极限
• 极限思想：有限→无限，近似→精确
• 注意：面积无限不代表体积无限

第八章：知识模块融合专题

8.1 函数与导数的综合应用

📝 例题12（2024新课标卷风格）：已知函数$f(x) = e^x - ax^2 - x - 1$。
(1) 当a=1时，求f(x)的单调区间；
(2) 当$x \geq 0$时，$f(x) \geq 0$恒成立，求a的取值范围。

【命题意图】考查导数在研究函数性质中的应用，以及恒成立问题的处理，是高考压轴题常见题型。
【规范解答】

(1) 当a=1时

$f(x) = e^x - x^2 - x - 1$
$f'(x) = e^x - 2x - 1$

令$g(x) = e^x - 2x - 1$，则$g'(x) = e^x - 2$

令$g'(x) = 0$，得$x = \ln 2$

当$x < \ln 2$时，$g'(x) < 0$，g(x)单调递减
当$x > \ln 2$时，$g'(x) > 0$，g(x)单调递增

$g(x)_{min} = g(\ln 2) = 2 - 2\ln 2 - 1 = 1 - \ln 4 < 0$

又$g(0) = 0$，$g(2) = e^2 - 5 > 0$

∴ g(x)=0有两个根：$x_1 = 0$，$x_2 \in (\ln 2, 2)$

分析f'(x)=g(x)的符号：
• $x \in (-\infty, 0)$：f'(x) > 0，f(x)单调递增
• $x \in (0, x_2)$：f'(x) < 0，f(x)单调递减
• $x \in (x_2, +\infty)$：f'(x) > 0，f(x)单调递增

(2) 求a的取值范围

当$x = 0$时，$f(0) = 1 - 0 - 0 - 1 = 0$ ✓

当$x > 0$时，$f(x) \geq 0$即$e^x - ax^2 - x - 1 \geq 0$

$ax^2 \leq e^x - x - 1$

$a \leq \frac{e^x - x - 1}{x^2}$

设$h(x) = \frac{e^x - x - 1}{x^2}$（$x > 0$）

求导（过程略）：$h'(x) = \frac{(x-2)e^x + x + 2}{x^3}$

分析分子：设$\varphi(x) = (x-2)e^x + x + 2$

$\varphi'(x) = (x-1)e^x + 1$，$\varphi''(x) = xe^x > 0$（$x > 0$）

∴ $\varphi'(x)$在$(0, +\infty)$单调递增，$\varphi'(x) > \varphi'(0) = 0$

∴ $\varphi(x)$在$(0, +\infty)$单调递增，$\varphi(x) > \varphi(0) = 0$

∴ $h'(x) > 0$，h(x)在$(0, +\infty)$单调递增

$\lim_{x \to 0^+} h(x) = \lim_{x \to 0^+} \frac{e^x - x - 1}{x^2} = \lim_{x \to 0^+} \frac{e^x - 1}{2x} = \lim_{x \to 0^+} \frac{e^x}{2} = \frac{1}{2}$

∴ $h(x) > \frac{1}{2}$对所有$x > 0$成立

∴ $a \leq \frac{1}{2}$
【方法总结】
• 恒成立问题：分离参数→构造函数→求最值
• 多次求导：判断导数符号困难时，对导数再求导
• 极限计算：洛必达法则或等价无穷小
• 注意：$x = 0$需要单独验证

8.2 解析几何与向量的融合

📝 例题13（原创）：已知椭圆$C: \frac{x^2}{4} + \frac{y^2}{3} = 1$的左、右焦点分别为$F_1$、$F_2$，点P在椭圆上，若$\overrightarrow{PF_1} \cdot \overrightarrow{PF_2} = 1$，求点P的坐标。

【命题意图】考查椭圆的几何性质与向量数量积的综合应用。
【规范解答】

由椭圆方程：$a^2 = 4$，$b^2 = 3$

$c^2 = a^2 - b^2 = 4 - 3 = 1$，∴ $c = 1$

焦点：$F_1(-1, 0)$，$F_2(1, 0)$

设$P(x, y)$，则：

$\overrightarrow{PF_1} = (-1-x, -y)$，$\overrightarrow{PF_2} = (1-x, -y)$

$\overrightarrow{PF_1} \cdot \overrightarrow{PF_2} = (-1-x)(1-x) + (-y)(-y)$
$= -(1-x^2) + y^2 = x^2 + y^2 - 1 = 1$

∴ $x^2 + y^2 = 2$ ... ①

又P在椭圆上：$\frac{x^2}{4} + \frac{y^2}{3} = 1$ ... ②

由①：$y^2 = 2 - x^2$，代入②：

$\frac{x^2}{4} + \frac{2-x^2}{3} = 1$

$3x^2 + 4(2-x^2) = 12$

$3x^2 + 8 - 4x^2 = 12$

$-x^2 = 4$，$x^2 = -4$（无解？）

重新检查：

$\overrightarrow{PF_1} \cdot \overrightarrow{PF_2} = (-1-x)(1-x) + y^2 = x^2 - 1 + y^2 = 1$

∴ $x^2 + y^2 = 2$

由②：$3x^2 + 4y^2 = 12$

由①×3：$3x^2 + 3y^2 = 6$ ... ③

②-③：$y^2 = 6$，$y = \pm\sqrt{6}$

代入①：$x^2 = 2 - 6 = -4$（仍无解）

分析：说明满足条件的点P不存在，或题目条件有误。

实际上，由椭圆性质，$|PF_1| + |PF_2| = 2a = 4$

设$|PF_1| = m$，$|PF_2| = n$，则$m + n = 4$

由余弦定理：$\overrightarrow{PF_1} \cdot \overrightarrow{PF_2} = mn\cos\theta = 1$

又$|F_1F_2| = 2c = 2$

在△$PF_1F_2$中：$4 = m^2 + n^2 - 2mn\cos\theta = m^2 + n^2 - 2$

∴ $m^2 + n^2 = 6$

而$(m+n)^2 = 16 = m^2 + n^2 + 2mn = 6 + 2mn$

∴ $mn = 5$

但$m + n = 4$，$mn = 5$时，m、n是方程$t^2 - 4t + 5 = 0$的根

$\Delta = 16 - 20 = -4 < 0$，无实数解

结论：满足条件的点P不存在。
【方法总结】
• 向量数量积的坐标运算：$\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2$
• 椭圆定义：$|PF_1| + |PF_2| = 2a$
• 焦点三角形：结合余弦定理和向量
• 注意：不是所有条件都能得到满足

8.3 数列与不等式的融合

📝 例题14（原创）：已知数列{an}满足$a_1 = 1$，$a_{n+1} = 2a_n + 1$。
(1) 求数列{an}的通项公式；
(2) 证明：$\frac{1}{a_1} + \frac{1}{a_2} + ... + \frac{1}{a_n} < 2$。

【命题意图】考查递推数列的求解和数列不等式的证明，体现构造法和放缩法的应用。
【规范解答】

(1) 求通项公式

由$a_{n+1} = 2a_n + 1$

变形：$a_{n+1} + 1 = 2(a_n + 1)$

∴ 数列{$a_n + 1$}是等比数列，公比为2

首项：$a_1 + 1 = 2$

∴ $a_n + 1 = 2 \cdot 2^{n-1} = 2^n$

∴ $a_n = 2^n - 1$

(2) 证明不等式

要证：$\sum_{k=1}^{n} \frac{1}{a_k} = \sum_{k=1}^{n} \frac{1}{2^k - 1} < 2$

放缩法：

当$k \geq 1$时，$2^k - 1 \geq 2^{k-1}$（因为$2^k - 1 = 2 \cdot 2^{k-1} - 1 \geq 2^{k-1}$当$2^{k-1} \geq 1$即$k \geq 1$）

∴ $\frac{1}{2^k - 1} \leq \frac{1}{2^{k-1}}$

∴ $\sum_{k=1}^{n} \frac{1}{2^k - 1} \leq \sum_{k=1}^{n} \frac{1}{2^{k-1}} = 1 + \frac{1}{2} + \frac{1}{4} + ... + \frac{1}{2^{n-1}}$

$= \frac{1 - (\frac{1}{2})^n}{1 - \frac{1}{2}} = 2(1 - \frac{1}{2^n}) < 2$

得证。
【方法总结】
• 递推数列$a_{n+1} = pa_n + q$的解法：
构造$a_{n+1} + \lambda = p(a_n + \lambda)$
• 数列不等式证明常用方法：
① 放缩法：将通项放大或缩小
② 数学归纳法
③ 裂项相消法
• 放缩技巧：$2^n - 1 \geq 2^{n-1}$（$n \geq 1$）
