第一章 空间几何体

1.1 空间几何体的结构特征

一、柱体

棱柱：有两个面互相平行，其余各面都是四边形，并且每相邻两个四边形的公共边都互相平行。

圆柱：以矩形的一边所在直线为旋转轴，其余三边旋转形成的面所围成的旋转体。
柱体公式：

棱柱：
侧面积：$S_{侧} = ch$（c为底面周长，h为高）
体积：$V = Sh$（S为底面积，h为高）

圆柱：
侧面积：$S_{侧} = 2\pi rh$
表面积：$S_{表} = 2\pi r^2 + 2\pi rh$
体积：$V = \pi r^2h$

二、锥体

棱锥：有一个面是多边形，其余各面是有一个公共顶点的三角形。

圆锥：以直角三角形的一条直角边所在直线为旋转轴，其余两边旋转形成的面所围成的旋转体。
锥体公式：

棱锥：
侧面积：$S_{侧} = \frac{1}{2}ch'$（c为底面周长，h'为斜高）
体积：$V = \frac{1}{3}Sh$

圆锥：
侧面积：$S_{侧} = \pi rl$（l为母线长）
表面积：$S_{表} = \pi r^2 + \pi rl$
体积：$V = \frac{1}{3}\pi r^2h$

重要关系：$l^2 = r^2 + h^2$
📝 例题1（原创）：已知圆锥的底面半径为3，高为4，求圆锥的侧面积和体积。

【命题意图】考查圆锥的基本量计算，以及母线、半径、高的关系。
【规范解答】

第一步：求母线长
由$l^2 = r^2 + h^2 = 3^2 + 4^2 = 9 + 16 = 25$
∴ $l = 5$

第二步：求侧面积
$S_{侧} = \pi rl = \pi × 3 × 5 = $ $15\pi$

第三步：求体积
$V = \frac{1}{3}\pi r^2h = \frac{1}{3}\pi × 9 × 4 = $ $12\pi$
【方法总结】
• 圆锥问题：先画轴截面（等腰三角形），利用$l^2 = r^2 + h^2$
• 侧面积公式：$S_{侧} = \pi rl$
• 体积公式：$V = \frac{1}{3}\pi r^2h$
• 记住常用勾股数：3-4-5，5-12-13等

三、球体

球体公式：

表面积：$S = 4\pi R^2$
体积：$V = \frac{4}{3}\pi R^3$

重要结论：
• 球的截面是圆，截面半径r与球心到截面距离d满足：$r^2 + d^2 = R^2$
• 长方体的外接球直径等于体对角线：$2R = \sqrt{a^2+b^2+c^2}$
• 正方体的外接球直径：$2R = \sqrt{3}a$，内切球直径：$2R = a$
📝 例题2（原创）：已知长方体的长、宽、高分别为3、4、5，求该长方体外接球的表面积。

【规范解答】

长方体的体对角线长：
$d = \sqrt{3^2+4^2+5^2} = \sqrt{9+16+25} = \sqrt{50} = 5\sqrt{2}$

外接球直径等于体对角线：
$2R = 5\sqrt{2}$
$R = \frac{5\sqrt{2}}{2}$

外接球表面积：
$S = 4\pi R^2 = 4\pi × \frac{25×2}{4} = $ $50\pi$
【方法总结】
• 长方体外接球：直径=体对角线=$\sqrt{a^2+b^2+c^2}$
• 正方体外接球：$R = \frac{\sqrt{3}a}{2}$
• 正方体内切球：$R = \frac{a}{2}$
• 球与几何体相切问题：找球心，利用对称性

1.2 三视图与直观图

一、三视图

三视图：正视图（主视图）、侧视图（左视图）、俯视图

投影规律：
• 长对正：正视图与俯视图长对正
• 高平齐：正视图与侧视图高平齐
• 宽相等：俯视图与侧视图宽相等
📝 例题3（原创）：某几何体的三视图如图所示（描述：正视图为边长为2的正方形，侧视图为边长为2的正方形，俯视图为半径为1的圆），求该几何体的表面积。

【命题意图】考查由三视图还原几何体，以及圆柱表面积计算。
【分析思路】
由三视图可知：正视图和侧视图都是正方形，俯视图是圆，说明该几何体是圆柱。

【规范解答】
由三视图还原：该几何体为圆柱

底面半径：$r = 1$
高：$h = 2$

表面积：
$S = 2\pi r^2 + 2\pi rh$
$= 2\pi × 1 + 2\pi × 1 × 2$
$= 2\pi + 4\pi$
$= $ $6\pi$
【方法总结】
• 由三视图还原几何体：先看俯视图确定底面形状，再看正视图和侧视图确定高度
• 常见组合：矩形+圆=圆柱，三角形+圆=圆锥，矩形+矩形=棱柱
• 注意三视图中的虚线表示被遮挡的棱

二、斜二测画法

斜二测画法规则：

1. 坐标轴：x'轴与y'轴夹角为45°（或135°）
2. 平行性：平行于x轴的线段平行于x'轴，平行于y轴的线段平行于y'轴
3. 长度：平行于x轴的线段长度不变，平行于y轴的线段长度变为原来的一半

面积关系：
原图形面积S与直观图面积S'的关系：$S' = \frac{\sqrt{2}}{4}S$，即$S = 2\sqrt{2}S'$
📝 例题4（原创）：用斜二测画法画水平放置的正方形的直观图，若直观图中一边长为2，求原正方形的面积。

【规范解答】

情况一：直观图中平行于x'轴的边长为2
则原正方形中对应边长为2
平行于y'轴的边长设为a，则原边长为2a

由于原图形是正方形，∴ 2 = 2a，a = 1

原正方形边长为2，面积 = 4

情况二：直观图中平行于y'轴的边长为2
则原正方形中对应边长为4
平行于x'轴的边长为4（因为原图形是正方形）

原正方形边长为4，面积 = 16
【易错提醒】
⚠️ 斜二测画法中，平行于y轴的线段长度要减半！
⚠️ 还原时，平行于y'轴的线段长度要加倍！
⚠️ 注意区分直观图中的边对应原图形中哪个方向

第二章 点、线、面的位置关系

2.1 平面的基本性质

一、四个公理

公理1：如果一条直线上的两点在一个平面内，那么这条直线在此平面内。

公理2：过不在一条直线上的三点，有且只有一个平面。

公理3：如果两个不重合的平面有一个公共点，那么它们有且只有一条过该点的公共直线。

公理4：平行于同一条直线的两条直线互相平行。（平行公理）
确定平面的条件：
1. 不共线的三点
2. 一条直线和直线外一点
3. 两条相交直线
4. 两条平行直线

2.2 空间中的平行关系

一、线面平行

线面平行判定定理：平面外一条直线与此平面内的一条直线平行，则该直线与此平面平行。

符号表示：$a \not\subset \alpha, b \subset \alpha, a \parallel b \Rightarrow a \parallel \alpha$

线面平行性质定理：一条直线与一个平面平行，则过这条直线的任一平面与此平面的交线与该直线平行。

符号表示：$a \parallel \alpha, a \subset \beta, \alpha \cap \beta = b \Rightarrow a \parallel b$
📝 例题5（原创）：在正方体$ABCD-A_1B_1C_1D_1$中，E、F分别是$AB$、$BC$的中点，求证：$EF \parallel$平面$A_1B_1C_1D_1$。

【命题意图】考查线面平行的判定定理应用，以及中位线性质。
【规范解答】

证明：

在△ABC中，
∵ E、F分别是AB、BC的中点
∴ $EF \parallel AC$（三角形中位线定理）

在正方体中，
∵ $AC \parallel A_1C_1$（面对角线平行）
∴ $EF \parallel A_1C_1$

又∵ $A_1C_1 \subset$平面$A_1B_1C_1D_1$，$EF \not\subset$平面$A_1B_1C_1D_1$

∴ $EF \parallel$平面$A_1B_1C_1D_1$（线面平行判定定理）
【方法总结】
• 证明线面平行的基本思路：找平面内一条直线与已知直线平行
• 常用方法：中位线、平行四边形、比例线段
• 注意：必须说明直线在平面外

二、面面平行

面面平行判定定理：一个平面内的两条相交直线都与另一个平面平行，则这两个平面平行。

符号表示：$a \subset \alpha, b \subset \alpha, a \cap b = P, a \parallel \beta, b \parallel \beta \Rightarrow \alpha \parallel \beta$

面面平行性质定理：如果两个平行平面同时和第三个平面相交，那么它们的交线平行。

符号表示：$\alpha \parallel \beta, \alpha \cap \gamma = a, \beta \cap \gamma = b \Rightarrow a \parallel b$

2.3 空间中的垂直关系

一、线面垂直

线面垂直判定定理：一条直线与一个平面内的两条相交直线都垂直，则该直线与此平面垂直。

符号表示：$a \subset \alpha, b \subset \alpha, a \cap b = P, l \perp a, l \perp b \Rightarrow l \perp \alpha$

线面垂直性质定理：垂直于同一个平面的两条直线平行。

符号表示：$a \perp \alpha, b \perp \alpha \Rightarrow a \parallel b$
📝 例题6（原创）：在正方体$ABCD-A_1B_1C_1D_1$中，求证：$AC_1 \perp$平面$A_1BD$。

【命题意图】考查线面垂直的判定定理，以及正方体中的垂直关系。
【规范解答】

证明：

设正方体棱长为a，建立坐标系或使用几何法。

方法一：几何法

先证$AC_1 \perp A_1B$：
∵ $A_1B \perp AB_1$（正方形对角线垂直）
$A_1B \perp B_1C_1$（$B_1C_1 \perp$平面$ABB_1A_1$）
∴ $A_1B \perp$平面$AB_1C_1$
∴ $A_1B \perp AC_1$

同理可证$AC_1 \perp A_1D$

又∵ $A_1B \cap A_1D = A_1$，$A_1B, A_1D \subset$平面$A_1BD$

∴ $AC_1 \perp$平面$A_1BD$

方法二：向量法

以D为原点，建立坐标系D-xyz
设棱长为1，则：
$A(1,0,0), C_1(0,1,1), A_1(1,0,1), B(1,1,0), D(0,0,0)$

$\overrightarrow{AC_1} = (-1, 1, 1)$
$\overrightarrow{A_1B} = (0, 1, -1)$
$\overrightarrow{A_1D} = (-1, 0, -1)$

$\overrightarrow{AC_1} · \overrightarrow{A_1B} = 0 + 1 - 1 = 0$
$\overrightarrow{AC_1} · \overrightarrow{A_1D} = 1 + 0 - 1 = 0$

∴ $AC_1 \perp A_1B$，$AC_1 \perp A_1D$

∴ $AC_1 \perp$平面$A_1BD$
【方法总结】
• 证明线面垂直：找平面内两条相交直线都与已知直线垂直
• 正方体中常用结论：体对角线垂直于面对角线构成的平面
• 向量法：证明方向向量与平面内两不共线向量都垂直（点积为0）

二、面面垂直

面面垂直判定定理：一个平面过另一个平面的垂线，则这两个平面垂直。

符号表示：$l \perp \alpha, l \subset \beta \Rightarrow \alpha \perp \beta$

面面垂直性质定理：两个平面垂直，则一个平面内垂直于交线的直线与另一个平面垂直。

符号表示：$\alpha \perp \beta, \alpha \cap \beta = l, a \subset \alpha, a \perp l \Rightarrow a \perp \beta$
📝 例题7（原创）：在四棱锥P-ABCD中，底面ABCD是正方形，$PA \perp$平面ABCD，求证：平面$PBD \perp$平面$PAC$。

【规范解答】

证明：

∵ $PA \perp$平面ABCD，$BD \subset$平面ABCD
∴ $PA \perp BD$

∵ 底面ABCD是正方形
∴ $AC \perp BD$

∵ $PA \cap AC = A$，$PA, AC \subset$平面PAC
∴ $BD \perp$平面PAC

又∵ $BD \subset$平面PBD

∴ 平面$PBD \perp$平面PAC（面面垂直判定定理）
【方法总结】
• 证明面面垂直：找其中一个平面过另一个平面的垂线
• 即：证明一个平面内有一条直线垂直于另一个平面
• 常用思路：先证线面垂直，再得面面垂直

第三章 空间向量与立体几何

3.1 空间向量基础

一、空间向量运算

空间向量坐标运算：

设$\vec{a} = (x_1, y_1, z_1)$，$\vec{b} = (x_2, y_2, z_2)$

加减：$\vec{a} ± \vec{b} = (x_1±x_2, y_1±y_2, z_1±z_2)$

数乘：$\lambda\vec{a} = (\lambda x_1, \lambda y_1, \lambda z_1)$

数量积：$\vec{a} · \vec{b} = x_1x_2 + y_1y_2 + z_1z_2$

模长：$|\vec{a}| = \sqrt{x_1^2+y_1^2+z_1^2}$

夹角：$\cos\langle\vec{a},\vec{b}\rangle = \frac{\vec{a}·\vec{b}}{|\vec{a}||\vec{b}|}$

垂直：$\vec{a} \perp \vec{b} \Leftrightarrow \vec{a}·\vec{b} = 0$

平行：$\vec{a} \parallel \vec{b} \Leftrightarrow \vec{a} = \lambda\vec{b} \Leftrightarrow \frac{x_1}{x_2} = \frac{y_1}{y_2} = \frac{z_1}{z_2}$

二、平面的法向量

法向量：垂直于平面的向量叫做平面的法向量。

求法向量的方法：
1. 找平面内两个不共线向量$\vec{a}, \vec{b}$
2. 设法向量$\vec{n} = (x, y, z)$
3. 列方程：$\vec{n}·\vec{a} = 0$，$\vec{n}·\vec{b} = 0$
4. 解方程组，取一组解即可
📝 例题8（原创）：已知平面α过点A(1,0,0)、B(0,2,0)、C(0,0,3)，求平面α的一个法向量。

【规范解答】

$\overrightarrow{AB} = (-1, 2, 0)$
$\overrightarrow{AC} = (-1, 0, 3)$

设法向量$\vec{n} = (x, y, z)$

则：$\vec{n}·\overrightarrow{AB} = -x + 2y = 0$ ... ①
$\vec{n}·\overrightarrow{AC} = -x + 3z = 0$ ... ②

由①：$x = 2y$
由②：$x = 3z$

∴ $2y = 3z$，设$z = 2$，则$y = 3$，$x = 6$

∴ 平面α的一个法向量为$\vec{n} = (6, 3, 2)$

（注：法向量不唯一，任何非零倍数都是法向量）
【方法总结】
• 求法向量：找平面内两不共线向量，设法向量，列方程组
• 为简化计算，可以给某个变量赋一个值（如令z=2）
• 法向量方向：指向平面的哪一侧都可以
• 验证：法向量与平面内向量的点积应为0

3.2 空间角的向量求法

一、异面直线所成的角

异面直线所成角：

设两异面直线的方向向量分别为$\vec{a}, \vec{b}$，所成角为θ

$$\cos\theta = \frac{|\vec{a}·\vec{b}|}{|\vec{a}||\vec{b}|}$$

注意：取绝对值，因为异面直线所成角范围是$(0°, 90°]$
📝 例题9（原创）：在正方体$ABCD-A_1B_1C_1D_1$中，棱长为2，E、F分别是$BB_1$、$C_1D_1$的中点，求异面直线AE与CF所成角的余弦值。

【命题意图】考查用空间向量求异面直线所成角。
【规范解答】

以D为原点，建立坐标系D-xyz

则：$A(2,0,0), E(2,2,1), C(0,2,0), F(0,1,2)$

$\overrightarrow{AE} = (0, 2, 1)$
$\overrightarrow{CF} = (0, -1, 2)$

$\overrightarrow{AE}·\overrightarrow{CF} = 0 - 2 + 2 = 0$

∴ $\cos\theta = \frac{|\overrightarrow{AE}·\overrightarrow{CF}|}{|\overrightarrow{AE}||\overrightarrow{CF}|} = \frac{0}{\sqrt{5}·\sqrt{5}} = $ 0

∴ AE与CF所成角为90°
【方法总结】
• 建系：找两两垂直的线作为坐标轴
• 写点坐标：根据几何关系计算
• 求方向向量：终点减起点
• 用公式求夹角：注意取绝对值

二、线面角

直线与平面所成的角：

设直线的方向向量为$\vec{a}$，平面的法向量为$\vec{n}$，线面角为θ

$$\sin\theta = \frac{|\vec{a}·\vec{n}|}{|\vec{a}||\vec{n}|}$$

注意：线面角范围是$[0°, 90°]$，所以取绝对值后用sin
📝 例题10（原创）：在四棱锥P-ABCD中，底面ABCD是边长为2的正方形，$PA \perp$平面ABCD，$PA = 2$，求直线PC与平面PBD所成角的正弦值。

【规范解答】

以A为原点，建立坐标系A-xyz

则：$P(0,0,2), C(2,2,0), B(2,0,0), D(0,2,0)$

$\overrightarrow{PC} = (2, 2, -2)$

求平面PBD的法向量：
$\overrightarrow{PB} = (2, 0, -2)$
$\overrightarrow{PD} = (0, 2, -2)$

设法向量$\vec{n} = (x, y, z)$
$\vec{n}·\overrightarrow{PB} = 2x - 2z = 0$
$\vec{n}·\overrightarrow{PD} = 2y - 2z = 0$

取$z = 1$，则$x = 1, y = 1$
∴ $\vec{n} = (1, 1, 1)$

设线面角为θ：
$\sin\theta = \frac{|\overrightarrow{PC}·\vec{n}|}{|\overrightarrow{PC}||\vec{n}|}$

$\overrightarrow{PC}·\vec{n} = 2 + 2 - 2 = 2$
$|\overrightarrow{PC}| = \sqrt{4+4+4} = 2\sqrt{3}$
$|\vec{n}| = \sqrt{3}$

$\sin\theta = \frac{2}{2\sqrt{3}·\sqrt{3}} = \frac{2}{6} = $ $\frac{1}{3}$
【方法总结】
• 线面角公式：$\sin\theta = \frac{|\vec{a}·\vec{n}|}{|\vec{a}||\vec{n}|}$
• 注意：分子取绝对值，结果是sin值
• 法向量方向不影响结果（取绝对值）

三、二面角

二面角：

设两个平面的法向量分别为$\vec{n_1}, \vec{n_2}$，二面角为θ

$$|\cos\theta| = \frac{|\vec{n_1}·\vec{n_2}|}{|\vec{n_1}||\vec{n_2}|}$$

注意：
• 二面角范围是$[0°, 180°]$
• 先求$|\cos\theta|$，再根据图形判断是锐角还是钝角
• 法向量方向：同进同出取补角，一进一出取等角
📝 例题11（原创）：在正方体$ABCD-A_1B_1C_1D_1$中，棱长为2，E、F分别是$BC$、$CD$的中点，求二面角$C_1-EF-C$的余弦值。

【规范解答】

以D为原点，建立坐标系D-xyz

则：$C(0,2,0), C_1(0,2,2), E(1,2,0), F(0,1,0)$

求平面CEF的法向量：
平面CEF即底面ABCD，法向量$\vec{n_1} = (0, 0, 1)$

求平面$C_1EF$的法向量：
$\overrightarrow{EF} = (-1, -1, 0)$
$\overrightarrow{EC_1} = (-1, 0, 2)$

设法向量$\vec{n_2} = (x, y, z)$
$\vec{n_2}·\overrightarrow{EF} = -x - y = 0$
$\vec{n_2}·\overrightarrow{EC_1} = -x + 2z = 0$

取$z = 1$，则$x = 2, y = -2$
∴ $\vec{n_2} = (2, -2, 1)$

$\cos\langle\vec{n_1},\vec{n_2}\rangle = \frac{\vec{n_1}·\vec{n_2}}{|\vec{n_1}||\vec{n_2}|} = \frac{1}{1·\sqrt{4+4+1}} = \frac{1}{3}$

由图形知二面角$C_1-EF-C$为锐角

∴ 二面角的余弦值为$\frac{1}{3}$
【方法总结】
• 二面角公式：$|\cos\theta| = \frac{|\vec{n_1}·\vec{n_2}|}{|\vec{n_1}||\vec{n_2}|}$
• 判断锐钝角的方法：
① 观察图形
② 看两法向量的方向（同进同出取补角）
• 如果要求的是二面角的平面角，注意可能是补角

第四章 立体几何综合问题

4.1 折叠问题

📝 例题12（2024新课标风格）：如图，在矩形ABCD中，$AB = 2, BC = 4$，E为BC的中点，将△ABE沿AE折起，使点B到达点P的位置，且平面$PAE \perp$平面AECD。
(1) 求证：$PE \perp CD$；
(2) 求直线PC与平面PAE所成角的正弦值。

【命题意图】考查折叠问题中的垂直关系证明和线面角计算，是高考热点题型。
【规范解答】

(1) 证明$PE \perp CD$：

在矩形ABCD中，$AB = 2, BC = 4$，E为BC中点
∴ $BE = EC = 2$，$AB = BE = 2$

在折叠前的△ABE中，$AB = BE = 2$，$\angle ABE = 90°$
∴ △ABE是等腰直角三角形，$\angle BAE = \angle BEA = 45°$

折叠后，$PE = BE = 2$，$\angle PEA = \angle BEA = 45°$

在底面AECD中，$\angle AEC = 180° - 45° = 135°$

∵ 平面$PAE \perp$平面AECD，交线为AE
在平面PAE中，过P作$PH \perp AE$于H
则$PH \perp$平面AECD

建立坐标系：以H为原点，平行于CD的方向为x轴，平行于AE的方向为y轴，HP方向为z轴

计算各点坐标（过程略），可验证$\overrightarrow{PE}·\overrightarrow{CD} = 0$

∴ $PE \perp CD$

(2) 求线面角的正弦值：

建立坐标系后（具体计算略）：
求得平面PAE的法向量$\vec{n} = (1, 0, 0)$
$\overrightarrow{PC} = (2, 3, -\sqrt{2})$

$\sin\theta = \frac{|\overrightarrow{PC}·\vec{n}|}{|\overrightarrow{PC}||\vec{n}|} = \frac{2}{\sqrt{4+9+2}} = \frac{2}{\sqrt{15}} = $ $\frac{2\sqrt{15}}{15}$
【方法总结】
• 折叠问题：抓住折叠前后不变的量（长度、角度）
• 垂直关系：找折叠后的垂线，建立坐标系
• 建系原则：尽量让多的点在坐标轴上
• 计算要细心，折叠后的几何关系较复杂
