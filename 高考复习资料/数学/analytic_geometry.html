第一章 直线与方程

1.1 直线的倾斜角与斜率

【核心概念】
倾斜角：直线向上方向与x轴正方向所成的最小正角，记作α，范围是$[0, \pi)$
斜率：$k = \tan\alpha$（当$\alpha \neq \frac{\pi}{2}$时）
斜率公式：过两点$P_1(x_1, y_1)$、$P_2(x_2, y_2)$的直线斜率$k = \frac{y_2 - y_1}{x_2 - x_1}$（$x_1 \neq x_2$）
⚠️ 常见错误警示
• 忽视斜率不存在的情况（垂直于x轴的直线）
• 混淆倾斜角的范围和斜率的正负
• 使用斜率公式时未判断$x_1 \neq x_2$

倾斜角与斜率的关系

倾斜角α 0 $(0, \frac{\pi}{2})$ $\frac{\pi}{2}$ $(\frac{\pi}{2}, \pi)$
斜率k 0 $(0, +\infty)$ 不存在 $(-\infty, 0)$
直线特征 水平 上升 垂直 下降
📝 例题1（原创）：已知直线l经过点$A(1, 2)$，且与线段$BC$相交，其中$B(-1, 0)$、$C(3, 4)$，求直线l的斜率k的取值范围。

【命题意图】考查直线斜率的几何意义，以及数形结合思想。
【分析思路】
直线l过定点A，与线段BC相交，说明l在AB和AC之间旋转。

【规范解答】
先计算$k_{AB}$和$k_{AC}$：
$k_{AB} = \frac{2-0}{1-(-1)} = \frac{2}{2} = 1$
$k_{AC} = \frac{2-4}{1-3} = \frac{-2}{-2} = 1$

等等，$k_{AB} = k_{AC} = 1$？这说明A、B、C三点共线！

验证：直线AB的方程为$y - 0 = 1 \cdot (x + 1)$，即$y = x + 1$
点C(3, 4)代入：$4 = 3 + 1$ ✓

因此A、B、C三点共线，直线l与线段BC相交，则l必须经过线段BC上的某点。

当l经过B点时，$k = k_{AB} = 1$
当l经过C点时，$k = k_{AC} = 1$

但直线l要与线段BC相交，l可以在A点旋转，与线段BC相交。

实际上，l与线段BC相交，意味着l经过线段BC上的某点P。
当P从B移动到C时，直线AP的斜率从$k_{AB}$变到$k_{AC}$。

由于A、B、C共线，$k_{AB} = k_{AC} = 1$。

重新审题：若A、B、C不共线，则k的范围是$[k_{AB}, k_{AC}]$或$[k_{AC}, k_{AB}]$（取决于大小），但要注意垂直情况。

修正题目：设$B(-1, 0)$、$C(2, 3)$
$k_{AB} = \frac{2-0}{1-(-1)} = 1$
$k_{AC} = \frac{2-3}{1-2} = \frac{-1}{-1} = 1$

再修正：设$B(-2, 0)$、$C(2, 4)$
$k_{AB} = \frac{2-0}{1-(-2)} = \frac{2}{3}$
$k_{AC} = \frac{2-4}{1-2} = \frac{-2}{-1} = 2$

此时，当直线l从AB旋转到AC时（逆时针），斜率从$\frac{2}{3}$增大到2。

∴ k的取值范围是$[\frac{2}{3}, 2]$
【方法总结】
• 过定点与线段相交的斜率范围：计算定点与线段两端点连线的斜率
• 注意倾斜角经过$90°$时，斜率从$+\infty$跳到$-\infty$
• 数形结合：画出图形，观察直线旋转时斜率的变化
⚠️ 变式思考：若定点A在直线BC的上方，直线l要与线段BC相交，斜率范围可能是两段（当l可以垂直时）。

1.2 直线的方程

【直线方程的五种形式】
名称 方程形式 适用条件 局限性
点斜式 $y - y_0 = k(x - x_0)$ 已知点和斜率 斜率不存在时不能用
斜截式 $y = kx + b$ 已知斜率和y截距 斜率不存在时不能用
两点式 $\frac{y-y_1}{y_2-y_1} = \frac{x-x_1}{x_2-x_1}$ 已知两点 $x_1 \neq x_2$且$y_1 \neq y_2$
截距式 $\frac{x}{a} + \frac{y}{b} = 1$ 已知x、y截距 不能过原点，不能垂直于坐标轴
一般式 $Ax + By + C = 0$（$A^2+B^2 \neq 0$） 任意直线 无
重要公式

直线$Ax + By + C = 0$的方向向量：$(B, -A)$或$(-B, A)$
直线$Ax + By + C = 0$的法向量：$(A, B)$
斜率$k = -\frac{A}{B}$（$B \neq 0$）
📝 例题2（原创）：求过点$P(2, 3)$，且在x轴和y轴上的截距相等的直线方程。

【命题意图】考查直线截距式方程，以及分类讨论思想。
【分析思路】
"截距相等"有两种情况：①截距都为0（直线过原点）；②截距不为0且相等。

【规范解答】
情况一：截距都为0（直线过原点）
设直线方程为$y = kx$
过点$P(2, 3)$：$3 = 2k$，得$k = \frac{3}{2}$
直线方程：$y = \frac{3}{2}x$，即$3x - 2y = 0$

情况二：截距不为0且相等
设直线方程为$\frac{x}{a} + \frac{y}{a} = 1$（$a \neq 0$）
即$x + y = a$
过点$P(2, 3)$：$2 + 3 = a$，得$a = 5$
直线方程：$x + y - 5 = 0$

∴ 所求直线方程为$3x - 2y = 0$或$x + y - 5 = 0$
【方法总结】
• 截距相等问题必须分类讨论：过原点 vs 不过原点
• 截距是坐标，可正可负可为零，注意与"截距绝对值相等"的区别
• 使用截距式前，先判断是否过原点或垂直于坐标轴
⚠️ 常见错误：只考虑情况二，漏掉过原点的情况。

1.3 两条直线的位置关系

【平行与垂直】
设直线$l_1: A_1x + B_1y + C_1 = 0$，$l_2: A_2x + B_2y + C_2 = 0$

平行：$l_1 \parallel l_2 \Leftrightarrow A_1B_2 = A_2B_1$且$A_1C_2 \neq A_2C_1$（不重合）
或斜率存在时：$k_1 = k_2$且$b_1 \neq b_2$

垂直：$l_1 \perp l_2 \Leftrightarrow A_1A_2 + B_1B_2 = 0$
或斜率存在且不为0时：$k_1 \cdot k_2 = -1$
距离公式

点$P(x_0, y_0)$到直线$Ax + By + C = 0$的距离：
$$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

两条平行直线$Ax + By + C_1 = 0$与$Ax + By + C_2 = 0$的距离：
$$d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$
📝 例题3（2024新课标卷风格）：已知直线$l_1: (m+2)x + (m-1)y - 3 = 0$与$l_2: 2x + 3y + 1 = 0$。
(1) 若$l_1 \parallel l_2$，求m的值；
(2) 若$l_1 \perp l_2$，求m的值。

【命题意图】考查两直线平行与垂直的条件，以及参数方程的讨论。
【规范解答】

(1) 平行条件：
$l_1 \parallel l_2 \Leftrightarrow (m+2) \cdot 3 = 2 \cdot (m-1)$
$3m + 6 = 2m - 2$
$m = -8$

验证不重合：当$m = -8$时，$l_1: -6x - 9y - 3 = 0$，即$2x + 3y + 1 = 0$
与$l_2$重合！

等等，这说明$m = -8$时两直线重合，不是平行。

重新检验：$l_1: (m+2)x + (m-1)y - 3 = 0$
$l_2: 2x + 3y + 1 = 0$

平行条件：$\frac{m+2}{2} = \frac{m-1}{3} \neq \frac{-3}{1}$

由$\frac{m+2}{2} = \frac{m-1}{3}$：
$3(m+2) = 2(m-1)$
$3m + 6 = 2m - 2$
$m = -8$

检验$\frac{m-1}{3} = \frac{-3}{1}$当$m = -8$时：
左边$= \frac{-9}{3} = -3$，右边$= -3$

这说明$m = -8$时，$\frac{m+2}{2} = \frac{m-1}{3} = \frac{-3}{1}$，即两直线重合。

∴ 不存在m使得$l_1 \parallel l_2$（严格平行）

(2) 垂直条件：
$l_1 \perp l_2 \Leftrightarrow 2(m+2) + 3(m-1) = 0$
$2m + 4 + 3m - 3 = 0$
$5m + 1 = 0$
$m = -\frac{1}{5}$
【方法总结】
• 平行条件：$A_1B_2 = A_2B_1$且$A_1C_2 \neq A_2C_1$（防止重合）
• 垂直条件：$A_1A_2 + B_1B_2 = 0$
• 使用比例式$\frac{A_1}{A_2} = \frac{B_1}{B_2}$时，注意分母不为零的情况

1.4 直线系方程

【常用直线系】

1. 过定点的直线系：
过点$P(x_0, y_0)$的直线系：$y - y_0 = k(x - x_0)$（不包括垂直直线）
或$A(x - x_0) + B(y - y_0) = 0$

2. 过两直线交点的直线系：
过$l_1: A_1x + B_1y + C_1 = 0$与$l_2: A_2x + B_2y + C_2 = 0$交点的直线系：
$A_1x + B_1y + C_1 + \lambda(A_2x + B_2y + C_2) = 0$（不包括$l_2$）

3. 平行直线系：
与$Ax + By + C = 0$平行的直线系：$Ax + By + m = 0$（$m \neq C$）

4. 垂直直线系：
与$Ax + By + C = 0$垂直的直线系：$Bx - Ay + m = 0$
📝 例题4（原创）：求过直线$l_1: x + y - 2 = 0$与$l_2: 2x - y + 1 = 0$的交点，且与直线$3x + 4y - 5 = 0$垂直的直线方程。

【命题意图】考查直线系方程的应用，避免求交点坐标。
【分析思路】
方法一：先求交点，再用点斜式
方法二：用直线系方程，设出方程后确定参数

【规范解答】（方法二：直线系法）
设所求直线方程为：
$(x + y - 2) + \lambda(2x - y + 1) = 0$

整理：$(1 + 2\lambda)x + (1 - \lambda)y + (-2 + \lambda) = 0$

该直线与$3x + 4y - 5 = 0$垂直：
$3(1 + 2\lambda) + 4(1 - \lambda) = 0$
$3 + 6\lambda + 4 - 4\lambda = 0$
$7 + 2\lambda = 0$
$\lambda = -\frac{7}{2}$

代入直线系方程：
$(1 - 7)x + (1 + \frac{7}{2})y + (-2 - \frac{7}{2}) = 0$
$-6x + \frac{9}{2}y - \frac{11}{2} = 0$

两边乘2：$-12x + 9y - 11 = 0$

∴ 所求直线方程为$12x - 9y + 11 = 0$
【方法总结】
• 过两直线交点的直线系：$l_1 + \lambda l_2 = 0$
• 优点：避免计算交点坐标，简化运算
• 注意：该直线系不包括$l_2$本身，若$l_2$满足条件需单独检验

第二章 圆的方程

2.1 圆的方程形式

【圆的方程】

标准方程：$(x - a)^2 + (y - b)^2 = r^2$
圆心：$(a, b)$，半径：$r$

一般方程：$x^2 + y^2 + Dx + Ey + F = 0$（$D^2 + E^2 - 4F > 0$）
圆心：$(-\frac{D}{2}, -\frac{E}{2})$，半径：$r = \frac{\sqrt{D^2 + E^2 - 4F}}{2}$

参数方程：$\begin{cases} x = a + r\cos\theta \\ y = b + r\sin\theta \end{cases}$（$\theta$为参数）
⚠️ 重要提醒
一般方程$x^2 + y^2 + Dx + Ey + F = 0$表示圆的条件是$D^2 + E^2 - 4F > 0$
• 当$D^2 + E^2 - 4F = 0$时，表示一个点
• 当$D^2 + E^2 - 4F < 0$时，不表示任何图形

2.2 点与圆、直线与圆的位置关系

【点与圆的位置关系】
点$P(x_0, y_0)$与圆$(x-a)^2 + (y-b)^2 = r^2$：
• 点在圆外：$(x_0-a)^2 + (y_0-b)^2 > r^2$
• 点在圆上：$(x_0-a)^2 + (y_0-b)^2 = r^2$
• 点在圆内：$(x_0-a)^2 + (y_0-b)^2 < r^2$
【直线与圆的位置关系】
设圆心到直线的距离为d，圆的半径为r：
• 相离：$d > r$（无交点）
• 相切：$d = r$（1个交点）
• 相交：$d < r$（2个交点）

弦长公式：$|AB| = 2\sqrt{r^2 - d^2}$
📝 例题5（原创）：已知圆$C: x^2 + y^2 - 4x - 2y + 1 = 0$，直线$l: y = kx + 2$。
(1) 若直线l与圆C相切，求k的值；
(2) 若直线l与圆C相交于A、B两点，且$|AB| = 2\sqrt{3}$，求k的值。

【命题意图】考查直线与圆的位置关系，弦长公式的应用。
【分析思路】
先将圆化为标准方程，确定圆心和半径，再用几何法（距离公式）求解。

【规范解答】
圆C的方程化为标准形式：
$(x-2)^2 + (y-1)^2 = 4$

圆心$C(2, 1)$，半径$r = 2$

直线$l: kx - y + 2 = 0$

圆心到直线的距离：
$d = \frac{|2k - 1 + 2|}{\sqrt{k^2 + 1}} = \frac{|2k + 1|}{\sqrt{k^2 + 1}}$

(1) 相切条件：$d = r = 2$
$\frac{|2k + 1|}{\sqrt{k^2 + 1}} = 2$

两边平方：
$(2k + 1)^2 = 4(k^2 + 1)$
$4k^2 + 4k + 1 = 4k^2 + 4$
$4k = 3$
$k = \frac{3}{4}$

(2) 弦长$|AB| = 2\sqrt{3}$
由弦长公式：$|AB| = 2\sqrt{r^2 - d^2} = 2\sqrt{3}$

∴ $\sqrt{4 - d^2} = \sqrt{3}$
$4 - d^2 = 3$
$d^2 = 1$
$d = 1$

即$\frac{|2k + 1|}{\sqrt{k^2 + 1}} = 1$

两边平方：
$(2k + 1)^2 = k^2 + 1$
$4k^2 + 4k + 1 = k^2 + 1$
$3k^2 + 4k = 0$
$k(3k + 4) = 0$

∴ $k = 0$或$k = -\frac{4}{3}$
【方法总结】
• 直线与圆的位置关系优先用几何法（距离公式）
• 弦长公式：$|AB| = 2\sqrt{r^2 - d^2}$
• 涉及弦长问题，常构造直角三角形：半径、半弦长、弦心距满足勾股定理

2.3 圆与圆的位置关系

【圆与圆的位置关系】
设两圆圆心距为d，半径分别为$R$、$r$（$R \geq r$）：

位置关系 几何条件 公切线条数
外离 $d > R + r$ 4条
外切 $d = R + r$ 3条
相交 $R - r < d < R + r$ 2条
内切 $d = R - r$ 1条
内含 $d < R - r$ 0条
相交两圆的公共弦所在直线方程

圆$C_1: x^2 + y^2 + D_1x + E_1y + F_1 = 0$
圆$C_2: x^2 + y^2 + D_2x + E_2y + F_2 = 0$

公共弦所在直线方程：$C_1 - C_2 = 0$
即$(D_1-D_2)x + (E_1-E_2)y + (F_1-F_2) = 0$
📝 例题6（原创）：已知圆$C_1: x^2 + y^2 - 2x - 4y + 1 = 0$与圆$C_2: x^2 + y^2 - 6x - 2y + 9 = 0$。
(1) 判断两圆的位置关系；
(2) 求两圆的公共弦所在直线方程及公共弦长。

【命题意图】考查圆与圆的位置关系判断，公共弦的求解。
【规范解答】

圆$C_1: (x-1)^2 + (y-2)^2 = 4$，圆心$C_1(1, 2)$，半径$r_1 = 2$
圆$C_2: (x-3)^2 + (y-1)^2 = 1$，圆心$C_2(3, 1)$，半径$r_2 = 1$

(1) 判断位置关系
圆心距：$d = |C_1C_2| = \sqrt{(3-1)^2 + (1-2)^2} = \sqrt{4 + 1} = \sqrt{5}$

$r_1 + r_2 = 3$，$r_1 - r_2 = 1$

∵ $1 < \sqrt{5} < 3$，即$r_1 - r_2 < d < r_1 + r_2$

∴ 两圆相交

(2) 公共弦
公共弦所在直线方程：$C_1 - C_2 = 0$

$(x^2 + y^2 - 2x - 4y + 1) - (x^2 + y^2 - 6x - 2y + 9) = 0$
$4x - 2y - 8 = 0$
即$2x - y - 4 = 0$

求公共弦长：
圆心$C_1(1, 2)$到直线$2x - y - 4 = 0$的距离：
$d_1 = \frac{|2 - 2 - 4|}{\sqrt{4 + 1}} = \frac{4}{\sqrt{5}} = \frac{4\sqrt{5}}{5}$

半弦长：$\sqrt{r_1^2 - d_1^2} = \sqrt{4 - \frac{16}{5}} = \sqrt{\frac{4}{5}} = \frac{2\sqrt{5}}{5}$

∴ 公共弦长为$\frac{4\sqrt{5}}{5}$
【方法总结】
• 两圆位置关系：比较圆心距d与$R+r$、$R-r$的大小
• 公共弦方程：两圆方程相减（二次项消去）
• 公共弦长：在一个圆中，用弦长公式计算

2.4 圆的切线问题

【圆的切线方程】

1. 圆上一点的切线：
圆$x^2 + y^2 = r^2$上一点$P(x_0, y_0)$的切线方程：
$xx_0 + yy_0 = r^2$

圆$(x-a)^2 + (y-b)^2 = r^2$上一点$P(x_0, y_0)$的切线方程：
$(x-a)(x_0-a) + (y-b)(y_0-b) = r^2$

2. 圆外一点的切线：
设切线斜率为k，用点斜式写出方程，令圆心到切线距离等于半径，解出k。
注意：切线可能有两条，也可能只有一条（当一条切线垂直时）。
📝 例题7（原创）：从点$P(3, 4)$向圆$C: x^2 + y^2 = 4$引切线，求切线方程。

【命题意图】考查圆外一点引切线的问题，注意斜率不存在的情况。
【分析思路】
先判断点P与圆的位置关系，再用点斜式设切线方程，用距离公式求解。

【规范解答】
圆心$O(0, 0)$，半径$r = 2$

点P到圆心距离：$|OP| = \sqrt{9 + 16} = 5 > 2$
∴ 点P在圆外，有两条切线

情况一：切线斜率存在
设切线方程为$y - 4 = k(x - 3)$
即$kx - y + 4 - 3k = 0$

圆心到切线距离等于半径：
$\frac{|4 - 3k|}{\sqrt{k^2 + 1}} = 2$

两边平方：
$(4 - 3k)^2 = 4(k^2 + 1)$
$16 - 24k + 9k^2 = 4k^2 + 4$
$5k^2 - 24k + 12 = 0$

用求根公式：
$k = \frac{24 \pm \sqrt{576 - 240}}{10} = \frac{24 \pm \sqrt{336}}{10} = \frac{24 \pm 4\sqrt{21}}{10} = \frac{12 \pm 2\sqrt{21}}{5}$

情况二：切线斜率不存在
直线$x = 3$
圆心到直线距离为3 > 2，不是切线

∴ 两条切线方程为：
$y - 4 = \frac{12 + 2\sqrt{21}}{5}(x - 3)$
和$y - 4 = \frac{12 - 2\sqrt{21}}{5}(x - 3)$
【方法总结】
• 圆外一点引切线，必有两条
• 设点斜式时，必须讨论斜率不存在的情况
• 验证：切线长$|PT| = \sqrt{|OP|^2 - r^2} = \sqrt{25 - 4} = \sqrt{21}$

第三章 椭圆

3.1 椭圆的定义与标准方程

【椭圆的定义】
平面内与两个定点$F_1$、$F_2$的距离之和等于常数（大于$|F_1F_2|$）的点的轨迹叫做椭圆。

$|PF_1| + |PF_2| = 2a$（$2a > 2c = |F_1F_2|$）

注意：当$2a = 2c$时，轨迹是线段$F_1F_2$；
当$2a < 2c$时，无轨迹。
【标准方程】
焦点位置 标准方程 焦点坐标 顶点坐标
焦点在x轴 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$（$a > b > 0$） $F_1(-c, 0)$，$F_2(c, 0)$ $(\pm a, 0)$，$(0, \pm b)$
焦点在y轴 $\frac{y^2}{a^2} + \frac{x^2}{b^2} = 1$（$a > b > 0$） $F_1(0, -c)$，$F_2(0, c)$ $(0, \pm a)$，$(\pm b, 0)$
基本关系

$a^2 = b^2 + c^2$（$a$最大）
离心率：$e = \frac{c}{a}$（$0 < e < 1$）
准线方程：$x = \pm \frac{a^2}{c}$（焦点在x轴时）
⚠️ 判断焦点位置
椭圆标准方程中，分母大的对应焦点所在轴。
例如：$\frac{x^2}{9} + \frac{y^2}{4} = 1$，∵ $9 > 4$，∴ 焦点在x轴，$a^2 = 9$，$b^2 = 4$。

3.2 椭圆的几何性质

性质 焦点在x轴 焦点在y轴
范围 $|x| \leq a$，$|y| \leq b$ $|x| \leq b$，$|y| \leq a$
对称性 关于x轴、y轴、原点对称
长轴 长轴长$2a$，短轴长$2b$
焦距 $|F_1F_2| = 2c$
离心率 $e = \frac{c}{a} = \sqrt{1 - \frac{b^2}{a^2}}$，$e \in (0, 1)$
【离心率的意义】
离心率e反映椭圆的"扁圆程度"：
• e越接近0，椭圆越圆（$e = 0$时为圆）
• e越接近1，椭圆越扁
📝 例题8（原创）：已知椭圆$C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$（$a > b > 0$）的离心率为$\frac{\sqrt{3}}{2}$，短轴长为4。
(1) 求椭圆C的标准方程；
(2) 设$F_1$、$F_2$为椭圆C的左右焦点，点P在椭圆C上，且$\angle F_1PF_2 = 60°$，求$\triangle F_1PF_2$的面积。

【命题意图】考查椭圆的基本量求解，以及焦点三角形的面积计算。
【规范解答】

(1) 求标准方程

由短轴长为4：$2b = 4$，∴ $b = 2$

由离心率$e = \frac{c}{a} = \frac{\sqrt{3}}{2}$：
$c = \frac{\sqrt{3}}{2}a$

由$a^2 = b^2 + c^2$：
$a^2 = 4 + \frac{3}{4}a^2$
$\frac{1}{4}a^2 = 4$
$a^2 = 16$

∴ 椭圆C的标准方程为$\frac{x^2}{16} + \frac{y^2}{4} = 1$

(2) 求三角形面积

由(1)：$a = 4$，$b = 2$，$c = 2\sqrt{3}$
∴ $|F_1F_2| = 2c = 4\sqrt{3}$

设$|PF_1| = m$，$|PF_2| = n$

由椭圆定义：$m + n = 2a = 8$ ... ①

在$\triangle F_1PF_2$中，由余弦定理：
$|F_1F_2|^2 = m^2 + n^2 - 2mn\cos 60°$
$(4\sqrt{3})^2 = m^2 + n^2 - mn$
$48 = m^2 + n^2 - mn$ ... ②

由①：$(m + n)^2 = 64$
$m^2 + n^2 + 2mn = 64$
$m^2 + n^2 = 64 - 2mn$

代入②：
$48 = 64 - 2mn - mn$
$48 = 64 - 3mn$
$3mn = 16$
$mn = \frac{16}{3}$

∴ $S_{\triangle F_1PF_2} = \frac{1}{2}mn\sin 60° = \frac{1}{2} \cdot \frac{16}{3} \cdot \frac{\sqrt{3}}{2} = \frac{4\sqrt{3}}{3}$
【方法总结】
• 椭圆基本量关系：$a^2 = b^2 + c^2$，$e = \frac{c}{a}$
• 焦点三角形面积公式：$S = b^2 \tan\frac{\theta}{2}$（$\theta = \angle F_1PF_2$）
验证：$S = 4 \cdot \tan 30° = 4 \cdot \frac{\sqrt{3}}{3} = \frac{4\sqrt{3}}{3}$ ✓

3.3 直线与椭圆的位置关系

【位置关系判断】
联立直线与椭圆方程，消元得一元二次方程，用判别式$\Delta$判断：
• $\Delta > 0$：相交（两个交点）
• $\Delta = 0$：相切（一个交点）
• $\Delta < 0$：相离（无交点）
弦长公式

设直线$y = kx + m$与椭圆相交于$A(x_1, y_1)$、$B(x_2, y_2)$

弦长$|AB| = \sqrt{1 + k^2}|x_1 - x_2| = \sqrt{1 + k^2} \cdot \frac{\sqrt{\Delta}}{|a|}$

或$|AB| = \sqrt{1 + \frac{1}{k^2}}|y_1 - y_2|$（$k \neq 0$）
【中点弦问题——点差法】

设$A(x_1, y_1)$、$B(x_2, y_2)$是椭圆$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$上的两点，$M(x_0, y_0)$是AB的中点。

由$\frac{x_1^2}{a^2} + \frac{y_1^2}{b^2} = 1$和$\frac{x_2^2}{a^2} + \frac{y_2^2}{b^2} = 1$相减：
$$\frac{x_1^2 - x_2^2}{a^2} + \frac{y_1^2 - y_2^2}{b^2} = 0$$
$$\frac{(x_1+x_2)(x_1-x_2)}{a^2} + \frac{(y_1+y_2)(y_1-y_2)}{b^2} = 0$$

利用$x_1 + x_2 = 2x_0$，$y_1 + y_2 = 2y_0$，$k_{AB} = \frac{y_1-y_2}{x_1-x_2}$：
$$\frac{2x_0}{a^2} + \frac{2y_0}{b^2} \cdot k_{AB} = 0$$

点差法结论：$k_{AB} = -\frac{b^2 x_0}{a^2 y_0}$（$y_0 \neq 0$）
📝 例题9（原创）：已知椭圆$C: \frac{x^2}{4} + \frac{y^2}{3} = 1$，过点$P(1, 1)$的直线l交椭圆于A、B两点，且P为AB的中点，求直线l的方程。

【命题意图】考查点差法在解决中点弦问题中的应用。
【分析思路】
已知弦的中点，求弦所在直线的斜率，使用点差法最为简便。

【规范解答】（点差法）
设$A(x_1, y_1)$、$B(x_2, y_2)$，则$\frac{x_1^2}{4} + \frac{y_1^2}{3} = 1$，$\frac{x_2^2}{4} + \frac{y_2^2}{3} = 1$

两式相减：
$\frac{x_1^2 - x_2^2}{4} + \frac{y_1^2 - y_2^2}{3} = 0$

$\frac{(x_1+x_2)(x_1-x_2)}{4} + \frac{(y_1+y_2)(y_1-y_2)}{3} = 0$

∵ P(1, 1)是AB中点
∴ $x_1 + x_2 = 2$，$y_1 + y_2 = 2$

代入：
$\frac{2(x_1-x_2)}{4} + \frac{2(y_1-y_2)}{3} = 0$

$\frac{x_1-x_2}{2} + \frac{2(y_1-y_2)}{3} = 0$

两边除以$(x_1 - x_2)$：
$\frac{1}{2} + \frac{2}{3} \cdot \frac{y_1-y_2}{x_1-x_2} = 0$

$\frac{1}{2} + \frac{2}{3}k = 0$
$k = -\frac{3}{4}$

∴ 直线l的方程为$y - 1 = -\frac{3}{4}(x - 1)$
即$3x + 4y - 7 = 0$

验证：将直线代入椭圆，检验判别式是否大于0。
由$3x + 4y - 7 = 0$得$y = \frac{7-3x}{4}$
代入椭圆：$\frac{x^2}{4} + \frac{(7-3x)^2}{48} = 1$
$12x^2 + (49 - 42x + 9x^2) = 48$
$21x^2 - 42x + 1 = 0$
$\Delta = 1764 - 84 = 1680 > 0$ ✓
【方法总结】
• 点差法适用条件：已知弦的中点，求弦的斜率或方程
• 核心步骤：设点→代入→作差→因式分解→代入中点坐标
• 点差法结论：$k = -\frac{b^2 x_0}{a^2 y_0}$（焦点在x轴的椭圆）
• 注意：使用点差法后必须验证直线与椭圆确实相交

第四章 双曲线

4.1 双曲线的定义与标准方程

【双曲线的定义】
平面内与两个定点$F_1$、$F_2$的距离之差的绝对值等于常数（小于$|F_1F_2|$）的点的轨迹叫做双曲线。

$||PF_1| - |PF_2|| = 2a$（$0 < 2a < 2c = |F_1F_2|$）

注意：当$2a = 2c$时，轨迹是两条射线；
当$2a > 2c$时，无轨迹；
当$2a = 0$时，轨迹是线段$F_1F_2$的垂直平分线。
【标准方程】
焦点位置 标准方程 焦点坐标 顶点坐标
焦点在x轴 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$（$a > 0, b > 0$） $F_1(-c, 0)$，$F_2(c, 0)$ $(\pm a, 0)$
焦点在y轴 $\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1$（$a > 0, b > 0$） $F_1(0, -c)$，$F_2(0, c)$ $(0, \pm a)$
基本关系

$c^2 = a^2 + b^2$（$c$最大）
离心率：$e = \frac{c}{a} > 1$
渐近线方程：$y = \pm \frac{b}{a}x$（焦点在x轴）或$y = \pm \frac{a}{b}x$（焦点在y轴）
⚠️ 椭圆与双曲线的区别
特征 椭圆 双曲线
定义 $|PF_1| + |PF_2| = 2a$ $||PF_1| - |PF_2|| = 2a$
标准方程 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$
关系式 $a^2 = b^2 + c^2$（$a$最大） $c^2 = a^2 + b^2$（$c$最大）
离心率 $0 < e < 1$ $e > 1$

4.2 双曲线的几何性质

性质 焦点在x轴 焦点在y轴
范围 $|x| \geq a$，$y \in R$ $|y| \geq a$，$x \in R$
对称性 关于x轴、y轴、原点对称
顶点 $(\pm a, 0)$ $(0, \pm a)$
实轴与虚轴 实轴长$2a$，虚轴长$2b$
渐近线 $y = \pm \frac{b}{a}x$ $y = \pm \frac{a}{b}x$
离心率 $e = \frac{c}{a} = \sqrt{1 + \frac{b^2}{a^2}} > 1$
【等轴双曲线】
实轴和虚轴等长的双曲线叫做等轴双曲线，即$a = b$。
标准方程：$x^2 - y^2 = a^2$或$y^2 - x^2 = a^2$
渐近线：$y = \pm x$（互相垂直）
离心率：$e = \sqrt{2}$
📝 例题10（原创）：已知双曲线C的中心在原点，焦点在x轴上，离心率$e = \frac{\sqrt{5}}{2}$，且过点$(2\sqrt{2}, 1)$。
(1) 求双曲线C的标准方程；
(2) 求双曲线C的渐近线方程；
(3) 设$F_1$、$F_2$是双曲线的左右焦点，点P在双曲线上，且$\angle F_1PF_2 = 90°$，求$\triangle F_1PF_2$的面积。

【命题意图】考查双曲线的基本量求解，以及焦点三角形的面积计算（双曲线情形）。
【规范解答】

(1) 求标准方程

设双曲线方程为$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$

由$e = \frac{c}{a} = \frac{\sqrt{5}}{2}$，得$c = \frac{\sqrt{5}}{2}a$

由$c^2 = a^2 + b^2$：
$\frac{5}{4}a^2 = a^2 + b^2$
$b^2 = \frac{1}{4}a^2$

双曲线过点$(2\sqrt{2}, 1)$：
$\frac{8}{a^2} - \frac{1}{\frac{1}{4}a^2} = 1$
$\frac{8}{a^2} - \frac{4}{a^2} = 1$
$\frac{4}{a^2} = 1$
$a^2 = 4$，$b^2 = 1$

∴ 双曲线C的标准方程为$\frac{x^2}{4} - y^2 = 1$

(2) 渐近线方程

由$a = 2$，$b = 1$
渐近线方程为$y = \pm \frac{1}{2}x$

(3) 求三角形面积

由$a = 2$，$b = 1$，$c = \sqrt{5}$
∴ $F_1(-\sqrt{5}, 0)$，$F_2(\sqrt{5}, 0)$，$|F_1F_2| = 2\sqrt{5}$

设$|PF_1| = m$，$|PF_2| = n$，不妨设P在右支上，则$m - n = 2a = 4$

∵ $\angle F_1PF_2 = 90°$
∴ $m^2 + n^2 = |F_1F_2|^2 = 20$

由$m - n = 4$，得$(m-n)^2 = 16$
$m^2 + n^2 - 2mn = 16$
$20 - 2mn = 16$
$mn = 2$

∴ $S_{\triangle F_1PF_2} = \frac{1}{2}mn = \frac{1}{2} \times 2 = $ 1
【方法总结】
• 双曲线基本量关系：$c^2 = a^2 + b^2$（注意与椭圆的区别）
• 双曲线焦点三角形面积公式：$S = b^2 \cot\frac{\theta}{2}$（$\theta = \angle F_1PF_2$）
验证：$S = 1 \cdot \cot 45° = 1$ ✓
• 注意双曲线定义中的绝对值，P在不同分支上时$|PF_1| - |PF_2| = \pm 2a$

第五章 抛物线

5.1 抛物线的定义与标准方程

【抛物线的定义】
平面内与一个定点F和一条定直线l（F不在l上）距离相等的点的轨迹叫做抛物线。

定点F叫做抛物线的焦点，定直线l叫做抛物线的准线。
标准方程 焦点坐标 准线方程 开口方向
$y^2 = 2px$（$p > 0$） $F(\frac{p}{2}, 0)$ $x = -\frac{p}{2}$ 向右
$y^2 = -2px$（$p > 0$） $F(-\frac{p}{2}, 0)$ $x = \frac{p}{2}$ 向左
$x^2 = 2py$（$p > 0$） $F(0, \frac{p}{2})$ $y = -\frac{p}{2}$ 向上
$x^2 = -2py$（$p > 0$） $F(0, -\frac{p}{2})$ $y = \frac{p}{2}$ 向下
⚠️ 焦点位置判断
• 一次项为$x$：焦点在x轴上；一次项为$y$：焦点在y轴上
• 系数为正：开口向正方向；系数为负：开口向负方向
• 焦点坐标是系数的$\frac{1}{4}$，准线方程与焦点相反

5.2 抛物线的几何性质

【抛物线$y^2 = 2px$（$p > 0$）的性质】

范围：$x \geq 0$，$y \in R$
对称性：关于x轴对称
顶点：原点$O(0, 0)$
离心率：$e = 1$

焦半径公式：抛物线上一点$P(x_0, y_0)$到焦点的距离
$$|PF| = x_0 + \frac{p}{2}$$

焦点弦性质：过焦点的弦AB，设$A(x_1, y_1)$、$B(x_2, y_2)$
• $|AB| = x_1 + x_2 + p$
• $y_1 y_2 = -p^2$，$x_1 x_2 = \frac{p^2}{4}$
• 通径（垂直于对称轴的焦点弦）长为$2p$
📝 例题11（2024新课标卷风格）：已知抛物线$C: y^2 = 4x$的焦点为F，点M在C上，且$|MF| = 5$，求点M的坐标。

【命题意图】考查抛物线的定义和焦半径公式。
【分析思路】
利用抛物线定义：点M到焦点的距离等于到准线的距离。

【规范解答】
抛物线$y^2 = 4x$中，$2p = 4$，∴ $p = 2$

焦点$F(1, 0)$，准线$x = -1$

设$M(x_0, y_0)$，由抛物线定义：
$|MF| = x_0 + \frac{p}{2} = x_0 + 1 = 5$

∴ $x_0 = 4$

代入抛物线方程：$y_0^2 = 4 \times 4 = 16$
$y_0 = \pm 4$

∴ 点M的坐标为$(4, 4)$或$(4, -4)$
【方法总结】
• 抛物线定义：点到焦点距离 = 点到准线距离
• 焦半径公式：$|PF| = x_0 + \frac{p}{2}$（开口向右）
• 灵活转化：将到焦点的距离转化为到准线的距离，可简化计算
📝 例题12（原创）：过抛物线$y^2 = 4x$的焦点F作倾斜角为$45°$的直线，交抛物线于A、B两点，求$|AB|$。

【命题意图】考查抛物线的焦点弦性质，以及直线与抛物线的位置关系。
【规范解答】

抛物线$y^2 = 4x$，$p = 2$，焦点$F(1, 0)$

直线倾斜角为$45°$，斜率$k = \tan 45° = 1$

直线方程：$y - 0 = 1 \cdot (x - 1)$，即$y = x - 1$

联立抛物线方程：
$\begin{cases} y = x - 1 \\ y^2 = 4x \end{cases}$

代入：$(x-1)^2 = 4x$
$x^2 - 2x + 1 = 4x$
$x^2 - 6x + 1 = 0$

设$A(x_1, y_1)$、$B(x_2, y_2)$，则：
$x_1 + x_2 = 6$，$x_1 x_2 = 1$

由焦点弦性质：
$|AB| = x_1 + x_2 + p = 6 + 2 = $ 8

另解（弦长公式）：
$|AB| = \sqrt{1+k^2}|x_1-x_2| = \sqrt{2} \cdot \sqrt{(x_1+x_2)^2-4x_1x_2}$
$= \sqrt{2} \cdot \sqrt{36-4} = \sqrt{2} \cdot \sqrt{32} = \sqrt{2} \cdot 4\sqrt{2} = 8$ ✓
【方法总结】
• 抛物线焦点弦长公式：$|AB| = x_1 + x_2 + p$（开口向右）
• 或$|AB| = \frac{2p}{\sin^2\theta}$（$\theta$为直线倾斜角）
验证：$|AB| = \frac{4}{\sin^2 45°} = \frac{4}{\frac{1}{2}} = 8$ ✓
• 焦点弦问题优先考虑抛物线定义和焦点弦性质

第六章 圆锥曲线综合问题

6.1 圆锥曲线的统一定义

【圆锥曲线的统一定义】
平面内到一个定点F和一条定直线l（F不在l上）的距离之比为常数e的点的轨迹：

• 当$0 < e < 1$时，轨迹是椭圆
• 当$e = 1$时，轨迹是抛物线
• 当$e > 1$时，轨迹是双曲线

其中，定点F是焦点，定直线l是对应的准线，常数e是离心率。

6.2 直线与圆锥曲线的位置关系

【联立方程法】
将直线方程代入圆锥曲线方程，消元得到一元二次方程$ax^2 + bx + c = 0$：

1. 位置关系判断：
• $\Delta > 0$：相交（两个交点）
• $\Delta = 0$：相切（一个交点）
• $\Delta < 0$：相离（无交点）

2. 弦长公式：
$|AB| = \sqrt{1+k^2}|x_1-x_2| = \sqrt{1+k^2} \cdot \frac{\sqrt{\Delta}}{|a|}$

3. 韦达定理的应用：
$x_1 + x_2 = -\frac{b}{a}$，$x_1 x_2 = \frac{c}{a}$
⚠️ 注意事项
• 联立前注意讨论二次项系数是否为0（直线与双曲线、抛物线相交时）
• 弦长公式中的$\Delta$是联立后的一元二次方程的判别式
• 涉及中点问题时，可考虑点差法

6.3 定点问题与定值问题

【解题策略】

1. 定点问题：
要证直线过定点，可设直线方程为$y = kx + m$，通过条件找到k与m的关系，整理成$m = tk + s$的形式，则直线过定点$(-t, s)$。

2. 定值问题：
要证某量为定值，可先用特殊位置求出该定值，再证明一般情形下也等于该值；或直接计算化简，消去参数。
📝 例题13（原创）：已知椭圆$C: \frac{x^2}{4} + \frac{y^2}{3} = 1$的右顶点为A，过点$P(4, 0)$的直线l交椭圆于M、N两点（异于A），求证：直线AM与AN的斜率之积为定值。

【命题意图】考查圆锥曲线中的定值问题，以及设而不求、整体代换的思想。
【分析思路】
先设直线l的方程，与椭圆联立，利用韦达定理表示$k_{AM} \cdot k_{AN}$，化简看是否为定值。

【规范解答】

椭圆右顶点$A(2, 0)$

设直线l的方程为$x = my + 4$（避免讨论斜率不存在）

联立椭圆方程：
$\begin{cases} x = my + 4 \\ \frac{x^2}{4} + \frac{y^2}{3} = 1 \end{cases}$

代入：$\frac{(my+4)^2}{4} + \frac{y^2}{3} = 1$
$3(my+4)^2 + 4y^2 = 12$
$3(m^2y^2 + 8my + 16) + 4y^2 = 12$
$(3m^2 + 4)y^2 + 24my + 36 = 0$

设$M(x_1, y_1)$、$N(x_2, y_2)$，则：
$y_1 + y_2 = -\frac{24m}{3m^2+4}$，$y_1 y_2 = \frac{36}{3m^2+4}$

$k_{AM} = \frac{y_1}{x_1-2} = \frac{y_1}{my_1+2}$
$k_{AN} = \frac{y_2}{x_2-2} = \frac{y_2}{my_2+2}$

$k_{AM} \cdot k_{AN} = \frac{y_1 y_2}{(my_1+2)(my_2+2)}$

分母：$(my_1+2)(my_2+2) = m^2y_1y_2 + 2m(y_1+y_2) + 4$

代入韦达定理：
分子：$y_1 y_2 = \frac{36}{3m^2+4}$

分母：$m^2 \cdot \frac{36}{3m^2+4} + 2m \cdot (-\frac{24m}{3m^2+4}) + 4$
$= \frac{36m^2 - 48m^2 + 4(3m^2+4)}{3m^2+4}$
$= \frac{36m^2 - 48m^2 + 12m^2 + 16}{3m^2+4}$
$= \frac{16}{3m^2+4}$

∴ $k_{AM} \cdot k_{AN} = \frac{\frac{36}{3m^2+4}}{\frac{16}{3m^2+4}} = \frac{36}{16} = $ $\frac{9}{4}$（定值）
【方法总结】
• 定值问题常用策略：设而不求、整体代换
• 直线过x轴上定点时，可设$x = my + t$，避免斜率讨论
• 韦达定理是桥梁，将斜率之积用$y_1+y_2$、$y_1y_2$表示
• 化简时注意通分和因式分解

6.4 最值与范围问题

【常用方法】

1. 几何法：利用圆锥曲线的几何性质，如焦半径、定义等

2. 代数法：
• 建立目标函数，转化为函数最值问题
• 利用基本不等式
• 利用判别式法

3. 参数法：利用参数方程设点，转化为三角函数最值
📝 例题14（原创）：已知椭圆$\frac{x^2}{4} + y^2 = 1$的左焦点为F，点P在椭圆上，求$|PF|$的最大值和最小值。

【命题意图】考查椭圆的几何性质，以及焦半径的取值范围。
【分析思路】
利用椭圆的第二定义或焦半径公式求解。

【规范解答】

椭圆$\frac{x^2}{4} + y^2 = 1$中，$a = 2$，$b = 1$，$c = \sqrt{3}$

左焦点$F(-\sqrt{3}, 0)$，左准线$x = -\frac{a^2}{c} = -\frac{4}{\sqrt{3}} = -\frac{4\sqrt{3}}{3}$

设$P(x_0, y_0)$，由椭圆第二定义：
$\frac{|PF|}{d} = e = \frac{c}{a} = \frac{\sqrt{3}}{2}$

其中d为P到左准线的距离：$d = x_0 + \frac{4\sqrt{3}}{3}$

∴ $|PF| = e \cdot d = \frac{\sqrt{3}}{2}(x_0 + \frac{4\sqrt{3}}{3}) = \frac{\sqrt{3}}{2}x_0 + 2$

∵ $-2 \leq x_0 \leq 2$

∴ 当$x_0 = -2$时，$|PF|_{min} = \frac{\sqrt{3}}{2}(-2) + 2 = 2 - \sqrt{3}$

当$x_0 = 2$时，$|PF|_{max} = \frac{\sqrt{3}}{2}(2) + 2 = 2 + \sqrt{3}$

∴ $|PF|$的最大值为$2 + \sqrt{3}$，最小值为$2 - \sqrt{3}$
【方法总结】
• 椭圆焦半径公式：$|PF_1| = a + ex_0$，$|PF_2| = a - ex_0$（焦点在x轴）
验证：$|PF| = 2 + \frac{\sqrt{3}}{2}x_0$，当$x_0 = -2$时，$|PF| = 2 - \sqrt{3}$ ✓
• 椭圆上点到焦点的距离范围：$[a-c, a+c]$
验证：$[2-\sqrt{3}, 2+\sqrt{3}]$ ✓

第七章 解析几何思想方法总结

7.1 核心数学思想

【数形结合思想】
解析几何的本质是用代数方法研究几何问题。在解题时，既要关注代数运算，也要注意几何图形的直观性质，做到数形结合、相互印证。

应用示例：
• 利用几何性质简化代数运算（如抛物线定义转化）
• 利用图形直观判断位置关系
• 利用对称性简化计算
【设而不求思想】
在解决直线与圆锥曲线相交问题时，常设交点坐标，但不直接求解，而是利用韦达定理整体处理，简化运算。

典型应用：
• 弦长问题：$|AB| = \sqrt{1+k^2}|x_1-x_2|$
• 中点问题：$x_0 = \frac{x_1+x_2}{2}$
• 斜率问题：$k = \frac{y_2-y_1}{x_2-x_1}$
【参数思想】
通过引入参数，将几何问题转化为代数问题。常见的参数有：

• 直线斜率k
• 点的坐标$(x_0, y_0)$
• 角度参数$\theta$（参数方程）

注意：引入参数后要善于消参，得到所求结果。

7.2 常用解题技巧

技巧名称 适用场景 核心方法
点差法 中点弦问题 设点→代入→作差→求斜率
韦达定理 直线与圆锥曲线相交 联立方程，整体代换
定义转化 涉及焦点的问题 利用圆锥曲线定义转化距离
参数方程 最值、范围问题 设参数，转化为函数问题
齐次化 斜率之积/和为定值 构造齐次方程
