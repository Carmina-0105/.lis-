第一章 三角函数概念与公式

1.1 任意角与弧度制

一、角的概念的推广

角的分类：
  • 正角：逆时针旋转形成的角
  • 负角：顺时针旋转形成的角
  • 零角：没有旋转
终边相同的角：与角α终边相同的角的集合为 {β|β = α + 2kπ, k∈Z}

象限角：角的顶点在原点，始边在x轴正半轴，终边在第几象限就是第几象限角
弧度制与角度制换算：
π rad = 180°
1 rad = (180/π)° ≈ 57.3°
1° = π/180 rad

常用换算：
角度0°30°45°60°90°180°270°360°
弧度0π/6π/4π/3π/2π3π/22π
📝 例题1（原创）：已知角α的终边在直线y = 2x上，求sinα、cosα、tanα的值。

【命题意图】考查三角函数定义，以及分类讨论思想（终边在第一或第三象限）。
【分析思路】
在终边上取点，利用三角函数定义求解。注意终边可能在第一或第三象限。

【规范解答】
在直线y = 2x上取点P(1, 2)，则r = √(1²+2²) = √5

情况一：终边在第一象限
sinα = y/r = 2/√5 = 2√5/5
cosα = x/r = 1/√5 = √5/5
tanα = y/x = 2

情况二：终边在第三象限
取点P(-1, -2)，则r = √5
sinα = -2/√5 = -2√5/5
cosα = -1/√5 = -√5/5
tanα = (-2)/(-1) = 2

答案：tanα = 2；当α在第一象限时，sinα = 2√5/5，cosα = √5/5；当α在第三象限时，sinα = -2√5/5，cosα = -√5/5
【方法总结】
• 已知终边求三角函数：在终边上取点(x,y)，r=√(x²+y²)，再用定义
• 直线y=kx上的角：终边可能在两个象限，要分类讨论
• tanα只与k有关，sinα和cosα与象限有关
【易错提醒】
⚠️ 终边在直线y=2x上≠终边在第一象限！直线经过第一、第三象限
⚠️ 不要遗漏第三象限的情况

二、扇形公式

扇形公式（弧度制下）：
设扇形半径为r，圆心角为α（弧度），弧长为l，面积为S

• 弧长公式：l = |α|·r
• 面积公式：S = (1/2)lr = (1/2)|α|r²

注意：公式中的α必须是弧度制！
📝 例题2（原创）：已知扇形的周长为8，面积为4，求扇形圆心角的弧度数。

【命题意图】考查扇形公式的应用，以及方程思想。
【分析思路】
设半径为r，圆心角为α，根据周长和面积列方程组求解。

【规范解答】
设扇形半径为r，圆心角为α（α > 0）

周长：2r + rα = 8 ... ①
面积：(1/2)r²α = 4 ... ②

由②：r²α = 8，得α = 8/r²

代入①：2r + r·(8/r²) = 8
2r + 8/r = 8
2r² + 8 = 8r
r² - 4r + 4 = 0
(r-2)² = 0
r = 2

∴ α = 8/4 = 2

答案：圆心角为2弧度
【方法总结】
• 扇形问题：设r和α，用周长、面积公式列方程
• 注意周长包括两条半径：C = 2r + l = 2r + rα
• 面积公式选择：已知l用S=(1/2)lr，已知α用S=(1/2)αr²

1.2 三角函数定义与同角关系

一、三角函数定义

单位圆定义：设角α的终边与单位圆交于点P(x,y)，则
sinα = y，cosα = x，tanα = y/x（x≠0）

一般定义：在角α终边上任取一点P(x,y)，r = √(x²+y²)，则
sinα = y/r，cosα = x/r，tanα = y/x（x≠0）

三角函数值符号（口诀）：
一全正，二正弦，三正切，四余弦
（第一象限全正，第二象限sin正，第三象限tan正，第四象限cos正）

二、同角三角函数基本关系

平方关系：sin²α + cos²α = 1

商数关系：tanα = sinα/cosα（cosα≠0）

常用变形：
• sin²α = 1 - cos²α = (1+cosα)(1-cosα)
• cos²α = 1 - sin²α = (1+sinα)(1-sinα)
• (sinα ± cosα)² = 1 ± 2sinαcosα
• sinαcosα = [(sinα+cosα)² - 1]/2

知一求二：已知sinα、cosα、tanα中的一个，可求另外两个（需确定象限）
📝 例题3（原创）：已知sinα + cosα = 1/3，α∈(0,π)，求：
(1) sinαcosα的值；
(2) sinα - cosα的值；
(3) tanα的值。

【命题意图】考查同角三角函数关系的灵活应用，以及(sinα±cosα)²的展开技巧。
【规范解答】

(1) 求sinαcosα：
(sinα + cosα)² = sin²α + 2sinαcosα + cos²α = 1 + 2sinαcosα

∴ (1/3)² = 1 + 2sinαcosα
1/9 = 1 + 2sinαcosα
2sinαcosα = 1/9 - 1 = -8/9

sinαcosα = -4/9

(2) 求sinα - cosα：
(sinα - cosα)² = sin²α - 2sinαcosα + cos²α = 1 - 2sinαcosα
= 1 - 2×(-4/9) = 1 + 8/9 = 17/9

∴ sinα - cosα = ±√17/3

确定符号：∵ sinαcosα = -4/9 < 0，且α∈(0,π)
∴ sinα > 0，cosα < 0（第二象限）
∴ sinα - cosα > 0

sinα - cosα = √17/3

(3) 求tanα：
由(1)(2)联立：
sinα + cosα = 1/3
sinα - cosα = √17/3

相加：2sinα = (1+√17)/3，sinα = (1+√17)/6
相减：2cosα = (1-√17)/3，cosα = (1-√17)/6

tanα = sinα/cosα = (1+√17)/(1-√17) = -(9+√17)/16（有理化后）
【方法总结】
• (sinα ± cosα)² = 1 ± 2sinαcosα（核心公式！）
• 已知sinα+cosα，可求sinαcosα，再求sinα-cosα
• 确定sinα-cosα符号：看象限（sinαcosα<0说明是第二或第四象限）
• 知sinα+cosα和sinα-cosα，联立可解sinα和cosα
【易错提醒】
⚠️ 开平方要确定符号！根据象限判断
⚠️ sinαcosα<0时，α在第二或第四象限，结合α∈(0,π)确定在第二象限
💪 变式训练（原创）：已知sinα - cosα = √2/2，α∈(0,π)，求sinαcosα和tanα的值。
（答案：sinαcosα = 1/4，tanα = 2+√3或2-√3，需进一步确定）

三、诱导公式

诱导公式口诀：奇变偶不变，符号看象限

解释：
• "奇偶"指π/2的奇数倍还是偶数倍
• "变"指函数名改变：sin↔cos，tan↔cot
• "符号看象限"：把α看成锐角，看原函数在相应象限的符号

常用诱导公式：
公式sincostan
-α-sinαcosα-tanα
π-αsinα-cosα-tanα
π+α-sinα-cosαtanα
2π-α-sinαcosα-tanα
π/2-αcosαsinαcotα
π/2+αcosα-sinα-cotα
📝 例题4（原创）：化简：sin(π-α)cos(2π+α)tan(-α)/[cos(π/2+α)sin(π+α)]

【规范解答】

分子：sin(π-α)cos(2π+α)tan(-α)
= sinα · cosα · (-tanα)
= -sinαcosα · (sinα/cosα)
= -sin²α

分母：cos(π/2+α)sin(π+α)
= (-sinα) · (-sinα)
= sin²α

原式 = (-sin²α)/(sin²α) = -1
【方法总结】
• 诱导公式化简：逐个化简每个三角函数
• 负角公式：sin(-α)=-sinα，cos(-α)=cosα，tan(-α)=-tanα
• 2π±α、π±α：函数名不变，符号看象限
• π/2±α：函数名改变，符号看象限

第二章 三角恒等变换

2.1 两角和与差的三角函数

一、和差角公式

两角和与差公式：

sin(α+β) = sinαcosβ + cosαsinβ
sin(α-β) = sinαcosβ - cosαsinβ

cos(α+β) = cosαcosβ - sinαsinβ
cos(α-β) = cosαcosβ + sinαsinβ

tan(α+β) = (tanα+tanβ)/(1-tanαtanβ)
tan(α-β) = (tanα-tanβ)/(1+tanαtanβ)

记忆口诀：正弦同号，余弦异号；正切分子同号，分母反号
📝 例题5（原创）：已知α、β都是锐角，sinα = 3/5，cos(α+β) = -5/13，求sinβ的值。

【命题意图】考查两角差的正弦公式，以及角的范围分析。
【分析思路】
注意到β = (α+β) - α，可以用两角差的正弦公式。

【规范解答】
∵ α是锐角，sinα = 3/5
∴ cosα = 4/5

∵ α、β都是锐角，∴ 0 < α+β < π
又cos(α+β) = -5/13 < 0
∴ π/2 < α+β < π（第二象限）

∴ sin(α+β) = √(1-cos²(α+β)) = √(1-25/169) = 12/13

sinβ = sin[(α+β)-α]
= sin(α+β)cosα - cos(α+β)sinα
= (12/13)×(4/5) - (-5/13)×(3/5)
= 48/65 + 15/65
= 63/65
【方法总结】
• 角的拆分：β = (α+β) - α，α = (α+β) - β，2α = (α+β) + (α-β)
• 注意分析角的范围，确定三角函数值的符号
• 给值求值问题：找已知角和所求角的关系
【易错提醒】
⚠️ 注意分析α+β的范围！虽然α、β都是锐角，但α+β可能在第二象限
⚠️ 开平方时根据象限确定符号

二、二倍角公式

二倍角公式：

sin2α = 2sinαcosα

cos2α = cos²α - sin²α = 2cos²α - 1 = 1 - 2sin²α

tan2α = 2tanα/(1-tan²α)

降幂公式（由二倍角公式变形）：
sin²α = (1-cos2α)/2
cos²α = (1+cos2α)/2

常用变形：
1 + cos2α = 2cos²α
1 - cos2α = 2sin²α
📝 例题6（2024新课标风格）：已知sin(α-π/6) = 1/3，求cos(2α+π/3)的值。

【命题意图】考查角的变换技巧，以及二倍角公式的灵活应用。
【分析思路】
观察角的关系：2α+π/3 = 2(α-π/6) + 2π/6 + π/3 = 2(α-π/6) + π/2？
验证：2(α-π/6) = 2α-π/3，再加π/2得2α+π/6，不对。

重新观察：2α+π/3 = 2(α+π/6)
而已知是α-π/6，所以：
2α+π/3 = 2(α-π/6) + π/3 + π/3 = 2(α-π/6) + 2π/3？
验证：2(α-π/6)+2π/3 = 2α-π/3+2π/3 = 2α+π/3 ✓

【规范解答】
设θ = α - π/6，则sinθ = 1/3

2α + π/3 = 2(α - π/6) + 2π/3 = 2θ + 2π/3

cos(2α+π/3) = cos(2θ + 2π/3)
= cos2θcos(2π/3) - sin2θsin(2π/3)
= cos2θ×(-1/2) - sin2θ×(√3/2)

其中：
cos2θ = 1 - 2sin²θ = 1 - 2×(1/9) = 7/9
sin2θ = 2sinθcosθ = 2×(1/3)×(2√2/3) = 4√2/9

（cosθ = ±2√2/3，这里取正值还是负值不影响最终结果，因为sin2θ=2sinθcosθ中cosθ的符号...实际上需要确定）

实际上cos(2α+π/3) = (-1/2)×(7/9) - (√3/2)×(4√2/9)
= -7/18 - 4√6/18
= -(7+4√6)/18
【方法总结】
• 角的变换技巧：把已知角看成一个整体，将所求角用已知角表示
• 常见变换：2α+β = 2(α+β/2)，α = (α+β) - β等
• 二倍角公式选择：已知sin用cos2α=1-2sin²α最方便

三、辅助角公式

辅助角公式：

asinx + bcosx = √(a²+b²)sin(x+φ)

其中tanφ = b/a，φ的象限由a、b的符号确定

特殊情况：
sinx ± cosx = √2sin(x ± π/4)
sinx ± √3cosx = 2sin(x ± π/3)
√3sinx ± cosx = 2sin(x ± π/6)

应用：求最值、周期、单调区间等
📝 例题7（原创）：求函数f(x) = sinx + √3cosx在[0,π/2]上的最大值和最小值。

【规范解答】

f(x) = sinx + √3cosx
= 2[(1/2)sinx + (√3/2)cosx]
= 2[sinxcos(π/3) + cosxsin(π/3)]
= 2sin(x + π/3)

∵ x∈[0,π/2]
∴ x+π/3 ∈ [π/3, 5π/6]

当x+π/3 = π/2，即x = π/6时，f(x)取得最大值2
当x+π/3 = 5π/6，即x = π/2时，f(x)取得最小值2×(1/2) = 1

最大值：2，最小值：1
【方法总结】
• asinx+bcosx型函数：用辅助角公式化为Asin(x+φ)
• 系数提取：A = √(a²+b²)
• 确定φ：cosφ=a/A，sinφ=b/A，tanφ=b/a
• 求最值：先确定x+φ的范围，再看sin(x+φ)的取值

3.2 函数y=Asin(ωx+φ)的图像与性质

一、参数的意义

y = Asin(ωx+φ)（A>0，ω>0）：

• A：振幅，最大值A，最小值-A
• ω：角频率，影响周期T = 2π/ω
• φ：初相，影响图像左右平移
• 周期：T = 2π/ω
• 频率：f = 1/T = ω/(2π)

图像变换：
y = sinx → y = sin(x+φ)：左加右减平移|φ|个单位
y = sinx → y = sin(ωx)：横坐标变为原来的1/ω倍
y = sinx → y = Asinx：纵坐标变为原来的A倍
📝 例题8（原创）：已知函数f(x) = Asin(ωx+φ)（A>0，ω>0，|φ|<π/2）的部分图像如图所示，求函数解析式。

【命题意图】考查由图像确定y=Asin(ωx+φ)的参数，是高考常考题型。
【分析思路】
从图像中读取：最大值确定A，周期确定ω，特殊点确定φ。

【规范解答】
（假设图像显示：最大值为2，最小值为-2，周期为π，过点(π/12, 2)）

由最大值2，得A = 2

由周期T = π，得ω = 2π/T = 2

∴ f(x) = 2sin(2x+φ)

代入点(π/12, 2)：
2sin(2×π/12+φ) = 2
sin(π/6+φ) = 1
π/6+φ = π/2 + 2kπ
φ = π/3 + 2kπ

∵ |φ| < π/2，∴ φ = π/3

解析式：f(x) = 2sin(2x+π/3)
【方法总结】
• 求A：看最大值（注意是否给出完整图像）
• 求ω：看周期T，用ω=2π/T
• 求φ：代入特殊点（最高点、零点、与y轴交点）
• 代入点后注意解的取舍（根据|φ|<π/2等条件）

第四章 解三角形

4.1 正弦定理与余弦定理

一、正弦定理

正弦定理：在△ABC中，

a/sinA = b/sinB = c/sinC = 2R

其中R为三角形外接圆半径

变形：
a = 2RsinA，b = 2RsinB，c = 2RsinC
sinA = a/(2R)，sinB = b/(2R)，sinC = c/(2R)
a:b:c = sinA:sinB:sinC

应用：已知两角一边、已知两边及其中一边的对角
📝 例题9（原创）：在△ABC中，a = √3，b = √2，B = 45°，求A、C和c。

【命题意图】考查正弦定理的应用，以及已知两边一对角时的多解情况判断。
【规范解答】

由正弦定理：a/sinA = b/sinB

sinA = asinB/b = √3×(√2/2)/√2 = √3/2

∵ a > b，∴ A > B = 45°
∴ A = 60° 或 A = 120°

情况一：A = 60°
C = 180° - 60° - 45° = 75°
c = bsinC/sinB = √2×sin75°/(√2/2) = 2sin75°
= 2×(√6+√2)/4 = (√6+√2)/2

情况二：A = 120°
C = 180° - 120° - 45° = 15°
c = 2sin15° = 2×(√6-√2)/4 = (√6-√2)/2

答案：有两解
① A=60°，C=75°，c=(√6+√2)/2
② A=120°，C=15°，c=(√6-√2)/2
【方法总结】
• 已知两边一对角（SSA）：可能有两解、一解或无解
• 判断方法：比较a与bsinA的大小
• a > b时，A有两个可能值（锐角和钝角），但需验证A+B<180°
• 牢记特殊角三角函数值
【易错提醒】
⚠️ 已知两边一对角时一定要注意多解情况！
⚠️ 求出角后一定要验证三角形内角和是否小于180°
⚠️ sinA = √3/2时，A可能是60°或120°，不要漏解

二、余弦定理

余弦定理：在△ABC中，

a² = b² + c² - 2bccosA
b² = a² + c² - 2accosB
c² = a² + b² - 2abcosC

变形（求角）：
cosA = (b²+c²-a²)/(2bc)
cosB = (a²+c²-b²)/(2ac)
cosC = (a²+b²-c²)/(2ab)

应用：已知三边、已知两边及其夹角
📝 例题10（原创）：在△ABC中，a = 2，b = 3，c = 4，求最大角的余弦值和三角形面积。

【规范解答】

第一步：判断最大角
∵ c > b > a，∴ C是最大角

第二步：用余弦定理求cosC
cosC = (a²+b²-c²)/(2ab)
= (4+9-16)/(2×2×3)
= (-3)/12
= -1/4

第三步：求面积
由cosC = -1/4，得sinC = √(1-1/16) = √15/4

面积S = (1/2)absinC
= (1/2)×2×3×(√15/4)
= 3√15/4
【方法总结】
• 已知三边：用余弦定理求角
• 最大角对大边，先判断哪个角最大
• 求面积：先用余弦定理求一个角，再求正弦，用S=(1/2)absinC
• 也可以用海伦公式：S = √[p(p-a)(p-b)(p-c)]，其中p=(a+b+c)/2

三、三角形面积公式

面积公式：

1. S = (1/2)ahₐ（底×高/2）

2. S = (1/2)absinC = (1/2)bcsinA = (1/2)acsinB

3. 海伦公式：S = √[p(p-a)(p-b)(p-c)]，其中p = (a+b+c)/2（半周长）

4. S = abc/(4R)（R为外接圆半径）

5. S = rp（r为内切圆半径，p为半周长）

4.2 解三角形综合应用

一、判断三角形形状

判断方法：

1. 角化边：用正弦定理把角的正弦化为边
2. 边化角：用正弦定理或余弦定理把边化为角

常见结论：
• a²+b²=c² ⇔ C=90° ⇔ 直角三角形
• a²+b²>c² ⇔ C<90° ⇔ 锐角三角形
• a²+b²<c² ⇔ C>90° ⇔ 钝角三角形
• a=b 或 A=B ⇔ 等腰三角形
• a=b=c 或 A=B=C=60° ⇔ 等边三角形
📝 例题11（原创）：在△ABC中，若acosA = bcosB，判断三角形的形状。

【命题意图】考查正弦定理和余弦定理在判断三角形形状中的应用。
【分析思路】
可以用正弦定理把边化为角，也可以用余弦定理把角化为边。

【规范解答】（方法一：边化角）
由正弦定理：a = 2RsinA，b = 2RsinB

代入acosA = bcosB：
2RsinAcosA = 2RsinBcosB
sin2A = sin2B

∴ 2A = 2B 或 2A = π - 2B
即 A = B 或 A + B = π/2

当A = B时，是等腰三角形；
当A + B = π/2时，C = π/2，是直角三角形。

答案：等腰三角形或直角三角形
【方法总结】
• 判断形状：边化角（用正弦定理）或角化边（用余弦定理）
• sin2A = sin2B ⇔ 2A=2B 或 2A=π-2B，不要漏解！
• 注意检验：如果是等腰直角三角形，两种条件都满足
【易错提醒】
⚠️ sin2A = sin2B 时，不要只得到A=B，还有2A=π-2B的情况！
⚠️ 结论要写完整："等腰或直角"，不是"等腰直角"

二、解三角形实际应用

常见应用问题：

1. 测量距离：两点间不可达的距离
2. 测量高度：底部或顶部不可达的高度
3. 角度问题：方位角、仰角、俯角

关键：画出示意图，标注已知量和未知量，选择合适定理
📝 例题12（原创）：某人在山顶观测山脚和山腰的两处目标，测得俯角分别为45°和60°，两处目标相距200米，求山的高度。

【命题意图】考查解三角形在实际测量问题中的应用，涉及俯角概念和正弦定理。
【分析思路】
画出示意图：设山顶为P，山脚目标为A，山腰目标为B。俯角是视线与水平线的夹角。

【规范解答】
设山高PQ = h，Q为山脚水平投影点。

俯角45°看A：∠PAQ = 45°，∴ QA = h
俯角60°看B：∠PBQ = 60°，∴ QB = h/√3 = h√3/3

在△PAB中（或在水平面上考虑）：
AB = 200，∠PAB = 45°，∠PBA = 180°-60° = 120°？

不对，应该在竖直平面内考虑。

实际上A、B在同一竖直平面内，P、A、B三点构成三角形。

∠PAQ = 45°，∠PBQ = 60°
∴ ∠PAB = 45°，∠PBA = 60°（内错角）

∠APB = 180° - 45° - 60° = 75°

由正弦定理：PA/sin60° = AB/sin75°

PA = 200×sin60°/sin75°
= 200×(√3/2)/[(√6+√2)/4]
= 100√3 × 4/(√6+√2)
= 400√3/(√6+√2)
= 400√3(√6-√2)/(6-2)
= 400√3(√6-√2)/4
= 100√3(√6-√2)
= 100(3√2-√6)

山高h = PA×sin45° = 100(3√2-√6)×(√2/2)
= 50(6-2√3)
= 100(3-√3)米 ≈ 126.8米
【方法总结】
• 实际应用问题：先画示意图，标出已知条件
• 俯角 = 仰角（内错角相等）
• 多个三角形问题：找公共边，逐步求解
• 复杂计算要有耐心，注意化简
