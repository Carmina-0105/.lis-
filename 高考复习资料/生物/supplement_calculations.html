🧮 计算题专项训练

生物高考常见计算类型及公式总结

一、蛋白质相关计算

核心公式

1. 肽键数 = 脱去水分子数 = 氨基酸数 - 肽链数
2. 蛋白质分子量 = 氨基酸平均分子量 × 氨基酸数 - 18 × 脱去水分子数
3. 基因碱基数 : mRNA碱基数 : 氨基酸数 = 6 : 3 : 1
4. 环肽：肽键数 = 氨基酸数

重点 例题1

某蛋白质由n条肽链组成，共含m个氨基酸，氨基酸的平均分子量为a。该蛋白质的分子量为多少？

解答：

肽键数 = m - n

脱去水分子数 = m - n

蛋白质分子量 = ma - 18(m - n)

二、DNA复制相关计算

核心公式

1. DNA复制n次后：
• DNA分子总数 = $2^n$
• 含亲代链的DNA分子数 = 2
• 只含新链的DNA分子数 = $2^n$ - 2

2. 需要游离的脱氧核苷酸数 = 亲代DNA中该核苷酸数 × ($2^n$ - 1)

三、遗传概率计算

核心公式

1. 分离定律：
Aa × Aa → 1AA : 2Aa : 1aa（表现型3:1）

2. 自由组合定律：
AaBb × AaBb → 9A_B_ : 3A_bb : 3aaB_ : 1aabb

3. 配子类型计算：
n对等位基因（独立遗传）产生的配子类型 = $2^n$

4. 后代基因型/表现型种类：
n对等位基因杂交，后代基因型种类 = $3^n$，表现型种类 = $2^n$

四、基因频率计算

核心公式

1. 基因频率 = (该基因纯合子个体数×2 + 杂合子个体数) / (总个体数×2)

2. 哈迪-温伯格定律：
设A的基因频率为p，a的基因频率为q
p + q = 1
p² + 2pq + q² = 1
AA频率 = p²，Aa频率 = 2pq，aa频率 = q²

五、种群密度计算

核心公式

1. 样方法：
种群密度 = 样方内个体总数 / 样方面积（或长度）

2. 标志重捕法：
N / M = n / m
N = (M × n) / m
（N：种群总数，M：初次捕获标记数，n：再次捕获总数，m：再次捕获标记数）

六、能量流动计算

核心公式

1. 能量传递效率 = (下一营养级同化量 / 上一营养级同化量) × 100%
一般为10%～20%

2. 最少消耗生产者 = 消费者获得量 ÷ 20%ⁿ
3. 最多消耗生产者 = 消费者获得量 ÷ 10%ⁿ
（n为营养级差）

📝 计算题解题技巧

1. 仔细审题：明确已知条件和所求问题

2. 选择合适公式：根据题目类型选择正确的计算公式

3. 注意单位换算：确保单位统一

4. 分步计算：复杂题目分步求解，避免出错

5. 验证结果：检查结果是否合理
