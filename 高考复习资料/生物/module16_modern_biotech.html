一、基因工程

1.1 基因工程的概念和工具

基因工程（基因拼接技术或DNA重组技术）：按照人们的意愿，把一种生物的某种基因提取出来，加以修饰改造，然后放到另一种生物的细胞里，定向地改造生物的遗传性状。

基因工程的基本工具：
1. 限制性核酸内切酶（限制酶）：识别双链DNA分子的某种特定核苷酸序列，并使每一条链中特定部位的两个核苷酸之间的磷酸二酯键断开
2. DNA连接酶：连接两个DNA片段之间的磷酸二酯键
3. 载体：将外源基因送入受体细胞的工具，常用的有质粒、λ噬菌体的衍生物、动植物病毒等

1.2 基因工程的基本操作程序

基因工程的四步操作

第一步：目的基因的获取
• 从基因文库中获取
• 利用PCR技术扩增
• 人工合成

第二步：基因表达载体的构建（核心步骤）
• 目的基因 + 启动子 + 终止子 + 标记基因
• 用同种限制酶切割，再用DNA连接酶连接

第三步：将目的基因导入受体细胞
• 植物：农杆菌转化法、基因枪法、花粉管通道法
• 动物：显微注射法
• 微生物：感受态细胞法

第四步：目的基因的检测与鉴定
• 分子水平：DNA分子杂交、分子杂交、抗原-抗体杂交
• 个体水平：抗虫鉴定、抗病鉴定等

1.3 基因工程的应用

应用领域 具体应用 实例
植物基因工程 抗虫、抗病、抗逆、改良品质 抗虫棉、转基因大豆
动物基因工程 提高生长速度、改善产品品质、生产药物 转基因鱼、乳腺生物反应器
基因工程药物 生产细胞因子、抗体、疫苗、激素 胰岛素、干扰素
基因治疗 把正常基因导入病人体内，使该基因的表达产物发挥功能 治疗遗传病

二、蛋白质工程

2.1 蛋白质工程的概念

蛋白质工程：以蛋白质分子的结构规律及其与生物功能的关系作为基础，通过基因修饰或基因合成，对现有蛋白质进行改造，或制造一种新的蛋白质，以满足人类的生产和生活需求。

蛋白质工程与基因工程的关系：
• 蛋白质工程是在基因工程的基础上延伸出来的第二代基因工程
• 基因工程生产的是自然界已有的蛋白质
• 蛋白质工程可以生产自然界不存在的蛋白质

三、细胞工程

3.1 植物细胞工程

植物组织培养技术：
• 原理：植物细胞的全能性
• 条件：离体、无菌、适宜的营养物质和激素
• 过程：外植体 → 愈伤组织 → 胚状体/丛芽 → 完整植株

植物体细胞杂交技术：
• 原理：细胞膜的流动性和植物细胞的全能性
• 过程：去壁 → 诱导融合 → 筛选 → 组织培养
• 意义：克服远缘杂交不亲和的障碍

3.2 动物细胞工程

动物细胞工程的主要技术
技术 原理 应用
动物细胞培养 细胞增殖 生产生物制品、检测有毒物质、医学研究
动物细胞核移植 动物细胞核的全能性 克隆动物、生产转基因动物
动物细胞融合 细胞膜的流动性 制备单克隆抗体
生产单克隆抗体 细胞增殖、细胞融合 诊断试剂、治疗疾病、运载药物

四、胚胎工程

4.1 体外受精

体外受精的主要步骤：
1. 卵母细胞的采集和培养
2. 精子的采集和获能
3. 受精

应用：试管动物技术

4.2 胚胎移植

胚胎移植：将雌性动物的早期胚胎，或者通过体外受精及其他方式得到的胚胎，移植到同种的、生理状态相同的其他雌性动物体内，使之继续发育为新个体的技术。

胚胎移植的基本程序：
供体、受体的选择和处理 → 配种或进行人工授精 → 胚胎的收集、检查、培养或保存 → 胚胎移植 → 移植后的检查

胚胎移植的生理学基础：
• 供体和受体的生理状态相同
• 早期胚胎处于游离状态
• 受体对移入子宫的外来胚胎基本上不发生免疫排斥反应
• 供体胚胎可与受体子宫建立正常的生理和组织联系

五、生物技术的安全性和伦理问题

5.1 转基因生物的安全性

转基因生物的安全性问题：
• 食物安全：滞后效应、过敏原、营养成分改变
• 生物安全：对生物多样性的影响
• 环境安全：对生态系统稳定性的影响

对待转基因生物的正确态度：趋利避害，不能因噎废食

5.2 生物技术的伦理问题

克隆人：中国政府禁止生殖性克隆人，不反对治疗性克隆

试管婴儿：设计试管婴儿需进行遗传学诊断

基因检测：基因歧视问题

生物武器：禁止生物武器公约

📝 高考真题演练

【2023年全国甲卷】下列关于基因工程的叙述，正确的是（ ）

A. 限制酶和DNA连接酶的作用部位不同

B. 基因工程的核心步骤是目的基因的获取

C. 质粒是基因工程常用的载体

D. 基因工程只能生产自然界已有的蛋白质

答案：C

解析：

A错误：限制酶和DNA连接酶都作用于磷酸二酯键，作用部位相同。

B错误：基因工程的核心步骤是基因表达载体的构建，不是目的基因的获取。

C正确：质粒是基因工程中最常用的载体。

D错误：基因工程生产的是自然界已有的蛋白质；蛋白质工程可以生产自然界不存在的蛋白质。
