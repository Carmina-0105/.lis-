一、通过神经系统的调节

1.1 神经调节的结构基础和反射

反射：在中枢神经系统参与下，动物体或人体对内外环境变化作出的规律性应答。

反射弧：完成反射的结构基础，由感受器、传入神经、神经中枢、传出神经和效应器五部分组成。

🔁 反射弧结构

感受器 → 传入神经 → 神经中枢 → 传出神经 → 效应器
（感觉神经末梢）（有神经节）（脊髓/脑）（运动神经末梢及所支配的肌肉或腺体）

⚠️ 反射弧完整性的重要性：
反射活动必须经过完整的反射弧来实现。反射弧任何一个环节受损，反射都不能完成。
• 感受器或传入神经受损：无感觉，无反应
• 神经中枢受损：无反应
• 传出神经或效应器受损：有感觉，无反应

1.2 兴奋在神经纤维上的传导

静息电位：神经纤维未受刺激时，细胞膜两侧的电位表现为外正内负。
形成原因：K⁺外流（协助扩散），使膜外阳离子浓度高于膜内。

动作电位：神经纤维受到刺激时，细胞膜两侧的电位表现为外负内正。
形成原因：Na⁺内流（协助扩散），使兴奋部位膜内侧阳离子浓度高于膜外侧。
兴奋传导的过程

刺激 → 兴奋部位（外负内正）→ 与相邻未兴奋部位（外正内负）形成电位差 → 局部电流 → 相邻部位产生兴奋 → 兴奋向前传导

特点：双向传导、速度快、不衰减

1.3 兴奋在神经元之间的传递

突触结构：
• 突触前膜：轴突末梢突触小体的膜
• 突触间隙：两膜之间的间隙（组织液）
• 突触后膜：与突触前膜相对应的细胞体膜或树突膜
兴奋在突触处的传递过程

兴奋到达突触前膜 → 突触小泡释放神经递质（胞吐）→ 神经递质扩散通过突触间隙 → 神经递质与突触后膜上的特异性受体结合 → 突触后膜电位变化 → 引起下一个神经元兴奋或抑制 → 神经递质被分解或回收

📡 兴奋在神经纤维上的传导

形式：电信号（神经冲动）

方向：双向传导

速度：快

特点：在同一神经元上进行

📨 兴奋在神经元之间的传递

形式：电信号→化学信号→电信号

方向：单向传递（只能从突触前膜到突触后膜）

速度：慢（有突触延搁）

特点：在神经元之间进行

⚠️ 神经递质相关知识：
1. 神经递质只存在于突触前膜的突触小泡中，只能由突触前膜释放，作用于突触后膜
2. 神经递质分为兴奋性递质（如乙酰胆碱）和抑制性递质（如γ-氨基丁酸）
3. 神经递质发挥作用后会被迅速分解或回收，以保证突触传递的灵敏性
4. 神经递质与激素的区别：神经递质作用于突触后膜，激素作用于靶细胞

二、通过激素的调节

2.1 激素调节的发现

促胰液素的发现（斯他林和贝利斯，1902年）：
实验表明，胰腺分泌胰液不是神经调节，而是化学调节——盐酸刺激小肠黏膜产生促胰液素，促胰液素随血液运输到胰腺，促进胰腺分泌胰液。

意义：发现了第一种激素，开辟了激素调节的新领域。

2.2 人体主要内分泌腺及其分泌的激素

内分泌腺 激素 主要功能 分泌异常
下丘脑 促甲状腺激素释放激素(TRH) 促进垂体分泌促甲状腺激素 -
抗利尿激素(ADH) 促进肾小管和集合管对水的重吸收 尿崩症（分泌不足）
垂体 生长激素(GH) 促进生长，促进蛋白质合成和骨生长 侏儒症（幼年不足）、巨人症（幼年过多）、肢端肥大症（成年过多）
促甲状腺激素(TSH) 促进甲状腺的生长发育和甲状腺激素的分泌 -
促性腺激素 促进性腺的生长发育和性激素的分泌 -
甲状腺 甲状腺激素 促进新陈代谢和生长发育，提高神经系统的兴奋性 呆小症（幼年不足）、甲亢（过多）、大脖子病（缺碘）
胰岛 胰岛素 降低血糖浓度 糖尿病（分泌不足）
胰岛 胰高血糖素 升高血糖浓度 -
肾上腺 肾上腺素 升高血糖，加快心跳，促进产热 -
性腺 性激素（雌激素/雄激素） 促进生殖器官发育和生殖细胞形成，激发并维持第二性征 性发育异常

2.3 激素调节的特点

1. 微量和高效：激素在血液中含量很低，但能产生显著的生理效应
2. 通过体液运输：激素分泌后进入体液，随血液流到全身
3. 作用于靶器官、靶细胞：激素只作用于含有特定受体的靶细胞
4. 作为信使传递信息：激素既不组成细胞结构，又不提供能量，也不起催化作用，只是调节生命活动

2.4 甲状腺激素分泌的分级调节

分级调节过程

寒冷/紧张刺激 → 下丘脑分泌促甲状腺激素释放激素(TRH) → 垂体分泌促甲状腺激素(TSH) → 甲状腺分泌甲状腺激素 → 促进新陈代谢，增加产热

反馈调节：当血液中甲状腺激素含量增加到一定程度时，会反过来抑制下丘脑和垂体分泌相关激素，进而使甲状腺激素分泌减少；当甲状腺激素含量降低时，对下丘脑和垂体的抑制作用减弱，使甲状腺激素分泌增加。

意义：维持血液中甲状腺激素含量的相对稳定。

三、神经调节与体液调节的关系

3.1 血糖平衡的调节

血糖的来源和去路

血糖来源（升高血糖）：

①食物中糖类的消化吸收（主要来源）

②肝糖原分解

③脂肪等非糖物质转化

血糖去路（降低血糖）：

①氧化分解供能（主要去路）

②合成肝糖原、肌糖原

③转化为脂肪、某些氨基酸等

🔽 胰岛素（降血糖）

分泌细胞：胰岛B细胞

作用机理：

  • 促进组织细胞加速摄取、利用和储存葡萄糖
  • 促进葡萄糖氧化分解
  • 促进葡萄糖合成糖原
  • 促进葡萄糖转化为脂肪等非糖物质
  • 抑制肝糖原分解和非糖物质转化为葡萄糖

🔼 胰高血糖素（升血糖）

分泌细胞：胰岛A细胞

作用机理：

  • 促进肝糖原分解为葡萄糖
  • 促进脂肪等非糖物质转化为葡萄糖

注意：不能促进肌糖原分解（肌糖原只能供肌肉自身使用）

3.2 体温调节

体温调节中枢：下丘脑
体温感觉中枢：大脑皮层
体温恒定原理：产热 = 散热

产热器官：安静时主要是肝脏，运动时主要是骨骼肌
散热器官：主要是皮肤（通过辐射、传导、对流、蒸发等方式）
寒冷环境下的体温调节

寒冷刺激 → 皮肤冷觉感受器 → 下丘脑体温调节中枢 →
①神经调节：骨骼肌战栗（快速产热）、皮肤血管收缩（减少散热）、立毛肌收缩（减少散热）
②体液调节：甲状腺激素和肾上腺素分泌增加（促进代谢，增加产热）

3.3 水盐平衡调节

水盐调节中枢：下丘脑
渴觉中枢：大脑皮层
调节激素：抗利尿激素（ADH）

抗利尿激素：
• 合成部位：下丘脑
• 释放部位：垂体后叶
• 作用：促进肾小管和集合管对水的重吸收，减少尿量

四、免疫调节

4.1 免疫系统的组成

组成 具体内容
免疫器官 骨髓、胸腺、脾、淋巴结、扁桃体等
免疫细胞 吞噬细胞、淋巴细胞（T细胞、B细胞）
免疫活性物质 抗体、淋巴因子、溶菌酶等

4.2 免疫系统的三道防线

防线 组成 特点
第一道防线 皮肤、黏膜 非特异性免疫（先天性免疫），对多种病原体都有防御作用
第二道防线 体液中的杀菌物质（如溶菌酶）和吞噬细胞 非特异性免疫，对多种病原体都有防御作用
第三道防线 免疫器官和免疫细胞（主要是淋巴细胞） 特异性免疫（后天性免疫），只对特定病原体起作用

4.3 特异性免疫

🛡️ 体液免疫

作用对象：细胞外的病原体和毒素

主要细胞：B细胞

过程：

  1. 抗原被吞噬细胞摄取、处理，呈递给T细胞
  2. T细胞产生淋巴因子，作用于B细胞
  3. B细胞增殖分化为浆细胞和记忆细胞
  4. 浆细胞分泌抗体，与抗原特异性结合

结果：抗原被清除或失去毒性

⚔️ 细胞免疫

作用对象：被病原体感染的细胞、癌细胞、移植器官的细胞

主要细胞：T细胞

过程：

  1. 抗原被吞噬细胞摄取、处理，呈递给T细胞
  2. T细胞增殖分化为效应T细胞和记忆细胞
  3. 效应T细胞与靶细胞密切接触
  4. 靶细胞裂解死亡，抗原暴露

结果：靶细胞裂解，抗原被释放

4.4 免疫失调引起的疾病

类型 疾病 病因
过敏反应 过敏性鼻炎、哮喘、荨麻疹等 已免疫的机体再次接受相同抗原刺激时，组织损伤或功能紊乱
特点：发作迅速、反应强烈、消退较快；一般不会破坏组织细胞；有明显的遗传倾向和个体差异
自身免疫病 类风湿性关节炎、系统性红斑狼疮、风湿性心脏病等 免疫系统异常敏感，将自身物质当作外来异物攻击
免疫缺陷病 艾滋病（AIDS） HIV攻击T细胞，使免疫功能丧失

📝 高考真题演练

【2023年全国甲卷】下列关于人体生命活动调节的叙述，正确的是（ ）

A. 兴奋在神经纤维上以电信号的形式双向传导

B. 胰岛素通过促进肝糖原分解来降低血糖浓度

C. 体温调节中枢位于大脑皮层

D. 抗体是由T细胞分泌的免疫活性物质

答案：A

解析：

A正确：兴奋在神经纤维上以电信号（神经冲动）的形式双向传导。

B错误：胰岛素通过促进组织细胞加速摄取、利用和储存葡萄糖，以及抑制肝糖原分解和非糖物质转化来降低血糖。

C错误：体温调节中枢位于下丘脑，体温感觉中枢位于大脑皮层。

D错误：抗体是由浆细胞（效应B细胞）分泌的，T细胞分泌的是淋巴因子。
