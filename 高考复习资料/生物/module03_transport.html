一、物质跨膜运输的实例

1.1 渗透作用

渗透作用：水分子（或其他溶剂分子）通过半透膜从低浓度溶液向高浓度溶液的扩散。

发生条件：
① 具有半透膜
② 半透膜两侧溶液具有浓度差

渗透方向：水分子从低浓度溶液 → 高浓度溶液（从水的相对含量高 → 相对含量低）
渗透作用的原理

半透膜两侧：
浓度高的一侧 → 渗透压高 → 吸水能力强
浓度低的一侧 → 渗透压低 → 吸水能力弱

结果：水分子从低浓度溶液向高浓度溶液移动，直到两侧渗透压平衡

1.2 动物细胞的吸水和失水

红细胞在不同溶液中的变化：

• 等渗溶液（0.9% NaCl）：细胞形态正常，水分进出平衡
• 低渗溶液（<0.9% NaCl）：细胞吸水膨胀，甚至涨破（溶血）
• 高渗溶液（>0.9% NaCl）：细胞失水皱缩

1.3 植物细胞的吸水和失水

成熟植物细胞的结构特点：

• 细胞壁：全透性，伸缩性小
• 原生质层：细胞膜 + 液泡膜 + 两层膜之间的细胞质，相当于半透膜
• 中央大液泡：含有细胞液，具有一定浓度

原生质层的特性：选择透过性，伸缩性大
植物细胞吸水与失水的条件

细胞液浓度 > 外界溶液浓度：
→ 细胞吸水 → 质壁分离复原或细胞膨胀（有细胞壁限制）

细胞液浓度 < 外界溶液浓度：
→ 细胞失水 → 质壁分离

细胞液浓度 = 外界溶液浓度：
→ 水分进出平衡 → 细胞维持原状

1.4 质壁分离与复原

质壁分离：植物细胞失水时，原生质层与细胞壁分离的现象。

质壁分离复原：已发生质壁分离的细胞，放入清水或低浓度溶液中，细胞吸水，原生质层恢复原状的现象。
质壁分离实验要点

实验材料：紫色洋葱鳞片叶外表皮（有紫色大液泡，便于观察）

实验步骤：
① 制作临时装片 → ② 观察正常细胞 → ③ 滴加蔗糖溶液（0.3g/mL）→ ④ 观察质壁分离 → ⑤ 滴加清水 → ⑥ 观察复原

实验现象：
• 质壁分离：细胞壁与原生质层分离，液泡变小，颜色变深
• 质壁分离复原：原生质层紧贴细胞壁，液泡变大，颜色变浅

⚠️ 质壁分离实验注意事项：
1. 必须使用活细胞（死细胞原生质层失去选择透过性）
2. 蔗糖溶液浓度不宜过高（会导致细胞过度失水死亡）
3. 质壁分离时间不宜过长（细胞会死亡无法复原）
4. 盐酸、酒精等溶液会导致细胞死亡，不能用于质壁分离实验

二、生物膜的流动镶嵌模型

2.1 对生物膜结构的探索历程

时间 科学家 实验/观点 结论
19世纪末 欧文顿 脂溶性物质更容易通过细胞膜 膜是由脂质组成的
20世纪初 科学家 化学分析 膜的主要成分是脂质和蛋白质
1925年 荷兰科学家 红细胞脂质铺展实验 脂质分子排列为连续的两层
1959年 罗伯特森 电镜观察 提出蛋白质-脂质-蛋白质三层结构
1970年 科学家 荧光标记的小鼠细胞和人细胞融合实验 细胞膜具有流动性
1972年 桑格和尼克森 提出流动镶嵌模型 被广泛接受的膜结构模型

2.2 流动镶嵌模型的基本内容

1. 磷脂双分子层：构成膜的基本支架，具有流动性

2. 蛋白质分子：
• 镶嵌、嵌入或横跨磷脂双分子层
• 大多数蛋白质分子是可以运动的

3. 糖类：
• 与蛋白质结合形成糖蛋白（糖被），位于细胞膜外侧
• 与脂质结合形成糖脂
• 糖被具有识别、保护、润滑等功能

三、物质跨膜运输的方式

3.1 小分子物质的跨膜运输

运输方式 运输方向 是否需要载体 是否消耗能量 实例
自由扩散 高浓度→低浓度 不需要 不消耗 O₂、CO₂、水、甘油、乙醇、苯、脂肪酸、维生素D、胆固醇
协助扩散 高浓度→低浓度 需要 不消耗 红细胞吸收葡萄糖、神经细胞Na⁺内流/K⁺外流
主动运输 低浓度→高浓度 需要 消耗ATP 小肠上皮细胞吸收葡萄糖、氨基酸、无机盐离子；植物根细胞吸收矿质元素

🔄 被动运输

特点：

  • 顺浓度梯度（高→低）
  • 不消耗能量
  • 自由扩散不需要载体
  • 协助扩散需要载体蛋白

实例：

O₂、CO₂、水的跨膜运输

⚡ 主动运输

特点：

  • 逆浓度梯度（低→高）
  • 消耗能量（ATP）
  • 需要载体蛋白
  • 具有选择性

意义：

保证细胞按照生命活动需要主动选择吸收营养物质，排出代谢废物

3.2 大分子物质的运输

胞吞：细胞摄取大分子时，大分子附着在细胞膜表面，细胞膜内陷形成小囊，包围大分子，然后小囊从细胞膜上分离下来，形成囊泡，进入细胞内部。

胞吐：细胞需要外排的大分子，先在细胞内形成囊泡，囊泡移动到细胞膜处，与细胞膜融合，将大分子排出细胞。

特点：
• 都消耗能量（ATP）
• 都体现细胞膜的流动性
• 不需要载体蛋白
物质跨膜运输方式的判断方法

第一步：看分子大小
大分子 → 胞吞/胞吐
小分子 → 继续判断

第二步：看浓度梯度
低→高 → 主动运输
高→低 → 继续判断

第三步：看是否需要载体
需要 → 协助扩散
不需要 → 自由扩散

📝 高考真题演练

【2023年全国甲卷】下列关于物质跨膜运输的叙述，正确的是（ ）

A. 葡萄糖进入红细胞需要消耗能量

B. 氧气进入肺泡细胞需要载体蛋白协助

C. 植物根细胞吸收矿质元素离子属于主动运输

D. 大分子物质通过主动运输进入细胞

答案：C

解析：

A错误：葡萄糖进入红细胞是协助扩散，不需要消耗能量。

B错误：氧气进入细胞是自由扩散，不需要载体蛋白。

C正确：植物根细胞吸收矿质元素离子是逆浓度梯度进行的，属于主动运输。

D错误：大分子物质通过胞吞进入细胞，不是主动运输。
