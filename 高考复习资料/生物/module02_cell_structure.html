一、细胞膜的结构与功能

1.1 细胞膜的成分

成分 含量 功能 特点
脂质 约50% 构成膜的基本骨架 主要是磷脂，动物细胞膜还含有胆固醇
蛋白质 约40% 承担膜的主要功能 功能越复杂的膜，蛋白质种类和数量越多
糖类 约2%-10% 识别、保护、润滑 与蛋白质或脂质结合形成糖蛋白或糖脂
重要结论：细胞膜的功能复杂程度与蛋白质的种类和数量成正比。
例如：线粒体内膜（有氧呼吸第三阶段）> 细胞膜 > 其他膜结构

1.2 流动镶嵌模型

流动镶嵌模型要点（辛格和尼科尔森，1972年）

1. 磷脂双分子层构成膜的基本支架

• 磷脂分子头部亲水，朝向膜的两侧（水环境）

• 磷脂分子尾部疏水，朝向膜的内部

2. 蛋白质分子镶嵌或贯穿其中

• 有的镶在表面，有的部分或全部嵌入，有的贯穿整个磷脂双分子层

3. 磷脂分子和大多数蛋白质分子可以运动

• 体现膜的流动性

细胞膜的结构特点 vs 功能特性

🔀 结构特点：具有一定的流动性

原因：

  • 磷脂分子可以侧向自由移动
  • 大多数蛋白质分子可以运动

实例：

  • 变形虫的变形运动
  • 细胞融合（如受精作用、杂交瘤细胞制备）
  • 胞吞和胞吐
  • 植物细胞的质壁分离与复原

🛡️ 功能特性：选择透过性

原因：

  • 磷脂双分子层的屏障作用
  • 载体蛋白的专一性

表现：

  • 细胞需要的营养物质可以进入
  • 细胞不需要或有害的物质不易进入
  • 细胞产生的废物可以排出
  • 细胞内的重要物质不会流失
⚠️ 易错提醒：
1. 流动性是结构特点，选择透过性是功能特性，不要混淆
2. 选择透过性的基础是流动性，但两者不是同一概念
3. 细胞死亡后，细胞膜失去选择透过性（变成全透性），但仍有流动性

1.3 细胞膜的功能

功能 具体内容 相关结构
将细胞与外界环境分隔开 保障细胞内部环境的相对稳定 磷脂双分子层
控制物质进出细胞 选择性吸收营养物质，排出代谢废物 载体蛋白、通道蛋白
进行细胞间的信息交流 细胞识别、信号传递 糖蛋白（受体）

细胞间信息交流的方式

方式一：通过化学物质传递信息

细胞分泌化学物质（如激素），通过血液运输，与靶细胞膜上的受体结合，传递信息。

实例：胰岛素与靶细胞受体结合，调节血糖。

方式二：细胞膜直接接触

相邻两个细胞的细胞膜直接接触，通过膜上的信号分子和受体传递信息。

实例：精子与卵细胞的识别和结合；效应T细胞与靶细胞的接触。

方式三：细胞间形成通道

相邻细胞之间形成通道，携带信息的物质通过通道进入另一个细胞。

实例：高等植物细胞之间的胞间连丝。

二、细胞器的结构与功能

细胞器分类（按膜结构）：
• 双层膜：线粒体、叶绿体（都有DNA、RNA，能半自主复制）
• 单层膜：内质网、高尔基体、溶酶体、液泡
• 无膜结构：核糖体、中心体

2.1 线粒体——"动力车间"

🔋 线粒体结构与功能

结构：

  • 外膜：平滑，将线粒体与细胞质基质分开
  • 内膜：向内折叠形成嵴，增大膜面积，附着有氧呼吸相关酶
  • 基质：含有DNA、RNA、核糖体、有氧呼吸相关酶

功能：有氧呼吸的主要场所，将有机物中的化学能转化为ATP中的能量

分布：动植物细胞都有，代谢旺盛的细胞中数量多

特点：含DNA、RNA，能半自主复制，属于半自主性细胞器

2.2 叶绿体——"养料制造车间"和"能量转换站"

🌿 叶绿体结构与功能

结构：

  • 外膜和内膜：双层膜结构
  • 基粒：由类囊体堆叠而成，类囊体薄膜上分布有光合色素和光反应相关酶
  • 基质：含有DNA、RNA、核糖体、暗反应相关酶

功能：光合作用的场所，将光能转化为化学能储存在有机物中

分布：主要存在于绿色植物的叶肉细胞和幼茎皮层细胞

特点：含DNA、RNA，能半自主复制，属于半自主性细胞器

线粒体 vs 叶绿体 比较

比较项目 线粒体 叶绿体
增大膜面积方式 内膜向内折叠形成嵴 类囊体堆叠形成基粒
功能 有氧呼吸，分解有机物 光合作用，合成有机物
ATP产生 产生ATP供细胞利用 产生ATP仅供暗反应使用
气体交换 O₂进入，CO₂排出 CO₂进入，O₂排出

2.3 内质网——蛋白质合成和加工，以及脂质合成的"车间"

🏭 内质网的类型与功能

粗面内质网（附着核糖体）：

  • 功能：分泌蛋白的合成和加工
  • 与分泌蛋白的运输有关

光面内质网（无核糖体附着）：

  • 功能：脂质合成（如性激素）
  • 解毒作用（如肝细胞）

分布：动植物细胞都有，是细胞内膜面积最大的细胞器

2.4 高尔基体——蛋白质加工、分类、包装的"车间"及"发送站"

📦 高尔基体功能

动物细胞：

  • 对来自内质网的蛋白质进行加工、分类和包装
  • 与分泌物的形成有关

植物细胞：

  • 与细胞壁的形成有关（合成纤维素）

分布：动植物细胞都有

2.5 核糖体——"生产蛋白质的机器"

⚙️ 核糖体结构与功能

结构：无膜结构，由rRNA和蛋白质组成

类型：

  • 游离核糖体：合成胞内蛋白（如细胞呼吸酶）
  • 附着核糖体：合成分泌蛋白（如抗体、消化酶）

功能：氨基酸脱水缩合形成肽链（翻译的场所）

分布：所有细胞（原核细胞和真核细胞）都有

2.6 溶酶体——"消化车间"

🗑️ 溶酶体功能

结构：单层膜包裹，内含多种水解酶

功能：

  • 分解衰老、损伤的细胞器
  • 吞噬并杀死侵入细胞的病毒或细菌
  • 细胞凋亡时与细胞膜融合，释放水解酶使细胞自溶

分布：主要存在于动物细胞

特点：膜破裂后水解酶释放，可导致细胞自溶

2.7 液泡——调节植物细胞内的环境

💧 液泡结构与功能

结构：单层膜，内有细胞液（含糖类、无机盐、色素、蛋白质等）

功能：

  • 调节细胞内的环境，维持细胞形态
  • 储存营养物质和代谢废物
  • 维持细胞渗透压，与质壁分离有关

分布：主要存在于植物细胞，成熟植物细胞有一个大液泡

注意：动物细胞也有液泡，但一般小而多，不明显

2.8 中心体——与细胞有丝分裂有关

🎯 中心体结构与功能

结构：无膜结构，由两个互相垂直排列的中心粒及周围物质组成

功能：与细胞的有丝分裂有关，发出星射线形成纺锤体

分布：动物细胞和某些低等植物细胞

复制：在间期复制，前期移向两极

三、细胞器的协调配合——以分泌蛋白为例

分泌蛋白：在细胞内合成，分泌到细胞外起作用的蛋白质。
实例：消化酶（如唾液淀粉酶）、抗体、部分激素（如胰岛素）、血浆蛋白等。

3.1 分泌蛋白的合成与分泌过程

分泌蛋白合成与分泌路径

核糖体（合成肽链）
↓
内质网（加工：折叠、组装、加糖基团）
↓ 囊泡运输
高尔基体（进一步加工、分类、包装）
↓ 囊泡运输
细胞膜（胞吐分泌到细胞外）

3.2 各细胞器的分工

细胞器 作用 说明
核糖体 合成肽链 氨基酸脱水缩合形成多肽
内质网 加工、运输 对肽链进行折叠、组装，加上糖基团等
高尔基体 加工、包装、分泌 对蛋白质进一步加工，分类包装，形成分泌小泡
线粒体 供能 为整个过程提供ATP
细胞膜 分泌 通过胞吐将蛋白质分泌到细胞外
💡 放射性同位素标记法研究分泌蛋白：
用³H标记亮氨酸，追踪放射性出现的先后顺序：
核糖体 → 内质网 → 高尔基体 → 细胞膜外
这证明了分泌蛋白的合成和运输路径。

四、细胞的生物膜系统

4.1 生物膜系统的组成

生物膜系统 = 细胞膜 + 核膜 + 细胞器膜

包括：细胞膜、核膜、线粒体膜、叶绿体膜、内质网膜、高尔基体膜、溶酶体膜、液泡膜等

4.2 生物膜系统的功能

功能 具体说明
细胞膜 物质运输、能量转换、信息传递
广阔的膜面积为酶提供附着位点 如内质网、线粒体内膜、叶绿体类囊体薄膜
将细胞分隔成小区室 使细胞内能够同时进行多种化学反应，互不干扰
⚠️ 注意：
1. 原核细胞没有生物膜系统（只有细胞膜，没有核膜和细胞器膜）
2. 生物膜系统是针对细胞而言的，不是针对生物体
3. 小肠黏膜、胃黏膜等不属于生物膜系统

五、细胞核——系统的控制中心

5.1 细胞核的结构

🏛️ 细胞核的结构组成

核膜：双层膜，外膜与内质网相连，上有核孔

核孔：实现核质之间频繁的物质交换和信息交流

核仁：与某种RNA（rRNA）的合成以及核糖体的形成有关

染色质：由DNA和蛋白质组成，是遗传信息的主要载体

5.2 染色质与染色体

染色质

状态：细丝状

时期：细胞分裂间期

形态：染色质丝（DNA+蛋白质）

染色体

状态：圆柱状或杆状

时期：细胞分裂期

形态：高度螺旋化、缩短变粗

染色质与染色体的关系：同一种物质在细胞不同时期的两种存在状态。
染色质（间期）→ 高度螺旋化、缩短变粗 → 染色体（分裂期）→ 解螺旋 → 染色质（末期）

5.3 细胞核的功能

细胞核是遗传信息库，是细胞代谢和遗传的控制中心。

实验证据：
1. 美西螈核移植实验：细胞核控制生物性状
2. 蝾螈受精卵横缢实验：细胞核控制细胞分裂和分化
3. 变形虫切割实验：细胞核控制细胞代谢
4. 伞藻嫁接与核移植实验：细胞核控制生物形态建成

六、原核细胞与真核细胞的比较

比较项目 原核细胞 真核细胞
细胞大小 较小（1-10μm） 较大（10-100μm）
细胞核 无核膜包被，无成形细胞核（拟核） 有核膜包被的成形细胞核
染色体 无染色体，有环状DNA（拟核DNA和质粒） 有染色体（DNA+蛋白质）
细胞器 只有核糖体一种细胞器 有多种细胞器
细胞壁 主要成分是肽聚糖（细菌）或几丁质（真菌） 植物：纤维素和果胶；真菌：几丁质
代表生物 细菌、蓝藻（蓝细菌）、放线菌、支原体、衣原体 动物、植物、真菌
⚠️ 常见误区：
1. 原核生物≠原生生物（原生生物是真核生物，如草履虫、变形虫）
2. 细菌名称中常带有"杆"、"球"、"螺旋"、"弧"等字样（如大肠杆菌、金黄色葡萄球菌）
3. 蓝藻（蓝细菌）是原核生物，包括颤藻、念珠藻、发菜等，能进行光合作用但没有叶绿体
4. 支原体是最小的原核生物，没有细胞壁

七、用高倍显微镜观察叶绿体和细胞质流动（实验）

🔬 实验：用高倍显微镜观察叶绿体和细胞质流动

实验原理：

  • 叶绿体呈绿色、扁平的椭球形或球形，散布于细胞质中
  • 活细胞中的细胞质处于不断流动的状态，可用叶绿体的运动作为标志观察细胞质流动

材料选择：

  • 观察叶绿体：藓类的小叶或菠菜叶稍带些叶肉的下表皮
  • 观察细胞质流动：黑藻叶片（叶片薄，只有1-2层叶肉细胞，叶绿体大且数量少）

实验步骤：

  1. 制作临时装片：在载玻片上滴一滴清水，将材料放入水滴中，盖上盖玻片
  2. 低倍镜观察：找到目标细胞，移至视野中央
  3. 高倍镜观察：转动转换器换高倍镜，调节细准焦螺旋使物像清晰
  4. 观察细胞质流动：寻找叶脉附近的细胞，观察叶绿体的运动方向
实验结果：
• 叶绿体呈绿色扁平椭球形，散布于细胞质中
• 细胞质流动方向：一般沿一定方向循环流动（如顺时针或逆时针）
注意事项：
• 临时装片要始终保持有水状态，保证细胞活性
• 光线不宜过强，否则叶绿体颜色太浅不易观察
• 若细胞质流动不明显，可适当提高温度或光照强度

📝 高考真题演练

【2023年全国乙卷】下列关于细胞结构和功能的叙述，正确的是（ ）

A. 细胞膜上的受体是细胞间信息交流所必需的结构

B. 核糖体是噬菌体、细菌、酵母菌唯一共有的细胞器

C. 溶酶体能合成多种水解酶，降解衰老损伤的细胞器

D. 细胞核是遗传信息库，是细胞代谢和遗传的控制中心

答案：D

解析：

A错误：细胞间信息交流不一定需要受体，如植物细胞通过胞间连丝进行信息交流。

B错误：噬菌体是病毒，没有细胞结构，没有核糖体。

C错误：溶酶体内的水解酶是在核糖体上合成的，溶酶体不能合成水解酶。

D正确：细胞核是遗传信息库，是细胞代谢和遗传的控制中心。
