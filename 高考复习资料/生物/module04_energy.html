一、降低化学反应活化能的酶

1.1 酶的本质与作用

酶的定义：酶是活细胞产生的具有催化作用的有机物，其中绝大多数酶是蛋白质，少数酶是RNA（核酶）。

酶的作用：降低化学反应的活化能，使反应在温和条件下快速进行。
活化能概念

分子从常态转变为容易发生化学反应的活跃状态所需要的能量称为活化能。

酶的作用机理：同无机催化剂相比，酶降低活化能的作用更显著，催化效率更高。

1.2 酶的特性

特性 含义 原因/实例
高效性 酶的催化效率远高于无机催化剂 酶的催化效率是无机催化剂的$10^7$～$10^{13}$倍
专一性 一种酶只能催化一种或一类化学反应 淀粉酶只能催化淀粉水解，不能催化蔗糖水解
作用条件温和 酶在常温、常压、适宜pH下活性最高 高温、过酸、过碱会使酶变性失活

1.3 影响酶活性的因素

温度对酶活性的影响

曲线特征：钟形曲线
• 在一定范围内，随温度升高，酶活性增强
• 达到最适温度时，酶活性最高
• 超过最适温度，随温度升高，酶活性下降，甚至失活

关键点：
• 低温：抑制酶活性，但酶的空间结构稳定，温度恢复后活性可恢复
• 高温：破坏酶的空间结构，使酶永久失活（变性）

pH对酶活性的影响

曲线特征：钟形曲线
• 在一定范围内，随pH升高或降低，酶活性变化
• 达到最适pH时，酶活性最高
• 过酸或过碱都会使酶变性失活

常见酶的最适pH：
• 胃蛋白酶：1.5-2.0（强酸性）
• 胰蛋白酶：8.0-9.0（弱碱性）
• 唾液淀粉酶：6.8左右（近中性）
• 过氧化氢酶：7.0左右
⚠️ 易错提醒：
1. 酶可以重复利用，反应前后性质和数量不变
2. 酶只能催化热力学上允许进行的反应
3. 酶不能为反应提供能量，只能降低活化能
4. 底物浓度和酶浓度影响反应速率，但不影响酶的活性

二、细胞的能量"货币"——ATP

2.1 ATP的结构

ATP的结构简式

A—P～P～P

• A：腺苷（腺嘌呤 + 核糖）

• P：磷酸基团

• —：普通磷酸键

• ～：高能磷酸键

ATP的组成：
1个腺嘌呤 + 1个核糖 + 3个磷酸基团 = ATP（三磷酸腺苷）

相关物质关系：
• 腺嘌呤（含氮碱基）+ 核糖 = 腺苷
• 腺苷 + 1个磷酸 = AMP（腺嘌呤核糖核苷酸，RNA基本单位）
• 腺苷 + 2个磷酸 = ADP（二磷酸腺苷）
• 腺苷 + 3个磷酸 = ATP（三磷酸腺苷）

2.2 ATP与ADP的相互转化

$$ATP \underset{酶2}{\stackrel{酶1}{\rightleftharpoons}} ADP + Pi + 能量$$

比较项目 ATP水解 ATP合成
反应式 ATP → ADP + Pi + 能量 ADP + Pi + 能量 → ATP
酶 ATP水解酶 ATP合成酶
能量来源 远离A的高能磷酸键断裂释放 细胞呼吸、光合作用
能量去向 各项生命活动 储存在高能磷酸键中
反应场所 所有需能部位 细胞质基质、线粒体、叶绿体
💡 ATP是直接的能源物质，但不是唯一的直接能源物质：
• GTP（三磷酸鸟苷）：蛋白质合成
• CTP（三磷酸胞苷）：脂质合成
• UTP（三磷酸尿苷）：糖原合成

三、ATP的主要来源——细胞呼吸

3.1 细胞呼吸的概念与类型

细胞呼吸：有机物在细胞内经过一系列的氧化分解，生成二氧化碳或其他产物，释放出能量并生成ATP的过程。

🫁 有氧呼吸

概念：细胞在氧的参与下，通过多种酶的催化作用，把葡萄糖等有机物彻底氧化分解，产生CO₂和H₂O，释放能量，生成大量ATP的过程。

主要场所：线粒体

产物：CO₂ + H₂O

能量：释放大量能量（1mol葡萄糖释放2870kJ，其中1161kJ储存在ATP中）

🦠 无氧呼吸

概念：细胞在无氧条件下，通过酶的催化作用，把葡萄糖等有机物不彻底地氧化分解，释放少量能量的过程。

场所：细胞质基质

产物：酒精+CO₂（植物）或乳酸（动物）

能量：释放少量能量（1mol葡萄糖释放196.65kJ，其中61.08kJ储存在ATP中）

3.2 有氧呼吸的过程（三个阶段）

有氧呼吸总反应式

$$C_6H_{12}O_6 + 6H_2O + 6O_2 \xrightarrow{酶} 6CO_2 + 12H_2O + 能量$$

阶段 场所 反应物 产物 ATP [H]
第一阶段（糖酵解） 细胞质基质 葡萄糖 丙酮酸 + [H] 少量（2个） 产生
第二阶段（柠檬酸循环） 线粒体基质 丙酮酸 + H₂O CO₂ + [H] 少量（2个） 产生
第三阶段（电子传递链） 线粒体内膜 [H] + O₂ H₂O 大量（34个） 消耗
有氧呼吸过程详解：

第一阶段（糖酵解）：
$$C_6H_{12}O_6 \xrightarrow{酶} 2丙酮酸 + 4[H] + 少量能量$$
不需要氧气参与，在细胞质基质中进行

第二阶段（柠檬酸循环）：
$$2丙酮酸 + 6H_2O \xrightarrow{酶} 6CO_2 + 20[H] + 少量能量$$
在线粒体基质中进行，产生CO₂

第三阶段（电子传递链）：
$$24[H] + 6O_2 \xrightarrow{酶} 12H_2O + 大量能量$$
在线粒体内膜上进行，需要氧气参与，产生大量ATP

3.3 无氧呼吸的过程

无氧呼吸总反应式

产生酒精（大多数植物、酵母菌）：

$$C_6H_{12}O_6 \xrightarrow{酶} 2C_2H_5OH + 2CO_2 + 少量能量$$

产生乳酸（动物、乳酸菌、少数植物）：

$$C_6H_{12}O_6 \xrightarrow{酶} 2C_3H_6O_3 + 少量能量$$

无氧呼吸过程：
第一阶段与有氧呼吸完全相同（在细胞质基质中进行）
第二阶段：丙酮酸在不同酶的催化下，分解为酒精和CO₂或乳酸
注意：无氧呼吸只在第一阶段释放少量能量，第二阶段不释放能量

3.4 有氧呼吸与无氧呼吸的比较

比较项目 有氧呼吸 无氧呼吸
条件 需要氧气 不需要氧气
场所 细胞质基质、线粒体 细胞质基质
分解程度 彻底分解 不彻底分解
产物 CO₂和H₂O 乳酸或酒精和CO₂
能量释放 大量（38个ATP） 少量（2个ATP）
相同点 第一阶段相同；都在细胞质基质中进行；都需要酶

四、能量之源——光合作用

4.1 光合作用的探究历程

科学家 实验 结论
普利斯特利 小鼠+绿色植物+蜡烛实验 植物可以更新因燃烧或呼吸而变污浊的空气
英格豪斯 重复普利斯特利实验 植物更新空气需要光，且只有绿叶才能更新空气
萨克斯 半叶遮光实验 光合作用产生淀粉，需要光
恩格尔曼 水绵和好氧细菌实验 光合作用的场所是叶绿体，光合作用释放O₂
鲁宾和卡门 同位素¹⁸O标记实验 光合作用释放的O₂来自H₂O，不是CO₂
卡尔文 ¹⁴C标记CO₂实验 CO₂中的C转化为有机物中的C（卡尔文循环）

4.2 光合作用的过程

光合作用总反应式

$$CO_2 + H_2O \xrightarrow[叶绿体]{光能} (CH_2O) + O_2$$

或

$$6CO_2 + 12H_2O \xrightarrow[叶绿体]{光能} C_6H_{12}O_6 + 6O_2 + 6H_2O$$

☀️ 光反应阶段

场所：叶绿体类囊体薄膜

条件：光、色素、酶

物质变化：

  • 水的光解：$2H_2O \xrightarrow{光} 4[H] + O_2$
  • ATP合成：ADP + Pi + 能量 → ATP

能量变化：光能 → ATP中活跃的化学能

🌙 暗反应阶段（卡尔文循环）

场所：叶绿体基质

条件：酶、[H]、ATP

物质变化：

  • CO₂固定：$CO_2 + C_5 \xrightarrow{酶} 2C_3$
  • C₃还原：$2C_3 \xrightarrow{[H], ATP, 酶} (CH_2O) + C_5$

能量变化：ATP中活跃的化学能 → 有机物中稳定的化学能

4.3 光反应与暗反应的联系

联系：
1. 光反应为暗反应提供[H]（NADPH）和ATP
2. 暗反应为光反应提供ADP、Pi和NADP⁺
3. 没有光反应，暗反应无法进行；没有暗反应，光反应也不能持续进行

4.4 影响光合作用的因素

光照强度

曲线分析：
• A点：光照强度为0，只进行细胞呼吸，释放CO₂
• B点（光补偿点）：光合作用速率 = 细胞呼吸速率，净光合速率为0
• C点（光饱和点）：再增加光照强度，光合速率不再增加
• A点以下：光合速率 < 呼吸速率，植物吸收O₂，释放CO₂
• AB段：光合速率 < 呼吸速率
• B点以后：光合速率 > 呼吸速率，植物吸收CO₂，释放O₂

CO₂浓度

曲线特征：
• 在一定范围内，随CO₂浓度升高，光合速率增强
• 达到CO₂饱和点后，再增加CO₂浓度，光合速率不再增加
• CO₂补偿点：光合速率 = 呼吸速率时的CO₂浓度
• 应用：温室栽培时适当提高CO₂浓度（如施放干冰、使用CO₂发生器）

温度

影响原理：温度通过影响酶的活性影响光合作用
• 最适温度：一般25-30℃
• 低温：酶活性降低，光合速率下降
• 高温：酶变性失活，光合速率急剧下降
• 应用：温室栽培时白天适当升温，夜间适当降温

五、光合作用与细胞呼吸的关系

光合作用与细胞呼吸的比较
比较项目 光合作用 细胞呼吸
场所 叶绿体 细胞质基质、线粒体
条件 光、色素、酶 酶（有氧/无氧）
原料 CO₂、H₂O C₆H₁₂O₆、O₂（有氧）
产物 (CH₂O)、O₂ CO₂、H₂O（有氧）
能量变化 光能 → 化学能（储能） 化学能 → ATP+热能（放能）
实质 合成有机物，储存能量 分解有机物，释放能量
光合作用与细胞呼吸的联系：
1. 光合作用制造的有机物和O₂可供细胞呼吸利用
2. 细胞呼吸产生的CO₂和H₂O可供光合作用利用
3. 光合作用光反应产生的ATP仅供暗反应使用，不用于其他生命活动
4. 细胞呼吸产生的ATP可用于各项生命活动（包括光合作用产物的运输）

六、实验专题

🧪 实验一：比较过氧化氢在不同条件下的分解

实验原理：$2H_2O_2 \xrightarrow{酶} 2H_2O + O_2$

实验设计：

试管 处理 现象 结论
1 常温 气泡很少 常温下过氧化氢分解缓慢
2 90℃水浴加热 气泡较多 加热能促进过氧化氢分解
3 加FeCl₃溶液 气泡较多 无机催化剂能加速反应
4 加肝脏研磨液（含过氧化氢酶） 气泡大量产生，带火星木条复燃 酶具有高效性

🧪 实验二：探究酵母菌细胞呼吸的方式

实验原理：

  • 酵母菌是兼性厌氧菌，在有氧和无氧条件下都能生存
  • CO₂可使澄清石灰水变浑浊，也可使溴麝香草酚蓝水溶液由蓝变绿再变黄
  • 酒精可使酸性重铬酸钾溶液由橙色变成灰绿色

实验装置：

有氧条件：空气→质量分数为10%的NaOH溶液（除去CO₂）→酵母菌培养液→澄清石灰水

无氧条件：酵母菌培养液（封口放置一段时间，消耗完氧气）→澄清石灰水

🧪 实验三：绿叶中色素的提取和分离

实验原理：

  • 提取原理：色素能溶解在有机溶剂无水乙醇中
  • 分离原理：不同色素在层析液中的溶解度不同，随层析液在滤纸上扩散速度不同

实验步骤：

  1. 提取色素：称取5g绿叶，剪碎，加入无水乙醇、SiO₂、CaCO₃，研磨，过滤
  2. 制备滤纸条：剪去两角，画铅笔线
  3. 画滤液细线：细、直、齐，重复2-3次
  4. 分离色素：滤液细线不能触及层析液
  5. 观察记录：从上到下依次为胡萝卜素、叶黄素、叶绿素a、叶绿素b

结果：

色素带 颜色 色素名称 含量 溶解度
1（最上） 橙黄色 胡萝卜素 最少 最高
2 黄色 叶黄素 较少 较高
3 蓝绿色 叶绿素a 最多 较低
4（最下） 黄绿色 叶绿素b 较多 最低

📝 高考真题演练

【2023年全国甲卷】下列关于光合作用和细胞呼吸的叙述，正确的是（ ）

A. 光合作用产生的ATP可用于各项生命活动

B. 有氧呼吸产生的[H]在线粒体内膜上与氧结合生成水

C. 无氧呼吸只在第一阶段释放少量能量，第二阶段不释放能量

D. 光合作用暗反应阶段C₃的还原需要光反应提供的[H]和ATP

答案：B、C、D（多选题）

解析：

A错误：光合作用光反应产生的ATP只能用于暗反应，不能用于其他生命活动。

B正确：有氧呼吸第三阶段，[H]与O₂结合生成水，发生在线粒体内膜上。

C正确：无氧呼吸只在第一阶段释放少量能量，第二阶段不释放能量。

D正确：暗反应中C₃的还原需要光反应提供的[H]（NADPH）和ATP。
