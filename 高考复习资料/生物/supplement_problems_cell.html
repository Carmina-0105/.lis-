🧬 细胞模块专项练习

涵盖模块1-5：细胞的分子组成、基本结构、物质运输、能量供应、生命历程

一、细胞的分子组成（模块1）

基础 例题1【2023年全国甲卷】
下列关于细胞中化合物的叙述，正确的是（ ）

A. 糖类都是能源物质

B. 脂质都只含有C、H、O三种元素

C. 蛋白质是生命活动的主要承担者

D. 核酸都只分布在细胞核中

答案：C
解析：
A错误：并非所有糖类都是能源物质。例如，纤维素是结构物质，构成植物细胞壁；五碳糖（核糖、脱氧核糖）是核酸的组成成分。
B错误：脂质中的磷脂含有C、H、O、N、P五种元素。
C正确：蛋白质功能多样，包括催化（酶）、运输（血红蛋白）、免疫（抗体）、调节（激素）、结构（胶原蛋白）等，是生命活动的主要承担者。
D错误：核酸包括DNA和RNA。DNA主要分布在细胞核，RNA主要分布在细胞质。
中等 重点 例题2【蛋白质计算】
某蛋白质由n条肽链组成，氨基酸的平均分子量为a，控制该蛋白质合成的基因含b个碱基对，则该蛋白质的分子量约为（ ）

A. $\\frac{2}{3}ab - 6b + 18n$

B. $\\frac{2}{3}ab - 18(\\frac{b}{3} - n)$

C. $\\frac{2}{3}ab - 18(b - n)$

D. $\\frac{1}{3}ab - 18(\\frac{b}{3} - n)$

答案：B
解析：
步骤1：计算氨基酸数目
基因中碱基对数 : mRNA碱基数 : 氨基酸数 = 6 : 3 : 1
氨基酸数 = $\\frac{b}{3}$

步骤2：计算脱去水分子数
脱去水分子数 = 氨基酸数 - 肽链数 = $\\frac{b}{3} - n$

步骤3：计算蛋白质分子量
蛋白质分子量 = 氨基酸总分子量 - 脱去水分子总分子量
= $a \\times \\frac{b}{3} - 18 \\times (\\frac{b}{3} - n)$
= $\\frac{2}{3}ab - 18(\\frac{b}{3} - n)$

关键公式：
• 蛋白质分子量 = 氨基酸平均分子量 × 氨基酸数 - 18 × 脱去水分子数
• 脱去水分子数 = 氨基酸数 - 肽链数 = 肽键数
⚠️ 易错警示：蛋白质计算常见错误

1. 混淆碱基与碱基对：基因中的"b个碱基对"等于"2b个碱基"
2. 忘记减去水分子：蛋白质分子量 = 氨基酸总分子量 - 脱去水分子总分子量
3. 环肽计算：环肽的肽键数 = 氨基酸数（不是氨基酸数-1）
4. 二硫键：每形成一个二硫键(-S-S-)脱去2个H，分子量减少2
较难 例题3【核酸综合】
某双链DNA分子含有1000个碱基对，其中一条链上A:T:G:C = 1:2:3:4，则该DNA分子（ ）

A. 含有1000个脱氧核糖

B. 含有200个腺嘌呤

C. 四种含氮碱基的比例为A:T:G:C = 3:3:7:7

D. 该DNA复制3次，需要游离的鸟嘌呤脱氧核苷酸2800个

答案：C
解析：
步骤1：计算一条链上各碱基数
一条链上碱基总数 = 1000
A = 1000 × 1/10 = 100
T = 1000 × 2/10 = 200
G = 1000 × 3/10 = 300
C = 1000 × 4/10 = 400

步骤2：计算整个DNA分子中各碱基数
根据碱基互补配对原则：
A总 = A₁ + T₁ = 100 + 200 = 300
T总 = T₁ + A₁ = 200 + 100 = 300
G总 = G₁ + C₁ = 300 + 400 = 700
C总 = C₁ + G₁ = 400 + 300 = 700

步骤3：验证各选项
A错误：含有2000个脱氧核糖（每个脱氧核苷酸含一个脱氧核糖）
B错误：含有300个腺嘌呤
C正确：A:T:G:C = 300:300:700:700 = 3:3:7:7
D错误：复制3次需要游离的鸟嘌呤脱氧核苷酸 = 700 × (2³-1) = 700 × 7 = 4900个

二、细胞的基本结构（模块2）

基础 例题4【细胞器功能】
下列关于细胞器的叙述，正确的是（ ）

A. 线粒体和叶绿体都含有DNA和核糖体，都能合成自身所需的全部蛋白质

B. 内质网是蛋白质合成、加工和运输的场所

C. 高尔基体在动植物细胞中的功能相同

D. 溶酶体可以消化细胞从外界吞入的颗粒和细胞自身产生的碎渣

答案：D
解析：
A错误：线粒体和叶绿体是半自主性细胞器，能合成部分蛋白质，但大部分蛋白质仍需核基因编码，在细胞质中合成后运入。
B错误：蛋白质合成的场所是核糖体，不是内质网。内质网是蛋白质加工和运输的场所。
C错误：高尔基体在动物细胞中与分泌物的形成有关，在植物细胞中与细胞壁的形成有关，功能不同。
D正确：溶酶体含有多种水解酶，可以消化细胞从外界吞入的颗粒（如细菌）和细胞自身产生的碎渣（如衰老的细胞器）。
中等 重点 例题5【分泌蛋白合成】
用³H标记的亮氨酸注射到胰腺腺泡细胞中，追踪分泌蛋白的合成和分泌过程。下列叙述正确的是（ ）

A. 放射性首先出现在高尔基体中

B. 内质网对该蛋白进行加工和包装

C. 分泌蛋白通过主动运输分泌到细胞外

D. 该过程体现了生物膜在结构和功能上的联系

答案：D
解析：
分泌蛋白的合成和分泌过程：
核糖体（合成肽链）→ 内质网（加工、折叠）→ 囊泡 → 高尔基体（进一步加工、分类、包装）→ 囊泡 → 细胞膜 → 胞吐 → 细胞外

A错误：放射性首先出现在核糖体中，因为氨基酸首先在核糖体上合成肽链。
B错误：内质网对蛋白质进行初步加工（如折叠、糖基化），高尔基体进行进一步加工和包装。
C错误：分泌蛋白通过胞吐（外排）分泌到细胞外，不是主动运输。胞吐需要消耗能量，但不需要载体蛋白。
D正确：该过程涉及核糖体、内质网、高尔基体、细胞膜等多种生物膜结构，体现了生物膜在结构上的联系（通过囊泡）和功能上的分工合作。

三、细胞的物质输入与输出（模块3）

中等 例题6【物质跨膜运输】
下列关于物质跨膜运输的叙述，正确的是（ ）

A. 葡萄糖进入红细胞需要载体蛋白协助，并消耗能量

B. 植物细胞发生质壁分离复原时，水分子只进入细胞

C. 主动运输使膜内外物质浓度趋于一致，维持了细胞的正常代谢

D. 抑制细胞膜上载体蛋白的活性，会影响细胞对离子的吸收

答案：D
解析：
A错误：葡萄糖进入红细胞是协助扩散，需要载体蛋白但不消耗能量。
B错误：质壁分离复原时，水分子既进入细胞也出细胞，只是进入细胞的水分子多于出细胞的水分子。
C错误：主动运输是逆浓度梯度运输，使膜内外物质浓度差增大，而不是趋于一致。
D正确：细胞对离子的吸收主要是主动运输，需要载体蛋白和能量。抑制载体蛋白活性会影响离子吸收。
📝 物质跨膜运输方式总结

运输方式 方向 载体 能量 实例
自由扩散 高→低 不需要 不消耗 O₂、CO₂、水、甘油、乙醇
协助扩散 高→低 需要 不消耗 葡萄糖进入红细胞
主动运输 低→高 需要 消耗 离子、氨基酸、葡萄糖进入小肠上皮细胞
胞吞/胞吐 - 不需要 消耗 蛋白质、神经递质、吞噬作用

四、细胞的能量供应与利用（模块4）

较难 重点 例题7【光合作用和呼吸作用综合】
将一株生长正常的绿色植物置于密闭的玻璃容器内，在适宜条件下光照培养。随培养时间的延长，玻璃容器内CO₂浓度可能出现的变化趋势是（ ）

A. 一直降低，直至为零

B. 一直保持稳定，不变化

C. 降低至一定水平时保持相对稳定

D. 升高至一定水平时保持相对稳定

答案：C
解析：
分析过程：
1. 初期：光照下植物进行光合作用，吸收CO₂，释放O₂。由于光合作用强度大于呼吸作用强度，容器内CO₂浓度降低。

2. 中期：随着CO₂浓度降低，光合作用强度逐渐减弱，而呼吸作用强度基本不变（温度等条件适宜且稳定）。

3. 平衡点：当光合作用强度等于呼吸作用强度时，CO₂的吸收量等于释放量，容器内CO₂浓度达到平衡，保持相对稳定。

4. 不可能为零：如果CO₂浓度为零，光合作用将无法进行，植物会死亡。实际上，当CO₂浓度降低到一定水平（CO₂补偿点）时，光合作用强度等于呼吸作用强度，CO₂浓度不再下降。

答案C正确。
📝 光合作用与呼吸作用的关系

1. 净光合速率 = 总光合速率 - 呼吸速率

2. CO₂补偿点：光合作用强度等于呼吸作用强度时的CO₂浓度。此时净光合速率为零，植物不吸收也不释放CO₂。

3. 光补偿点：光合作用强度等于呼吸作用强度时的光照强度。

4. 密闭容器中气体变化规律：
• 光照下：CO₂浓度先降低后稳定（在CO₂补偿点）
• 黑暗中：CO₂浓度持续升高，O₂浓度持续降低

五、细胞的生命历程（模块5）

中等 例题8【细胞分裂综合】
下列关于细胞分裂的叙述，正确的是（ ）

A. 有丝分裂和减数分裂过程中都有同源染色体的联会

B. 有丝分裂后期和减数第二次分裂后期都有着丝点的分裂

C. 有丝分裂中期和减数第一次分裂中期染色体数目相同

D. 有丝分裂和减数分裂过程中DNA复制次数相同

答案：B
解析：
A错误：同源染色体的联会只发生在减数第一次分裂前期，有丝分裂过程中没有联会。

B正确：有丝分裂后期和减数第二次分裂后期都发生着丝点分裂，姐妹染色单体分开，染色体数目暂时加倍。

C错误：假设正常体细胞染色体数为2N。
• 有丝分裂中期：染色体数为2N
• 减数第一次分裂中期：染色体数为2N
虽然染色体数目相同，但染色单体数目不同（有丝分裂中期有4N条染色单体，减数第一次分裂中期也有4N条染色单体）。

D错误：有丝分裂过程中DNA复制1次；减数分裂过程中DNA也只复制1次（在减数第一次分裂前的间期），但细胞分裂2次。

注意：本题C选项容易误判。虽然染色体数目相同，但题目问的是"正确的是"，B选项更加准确无误。
⚠️ 易错警示：有丝分裂 vs 减数分裂

比较项目 有丝分裂 减数分裂
发生部位 体细胞、原始生殖细胞 生殖细胞
分裂次数 1次 2次
DNA复制 1次 1次
同源染色体联会 无 有（减Ⅰ前期）
同源染色体分离 无 有（减Ⅰ后期）
子细胞染色体数 与母细胞相同（2N） 减半（N）
