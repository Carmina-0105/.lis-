一、基因突变

1.1 基因突变的概念

基因突变：DNA分子中发生碱基对的替换、增添和缺失，而引起的基因结构的改变。

基因突变的时期：主要发生在DNA复制时（有丝分裂间期或减数第一次分裂前的间期）

1.2 基因突变的类型

突变类型 概念 实例
替换 一个碱基对被另一个碱基对替换 镰刀型细胞贫血症（A→T）
增添 增加一个或多个碱基对 某些遗传病
缺失 缺失一个或多个碱基对 囊性纤维病

1.3 基因突变的特点

1. 普遍性：在生物界普遍存在
2. 随机性：可以发生在生物个体发育的任何时期、任何细胞、任何DNA分子、任何部位
3. 低频性：自然状态下，突变频率很低（10⁻⁵~10⁻⁸）
4. 不定向性：一个基因可以向不同方向突变，产生一个以上的等位基因
5. 多害少利性：大多数突变是有害的，少数是有利的

二、基因重组

2.1 基因重组的概念

基因重组：生物体进行有性生殖的过程中，控制不同性状的基因的重新组合。

2.2 基因重组的类型

基因重组的两种类型

类型一：自由组合型
时期：减数第一次分裂后期
机制：非同源染色体自由组合，导致非同源染色体上的非等位基因自由组合

类型二：交叉互换型
时期：减数第一次分裂前期（四分体时期）
机制：同源染色体的非姐妹染色单体之间发生交叉互换，导致等位基因交换

✅ 基因重组的意义：
基因重组是生物变异的来源之一，对生物的进化具有重要意义。通过基因重组，可以产生多样化的基因组合的子代，其中可能有一些子代会含有适应某种变化的、生存所必需的基因组合。

三、染色体变异

3.1 染色体结构变异

变异类型 概念 图示 实例
缺失 染色体某一片段缺失 ab·cdef → ab·cef 猫叫综合征（5号染色体缺失）
重复 染色体增加某一片段 ab·cdef → ab·cdcdef 果蝇棒眼
倒位 染色体某一片段位置颠倒180° ab·cdef → ab·edcf 某些遗传病
易位 染色体的某一片段移接到另一条非同源染色体上 ab·cdef + wx·yz → ab·yz + wx·cdef 慢性粒细胞白血病

3.2 染色体数目变异

染色体组：细胞中的一组非同源染色体，在形态和功能上各不相同，携带着控制生物生长发育的全部遗传信息。

二倍体：由受精卵发育而来，体细胞中含有两个染色体组的个体。

多倍体：由受精卵发育而来，体细胞中含有三个或三个以上染色体组的个体。

单倍体：体细胞中含有本物种配子染色体数目的个体。
⚠️ 关于单倍体的理解：
• 单倍体不一定只含一个染色体组（如四倍体的单倍体含两个染色体组）
• 单倍体高度不育（但雄蜂是例外，可进行假减数分裂）
• 单倍体育种可以明显缩短育种年限

四、人类遗传病

4.1 人类遗传病的类型

类型 概念 特点 实例
单基因遗传病 受一对等位基因控制的遗传病 符合孟德尔遗传定律 白化病、红绿色盲、血友病
多基因遗传病 受两对以上等位基因控制的遗传病 群体中发病率高，易受环境影响 原发性高血压、青少年型糖尿病
染色体异常遗传病 由染色体异常引起的遗传病 往往造成严重后果 21三体综合征、猫叫综合征

五、育种方法

5.1 常见育种方法比较

育种方法 原理 方法 优点 缺点 实例
杂交育种 基因重组 杂交→自交→选优→自交 集优，操作简单 育种年限长 杂交水稻
诱变育种 基因突变 物理/化学因素处理 提高突变率，缩短育种周期 有利变异少，需大量处理材料 青霉素高产菌株
单倍体育种 染色体变异 花药离体培养→秋水仙素加倍 明显缩短育种年限 技术复杂 快速纯合小麦
多倍体育种 染色体变异 秋水仙素处理萌发的种子或幼苗 茎秆粗壮，果实种子大 发育延迟，结实率低 三倍体无子西瓜
基因工程育种 基因重组 提取→拼接→导入→检测 定向改造，克服远缘杂交障碍 技术复杂，安全性争议 抗虫棉

📝 高考真题演练

【2023年全国甲卷】下列关于生物变异的叙述，正确的是（ ）

A. 基因突变一定能改变生物的表现型

B. 基因重组可以产生新的基因

C. 染色体结构变异会导致基因数目或排列顺序改变

D. 单倍体只含有一个染色体组

答案：C

解析：

A错误：由于密码子的简并性，基因突变不一定改变表现型；隐性突变在杂合状态下也不表现。

B错误：基因重组只能产生新的基因型，不能产生新的基因；只有基因突变能产生新基因。

C正确：染色体结构变异（缺失、重复、倒位、易位）会导致基因数目或排列顺序改变。

D错误：单倍体是指体细胞中含有本物种配子染色体数目的个体，不一定只含一个染色体组。
