一、组成细胞的元素

1.1 元素的分类

核心概念：细胞中的元素按含量分为大量元素和微量元素，按作用分为最基本元素、基本元素和主要元素。
分类依据 类别 包含元素 含量特点
按含量分 大量元素 C、H、O、N、P、S、K、Ca、Mg 占生物体总重量万分之一以上
微量元素 Fe、Mn、Zn、Cu、B、Mo 含量很少，但必不可少
按作用分 最基本元素 C 碳链是生物大分子的骨架
基本元素 C、H、O、N 占细胞鲜重90%以上
主要元素 C、H、O、N、P、S 占细胞鲜重97%以上
📊 元素含量记忆口诀

鲜重时：O > C > H > N（氧碳氢氮，养碳氢蛋）

干重时：C > O > N > H（碳氧氮氢，碳优先）

原因：鲜重中水占85%-90%，所以O最多；干重中蛋白质等有机物占主要，所以C最多

1.2 元素的统一性与差异性

🔬 统一性（生物界与非生物界）

  • 组成生物体的化学元素在无机自然界都能找到
  • 没有一种元素是生物界特有的
  • 说明：生物起源于非生物环境

🎯 差异性（生物界与非生物界）

  • 生物体中元素含量与无机自然界差异很大
  • 如：C、H、O、N在生物体中含量高
  • 说明：生物有选择地从环境中吸收元素
⚠️ 常见误区：
1. 微量元素含量少≠作用小（如Fe是血红蛋白的组成成分，缺Fe会导致贫血）
2. 大量元素作用大≠可以缺少微量元素
3. 最基本元素是C，但占细胞鲜重最多的是O（因为水的存在）

二、组成细胞的化合物

2.1 化合物的分类与含量

类别 化合物 含量（鲜重） 主要功能
无机物 水 85%-90% 良好溶剂、反应介质、物质运输
无机盐 1%-1.5% 组成复杂化合物、维持生命活动
有机物 蛋白质 7%-10% 生命活动的主要承担者
脂质 1%-2% 储能、膜结构、调节
糖类 1%-1.5% 主要能源物质
核酸 1%-1.5% 遗传信息的携带者

2.2 水的存在形式与功能

💧 自由水

定义：以游离形式存在，可自由流动

含量：约占细胞内全部水分的95.5%

功能：

  • 细胞内的良好溶剂
  • 参与许多生物化学反应
  • 为细胞提供液体环境
  • 运输营养物质和代谢废物

🔒 结合水

定义：与细胞内的其他物质相结合

含量：约占细胞内全部水分的4.5%

功能：

  • 细胞结构的重要组成成分
  • 维持细胞的形态和结构
  • 增强细胞的抗逆性
自由水/结合水比值与代谢的关系：
• 比值高 → 代谢旺盛，生长迅速，抗逆性弱
• 比值低 → 代谢缓慢，休眠状态，抗逆性强
• 种子萌发时：结合水→自由水，代谢增强
• 种子储存时：自由水→结合水，降低代谢

2.3 无机盐的存在形式与功能

存在形式：大多数以离子形式存在（如Na⁺、K⁺、Ca²⁺、Mg²⁺、Cl⁻、HCO₃⁻、HPO₄²⁻等）
少数以化合物形式存在：如骨骼中的CaCO₃
无机盐 功能 缺乏症
Fe²⁺ 血红蛋白的组成成分 缺铁性贫血
Mg²⁺ 叶绿素的组成成分 叶片发黄，影响光合作用
I⁻ 甲状腺激素的组成成分 地方性甲状腺肿（大脖子病）
Ca²⁺ 骨骼、牙齿的组成成分；调节神经肌肉兴奋性 佝偻病（幼年）、骨质疏松（老年）；血钙过低抽搐
Na⁺、K⁺ 维持细胞外液和细胞内液的渗透压 渗透压失衡，细胞形态改变
HCO₃⁻、HPO₄²⁻ 维持血浆的酸碱平衡 酸碱失衡
B 促进花粉萌发和花粉管伸长 花而不实（只开花不结果）
Zn 多种酶的组成成分 生长发育不良，味觉障碍

三、蛋白质——生命活动的主要承担者

3.1 氨基酸：蛋白质的基本单位

氨基酸的结构通式

$$\underset{R}{\underset{|}{H_2N-C-COOH}}$$ $$\quad\quad\quad|$$ $$\quad\quad\quad H$$

记忆口诀："一个中心，两个基本点"

• 一个中心：中心碳原子（C）

• 两个基本点：氨基（-NH₂）和羧基（-COOH）

• 一个氢原子（-H）和一个R基（决定氨基酸种类）

氨基酸的结构特点：
1. 至少含有一个氨基（-NH₂）和一个羧基（-COOH）
2. 都有一个氨基和一个羧基连接在同一个碳原子上
3. 各种氨基酸的区别在于R基的不同

氨基酸的分类（根据R基性质）

分类 特点 举例
必需氨基酸 人体不能合成，必须从食物中获取（8种） 甲硫氨酸、缬氨酸、赖氨酸、异亮氨酸、苯丙氨酸、亮氨酸、色氨酸、苏氨酸
非必需氨基酸 人体可以合成（12种） 甘氨酸、丙氨酸、丝氨酸等
💡 记忆口诀：必需氨基酸（8种）
"甲携来一本亮色书"
甲（甲硫氨酸）携（缬氨酸）来（赖氨酸）一（异亮氨酸）本（苯丙氨酸）亮（亮氨酸）色（色氨酸）书（苏氨酸）

3.2 蛋白质的结构层次

🏗️ 蛋白质的结构层次（从简单到复杂）

氨基酸（基本单位）
↓ 脱水缩合（肽键连接）
多肽（肽链）
↓ 盘曲、折叠
蛋白质（具有一定空间结构）

脱水缩合反应详解

脱水缩合过程

一个氨基酸的羧基（-COOH）与另一个氨基酸的氨基（-NH₂）相连接

同时脱去一分子水，形成肽键（-CO-NH-）

$$H_2N-CH(R_1)-COOH + H_2N-CH(R_2)-COOH$$ $$\downarrow$$ $$H_2N-CH(R_1)-CO-NH-CH(R_2)-COOH + H_2O$$

蛋白质相关计算公式：
必背公式

1. 肽键数 = 脱去水分子数 = 氨基酸数 - 肽链数
$$肽键数 = n - m$$ （n为氨基酸数，m为肽链数）

2. 蛋白质相对分子质量计算：
$$蛋白质分子量 = 氨基酸平均分子量 \times 氨基酸数 - 18 \times 脱去水分子数$$ （氨基酸平均分子量按110或128计算，具体看题目要求）

3. 游离氨基/羧基数目：
• 游离氨基数 = 肽链数 + R基中的氨基数
• 游离羧基数 = 肽链数 + R基中的羧基数

4. N原子数 = 肽键数 + 肽链数 + R基中的N原子数
5. O原子数 = 肽键数 + 2×肽链数 + R基中的O原子数
📝 例题1：蛋白质结构计算

题目：某蛋白质由n条肽链组成，共有m个氨基酸，氨基酸的平均分子量为a，求：

(1) 该蛋白质中的肽键数

(2) 形成该蛋白质时脱去的水分子数

(3) 该蛋白质的相对分子质量

(4) 该蛋白质至少含有的游离氨基数和羧基数

解析：

(1) 肽键数 = 氨基酸数 - 肽链数 = m - n

(2) 脱去水分子数 = 肽键数 = m - n

(3) 蛋白质分子量 = m×a - 18×(m-n) = ma - 18(m-n)

(4) 至少含有的游离氨基数 = 肽链数 = n；至少含有的游离羧基数 = n

3.3 蛋白质的结构多样性

蛋白质结构多样性的原因（四个层次）：
1. 氨基酸水平：氨基酸的种类不同
2. 氨基酸水平：氨基酸的数目成百上千
3. 氨基酸水平：氨基酸的排列顺序千变万化
4. 肽链水平：肽链的盘曲、折叠方式及其形成的空间结构千差万别

3.4 蛋白质的功能多样性

功能 举例 说明
结构蛋白 肌肉蛋白、胶原蛋白、头发中的角蛋白 构成细胞和生物体的结构
催化功能 绝大多数酶 降低反应活化能，加速化学反应
运输功能 血红蛋白（运输O₂）、载体蛋白 运输物质
调节功能 胰岛素、生长激素等蛋白质类激素 调节生命活动
免疫功能 抗体（免疫球蛋白） 抵御病原体
信息传递 细胞膜上的受体蛋白 识别信号分子
⚠️ 易错提醒：
1. 并非所有酶都是蛋白质（少数酶是RNA）
2. 并非所有激素都是蛋白质（如性激素是脂质，甲状腺激素是氨基酸衍生物）
3. 抗体一定是蛋白质，且都是球蛋白
4. 载体蛋白和通道蛋白都有特异性，但作用机制不同

3.5 蛋白质的变性

蛋白质变性的概念：蛋白质分子的空间结构被破坏，导致其理化性质改变和生物活性丧失的过程。

引起变性的因素：
• 物理因素：高温、紫外线、超声波、剧烈振荡等
• 化学因素：强酸、强碱、重金属盐、有机溶剂（如酒精）、尿素等

变性的特点：
• 空间结构破坏（不可逆，但肽键未断裂）
• 生物活性丧失
• 易被蛋白酶水解
• 溶解度降低，易沉淀
💡 生活中的蛋白质变性：
• 煮熟的鸡蛋：高温使蛋白质变性，更易消化
• 酒精消毒：使细菌蛋白质变性
• 豆腐制作：盐卤使蛋白质变性凝固
• 疫苗冷藏：防止蛋白质变性失活

四、核酸——遗传信息的携带者

4.1 核酸的种类与分布

比较项目 DNA（脱氧核糖核酸） RNA（核糖核酸）
全称 脱氧核糖核酸 核糖核酸
基本单位 脱氧核苷酸 核糖核苷酸
五碳糖 脱氧核糖 核糖
含氮碱基 A、T、G、C A、U、G、C
结构 双螺旋结构 通常为单链
主要分布 细胞核（主要）、线粒体、叶绿体 细胞质（主要）、细胞核
功能 储存遗传信息 传递遗传信息、参与蛋白质合成

4.2 核苷酸的结构

核苷酸的组成

核苷酸 = 磷酸 + 五碳糖 + 含氮碱基

• 磷酸：连接在5号碳原子上

• 五碳糖：核糖（RNA）或脱氧核糖（DNA）

• 含氮碱基：连接在1号碳原子上

DNA与RNA的核苷酸种类

🧬 DNA的4种脱氧核苷酸

  • 腺嘌呤脱氧核苷酸（dAMP）
  • 鸟嘌呤脱氧核苷酸（dGMP）
  • 胞嘧啶脱氧核苷酸（dCMP）
  • 胸腺嘧啶脱氧核苷酸（dTMP）

📝 RNA的4种核糖核苷酸

  • 腺嘌呤核糖核苷酸（AMP）
  • 鸟嘌呤核糖核苷酸（GMP）
  • 胞嘧啶核糖核苷酸（CMP）
  • 尿嘧啶核糖核苷酸（UMP）
核酸中的碱基互补配对原则：
• DNA中：A = T，G = C（所以 A + G = T + C，即嘌呤数 = 嘧啶数）
• RNA中：A与U配对，G与C配对
• 双链DNA中：(A + T) / (G + C) 的比值因物种而异，可用于物种鉴定

4.3 DNA的结构特点

DNA双螺旋结构的主要特点（沃森和克里克，1953年）：
1. DNA分子是由两条反向平行的脱氧核苷酸长链盘旋而成的双螺旋结构
2. 脱氧核糖和磷酸交替连接，排列在外侧，构成基本骨架
3. 碱基排列在内侧，通过氢键连接成碱基对
4. 碱基互补配对原则：A与T配对（2个氢键），G与C配对（3个氢键）
📊 DNA结构相关计算

1. 碱基数量关系（双链DNA）：

$$A = T, \quad G = C$$

$$A + G = T + C = A + C = T + G = 50\%$$

$$\frac{A + T}{G + C} = \frac{A_1 + T_1}{G_1 + C_1} = \frac{A_2 + T_2}{G_2 + C_2}$$

2. DNA复制相关计算：

复制n次后：DNA分子数 = $2^n$

含亲代链的DNA分子数 = 2（始终是2个）

需要游离的脱氧核苷酸数 = $a \times (2^n - 1)$（a为DNA分子中该脱氧核苷酸数）

五、糖类和脂质

5.1 糖类的种类与功能

分类 种类 分布 功能
单糖 葡萄糖 动植物细胞 细胞的主要能源物质（"生命的燃料"）
果糖 植物细胞 提供能量
半乳糖 动物细胞 提供能量
二糖 蔗糖 植物细胞（甘蔗、甜菜） 储能，运输形式
麦芽糖 植物细胞（发芽的小麦） 淀粉水解产物
乳糖 动物细胞（乳汁） 储能
多糖 淀粉 植物细胞 植物细胞的储能物质
纤维素 植物细胞（细胞壁） 结构物质，支持保护
糖原 动物细胞（肝脏、肌肉） 动物细胞的储能物质
糖类的重要知识点：
1. 葡萄糖是细胞的主要能源物质，但主要的储能物质是脂肪
2. 淀粉是植物细胞储能物质，糖原是动物细胞储能物质
3. 纤维素是植物细胞壁的主要成分，人体不能消化（膳食纤维）
4. 几丁质（壳多糖）是真菌细胞壁和昆虫外骨骼的成分

5.2 脂质的种类与功能

类别 亚类 功能 举例
脂肪 甘油三酯 良好的储能物质；绝热体，保温；缓冲减压，保护内脏 植物油、动物脂肪
相同质量下，脂肪氧化分解释放能量是糖类的2倍多
磷脂 磷脂分子 构成细胞膜和多种细胞器膜的重要成分 卵磷脂、脑磷脂
磷脂双分子层是生物膜的基本支架
固醇 胆固醇 构成动物细胞膜的重要成分；参与血液中脂质的运输 动物细胞膜
性激素 促进人和动物生殖器官的发育以及生殖细胞的形成 雌激素、雄激素
维生素D 促进人和动物肠道对钙和磷的吸收 鱼肝油、蛋黄
⚠️ 糖类与脂质的比较误区：
1. 并非所有糖类都是能源物质（如纤维素是结构物质）
2. 并非所有细胞都含有胆固醇（植物细胞膜一般不含胆固醇）
3. 脂肪是主要储能物质，但主要的能源物质是糖类（葡萄糖）
4. 磷脂是所有细胞都含有的脂质（因为有细胞膜）

六、检测生物组织中的有机物（实验）

🧪 实验一：检测生物组织中的还原糖

实验原理：还原糖（如葡萄糖、果糖、麦芽糖）中的醛基（-CHO）在加热条件下能将斐林试剂中的Cu²⁺还原为Cu⁺，生成砖红色的Cu₂O沉淀。

试剂配制：

  • 斐林试剂甲液：0.1g/mL NaOH溶液
  • 斐林试剂乙液：0.05g/mL CuSO₄溶液
  • 使用时甲液和乙液等量混合均匀

实验步骤：

  1. 向试管内注入2mL待测组织样液（如苹果汁、梨汁）
  2. 向试管内注入1mL斐林试剂（甲液和乙液等量混合均匀后再注入）
  3. 将试管放入盛有50-65℃温水的大烧杯中加热约2分钟
  4. 观察试管中出现的颜色变化

实验结果：蓝色 → 棕色 → 砖红色沉淀

注意事项：
• 选材：选择还原糖含量高、颜色浅的组织（苹果、梨），避免西瓜汁（红色干扰）
• 斐林试剂必须现配现用，甲液乙液先混合再加入
• 必须水浴加热，不能直接加热
• 蔗糖、淀粉是非还原糖，不能发生此反应

🧪 实验二：检测生物组织中的脂肪

实验原理：脂肪可以被苏丹Ⅲ染液染成橘黄色（或被苏丹Ⅳ染液染成红色）。

方法一：花生子叶切片法（显微镜观察）

  1. 向待测组织样液（如花生子叶）滴加2-3滴苏丹Ⅲ染液，染色3分钟
  2. 用吸水纸吸去染液，滴加1-2滴50%酒精溶液，洗去浮色
  3. 用吸水纸吸去酒精，滴加1滴蒸馏水，盖上盖玻片
  4. 在低倍镜下找到花生子叶最薄处，移至视野中央
  5. 换高倍镜观察，脂肪颗粒被染成橘黄色

方法二：组织样液法（直接观察）

  1. 向待测组织样液（如花生油）滴加3滴苏丹Ⅲ染液
  2. 观察样液颜色变化，溶液变成橘黄色
注意事项：
• 切片要薄而均匀，便于观察
• 染色后必须用50%酒精洗去浮色，否则影响观察
• 苏丹Ⅲ染成橘黄色，苏丹Ⅳ染成红色，不要混淆

🧪 实验三：检测生物组织中的蛋白质

实验原理：蛋白质中的肽键（-CO-NH-）在碱性条件下能与Cu²⁺结合，形成紫色络合物。

试剂配制：

  • 双缩脲试剂A液：0.1g/mL NaOH溶液
  • 双缩脲试剂B液：0.01g/mL CuSO₄溶液
  • 先加A液创造碱性环境，再加B液

实验步骤：

  1. 向试管内注入2mL待测组织样液（如豆浆、蛋清稀释液）
  2. 向试管内注入双缩脲试剂A液1mL，摇匀
  3. 向试管内注入双缩脲试剂B液4滴，摇匀
  4. 观察试管中出现的颜色变化

实验结果：溶液变成紫色

注意事项：
• 选材：蛋清必须稀释（1:10），否则反应后粘附在试管壁上不易清洗
• 双缩脲试剂要先加A液（NaOH），后加B液（CuSO₄），顺序不能颠倒
• B液不能加太多（4滴即可），否则蓝色会掩盖紫色
• 不需要加热，常温下即可反应

🧪 实验四：检测生物组织中的淀粉

实验原理：淀粉遇碘液变蓝色。

实验步骤：

  1. 向试管内注入2mL待测组织样液（如马铃薯匀浆）
  2. 向试管内滴加2滴碘液
  3. 观察试管中出现的颜色变化

实验结果：溶液变成蓝色

注意事项：
• 碘液浓度不宜过高，2滴即可
• 反应非常灵敏，微量淀粉即可显色
• 颜色为蓝色，不要描述为"蓝黑色"或"紫蓝色"

📝 高考真题演练

【2023年全国甲卷】下列关于细胞中化合物的叙述，正确的是（ ）

A. 胆固醇是构成植物细胞膜的重要成分

B. 脂肪是细胞内良好的储能物质，其元素组成与糖类不同

C. 蛋白质变性后，其特定功能丧失，但肽键并未断裂

D. 无机盐在细胞中都以离子形式存在

答案：C

解析：

A错误：胆固醇是构成动物细胞膜的重要成分，植物细胞膜一般不含胆固醇。

B错误：脂肪和糖类的元素组成相同，都是C、H、O，只是比例不同（脂肪中H比例高，O比例低）。

C正确：蛋白质变性是空间结构被破坏，肽键并未断裂，特定功能丧失。

D错误：无机盐大多数以离子形式存在，少数以化合物形式存在（如骨骼中的CaCO₃）。
