一、拉马克的进化学说

1.1 拉马克学说的主要内容

拉马克（法国，1744-1829）：历史上第一个提出比较完整的进化学说的科学家。

主要观点：
1. 用进废退：器官用得越多越发达，不用就会退化
2. 获得性遗传：后天获得的性状可以遗传给后代

意义：否定了神创论和物种不变论，奠定了科学生物进化论的基础。

二、达尔文的自然选择学说

2.1 自然选择学说的主要内容

达尔文（英国，1809-1882）：在《物种起源》中提出了以自然选择学说为核心的生物进化理论。

自然选择学说的主要内容：
1. 过度繁殖：生物具有很强的繁殖能力，产生大量后代
2. 生存斗争：生物与无机环境、种内、种间的斗争
3. 遗传变异：生物普遍存在变异，且许多变异可以遗传
4. 适者生存：具有有利变异的个体容易生存并繁殖后代
自然选择学说的解释模型

过度繁殖 → 生存斗争 → 遗传变异 → 适者生存
                                      ↓
                                    生物进化

2.2 达尔文自然选择学说的评价

✅ 贡献

  • 科学地解释了生物进化的原因
  • 解释了生物的多样性和适应性
  • 否定了神创论和物种不变论
  • 使生物学第一次摆脱了神学的束缚

⚠️ 局限性

  • 对遗传和变异的本质不能做出科学解释
  • 局限于个体水平
  • 强调物种形成是渐变的结果，不能解释物种大爆发

三、现代生物进化理论

3.1 种群是生物进化的基本单位

种群：生活在一定区域的同种生物的全部个体。

基因库：一个种群中全部个体所含有的全部基因。

基因频率：在一个种群基因库中，某个基因占全部等位基因数的比率。

基因型频率：某种基因型的个体在种群中所占的比率。
基因频率和基因型频率的计算

基因频率计算公式：
某基因频率 = (该基因的总数 / 该基因及其等位基因的总数) × 100%

基因型频率计算公式：
某基因型频率 = (该基因型的个体数 / 种群总个体数) × 100%

哈迪-温伯格定律（理想条件下）：
设A的基因频率为p，a的基因频率为q，则：
p + q = 1
(p + q)² = p² + 2pq + q² = 1
即：AA的频率 = p²，Aa的频率 = 2pq，aa的频率 = q²

3.2 突变和基因重组产生进化的原材料

可遗传变异的来源：
• 基因突变：产生新的等位基因
• 基因重组：产生新的基因型
• 染色体变异：改变基因的数量或排列顺序

注意：
• 突变和重组都是随机的、不定向的
• 它们只是为进化提供原材料，不能决定进化的方向

3.3 自然选择决定生物进化的方向

自然选择的作用：
在自然选择的作用下，种群的基因频率会发生定向改变，导致生物朝着一定的方向不断进化。

自然选择对基因频率的影响：
• 有利变异的基因频率增加
• 不利变异的基因频率减少

生物进化的实质：种群基因频率的改变

3.4 隔离导致物种的形成

物种：能够在自然状态下相互交配并且产生可育后代的一群生物。

隔离：不同种群间的个体，在自然条件下基因不能自由交流的现象。

隔离的类型：
• 地理隔离：同一种生物由于地理上的障碍而分成不同的种群
• 生殖隔离：不同物种之间一般不能相互交配，或交配后不能产生可育后代
物种形成的一般过程

地理隔离 → 阻断基因交流 → 不同突变和基因重组 → 不同自然选择 →
基因频率定向改变 → 生殖隔离形成 → 新物种形成

注意：生殖隔离是物种形成的标志，但不是所有物种形成都需要地理隔离（如多倍体植物）

四、共同进化与生物多样性的形成

4.1 共同进化

共同进化：不同物种之间、生物与无机环境之间在相互影响中不断进化和发展。

实例：
• 捕食者与被捕食者之间的相互选择
• 传粉昆虫与开花植物之间的相互适应
• 寄生生物与宿主之间的相互斗争
• 生物与无机环境之间的相互影响

4.2 生物多样性的形成

生物多样性的三个层次：
• 基因多样性：物种内基因的多样性
• 物种多样性：一定区域内物种的丰富程度
• 生态系统多样性：生物群落和生态过程的多样性

生物多样性的形成：是漫长进化历程中不同物种之间、生物与无机环境之间共同进化的结果。

📝 高考真题演练

【2023年全国甲卷】下列关于现代生物进化理论的叙述，正确的是（ ）

A. 生物进化的实质是种群基因型频率的改变

B. 突变和基因重组可以决定生物进化的方向

C. 地理隔离是物种形成的必要条件

D. 自然选择使种群的基因频率发生定向改变

答案：D

解析：

A错误：生物进化的实质是种群基因频率的改变，不是基因型频率的改变。

B错误：突变和基因重组为进化提供原材料，但不能决定进化方向；自然选择决定进化方向。

C错误：隔离是物种形成的必要条件，但地理隔离不是必要条件（如多倍体植物可在同地形成新物种）。

D正确：自然选择使种群的基因频率发生定向改变，从而决定生物进化的方向。
