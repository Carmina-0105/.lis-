一、遗传因子的发现（模块6）

基础 例题1【分离定律应用】
豌豆的高茎(D)对矮茎(d)为显性。现有一株高茎豌豆，要确定其基因型，最简便的方法是（ ）

A. 用基因型为DD的豌豆与其杂交

B. 用基因型为Dd的豌豆与其杂交

C. 用基因型为dd的豌豆与其测交

D. 让其自交，观察后代是否性状分离

答案：D
解析：
确定显性个体基因型的方法有两种：
1. 测交法：与隐性纯合子(dd)杂交。若后代全为高茎，则为DD；若后代出现矮茎，则为Dd。
2. 自交法：让该高茎豌豆自交。若后代全为高茎，则为DD；若后代出现矮茎（性状分离），则为Dd。

对于豌豆而言，自交是最简便的方法，因为豌豆是自花传粉、闭花受粉的植物，自然状态下就是自交。不需要人工去雄、套袋等复杂操作。

答案D正确。
较难 重点 例题2【自由组合定律计算】
某植物的花色由两对等位基因(A/a和B/b)控制，遵循自由组合定律。基因型为A_B_的植株开红花，A_bb的植株开粉花，aaB_和aabb的植株开白花。现有基因型为AaBb的植株自交，后代的表现型及比例为（ ）

A. 红花:粉花:白花 = 9:3:4

B. 红花:粉花:白花 = 9:4:3

C. 红花:粉花:白花 = 12:3:1

D. 红花:粉花:白花 = 9:6:1

答案：A
解析：
步骤1：分析基因型与表现型的对应关系
• 红花：A_B_（双显性）
• 粉花：A_bb（单显性A）
• 白花：aaB_ + aabb（单显性B + 双隐性）

步骤2：计算AaBb自交后代各基因型比例
AaBb自交，后代基因型比例为：
A_B_ : A_bb : aaB_ : aabb = 9 : 3 : 3 : 1

步骤3：合并表现型
• 红花(A_B_)：9/16
• 粉花(A_bb)：3/16
• 白花(aaB_ + aabb)：3/16 + 1/16 = 4/16

表现型比例为：红花:粉花:白花 = 9:3:4

注意：这是9:3:3:1的变式，属于"隐性上位"效应。aa基因掩盖了B/b基因的作用。
📝 9:3:3:1的变式类型总结

类型 条件 比例 实例
互补作用 双显性才表现某性状 9:7 香豌豆花色
累加作用 显性基因累加效应 9:6:1 南瓜果形
重叠作用 只要有显性基因就表现 15:1 荠菜果形
隐性上位 aa掩盖B的作用 9:3:4 家兔毛色
显性上位 A掩盖B的作用 12:3:1 燕麦颖色

二、基因和染色体的关系（模块7）

中等 重点 例题3【伴性遗传】
人类红绿色盲是伴X染色体隐性遗传病。一对表现型正常的夫妇，生了一个患红绿色盲的孩子。下列叙述正确的是（ ）

A. 该患病孩子一定是男孩

B. 该夫妇再生一个孩子，患病的概率为1/4

C. 该夫妇的女儿一定是携带者

D. 该患病孩子的色盲基因来自父亲

答案：A
解析：
分析遗传方式：
红绿色盲是伴X染色体隐性遗传病，设基因为Xᵇ。

确定亲本基因型：
• 父亲正常：XᴮY
• 母亲正常但能生出色盲孩子：XᴮXᵇ（携带者）

分析各选项：
A正确：色盲是X染色体隐性遗传，男性(XᵇY)只要有一个色盲基因就患病，女性(XᵇXᵇ)需要两个色盲基因才患病。父亲正常(XᴮY)，女儿至少从父亲获得一个Xᴮ，所以女儿不会患病。患病孩子一定是男孩(XᵇY)。

B错误：再生一个孩子，患病概率为1/4（XᵇY），但题目说的是"患病"，没有限定性别。如果考虑所有孩子，男孩患病概率1/2，女孩患病概率0，总体患病概率1/4。但选项表述不够准确。

C错误：女儿基因型为XᴮXᴮ或XᴮXᵇ，概率各为1/2。女儿不一定是携带者。

D错误：患病男孩的色盲基因Xᵇ只能来自母亲（父亲提供的是Y染色体）。
⚠️ 易错警示：伴性遗传特点

X染色体隐性遗传病（如红绿色盲、血友病）：
• 男性患者多于女性患者
• 隔代交叉遗传：男性患者的致病基因通过女儿传给外孙
• 女性患者的父亲和儿子一定患病

X染色体显性遗传病（如抗维生素D佝偻病）：
• 女性患者多于男性患者
• 世代连续遗传
• 男性患者的母亲和女儿一定患病

Y染色体遗传病（如外耳道多毛症）：
• 只有男性患病
• 父传子，子传孙

三、基因的本质与表达（模块8）

较难 重点 例题4【DNA复制与细胞分裂综合】
将某一经³H充分标记DNA的雄性动物细胞（染色体数为2N）置于不含³H的培养基中培养，经过两次连续分裂。下列有关说法正确的是（ ）

A. 若进行有丝分裂，则子细胞含³H的染色体数一定为N

B. 若进行减数分裂，则子细胞含³H的DNA分子数为N

C. 若进行有丝分裂，则第二次分裂后期每个细胞中含³H的染色体数为2N

D. 若进行减数分裂，则减数第二次分裂后期每个细胞中含³H的染色体数为N

答案：B
解析：
DNA复制的特点：半保留复制

情况一：有丝分裂两次
• 第一次分裂：DNA复制后，每条染色体含2个DNA分子，都是一条链³H、一条链不含³H。分裂后，每个子细胞的每条染色体都含³H。
• 第二次分裂：DNA再次复制，每条染色体的2个DNA分子中，1个含³H（一条链³H），1个不含³H。分裂时，染色体随机分配，子细胞含³H的染色体数为0～2N，不一定。

情况二：减数分裂
• DNA只复制1次，细胞分裂2次。
• 复制后，每条染色体含2条染色单体，每条单体的DNA都是一条链³H、一条链不含³H。
• 减数第一次分裂后，每条染色体仍含2条染色单体，都含³H。
• 减数第二次分裂后，每个子细胞含N条染色体，每条染色体的DNA都含³H。
• 所以子细胞含³H的DNA分子数为N。

答案B正确。

四、变异与育种（模块9）

中等 例题5【育种方法选择】
某自花传粉植物的矮茎(a)对高茎(A)为隐性，抗病(B)对感病(b)为显性。现有矮茎抗病(aaBB)和高茎感病(AAbb)两个纯合品种，要培育出高茎抗病(AABB)的新品种，最快捷的育种方法是（ ）

A. 杂交育种

B. 诱变育种

C. 单倍体育种

D. 多倍体育种

答案：C
解析：
分析各育种方法：

A. 杂交育种：
aaBB × AAbb → AaBb（F₁）→ 自交 → F₂中选高茎抗病(A_B_) → 连续自交纯化
需要至少3-4年，时间较长。

B. 诱变育种：
原理是基因突变，具有不定向性和低频性，无法定向获得所需品种。

C. 单倍体育种：
aaBB × AAbb → AaBb（F₁）→ 花药离体培养 → 单倍体(AB、Ab、aB、ab) → 秋水仙素加倍 → 纯合二倍体(AABB、AAbb、aaBB、aabb) → 选择高茎抗病(AABB)
只需2年，能明显缩短育种年限，且后代都是纯合子。

D. 多倍体育种：
主要用于获得茎秆粗壮、果实种子大的品种，不适用于此情况。

答案C正确。单倍体育种最快捷。

五、现代生物进化理论（模块10）

中等 例题6【基因频率计算】
某植物种群中，AA个体占16%，aa个体占36%，该种群随机交配产生的后代中AA个体百分比、A基因频率和自交产生的后代中AA个体百分比、A基因频率的变化依次为（ ）

A. 增大，不变；不变，不变

B. 不变，增大；增大，不变

C. 不变，不变；增大，不变

D. 不变，不变；不变，增大

答案：C
解析：
步骤1：计算亲代基因频率
AA = 16%，aa = 36%，则Aa = 48%
A基因频率 = 16% + 48%/2 = 40%
a基因频率 = 36% + 48%/2 = 60%

步骤2：随机交配后代（哈迪-温伯格平衡）
AA = p² = 0.4² = 16%
Aa = 2pq = 2 × 0.4 × 0.6 = 48%
aa = q² = 0.6² = 36%
A基因频率仍为40%

随机交配结果：AA个体百分比不变，A基因频率不变。

步骤3：自交后代
AA自交 → 全部AA
aa自交 → 全部aa
Aa自交 → 1/4 AA : 1/2 Aa : 1/4 aa

后代AA = 16% + 48% × 1/4 = 16% + 12% = 28%（增大）
A基因频率 = 28% + (48% × 1/2)/2 = 28% + 12% = 40%（不变）

答案C正确：随机交配：不变，不变；自交：增大，不变。
📝 基因频率计算要点

1. 基因频率计算公式：
某基因频率 = (该基因纯合子个体数×2 + 杂合子个体数) / (总个体数×2)

2. 哈迪-温伯格定律（理想条件下）：
• 种群足够大
• 随机交配
• 无突变、无迁移、无自然选择
满足条件时：p² + 2pq + q² = 1，基因频率保持不变

3. 自交与随机交配的区别：
• 自交：基因型频率改变（纯合子增多），基因频率不变
• 随机交配：基因型频率和基因频率都不变（哈迪-温伯格平衡）
