一、DNA分子的结构

1.1 DNA双螺旋结构模型

DNA双螺旋结构的特点（沃森和克里克，1953年）：

1. 反向平行：两条脱氧核苷酸链按反向平行方式盘旋成双螺旋结构
2. 外侧骨架：脱氧核糖和磷酸交替连接，排列在外侧，构成基本骨架
3. 内侧碱基：碱基排列在内侧，通过氢键连接成碱基对
4. 碱基互补配对：A与T配对（2个氢键），G与C配对（3个氢键）

🧬 DNA双螺旋结构示意

5'-磷酸-脱氧核糖-碱基-3'
        ↑            ↓
3'-碱基-脱氧核糖-磷酸-5'

A ═══ T（2个氢键）
G ≡≡≡ C（3个氢键）

1.2 DNA分子的特性

1. 多样性：碱基对的排列顺序千变万化
若有n个碱基对，可能的排列方式有4ⁿ种

2. 特异性：每个DNA分子有特定的碱基排列顺序

3. 稳定性：磷酸和脱氧核糖交替连接形成稳定骨架；碱基互补配对原则保证了复制的准确

1.3 DNA分子中的碱基数量关系

碱基互补配对原则的应用

在双链DNA分子中：
• A = T，G = C
• A + G = T + C = A + C = T + G = 50%（嘌呤 = 嘧啶）
• (A + T) / (G + C) 比值因DNA而异

在单链DNA中：
• A₁ = T₂，T₁ = A₂，G₁ = C₂，C₁ = G₂
• 一条链的(A+T)/(G+C) = 另一条链的(A+T)/(G+C) = 双链的(A+T)/(G+C)

二、DNA分子的复制

2.1 DNA复制的基本概念

复制时间：有丝分裂间期和减数第一次分裂前的间期

复制场所：细胞核（主要）、线粒体、叶绿体

复制条件：
• 模板：DNA的两条链
• 原料：四种游离的脱氧核苷酸
• 能量：ATP
• 酶：解旋酶、DNA聚合酶等

2.2 DNA复制的过程

DNA复制过程（边解旋边复制）

① 解旋：在解旋酶作用下，DNA双链解开，形成两条单链
② 合成子链：以两条母链为模板，在DNA聚合酶作用下，按照碱基互补配对原则合成子链
③ 形成子代DNA：每条子链与对应的母链盘绕成双螺旋结构，形成两个子代DNA分子

特点：边解旋边复制、半保留复制、多起点复制（真核生物）

2.3 DNA复制的特点——半保留复制

半保留复制：新合成的每个DNA分子中，都保留了原来DNA分子中的一条链。

证明实验（梅塞尔森和斯塔尔，1958年）：
用¹⁵N标记大肠杆菌的DNA，然后转移到¹⁴N培养基中培养，通过密度梯度离心分析DNA密度变化，证明了DNA的半保留复制。
DNA复制相关计算

1. DNA复制n次后：
• DNA分子总数：2ⁿ个
• 含母链的DNA分子：2个
• 只含母链的DNA分子：0个
• 含子链的DNA分子：2ⁿ个
• 只含子链的DNA分子：2ⁿ - 2个

2. 所需原料计算：
若DNA分子中某碱基为m个，复制n次需要该游离脱氧核苷酸：m × (2ⁿ - 1)个
第n次复制需要：m × 2ⁿ⁻¹个

三、基因的表达

3.1 基因指导蛋白质的合成

基因表达：基因通过指导蛋白质的合成来控制性状的过程，包括转录和翻译两个阶段。

3.2 转录

转录过程（DNA → RNA）

场所：细胞核（主要）
模板：DNA的一条链（模板链/反义链）
原料：四种核糖核苷酸
酶：RNA聚合酶
能量：ATP

过程：
① DNA双链解开，RNA聚合酶结合到启动子
② 按照碱基互补配对原则合成RNA（A-U，T-A，G-C，C-G）
③ RNA链延伸，到达终止子后释放
④ DNA双链恢复

3.3 翻译

翻译过程（RNA → 蛋白质）

场所：核糖体
模板：mRNA
原料：20种氨基酸
转运工具：tRNA
酶和能量：多种酶、ATP

过程：
① mRNA与核糖体结合
② tRNA携带氨基酸进入核糖体，反密码子与mRNA上的密码子配对
③ 氨基酸之间形成肽键
④ 核糖体沿mRNA移动，读取下一个密码子
⑤ 到达终止密码子，肽链释放，盘曲折叠形成蛋白质

📝 转录 vs 复制

相同点：

  • 都以DNA为模板
  • 都遵循碱基互补配对
  • 都需要酶和能量

不同点：

  • 复制：两条链都作模板，产物是DNA
  • 转录：一条链作模板，产物是RNA

🔄 转录 vs 翻译

项目 转录 翻译
场所 细胞核 核糖体
模板 DNA一条链 mRNA
原料 核糖核苷酸 氨基酸
产物 RNA 蛋白质

3.4 遗传密码子

密码子：mRNA上决定一个氨基酸的三个相邻碱基。

特点：
• 64种密码子，其中61种编码氨基酸，3种终止密码子（UAA、UAG、UGA）
• 起始密码子：AUG（编码甲硫氨酸）
• 一种氨基酸可对应多种密码子（密码子的简并性）
• 所有生物共用一套遗传密码（密码子的通用性）

四、中心法则

4.1 中心法则的内容

中心法则图解

                复制
         DNA ─────────→ DNA
          ↓转录    ↑逆转录
          ↓      ↓
         RNA ──翻译──→ 蛋白质
          ↓复制
         RNA

中心法则各过程的适用范围：
• DNA复制：绝大多数生物
• 转录和翻译：绝大多数生物
• RNA复制：RNA病毒（如烟草花叶病毒）
• 逆转录：逆转录病毒（如HIV）

五、基因对性状的控制

5.1 基因控制性状的两条途径

🔧 直接控制

途径：基因 → 蛋白质结构 → 性状

实例：

  • 镰刀型细胞贫血症：血红蛋白基因突变 → 血红蛋白结构异常 → 红细胞呈镰刀状
  • 囊性纤维病：CFTR基因突变 → CFTR蛋白结构异常 → 肺部等疾病

⚙️ 间接控制

途径：基因 → 酶的合成 → 代谢过程 → 性状

实例：

  • 白化病：酪氨酸酶基因突变 → 不能合成酪氨酸酶 → 不能合成黑色素 → 白化
  • 苯丙酮尿症：苯丙氨酸羟化酶基因突变 → 代谢异常 → 智力低下

5.2 基因与性状的关系

1. 一个基因控制一个性状（简单情况）

2. 一个基因影响多个性状（一因多效）：
如豌豆的圆皱基因不仅控制粒形，还影响淀粉代谢和甜度

3. 多个基因控制一个性状（多因一效）：
如人的身高、肤色等数量性状

4. 基因与环境共同决定性状：
表现型 = 基因型 + 环境条件

📝 高考真题演练

【2023年全国乙卷】一个DNA分子中，G和C之和占全部碱基的46%，其中一条链的A和C分别占该链碱基总数的28%和22%，则该DNA分子的另一条链中，A和C分别占该链碱基总数的（ ）

A. 26%、24%    B. 24%、26%    C. 28%、22%    D. 22%、28%

答案：A

解析：

已知：G+C = 46%，则A+T = 54%

在双链中：A = T = 27%，G = C = 23%

设该DNA的两条链为1链和2链

已知1链：A₁ = 28%，C₁ = 22%

因为A = (A₁ + A₂)/2，所以27% = (28% + A₂)/2，解得A₂ = 26%

因为C = (C₁ + C₂)/2，所以23% = (22% + C₂)/2，解得C₂ = 24%
