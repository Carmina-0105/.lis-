一、种群的特征

1.1 种群的数量特征

种群：在一定的自然区域内，同种生物的全部个体。

种群的数量特征：
1. 种群密度：种群最基本的数量特征
2. 出生率和死亡率：决定种群大小和密度的重要因素
3. 迁入率和迁出率：影响种群大小和密度
4. 年龄组成：预测种群数量变化趋势
5. 性别比例：影响种群密度

1.2 种群密度的调查方法

调查方法 适用范围 计算方法 注意事项
样方法 植物或活动能力弱、活动范围小的动物 种群密度 = 样方内个体总数 / 样方面积 随机取样；样方大小适中；取平均值
标志重捕法 活动能力强、活动范围大的动物 N = (初次捕获标记数 × 再次捕获总数) / 再次捕获标记数 标记不能影响动物生存；标记不易脱落

二、种群数量的变化

2.1 "J"型增长曲线

条件：食物和空间条件充裕、气候适宜、没有敌害等理想条件

数学模型：
$N_t = N_0 \lambda^t$

其中：$N_0$ 为起始数量，$t$ 为时间，$N_t$ 为t年后种群数量，$\lambda$ 为年均增长率

特点：增长率保持不变，种群数量连续增长

2.2 "S"型增长曲线

条件：自然条件（资源和空间有限）

特点：
• 种群数量达到环境容纳量（K值）后停止增长
• 增长速率先增大后减小，在K/2时增长速率最大
• K值不是固定不变的，会随环境变化而变化

K值（环境容纳量）：在环境条件不受破坏的情况下，一定空间中所能维持的种群最大数量。
两种增长曲线的比较
项目 J型曲线 S型曲线
前提条件 理想条件 自然条件
增长特点 连续增长，无上限 增长到一定数量后保持相对稳定
增长速率 逐渐增大 先增大后减小
K值 无K值 有K值
曲线形状 指数曲线 逻辑斯蒂曲线

2.3 种群数量的波动和下降

影响因素：
• 气候、食物、天敌、传染病等
• 人类活动的影响

种群数量的波动：
• 大多数种群数量总是在波动中
• 某些种群可能出现周期性波动

三、群落的结构

3.1 群落的概念

群落：同一时间内聚集在一定区域中各种生物种群的集合。

群落的基本特征：
• 物种组成（丰富度）
• 种间关系
• 空间结构
• 群落演替

3.2 群落的物种组成

丰富度：群落中物种数目的多少。

优势种：对群落结构和环境形成有明显控制作用的物种。

调查方法：常用取样器取样法调查土壤小动物类群丰富度。

3.3 种间关系

关系类型 特点 实例
互利共生 两种生物共同生活，相互依存，彼此有利 豆科植物与根瘤菌、地衣
寄生 一种生物寄居于另一种生物体内或体表，摄取寄主的养分 蛔虫与人、菟丝子与植物
竞争 两种或两种以上生物相互争夺资源和空间等 牛与羊、水稻与杂草
捕食 一种生物以另一种生物为食 狼与兔、猫头鹰与鼠

3.4 群落的空间结构

垂直结构：
• 概念：群落在垂直方向上的分层现象
• 原因：光照（植物）、食物和栖息空间（动物）
• 意义：提高对环境资源的利用能力

水平结构：
• 概念：群落中生物在水平方向上的配置状况
• 特点：呈镶嵌分布
• 原因：地形变化、土壤湿度和盐碱度差异、光照强度不同、生物自身生长特点、人类活动的影响

四、群落的演替

4.1 演替的概念和类型

演替：随着时间的推移，一个群落被另一个群落代替的过程。

演替的类型：
• 初生演替：在一个从来没有被植物覆盖的地面，或者是原来存在过植被、但被彻底消灭了的地方发生的演替
• 次生演替：在原有植被虽已不存在，但原有土壤条件基本保留，甚至还保留了植物的种子或其他繁殖体的地方发生的演替
初生演替和次生演替的比较
项目 初生演替 次生演替
起点 无植被或植被被彻底消灭 原有土壤条件基本保留
时间 长 短
速度 慢 快
实例 裸岩、沙丘、火山岩 火灾后的草原、弃耕的农田

4.2 演替的过程

初生演替的一般过程：
裸岩阶段 → 地衣阶段 → 苔藓阶段 → 草本植物阶段 → 灌木阶段 → 森林阶段

演替的趋势：
• 物种丰富度增加
• 群落结构复杂化
• 稳定性增强

注意：演替的最终阶段不一定都是森林，取决于当地的气候条件

📝 高考真题演练

【2023年全国甲卷】下列关于种群和群落的叙述，正确的是（ ）

A. 种群密度是种群最基本的数量特征

B. 种群数量达到K值后不再变化

C. 群落演替的最终阶段一定是森林

D. 竞争的结果一定是一方获胜，另一方被淘汰

答案：A

解析：

A正确：种群密度是种群最基本的数量特征。

B错误：种群数量达到K值后会在K值附近波动，不是不再变化。

C错误：群落演替的最终阶段取决于当地的气候条件，不一定都是森林（如干旱地区可能是草原）。

D错误：竞争的结果可能是一方获胜，也可能是双方共存（如生态位分化）。
