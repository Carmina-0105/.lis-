⚡ 模块七：静电场

7.1 电场强度与电场线

定义式：$\displaystyle E = \frac{F}{q}$（$q$ 为试探电荷，$E$ 与 $q$ 无关）

点电荷：$\displaystyle E = k\frac{Q}{r^2}$，$k = 9.0 \times 10^9 \text{N·m}^2/\text{C}^2$

匀强电场：$\displaystyle E = \frac{U}{d}$（$d$ 为沿电场线方向距离）

叠加原理：$\vec{E} = \vec{E}_1 + \vec{E}_2 + \cdots$（矢量合成）

🔌 电场强度三公式对比：
• 定义式：$E = F/q$ → 普适，$E$ 与 $F$、$q$ 无关
• 决定式：$E = kQ/r^2$ → 仅点电荷，$E \propto Q$，$E \propto 1/r^2$
• 关系式：$E = U/d$ → 仅匀强电场，$d$ 必须是沿场强方向距离

常见电场线分布：点电荷、等量异种/同种电荷

7.2 电势能与电势

电势能：$E_p = q\varphi$（$q$、$\varphi$ 均带符号）

电势：$\displaystyle \varphi = \frac{E_p}{q} = k\frac{Q}{r}$（点电荷，取无穷远为零势面）

电势差：$U_{AB} = \varphi_A - \varphi_B = \frac{W_{AB}}{q}$

电场力做功：$W_{AB} = qU_{AB} = -\Delta E_p$

物理量 定义 标矢性 零势点 与电场线关系
电场强度 $E$ $F/q$ 矢量 无 切线方向
电势 $\varphi$ $E_p/q$ 标量（有正负） 人为规定 沿电场线降低
电势差 $U$ $W/q$ 标量 与零势点无关 $U = Ed$（匀强）
🎯 电场能性质判断口诀：
"正电荷，高势高能；负电荷，高势低能。
电场力做正功，电势能减小；
电场力做负功，电势能增加。"

7.3 电容器与电容

定义：$\displaystyle C = \frac{Q}{U}$（$C$ 与 $Q$、$U$ 无关）

平行板电容器：$\displaystyle C = \frac{\varepsilon_r S}{4\pi kd} = \frac{\varepsilon S}{d}$

电场强度：$\displaystyle E = \frac{U}{d} = \frac{Q}{Cd} = \frac{4\pi kQ}{\varepsilon_r S}$

储能：$\displaystyle W = \frac{1}{2}QU = \frac{1}{2}CU^2 = \frac{Q^2}{2C}$

⚠️ 电容器动态分析两类问题：
1. 与电源相连（$U$ 不变）：
   • $d \uparrow$ → $C \downarrow$ → $Q \downarrow$ → $E \downarrow$
   • $S \downarrow$ → $C \downarrow$ → $Q \downarrow$ → $E$ 不变（$E = U/d$）
2. 与电源断开（$Q$ 不变）：
   • $d \uparrow$ → $C \downarrow$ → $U \uparrow$ → $E$ 不变（$E \propto Q/S$）
   • $S \downarrow$ → $C \downarrow$ → $U \uparrow$ → $E \uparrow$

7.4 带电粒子在电场中的运动

加速：$qU = \frac{1}{2}mv^2 - \frac{1}{2}mv_0^2$

偏转（类平抛）：$y = \frac{1}{2}at^2 = \frac{qUL^2}{2mdv_0^2}$

偏转角：$\tan\theta = \frac{v_y}{v_0} = \frac{qUL}{mdv_0^2}$

重要推论：$\tan\theta = \frac{2y}{L}$（速度反向延长线过水平位移中点）

🚀 带电粒子偏转问题解题步骤：
1. 加速阶段：用动能定理求 $v_0$
2. 偏转阶段：分解为水平匀速、竖直匀加速
3. 离开电场后：匀速直线运动
4. 示波器问题：$Y = y + L'\tan\theta$（$L'$ 为偏转板到屏距离）
🔋 模块八：恒定电流

8.1 电路基础

电流：$\displaystyle I = \frac{q}{t} = nqSv$（$n$ 为单位体积自由电荷数）

电阻定律：$\displaystyle R = \rho\frac{L}{S}$（$\rho$ 为电阻率，与材料、温度有关）

欧姆定律：$\displaystyle I = \frac{U}{R}$（纯电阻电路）

电功率：$P = UI = I^2R = \frac{U^2}{R}$

连接方式 电流 电压 电阻 功率分配
串联 $I = I_1 = I_2 = \cdots$ $U = U_1 + U_2 + \cdots$ $R = R_1 + R_2 + \cdots$ $P \propto R$
并联 $I = I_1 + I_2 + \cdots$ $U = U_1 = U_2 = \cdots$ $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots$ $P \propto \frac{1}{R}$

8.2 闭合电路欧姆定律

电流：$\displaystyle I = \frac{E}{R+r}$（$E$ 为电动势，$r$ 为内阻）

路端电压：$U = E - Ir = IR = \frac{R}{R+r}E$

输出功率：$P_{出} = I^2R = \frac{E^2R}{(R+r)^2}$

最大输出功率：当 $R = r$ 时，$P_{max} = \frac{E^2}{4r}$

效率：$\eta = \frac{P_{出}}{P_{总}} = \frac{R}{R+r} = \frac{U}{E}$

电源输出功率随外电阻变化曲线

🔋 电源功率关系：
• 总功率：$P_{总} = EI$（电源消耗总功率）
• 内耗功率：$P_{内} = I^2r$（内阻发热）
• 输出功率：$P_{出} = UI = I^2R$（外电路获得功率）
• 关系：$P_{总} = P_{出} + P_{内}$

8.3 电路动态分析

🔄 动态分析口诀（串反并同）：
串反：与变化电阻串联的 $I$、$U$、$P$ 变化相反
并同：与变化电阻并联的 $I$、$U$、$P$ 变化相同
注意：适用于电源有内阻（$r \neq 0$）的情况
⚠️ 电路故障分析：
• 断路：电流为零，断路点两端电压等于电源电压
• 短路：电压为零，电流很大
• 检测方法：电压表法（并联）、电流表法（串联）、欧姆表法（断电）

8.4 电学实验基础

实验 原理 电路图 关键操作 误差分析
测电阻（伏安法） $R = U/I$ 内接/外接 选量程、接法判断 内接偏大，外接偏小
测电源电动势和内阻 $U = E - Ir$ 电流表内接 作 $U-I$ 图 $E_{测} \leq E_{真}$，$r_{测} \leq r_{真}$
描绘小灯泡伏安特性 $R = U/I$ 分压外接 从零开始调节 温度影响电阻
电表改装 串/并联电阻 - 计算分压/分流电阻 实际电阻与计算值偏差
🎯 内接法与外接法选择：
• 大内偏大：$R_x \gg R_A$ 用内接，测量值 $R_{测} = R_x + R_A > R_x$
• 小外偏小：$R_x \ll R_V$ 用外接，测量值 $\frac{1}{R_{测}} = \frac{1}{R_x} + \frac{1}{R_V}$，$R_{测} < R_x$
• 临界值：$\sqrt{R_A R_V}$，大于用内接，小于用外接

🔗 电磁学（上）融合总结

电场与电路综合

1. 带电粒子在电场中加速+偏转：加速用动能定理，偏转用类平抛，注意示波器灵敏度 $\frac{h}{U_1}$

2. 电容器+电路：含容电路稳定时电容视为断路，$Q = CU$，变化量 $\Delta Q = C\Delta U$

3. 电场图像：$\varphi-x$ 图斜率 = $-E$，$E_p-x$ 图斜率 = $-F_{电}$
