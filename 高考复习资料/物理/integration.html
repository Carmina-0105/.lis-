🎯 带电粒子在复合场中的运动

模型1：电场+重力场

• 等效重力法：$F_{等效} = qE + mg$，$g' = g + \frac{qE}{m}$（同向）或 $|g - \frac{qE}{m}|$（反向）

• 应用：单摆周期 $T = 2\pi\sqrt{\frac{L}{g'}}$，竖直圆周运动临界条件

模型2：电场+磁场（速度选择器）

• 平衡条件：$qE = qvB$ → $v = \frac{E}{B}$

• 应用：筛选特定速度粒子，质谱仪前置

模型3：电磁感应+力学

• 单棒模型：$F_{安} = \frac{B^2L^2v}{R}$，牛顿第二定律 $F - \frac{B^2L^2v}{R} = ma$

• 能量转化：$W_{外} = \Delta E_k + Q$，$Q = \frac{B^2L^2v^2}{R}t$

⚡ 电磁感应+电路+能量

解题流程：

1. 找电源：切割磁感线的部分相当于电源，$E = BLv$ 或 $E = n\frac{\Delta\Phi}{\Delta t}$

2. 画电路：确定内外电路，求总电阻 $R_{总}$

3. 求电流：$I = \frac{E}{R_{总}}$，安培力 $F = BIL$

4. 列方程：牛顿第二定律或平衡方程

5. 算能量：$Q = I^2Rt$ 或能量守恒

能量观点贯穿始终

🎯 分题型解题策略

选择题解题技巧（40分左右）

1. 排除法：先排除明显错误的选项，缩小范围
2. 特殊值法：取 $m=1$、$\theta=0°$、$t=0$ 等简化计算
3. 量纲法：检查单位是否匹配（如求速度，选项中出现 $\text{m}^2/\text{s}$ 必错）
4. 极限法：让某个量趋于0或无穷大，看结果是否合理
5. 图像法：利用 $v-t$ 图、$F-x$ 图等面积、斜率物理意义
6. 对称性法：利用对称性简化问题（如等势面对称、运动对称）
🚀 选择题抢分策略：
• 单选题：确保前6题全对（基础题），第7-8题尽量拿分
• 多选题：宁缺毋滥，不确定的选项不选，确保拿部分分
• 时间控制：平均每题2-3分钟，难的先跳过

实验题解题模板（15分左右）

📝 力学实验答题模板

纸带数据处理：

① 求速度：$v_n = \frac{x_n + x_{n+1}}{2T}$（注意单位换算 $\text{cm} \rightarrow \text{m}$）

② 求加速度：逐差法 $a = \frac{(x_4+x_5+x_6)-(x_1+x_2+x_3)}{9T^2}$

③ 作图：$v-t$ 图，描点连线，斜率即加速度

验证机械能守恒：

① 计算 $\Delta E_p = mg\Delta h$

② 计算 $\Delta E_k = \frac{1}{2}m(v_2^2 - v_1^2)$

③ 比较 $\Delta E_p$ 与 $\Delta E_k$（允许 $\Delta E_p > \Delta E_k$，因存在阻力）

⚡ 电学实验答题模板

电路连接：先画电路图，再连实物图（先串后并，注意正负极）

器材选择：

• 电压表/电流表：指针偏转在 $\frac{1}{3} \sim \frac{2}{3}$ 量程最佳

• 滑动变阻器：限流式选阻值相近，分压式选阻值小的

数据处理：

• 作图：坐标轴标注物理量和单位，描点连线（直线用直尺，曲线用平滑曲线）

• 求斜率：在直线上取两点（不是数据点），$k = \frac{\Delta y}{\Delta x}$

计算题答题规范（32分左右）

⚠️ 计算题失分重灾区：
1. 只有结果没有过程 → 过程分全丢
2. 不写原始公式直接代数据 → 扣公式分
3. 不写单位或单位错误 → 扣结果分
4. 字母符号混用，下标混乱 → 扣卷面分
5. 不写"解"和"答"，步骤跳跃 → 印象分低
✅ 计算题标准答题步骤：
  • 写"解："（或"解：设..."）
  • 写原始公式（用题目给的字母）
  • 代入数据（带单位）
  • 计算结果（带单位，科学计数法）
  • 写"答：..."
    • 📝 示例（规范写法）：
    解：由牛顿第二定律得
    $F - \mu mg = ma$

    代入数据
    $10 - 0.2 \times 2 \times 10 = 2a$

    解得
    $a = 3 \text{m/s}^2$

    由运动学公式
    $v = v_0 + at = 0 + 3 \times 5 = 15 \text{m/s}$

    答：物体的加速度为 $3 \text{m/s}^2$，5s末的速度为 $15 \text{m/s}$。

    🔧 物理模型大全（高考必考）

    模型1：传送带模型

    🎯 水平传送带

    情景1：物块初速度为0

    • 加速度：$a = \mu g$

    • 共速时间：$t = \frac{v_{带}}{\mu g}$

    • 相对位移：$\Delta x = v_{带}t - \frac{1}{2}\mu gt^2 = \frac{v_{带}^2}{2\mu g}$

    • 痕迹长度（摩擦生热）：$Q = \mu mg \cdot \Delta x$

    情景2：物块有初速度（与传送带同向）

    • 若 $v_{物} < v_{带}$：先加速后匀速，$t_{加} = \frac{v_{带}-v_{物}}{\mu g}$

    • 若 $v_{物} > v_{带}$：先减速后匀速，$t_{减} = \frac{v_{物}-v_{带}}{\mu g}$

    情景3：物块初速度与传送带反向

    • 先减速至0，再反向加速，最终共速或掉下

    • 判断是否掉下：减速位移 $x = \frac{v_{物}^2}{2\mu g}$ 与传送带长度比较

    🎯 倾斜传送带

    关键：比较 $\mu$ 与 $\tan\theta$

    • 若 $\mu > \tan\theta$：物块最终能与传送带相对静止

    • 若 $\mu < \tan\theta$：物块一直加速，不能共速

    • 上滑时：$a = g(\sin\theta + \mu\cos\theta)$（相对下滑）

    • 下滑时：$a = g(\sin\theta - \mu\cos\theta)$（相对上滑）

    💡 解题步骤：
    1. 确定相对运动方向，判断摩擦力方向
    2. 求加速度 $a = \frac{F_{合}}{m}$
    3. 计算共速时间和位移
    4. 判断是否共速，计算总时间和痕迹长度

    模型2：板块模型（滑块-滑板）

    🎯 地面光滑情况

    基本方程：

    • 滑块：$a_1 = \frac{F - \mu mg}{m_1}$ 或 $a_1 = \mu g$（仅摩擦力）

    • 滑板：$a_2 = \frac{\mu mg}{m_2}$

    • 共速条件：$v_1 + a_1t = a_2t$（假设滑块有初速度）

    • 相对位移：$\Delta x = x_1 - x_2$

    临界条件：

    • 不滑落条件：$\Delta x \leq L$（滑板长度）

    • 恰好不掉下：$\Delta x = L$，共速时到达边缘

    🎯 地面粗糙情况

    滑板受力分析：

    • 水平方向：$\mu_1m_1g - \mu_2(m_1+m_2)g = m_2a_2$

    • 判断滑板是否运动：比较 $\mu_1m_1g$ 与 $\mu_2(m_1+m_2)g$

    两种情景：

    • 若 $\mu_1m_1g \leq \mu_2(m_1+m_2)g$：滑板静止，滑块减速

    • 若 $\mu_1m_1g > \mu_2(m_1+m_2)g$：滑板加速，系统复杂

    💡 解题技巧：
    1. 画受力分析图，标出各物体加速度方向
    2. 分别列牛顿第二定律方程
    3. 找位移关系（相对位移 = 板长）
    4. 用v-t图像辅助分析更直观

    模型3：弹簧模型

    🎯 弹簧分离问题

    分离条件：两物体间弹力 $N = 0$，加速度相同

    • 对A：$F_{弹} - m_Ag = m_Aa$

    • 对B：$F - m_Bg = m_Ba$

    • 分离时：$a_A = a_B$，$N = 0$

    常见情景：

    • 竖直弹簧：原长位置分离（若初始压缩）

    • 水平弹簧：弹力提供加速度时分离

    🎯 弹簧能量问题

    能量守恒：$E_{k1} + E_{p1} + E_{弹1} = E_{k2} + E_{p2} + E_{弹2}$

    • 弹性势能：$E_p = \frac{1}{2}kx^2$（x为形变量）

    • 最大弹性势能：速度为0时，或共速时（碰撞模型）

    • 弹簧做功：$W = -\Delta E_p = \frac{1}{2}kx_1^2 - \frac{1}{2}kx_2^2$

    📝 重要结论：
    • 弹簧振子周期：$T = 2\pi\sqrt{\frac{m}{k}}$
    • 简谐运动对称性：关于平衡位置对称的两点，加速度大小相等、速度大小相等
    • 弹簧连接体：加速度相同，内力按质量分配

    模型4：碰撞模型

    🎯 弹性碰撞（动量守恒+动能守恒）

    基本方程：

    • 动量守恒：$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$

    • 动能守恒：$\frac{1}{2}m_1v_1^2 + \frac{1}{2}m_2v_2^2 = \frac{1}{2}m_1v_1'^2 + \frac{1}{2}m_2v_2'^2$

    速度公式（一静一动）：

    • $v_1' = \frac{m_1-m_2}{m_1+m_2}v_1$

    • $v_2' = \frac{2m_1}{m_1+m_2}v_1$

    特殊情况：

    • $m_1 = m_2$：交换速度，$v_1' = 0$，$v_2' = v_1$

    • $m_1 \gg m_2$：$v_1' \approx v_1$，$v_2' \approx 2v_1$

    • $m_1 \ll m_2$：$v_1' \approx -v_1$，$v_2' \approx 0$

    🎯 完全非弹性碰撞（共速）

    特点：碰后共速，动能损失最大

    • 动量守恒：$m_1v_1 + m_2v_2 = (m_1+m_2)v_{共}$

    • 共速：$v_{共} = \frac{m_1v_1 + m_2v_2}{m_1+m_2}$

    • 动能损失：$\Delta E_k = \frac{1}{2}m_1v_1^2 + \frac{1}{2}m_2v_2^2 - \frac{1}{2}(m_1+m_2)v_{共}^2$

    • 转化为内能或弹性势能：$Q = \Delta E_k$

    🎯 非弹性碰撞（一般情况）

    恢复系数：$e = \frac{v_2' - v_1'}{v_1 - v_2}$

    • $e = 1$：弹性碰撞

    • $e = 0$：完全非弹性碰撞

    • $0 < e < 1$：一般非弹性碰撞

    模型5：电磁感应单棒/双棒模型

    🎯 单棒模型（电阻棒或电容棒）

    电阻棒（受恒力F）：

    • 感应电动势：$E = BLv$

    • 感应电流：$I = \frac{BLv}{R}$

    • 安培力：$F_安 = \frac{B^2L^2v}{R}$（阻力）

    • 牛顿定律：$F - \frac{B^2L^2v}{R} = ma$

    • 最终匀速：$v_m = \frac{FR}{B^2L^2}$，$a = 0$

    电容棒：

    • 充电电流：$I = \frac{\Delta Q}{\Delta t} = C\frac{\Delta U}{\Delta t} = CBL\frac{\Delta v}{\Delta t} = CBLa$

    • 安培力：$F_安 = BIL = B^2L^2Ca$

    • 等效质量：$m' = m + B^2L^2C$，匀加速运动

    🎯 双棒模型（光滑导轨）

    等距双棒（一动一静）：

    • 动量守恒：$m_1v_0 = m_1v_1 + m_2v_2$

    • 最终共速：$v_{共} = \frac{m_1v_0}{m_1+m_2}$

    • 回路电荷量：$q = \frac{m_1(v_0-v_{共})}{BL} = \frac{m_1m_2v_0}{(m_1+m_2)BL}$

    • 焦耳热：$Q = \frac{1}{2}m_1v_0^2 - \frac{1}{2}(m_1+m_2)v_{共}^2$

    不等距双棒：

    • 动量不守恒（安培力不等）

    • 用动量定理分别列方程

    • 稳定条件：$E_1 = E_2$，即 $B_1L_1v_1 = B_2L_2v_2$

    💡 电磁感应解题流程：
    1. 确定电源：切割磁感线的部分
    2. 画等效电路：内阻、外阻
    3. 求感应电流：$I = \frac{E}{R_{总}}$
    4. 求安培力：$F = BIL$
    5. 列牛顿定律或能量方程

    模型6：带电粒子在复合场中的运动

    🎯 电场+重力场

    等效重力法：

    • 等效重力加速度：$g' = \frac{\sqrt{(mg)^2+(qE)^2}}{m}$

    • 等效重力方向：与竖直方向夹角 $\tan\theta = \frac{qE}{mg}$

    • 应用：单摆周期 $T = 2\pi\sqrt{\frac{L}{g'}}$

    • 竖直圆周运动临界速度：$v_{临} = \sqrt{g'R}$（最高点）

    🎯 电场+磁场（速度选择器）

    直线运动条件：$qE = qvB$，即 $v = \frac{E}{B}$

    • 与电荷量、质量无关

    • 只选择速度，不选择粒子种类

    • 应用：质谱仪前置筛选

    🎯 磁场中的圆周运动

    基本公式：

    • 半径：$r = \frac{mv}{qB} = \frac{p}{qB} = \frac{\sqrt{2mE_k}}{qB}$

    • 周期：$T = \frac{2\pi m}{qB}$（与速度无关）

    • 运动时间：$t = \frac{\theta}{2\pi}T = \frac{\theta m}{qB}$

    边界问题：

    • 直线边界：入射角 = 出射角，对称性

    • 圆形边界：指向圆心入射，背离圆心出射

    • 矩形边界：找圆心，定半径，画轨迹

    加速度减小≠速度减小    		 加速度减小就认为速度减小
    向心力
    vs
    离心力    		 向心力是实际受力
    离心力是惯性效应，不是真实力    		 画受力分析时画离心力
    功
    vs
    功率    		 功是能量转化的量度
    功率是做功的快慢
    $W = Pt$，但$W$大不一定$P$大    		 功率大就认为做功多
    场强
    vs
    电势    		 场强是矢量，描述力的性质
    电势是标量，描述能的性质
    $E$大≠$\varphi$大    		 认为场强大的地方电势高
    磁通量
    vs
    磁通量变化率    		 $\Phi = BS$
    $\frac{\Delta\Phi}{\Delta t}$
    感应电动势与变化率成正比    		 磁通量为0就认为感应电动势为0

    二、常见计算错误

    1. 单位换算错误
    • cm→m忘记除以100
    • g→kg忘记除以1000
    • km/h→m/s乘以3.6（应该是除以3.6）
    • 角度→弧度：$\theta(rad) = \theta(°) \times \frac{\pi}{180}$

    2. 矢量方向错误
    • 规定正方向后，与正方向相反的物理量要加负号
    • 功的正负：力与位移夹角大于90°做负功
    • 电势的正负：与零势面选取有关

    3. 有效数字错误
    • 题目要求保留2位有效数字，结果写成3位
    • 中间计算保留过多位数，最后才四舍五入
    • 测量结果的有效数字与测量工具精度不匹配

    4. 公式适用条件不清
    • $v^2 - v_0^2 = 2ax$只适用于匀变速直线运动
    • $F = ma$只适用于惯性系
    • $E = BLv$要求B、L、v三者互相垂直

    三、图像理解误区

    图像类型 横纵坐标 斜率意义 面积意义 常见错误
    $v-t$图 t-v 加速度a 位移x 把曲线当成运动轨迹
    $x-t$图 t-x 速度v 无 斜率为负就认为物体反向运动（可能正方向选取问题）
    $a-t$图 t-a 加速度变化率 速度变化量$\Delta v$ 面积意义记错
    $F-x$图 x-F 无 功W 力随位移变化时，直接用$W=Fx$计算
    $E_k-x$图 x-$E_k$ 合外力F 无 由$E_k = \frac{1}{2}mv^2$认为斜率是mv
    $\Phi-t$图 t-$\Phi$ 感应电动势E（负值） 无 斜率绝对值就是E，不考虑方向

    四、实验常见错误

    1. 纸带数据处理
    • 单位未换算：纸带上标的是cm，计算时要用m
    • 时间间隔搞错：若每5个点取一个计数点，$T = 0.1s$不是$0.02s$
    • 逐差法用错：应该是$(x_4+x_5+x_6)-(x_1+x_2+x_3)$除以$9T^2$，不是$3T^2$

    2. 电路实验
    • 电表量程选错：指针偏转太小（量程太大）或超过满偏（量程太小）
    • 滑动变阻器接错：限流式接成$-\frac{1}{1}$，分压式接成$-\frac{1}{1}$
    • 正负极接反：电流从负接线柱流入
    • 读数错误：电压表/电流表最小刻度是0.1还是0.2，估读位数不同

    3. 作图问题
    • 坐标轴未标注物理量和单位
    • 描点过大或过小，看不清
    • 连线过所有点（应该是平滑曲线或直线，不要求过所有点）
    • 求斜率时取的数据点不是图线上的点

    五、审题常见失误

    1. 忽略关键条件
    • "光滑"→忽略摩擦力
    • "从静止开始"→$v_0 = 0$
    • "恰好"、"刚好"→临界条件
    • "不计空气阻力"→只受重力
    • "弹性碰撞"→动能守恒

    2. 误解题意
    • "第几秒内"vs"几秒内"：第3秒内是$t \in [2,3]$，3秒内是$t \in [0,3]$
    • "位移"vs"路程"：位移是矢量，路程是标量
    • "平均速度"vs"瞬时速度"：平均速度$\bar{v} = \frac{\Delta x}{\Delta t}$
    • "增加"vs"增加到"：增加3倍是原来的4倍，增加到3倍是原来的3倍

    3. 漏看隐含条件
    • 圆周运动最高点：$v \geq \sqrt{gR}$（绳模型）或$v \geq 0$（杆模型）
    • 恰好不滑出：到达边缘时共速
    • 弹簧原长：弹性势能为0，但可能有弹力

    六、答题规范问题

    1. 计算题失分点
    • 不写"解"和"答"
    • 只有公式没有代入数据的过程
    • 结果不带单位或单位错误
    • 有效数字与题目要求不符
    • 字母符号混用（如$m$和$M$、$v$和$V$）

    2. 符号规范
    • 物理量用斜体：$m$、$v$、$a$
    • 单位用正体：m、s、kg
    • 下标规范：$v_0$（初速度）、$v_t$（末速度）、$E_k$（动能）
    • 不要自创符号，用题目给的或通用符号

    3. 文字表述
    • "由牛顿第二定律得"→"由牛顿第二定律$F=ma$得"
    • "由图可知"→要具体说明从图中得到什么信息
    • "解得"→要有计算过程，不能直接写答案
