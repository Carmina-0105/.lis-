📐 模块一：匀变速直线运动

1.1 五大基本公式（必须烂熟于心）

速度公式：$v = v_0 + at$

位移公式：$x = v_0t + \frac{1}{2}at^2$

速度-位移关系：$v^2 - v_0^2 = 2ax$

平均速度公式：$\bar{v} = \frac{v_0 + v}{2} = \frac{x}{t}$

位移差公式：$\Delta x = aT^2$（逐差法核心）

🔑 公式选择黄金法则：
• 缺位移 $x$ → 用 $v = v_0 + at$
• 缺末速度 $v$ → 用 $x = v_0t + \frac{1}{2}at^2$
• 缺时间 $t$ → 用 $v^2 - v_0^2 = 2ax$
• 缺加速度 $a$ → 用 $\bar{v} = \frac{x}{t}$
• 纸带数据处理 → 用 $\Delta x = aT^2$
⚠️ 常见陷阱：
1. 刹车问题：先判断停止时间 $t_0 = \frac{v_0}{a}$，若 $t > t_0$ 则位移按 $t_0$ 计算
2. 往返运动：注意速度方向变化，建议设定正方向（通常取初速度方向）
3. 竖直上抛：$a = -g$，上升到最高点 $v=0$，时间 $t = \frac{v_0}{g}$

1.2 v-t图像深度解析

v-t图像：斜率=加速度，面积=位移，交点=速度相等（距离极值点）

图像要素 物理意义 数学表达 应用技巧
斜率 加速度 $a$ $a = \frac{\Delta v}{\Delta t}$ 斜率正负表方向，绝对值表大小
面积 位移 $x$ $x = \int v \, dt$ t轴上方面积为正，下方为负
截距 初速度 $v_0$ $v(0) = v_0$ 注意单位换算（如km/h→m/s）
交点 速度相等 $v_1 = v_2$ 追及问题中距离极值点
💡 图像法解题三步走：
1. 定性分析：看趋势（加速/减速/匀速）
2. 定量计算：利用几何关系（相似三角形、梯形面积）
3. 特殊点：拐点（加速度突变）、交点（速度相等）、零点（反向运动）

1.3 追及相遇问题（高频考点）

临界条件：$\Delta x = x_{前} - x_{后} = d_0$（初始距离）

恰好追上：$v_{后} = v_{前}$ 且 $x_{后} = x_{前} + d_0$

恰好不撞：$v_{后} = v_{前}$ 且 $x_{后} = x_{前} + d_0 - L$（车长）

🎯 追及问题四步法：
1. 写出两物体位移表达式 $x_1(t)$、$x_2(t)$
2. 建立方程 $x_2 - x_1 = d_0$
3. 讨论判别式 $\Delta$：$\Delta > 0$（两次相遇）、$\Delta = 0$（一次相遇）、$\Delta < 0$（不相遇）
4. 检验解的合理性（时间 $t > 0$，速度方向等）
⚖️ 模块二：相互作用与物体平衡

2.1 力的合成与分解

平行四边形定则：$F = \sqrt{F_1^2 + F_2^2 + 2F_1F_2\cos\theta}$

正交分解：$F_x = F\cos\theta$，$F_y = F\sin\theta$

三力平衡：任意两力合力与第三力等大反向

力的合成与分解示意图

方法 适用条件 操作步骤 注意事项
平行四边形法 两个已知力 共起点作平行四边形，对角线为合力 注意角度是力的夹角
三角形法 三力平衡 三力首尾相接构成闭合三角形 可用正弦定理、余弦定理
正交分解法 多力作用（≥3个） 建立坐标系→分解→列平衡方程 坐标轴尽量与多数力重合
相似三角形法 几何关系明显 找力三角形与几何三角形相似 对应边成比例

2.2 动态平衡问题（极重要）

🔄 动态平衡三模型：
1. 一力恒定、一力方向不变：用三角形法则，观察第三力变化
2. 一力恒定、两力方向都变：找相似三角形，利用几何关系
3. 一力恒定、另两力夹角不变：画圆法（辅助圆），力矢量端点在圆上
🎨 图解法口诀：
"恒力为基画始终，变力方向定轨迹，
三角相似比例定，辅助圆里找极值。"

2.3 摩擦力深度剖析

滑动摩擦力：$f = \mu N$（与相对运动方向相反）

静摩擦力：$0 \leq f_{静} \leq f_{max} = \mu_s N$（与相对运动趋势方向相反）

最大静摩擦力略大于滑动摩擦力：$f_{max} \approx 1.1f_{滑}$

⚠️ 摩擦力判断四步法：
1. 先看运动状态：静止→静摩擦，滑动→滑动摩擦
2. 再用平衡/牛顿第二定律求大小
3. 方向判断：与相对运动（趋势）方向相反
4. 注意：摩擦力可以是动力，也可以是阻力！
🚀 模块三：牛顿运动定律

3.1 牛顿三定律核心

第一定律（惯性定律）： $F_{合} = 0 \Leftrightarrow$ 静止或匀速直线运动

第二定律（动力学核心）： $F_{合} = ma$（瞬时性、矢量性、独立性）

第三定律（作用力与反作用力）： $F_{12} = -F_{21}$（等大、反向、共线、异体）

🔑 第二定律理解的五个维度：
• 瞬时性：$a$ 与 $F$ 同时产生、同时变化、同时消失
• 矢量性：$a$ 方向与 $F_{合}$ 方向相同
• 独立性：每个力独立产生加速度，$a_{合} = \sum a_i$
• 同体性：$F$、$m$、$a$ 对应同一物体、同一参考系
• 局限性：仅适用于宏观、低速、惯性系

3.2 连接体问题（整体法与隔离法）

整体法：$F_{合外} = (m_1 + m_2 + \cdots)a$（求加速度、外力）

隔离法：$F_{合i} = m_i a$（求内力）

内力公式：$T = \frac{m_{远}}{m_{总}}F$（水平光滑连接体）

🎯 连接体问题解题策略：
1. 求加速度 $a$：整体法，忽略内力
2. 求内力 $T$：隔离受力少的物体
3. 临界问题：$f_{max} = \mu N$，恰好滑动时 $f = f_{max}$
4. 多过程问题：分段分析，注意连接点速度、加速度关系
模型 示意图 加速度 绳张力
水平光滑 $F \rightarrow m_1 - m_2$ $a = \frac{F}{m_1+m_2}$ $T = \frac{m_2}{m_1+m_2}F$
竖直提升 $F \uparrow m_1 - m_2$ $a = \frac{F-(m_1+m_2)g}{m_1+m_2}$ $T = m_2(g+a)$
斜面光滑 $F \rightarrow m_1 /\ m_2$（斜面） $a = \frac{F-(m_1+m_2)g\sin\theta}{m_1+m_2}$ $T = m_2(g\sin\theta + a)$

3.3 传送带与板块模型

🚧 传送带问题分析框架：
1. 初态分析：$v_0$ 与 $v_{带}$ 关系，判断相对运动方向
2. 受力分析：滑动摩擦力 $f = \mu mg$，方向与相对运动方向相反
3. 运动分析：匀变速直到共速，之后可能匀速
4. 痕迹计算：$\Delta x = x_{带} - x_{物}$（相对位移）
5. 能量分析：$Q = f \cdot \Delta x$（摩擦生热）
📊 传送带问题速查表：
• 水平传送带：加速段 $x = \frac{v^2-v_0^2}{2\mu g}$，若 $x < L$ 则先加速后匀速
• 倾斜传送带：比较 $\mu$ 与 $\tan\theta$，判断能否相对静止
• 划痕长度 = 相对位移（注意往返情况取最大值）

🔗 模块融合：运动学+力学+牛顿定律

融合使用技巧

1. 已知力求运动：受力分析 → $F_{合} = ma$ → 运动学公式 → 求解

2. 已知运动求力：运动学公式求 $a$ → $F_{合} = ma$ → 受力分析 → 求解

3. 多过程问题：划分阶段 → 找连接点（速度、位移）→ 分段列方程

4. 临界极值：找出临界条件（$f_{max}$、$T_{max}$、$v_{共}$）→ 列方程求解
