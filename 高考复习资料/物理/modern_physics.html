🌡️ 模块十二：热学

12.1 分子动理论

阿伏伽德罗常数：$N_A = 6.02 \times 10^{23} \text{mol}^{-1}$

分子质量：$\displaystyle m_0 = \frac{M}{N_A} = \frac{\rho V_m}{N_A}$

分子体积（固体/液体）：$\displaystyle V_0 = \frac{V_m}{N_A} = \frac{M}{\rho N_A}$

分子数：$N = nN_A = \frac{m}{M}N_A = \frac{V}{V_m}N_A$

🧬 分子动理论三要点：
• 物质由大量分子组成（$N_A$ 是桥梁）
• 分子永不停息做无规则运动（扩散、布朗运动）
• 分子间存在相互作用力（引力和斥力同时存在）
⚠️ 分子力与分子势能：
• 平衡位置 $r = r_0$：$F_{引} = F_{斥}$，$F_{合} = 0$，$E_p$ 最小
• $r < r_0$：斥力为主，$r \downarrow$ → $F_{斥} \uparrow$，$E_p \uparrow$
• $r > r_0$：引力为主，$r \uparrow$ → $F_{引}$ 先增后减，$E_p \uparrow$
• $r > 10r_0$：$F \approx 0$，$E_p \approx 0$

12.2 气体实验定律与理想气体状态方程

玻意耳定律（等温）：$p_1V_1 = p_2V_2$ 或 $pV = C$

查理定律（等容）：$\displaystyle \frac{p_1}{T_1} = \frac{p_2}{T_2}$ 或 $p = CT$（$T$ 为热力学温度）

盖-吕萨克定律（等压）：$\displaystyle \frac{V_1}{T_1} = \frac{V_2}{T_2}$ 或 $V = CT$

理想气体状态方程：$\displaystyle \frac{p_1V_1}{T_1} = \frac{p_2V_2}{T_2}$ 或 $pV = nRT = \frac{m}{M}RT$

克拉珀龙方程：$pV = NkT$（$k = 1.38 \times 10^{-23} \text{J/K}$ 为玻尔兹曼常量）

🎯 气体问题解题模板：
1. 选对象：确定研究的气体（质量是否变化）
2. 找状态：明确初态 $(p_1, V_1, T_1)$ 和末态 $(p_2, V_2, T_2)$
3. 列方程：根据过程特点选定律（等温/等容/等压/绝热）
4. 解方程：注意单位统一（$T$ 必须用开尔文）

12.3 热力学定律

热力学第一定律：$\Delta U = Q + W$

符号规定：吸热 $Q > 0$，放热 $Q < 0$；外界对系统做功 $W > 0$，系统对外做功 $W < 0$

理想气体内能：$U = N \cdot \frac{i}{2}kT = \frac{m}{M} \cdot \frac{i}{2}RT$（$i$ 为自由度，仅与温度有关）

热力学第二定律：热量不能自发地从低温传向高温（克劳修斯表述）

过程 特点 $\Delta U$ $Q$ $W$
等温过程 $T = $ 常数 0 $-W$ $-nRT\ln\frac{V_2}{V_1}$
等容过程 $V = $ 常数 $Q$ $nC_V\Delta T$ 0
等压过程 $p = $ 常数 $nC_V\Delta T$ $nC_p\Delta T$ $-p\Delta V$
绝热过程 $Q = 0$ $W$ 0 $\Delta U$
💡 模块十三：光学

13.1 几何光学

折射定律：$n_1\sin\theta_1 = n_2\sin\theta_2$（斯涅尔定律）

折射率：$\displaystyle n = \frac{c}{v} = \frac{\sin i}{\sin r} = \frac{1}{\sin C}$（$C$ 为临界角）

全反射条件：光密→光疏，$i \geq C$，$\sin C = \frac{1}{n}$

视深公式：$h' = \frac{h}{n}$（从光疏看光密，视深变浅）

光的折射与全反射

🪞 光学元件成像规律：
• 平面镜：等大、等距、虚像、左右相反
• 平行玻璃砖：侧移但不改变传播方向，出射光与入射光平行
• 三棱镜：向底边偏折，色散（$n_紫 > n_红$，$v_紫 < v_红$）
• 光导纤维：全反射原理，内芯折射率 > 外套折射率

13.2 物理光学

双缝干涉：$\Delta x = \frac{L}{d}\lambda$（条纹间距，$L$ 为缝到屏距离，$d$ 为缝间距）

亮纹条件：$\Delta r = n\lambda$（$n = 0, 1, 2, \cdots$）

暗纹条件：$\Delta r = (2n+1)\frac{\lambda}{2}$（$n = 0, 1, 2, \cdots$）

薄膜干涉：光程差 $\Delta = 2nd + \frac{\lambda}{2}$（半波损失）

现象 条件 条纹特点 应用
双缝干涉 相干光（频率相同、相位差恒定） 等间距、等亮度、明暗相间 测波长、检查平整度
薄膜干涉 薄膜前后表面反射光叠加 等厚条纹（劈尖）或等倾条纹 增透膜、检查表面平整
单缝衍射 缝宽与波长可比拟 中央亮纹最宽最亮，两侧不等距 -
光的偏振 横波特性 只有特定振动方向的光通过 偏振片、3D眼镜
⚠️ 电磁波谱（按波长从大到小）：
无线电波 → 微波 → 红外线 → 可见光（红橙黄绿蓝靛紫）→ 紫外线 → X射线 → $\gamma$射线
• 波长 $\lambda$ 越大，频率 $f$ 越小，能量 $E = hf$ 越小，波动性越强
• 波长 $\lambda$ 越小，粒子性越强，穿透能力越强（$\gamma$ 射线最强）
⚛️ 模块十四：近代物理

14.1 光电效应

爱因斯坦光电效应方程：$h\nu = E_{km} + W_0$ 或 $\frac{1}{2}mv_m^2 = h\nu - W_0$

截止频率：$\nu_0 = \frac{W_0}{h}$（入射光频率必须 $\nu > \nu_0$ 才能发生光电效应）

遏止电压：$eU_c = E_{km} = h\nu - W_0$

普朗克常量：$h = 6.63 \times 10^{-34} \text{J·s}$

🔆 光电效应四规律：
1. 瞬时性：$\tau < 10^{-9} \text{s}$，光电子立即逸出
2. 截止频率：只有 $\nu > \nu_0$ 才能发生，与光强无关
3. 最大初动能：$E_{km}$ 只与 $\nu$ 有关，与光强无关
4. 光电流强度：饱和光电流与入射光强度成正比

14.2 原子结构

玻尔理论假设：定态假设、跃迁假设、轨道量子化假设

能级公式：$\displaystyle E_n = \frac{E_1}{n^2}$（氢原子，$E_1 = -13.6 \text{eV}$）

半径公式：$\displaystyle r_n = n^2r_1$（$r_1 = 0.53 \times 10^{-10} \text{m}$）

跃迁频率：$h\nu = E_m - E_n$（$m > n$，发射光子；$m < n$，吸收光子）

🎯 氢原子跃迁问题：
• 从 $n$ 能级向低能级跃迁：最多发射 $C_n^2 = \frac{n(n-1)}{2}$ 种频率光子
• 电离能：从 $n$ 能级电离需要能量 $\geq |E_n|$
• 光子吸收：能量必须恰好等于能级差（共振吸收）或大于电离能
• 电子碰撞：电子能量 $\geq$ 能级差即可（可剩余动能）

14.3 原子核与核反应

质能方程：$E = mc^2$，$\Delta E = \Delta mc^2$

质量亏损：$\Delta m = (m_{反应前} - m_{反应后}) > 0$ 时释放能量

半衰期：$\displaystyle N = N_0\left(\frac{1}{2}\right)^{t/\tau}$，$m = m_0\left(\frac{1}{2}\right)^{t/\tau}$

平均结合能：越大核越稳定，铁最稳定

核反应类型 方程 特点 实例
$\alpha$衰变 $^A_Z X \rightarrow ^{A-4}_{Z-2}Y + ^4_2He$ 质量数-4，电荷数-2 $^{238}_{92}U \rightarrow ^{234}_{90}Th + \alpha$
$\beta$衰变 $^A_Z X \rightarrow ^A_{Z+1}Y + ^0_{-1}e$ 质量数不变，电荷数+1 $^{14}_6C \rightarrow ^{14}_7N + \beta$
核裂变 重核分裂为中等质量核 链式反应，释放大量能量 $^{235}_{92}U + ^1_0n \rightarrow \cdots$
核聚变 轻核聚变为较重核 需要高温高压，清洁能源 $^2_1H + ^3_1H \rightarrow ^4_2He + n$
⚠️ 核反应守恒律：
• 质量数守恒（核子数守恒）
• 电荷数守恒（质子数守恒）
• 能量守恒（包括质能转换）
• 动量守恒

14.4 波粒二象性与物质波

光子能量：$E = h\nu = \frac{hc}{\lambda}$

光子动量：$\displaystyle p = \frac{h}{\lambda} = \frac{E}{c}$

物质波（德布罗意波）：$\displaystyle \lambda = \frac{h}{p} = \frac{h}{mv}$

不确定关系：$\Delta x\Delta p \geq \frac{h}{4\pi}$

🌊 波粒二象性理解：
• 光：干涉衍射证明波动性，光电效应证明粒子性
• 实物粒子：电子衍射实验证明波动性
• 概率波：物质波是概率波，波强代表粒子出现概率
• 互补原理：波动性和粒子性不会在同一实验中同时表现

跨模块联系

选考模块一：热学（3-3）

🎯 分子动理论要点

1. 分子大小与数量

• 分子直径数量级：$10^{-10}m$（0.1nm）

• 阿伏伽德罗常数：$N_A = 6.02 \times 10^{23} mol^{-1}$

• 分子质量：$m_0 = \frac{M}{N_A}$（M为摩尔质量）

• 分子体积：$V_0 = \frac{V_m}{N_A} = \frac{M}{\rho N_A}$（固体、液体）

2. 分子热运动

• 扩散现象：分子永不停息地做无规则运动

• 布朗运动：悬浮微粒的无规则运动，反映液体分子的无规则运动

• 温度是分子平均动能的标志：$\bar{E_k} = \frac{3}{2}kT$

3. 分子间作用力

• 引力和斥力同时存在，都随距离增大而减小

• $r = r_0$（$10^{-10}m$）：引力=斥力，分子力为0

• $r < r_0$：斥力>引力，表现为斥力

• $r > r_0$：引力>斥力，表现为引力

🎯 固体、液体和气体性质

1. 固体分类

性质 晶体 非晶体
外形 规则几何形状 无规则形状
熔点 有确定熔点 无确定熔点
物理性质 各向异性 各向同性
典型物质 石英、云母、食盐、金属 玻璃、松香、沥青

2. 液晶特性

• 具有液体的流动性

• 具有晶体的各向异性

• 对光、电、磁敏感，用于显示器

3. 液体表面张力

• 产生原因：表面层分子间距大，表现为引力

• 方向：与液面相切，垂直于分界线

• 效果：使液面收缩，形成球形（体积一定时表面积最小）

• 应用：毛细现象、肥皂泡、昆虫在水面行走

🎯 气体实验定律

玻意耳定律（等温变化）：
$p_1V_1 = p_2V_2$ 或 $pV = C$

查理定律（等容变化）：
$\frac{p_1}{T_1} = \frac{p_2}{T_2}$ 或 $\frac{p}{T} = C$

盖-吕萨克定律（等压变化）：
$\frac{V_1}{T_1} = \frac{V_2}{T_2}$ 或 $\frac{V}{T} = C$

理想气体状态方程：
$\frac{p_1V_1}{T_1} = \frac{p_2V_2}{T_2}$ 或 $pV = nRT = \frac{m}{M}RT$

解题步骤：

1. 确定研究对象（哪部分气体）

2. 确定初末状态的p、V、T

3. 判断变化过程（等温、等容、等压）

4. 选择相应定律列方程

5. 注意：T必须用热力学温度（K），$T = t + 273$

🎯 热力学定律

热力学第一定律：$\Delta U = Q + W$

• $\Delta U$：内能变化（增加为正，减少为负）

• $Q$：热量（吸热为正，放热为负）

• $W$：做功（外界对系统做功为正，系统对外做功为负）

热力学第二定律（两种表述）：

• 克劳修斯表述：热量不能自发地从低温物体传到高温物体

• 开尔文表述：不可能从单一热源吸热，使之完全变为功而不产生其他影响

• 本质：与热现象有关的宏观过程具有方向性

热机效率：$\eta = \frac{W}{Q_1} = \frac{Q_1 - Q_2}{Q_1} = 1 - \frac{Q_2}{Q_1} < 1$

选考模块二：机械振动与机械波（3-4）

🎯 简谐运动

1. 描述简谐运动的物理量

• 振幅A：离开平衡位置的最大距离，表示振动强弱

• 周期T：完成一次全振动的时间，$T = \frac{1}{f}$

• 频率f：单位时间内完成全振动的次数

• 相位：描述振动状态的物理量

2. 简谐运动的表达式

位移：$x = A\sin(\omega t + \varphi_0)$
其中 $\omega = \frac{2\pi}{T} = 2\pi f$

速度和加速度：
$v = \frac{dx}{dt} = A\omega\cos(\omega t + \varphi_0)$
$a = \frac{dv}{dt} = -A\omega^2\sin(\omega t + \varphi_0) = -\omega^2 x$

3. 简谐运动的能量

• 动能：$E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2(A^2 - x^2)$

• 势能：$E_p = \frac{1}{2}kx^2 = \frac{1}{2}m\omega^2x^2$

• 总能量：$E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2A^2$（守恒）

• 平衡位置：动能最大，势能为0

• 最大位移处：动能为0，势能最大

🎯 单摆

周期公式：
$T = 2\pi\sqrt{\frac{L}{g}}$

适用条件：摆角 $\theta < 5°$，$L \gg$ 摆球直径

应用：

• 测重力加速度：$g = \frac{4\pi^2L}{T^2}$

• 等效摆长：$L_{等效} = L\sin\theta$（圆锥摆）

• 等效重力加速度：$g' = g + a$（加速上升）或 $g' = |g - a|$（加速下降）

🎯 机械波

1. 波的基本概念

• 波速：$v = \frac{\lambda}{T} = \lambda f$

• 波长$\lambda$：相邻两个振动情况完全相同的质点间距离

• 波的传播：振动形式和能量的传播，介质质点不随波迁移

2. 波的图像（y-x图）

• 横轴：各质点的平衡位置坐标

• 纵轴：各质点的位移

• 图像表示某一时刻各质点的位移分布

3. 振动图像（y-t图）vs 波动图像（y-x图）

比较 振动图像 波动图像
研究对象 一个质点 介质中所有质点
横轴 时间t 位置x
纵轴 位移y 位移y
物理意义 一个质点不同时刻的位移 不同时刻各质点的位移
相邻峰值间距 周期T 波长λ

🎯 波的特有现象

1. 波的叠加

• 几列波相遇时，每列波保持各自状态继续传播

• 相遇区域，质点的位移等于各列波单独引起的位移的矢量和

2. 波的干涉

• 条件：频率相同、振动方向相同、相位差恒定

• 加强点：波程差 $\Delta r = n\lambda$（$n = 0, 1, 2...$），振幅最大

• 减弱点：波程差 $\Delta r = (2n+1)\frac{\lambda}{2}$，振幅最小（可能为0）

3. 波的衍射

• 明显衍射条件：障碍物或孔的尺寸比波长小或与波长相差不多

• 衍射是波特有的现象，证明波具有波动性

4. 多普勒效应

• 波源与观察者相对运动时，观察者接收到的频率发生变化

• 相互靠近：接收频率 $f' > f$（音调变高）

• 相互远离：接收频率 $f' < f$（音调变低）

• 应用：测速雷达、医学超声（B超）

选考模块三：光学（3-4）

🎯 光的折射与全反射

1. 折射定律

$n_1\sin\theta_1 = n_2\sin\theta_2$

折射率：$n = \frac{\sin i}{\sin r} = \frac{c}{v} = \frac{\lambda_0}{\lambda}$

2. 全反射

• 条件：光从光密介质射向光疏介质，入射角 $i \geq$ 临界角C

• 临界角：$\sin C = \frac{1}{n}$

• 应用：光纤通信、内窥镜、海市蜃楼

3. 棱镜与色散

• 白光通过棱镜发生色散，分解成七色光

• 折射率：$n_{紫} > n_{红}$

• 传播速度：$v_{紫} < v_{红}$

• 临界角：$C_{紫} < C_{红}$

🎯 光的干涉、衍射和偏振

1. 双缝干涉

条纹间距：
$\Delta x = \frac{L}{d}\lambda$

L：双缝到屏的距离
d：双缝间距
λ：光的波长

• 亮纹：波程差 $\Delta r = n\lambda$（$n = 0, \pm1, \pm2...$）

• 暗纹：波程差 $\Delta r = (2n+1)\frac{\lambda}{2}$

• 条纹等间距、等宽度、等亮度

2. 薄膜干涉

• 原理：薄膜前后表面反射光叠加

• 光程差：$\Delta = 2nd + \frac{\lambda}{2}$（考虑半波损失）

• 应用：增透膜、增反膜、检查平面平整度

3. 光的衍射

• 单缝衍射：中央亮纹最宽最亮，两侧条纹不等间距

• 圆孔衍射：出现明暗相间的圆环

• 泊松亮斑：圆盘阴影中心出现亮斑，证明光的波动性

4. 光的偏振

• 自然光：各个方向振动，强度相同

• 偏振光：只沿某一特定方向振动

• 偏振片：只允许特定方向振动的光通过

• 应用：太阳镜、3D电影、液晶显示器
