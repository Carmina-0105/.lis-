🧲 模块九：磁场与安培力

9.1 磁感应强度与安培力

定义：$\displaystyle B = \frac{F}{IL}$（$B \perp I$ 时，$B$ 与 $F$、$I$、$L$ 无关）

安培力：$F = BIL\sin\theta$（$\theta$ 为 $B$ 与 $I$ 夹角）

方向：左手定则（$B$ 穿手心，四指 $I$，拇指 $F$）

磁通量：$\Phi = BS\cos\theta = BS_\perp$（$\theta$ 为 $B$ 与法线夹角）

🖐️ 左手定则应用要点：
• 磁感线垂直穿入手心（$B$ 的方向）
• 四指指向电流方向（正电荷运动方向）
• 拇指指向安培力（洛伦兹力）方向
• 注意：$F$ 始终垂直于 $B$ 和 $I$ 所确定的平面

9.2 带电粒子在磁场中的运动

洛伦兹力：$f = qvB\sin\theta$（始终不做功！）

圆周运动半径：$\displaystyle r = \frac{mv}{qB}$

周期：$\displaystyle T = \frac{2\pi m}{qB}$（与速度无关！）

运动时间：$\displaystyle t = \frac{\theta}{2\pi}T = \frac{\theta m}{qB}$（$\theta$ 为圆心角，弧度制）

带电粒子在匀强磁场中的圆周运动

🎯 带电粒子圆周运动解题步骤：
1. 找圆心：$f \perp v$，圆心在洛伦兹力垂线上；或弦的中垂线
2. 求半径：$r = \frac{mv}{qB}$，或几何关系 $r = \frac{L}{2\sin\theta}$
3. 定圆心角：速度偏转角 = 圆心角 = 2×弦切角
4. 求时间：$t = \frac{\theta m}{qB}$ 或 $t = \frac{s}{v} = \frac{r\theta}{v}$
⚠️ 常见边界磁场问题：
• 直线边界：入射角 = 出射角，对称性
• 平行边界：临界条件（与边界相切）
• 圆形边界：沿径向射入必沿径向射出
• 矩形边界：找临界轨迹（与边相切）

9.3 带电粒子在复合场中的运动

速度选择器：$qE = qvB$ → $\displaystyle v = \frac{E}{B}$（与 $q$、$m$ 无关）

质谱仪：$r = \frac{mv}{qB}$，$\frac{q}{m} = \frac{v}{Br}$

霍尔效应：$U_H = \frac{IB}{nqd} = \frac{RB}{d}I$（$R_H = \frac{1}{nq}$ 为霍尔系数）

回旋加速器：$T = \frac{2\pi m}{qB}$，$E_{kmax} = \frac{q^2B^2R^2}{2m}$

装置 原理 关键公式 应用
速度选择器 正交电磁场 $v = E/B$ 筛选特定速度粒子
质谱仪 电场加速+磁场偏转 $\frac{q}{m} = \frac{2U}{B^2r^2}$ 测比荷、分析同位素
回旋加速器 电场加速+磁场偏转 $E_k = \frac{q^2B^2R^2}{2m}$ 高能粒子加速
磁流体发电机 等离子体在磁场中偏转 $E = Bdv$ 发电
⚡ 模块十：电磁感应

10.1 法拉第电磁感应定律

感应电动势：$\displaystyle E = n\frac{\Delta\Phi}{\Delta t}$（$n$ 为匝数）

动生电动势：$E = BLv$（$B$、$L$、$v$ 两两垂直）

转动切割：$\displaystyle E = \frac{1}{2}BL^2\omega$（绕一端转动）

感应电荷量：$\displaystyle q = n\frac{\Delta\Phi}{R_{总}}$（与过程无关，只看初末）

🔑 感应电动势两类计算：
• 感生电动势：$E = nS\frac{\Delta B}{\Delta t}$（磁场变化，$B-t$ 图斜率）
• 动生电动势：$E = BLv$（切割磁感线）
• 有效长度：导体两端连线在垂直于 $v$ 方向的投影

10.2 楞次定律与右手定则

楞次定律：感应电流的磁场总是阻碍引起感应电流的磁通量变化

"增反减同"：$\Phi$ 增加，$B_{感}$ 与 $B_{原}$ 反向；$\Phi$ 减少，同向

"来拒去留"：相对运动，安培力阻碍相对运动

右手定则：$B$ 穿手心，拇指 $v$，四指 $I$（仅切割情况）

🎯 楞次定律判断三步法：
1. 确定原磁场 $B_{原}$ 方向
2. 判断 $\Phi$ 变化（增加或减少）
3. 用"增反减同"确定 $B_{感}$ 方向，进而确定 $I_{感}$ 方向

10.3 自感与互感

自感电动势：$\displaystyle E = L\frac{\Delta I}{\Delta t}$（$L$ 为自感系数）

自感系数：与线圈匝数、大小、形状、有无铁芯有关

通电自感：电流逐渐增大，$L$ 相当于断路 → 电阻 → 导线

断电自感：$L$ 相当于电源，与 $L$ 构成回路的灯可能闪亮

⚠️ 自感现象分析要点：
• 稳定时：$L$ 相当于导线（电阻为 $R_L$）
• 变化时：$L$ 阻碍电流变化（$I$ 只能渐变，不能突变）
• 断电闪亮条件：$I_L > I_{灯}$（稳定时通过 $L$ 的电流大于灯）

10.4 电磁感应中的动力学与能量

安培力：$F = BIL = \frac{B^2L^2v}{R}$（阻碍相对运动）

牛顿第二定律：$F - \frac{B^2L^2v}{R} = ma$（导体棒加速）

能量守恒：$W_{外} = \Delta E_k + Q$（$Q$ 为焦耳热）

电荷量：$q = \frac{BLx}{R}$（$x$ 为位移）

🚀 单棒模型总结：
1. 阻尼式：给初速度，$F_安 = \frac{B^2L^2v}{R}$，减速至静止
2. 发电式：恒力拉动，$F - \frac{B^2L^2v}{R} = ma$，最终匀速 $v_m = \frac{FR}{B^2L^2}$
3. 电动式：含电源，类似发电式
4. 电容式：$I = \frac{\Delta Q}{\Delta t} = CBL\frac{\Delta v}{\Delta t} = CBLa$，匀加速运动
〰️ 模块十一：交变电流

11.1 正弦式交变电流

瞬时值：$e = E_m\sin\omega t = NBS\omega\sin\omega t$

峰值：$E_m = NBS\omega = N\Phi_m\omega$

有效值：$\displaystyle E = \frac{E_m}{\sqrt{2}}$（正弦/余弦，一个周期内）

平均值：$\bar{E} = n\frac{\Delta\Phi}{\Delta t}$（计算电荷量）

正弦交流电波形与有效值

物理量 公式 适用场景
瞬时值 $e = E_m\sin\omega t$ 计算某时刻受力、电流
峰值 $E_m = NBS\omega$ 电容器耐压值
有效值 $E = E_m/\sqrt{2}$ 电表读数、功率计算、铭牌标注
平均值 $\bar{E} = n\Delta\Phi/\Delta t$ 计算通过导体的电荷量

11.2 变压器与远距离输电

变压比：$\displaystyle \frac{U_1}{U_2} = \frac{n_1}{n_2}$

变流比：$\displaystyle \frac{I_1}{I_2} = \frac{n_2}{n_1}$（理想变压器，一个副线圈）

功率关系：$P_入 = P_出$（理想变压器无损耗）

输电损失：$P_{损} = I^2R = \left(\frac{P}{U}\right)^2R$（高压输电减小损耗）

🎯 变压器动态分析：
1. $U_1$ 决定 $U_2$（$U_2 = \frac{n_2}{n_1}U_1$）
2. $I_2$ 决定 $I_1$（$I_1 = \frac{n_2}{n_1}I_2$）
3. $P_2$ 决定 $P_1$（$P_1 = P_2$）
口诀："电压原决定副，电流副决定原，功率副决定原"
⚠️ 远距离输电要点：
• 升压变压器：减小输电电流，$P_{损} = I^2R$ 大幅减小
• 降压变压器：将高压降至用户电压
• 输电电流：$I = \frac{P_{输送}}{U_{输送}}$
• 电压损失：$\Delta U = IR$，功率损失：$\Delta P = I^2R$

🔗 电磁学（下）融合总结

磁场+电磁感应+电路综合

1. 电磁感应+电路：感应电动势作为电源，$E = BLv$ 或 $E = n\Delta\Phi/\Delta t$，画等效电路

2. 电磁感应+力学：$F_安 = BIL = \frac{B^2L^2v}{R}$，牛顿第二定律，动量定理（$BLq = m\Delta v$）

3. 电磁感应+能量：$Q = I^2Rt$ 或 $Q = W_{安}$（克服安培力做功），能量守恒

4. 交变电流+变压器：注意有效值计算，远距离输电功率损失 $P_{损} = (P/U)^2R$
