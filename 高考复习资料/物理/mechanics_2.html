🎯 模块四：曲线运动与抛体运动

4.1 曲线运动基础

条件：$F_{合}$ 与 $v$ 不共线（$F_{合} \perp v$ 时速度大小不变）

特点：速度方向沿切线，$F_{合}$ 指向轨迹凹侧

分解：$a_x = \frac{F_x}{m}$，$a_y = \frac{F_y}{m}$（独立性原理）

🎪 曲线运动三要素：
• 切向加速度 $a_\tau$：改变速度大小（$a_\tau = \frac{dv}{dt}$）
• 法向加速度 $a_n$：改变速度方向（$a_n = \frac{v^2}{\rho}$，$\rho$为曲率半径）
• 合加速度：$a = \sqrt{a_\tau^2 + a_n^2}$

4.2 平抛运动（分解思想典范）

水平方向：$x = v_0t$，$v_x = v_0$（匀速直线）

竖直方向：$y = \frac{1}{2}gt^2$，$v_y = gt$（自由落体）

合速度：$v = \sqrt{v_0^2 + (gt)^2}$，$\tan\theta = \frac{gt}{v_0}$

合位移：$s = \sqrt{x^2 + y^2}$，$\tan\alpha = \frac{gt}{2v_0}$

重要推论：$\tan\theta = 2\tan\alpha$（速度偏角=2×位移偏角）

平抛运动轨迹与分解

🎯 平抛运动解题模板：
1. 建立坐标系：原点为抛出点，x水平，y竖直向下
2. 列方程：$x = v_0t$，$y = \frac{1}{2}gt^2$
3. 消去 $t$ 得轨迹方程：$y = \frac{g}{2v_0^2}x^2$（抛物线）
4. 几何关系：利用斜面倾角 $\theta$，有 $\tan\theta = \frac{y}{x}$
⚠️ 平抛常见变式：
• 斜面平抛：落在斜面上时 $\tan\theta = \frac{y}{x}$，时间 $t = \frac{2v_0\tan\theta}{g}$
• 垂直撞斜面：$v_x = v_0$，$v_y = gt$，$\tan\theta = \frac{v_x}{v_y}$
• 最小位移：垂直斜面抛出时位移最小

4.3 圆周运动

线速度：$v = \frac{2\pi r}{T} = \omega r$

角速度：$\omega = \frac{2\pi}{T} = 2\pi f$

向心加速度：$a = \frac{v^2}{r} = \omega^2 r = \omega v = \frac{4\pi^2}{T^2}r$

向心力：$F = m\frac{v^2}{r} = m\omega^2 r = m\frac{4\pi^2}{T^2}r$

传动类型 特点 关系 应用场景
同轴转动 同一转轴 $\omega$ 相同，$v \propto r$ 地球自转、陀螺
皮带传动 边缘接触 $v$ 相同，$\omega \propto \frac{1}{r}$ 自行车轮、传送带
齿轮传动 齿合接触 $\frac{\omega_1}{\omega_2} = \frac{r_2}{r_1} = \frac{N_2}{N_1}$ 机械钟表、汽车变速箱
🎡 竖直平面圆周运动临界条件：
1. 绳模型（无支撑）：最高点 $v_{min} = \sqrt{gr}$，$T = 0$
2. 杆模型（有支撑）：最高点 $v_{min} = 0$，$N = mg$（支持力）
3. 拱桥模型：最高点 $v_{max} = \sqrt{gr}$，$N = 0$（飞起临界）
4. 管道模型：同杆模型，内外壁均可施力
🌍 模块五：万有引力与航天

5.1 万有引力定律

$\displaystyle F = G\frac{Mm}{r^2}$，$G = 6.67 \times 10^{-11} \text{N·m}^2/\text{kg}^2$

黄金代换：$GM = gR^2$（近地时 $r \approx R$）

密度公式：$\rho = \frac{3g}{4\pi GR}$ 或 $\rho = \frac{3\pi}{GT^2}$（近地卫星）

🚀 天体运动核心思想：
万有引力提供向心力：$G\frac{Mm}{r^2} = m\frac{v^2}{r} = m\omega^2 r = m\frac{4\pi^2}{T^2}r = ma$
口诀："高轨低速大周期，大机大势小动能"

5.2 卫星运动参数（记住结论）

物理量 公式 与 $r$ 关系
线速度 $v$ $v = \sqrt{\frac{GM}{r}}$ $v \propto \frac{1}{\sqrt{r}}$
角速度 $\omega$ $\omega = \sqrt{\frac{GM}{r^3}}$ $\omega \propto \frac{1}{r^{3/2}}$
周期 $T$ $T = 2\pi\sqrt{\frac{r^3}{GM}}$ $T \propto r^{3/2}$（开普勒第三定律）
向心加速度 $a$ $a = \frac{GM}{r^2}$ $a \propto \frac{1}{r^2}$

5.3 宇宙速度

第一宇宙速度（环绕速度）：$v_1 = \sqrt{gR} = \sqrt{\frac{GM}{R}} \approx 7.9 \text{km/s}$

第二宇宙速度（脱离速度）：$v_2 = \sqrt{2}v_1 \approx 11.2 \text{km/s}$

第三宇宙速度（逃逸速度）：$v_3 \approx 16.7 \text{km/s}$

⚠️ 卫星变轨问题：
• 加速升轨：$v \uparrow$ → 离心运动 → $r \uparrow$ → $v \downarrow$（最终 $v$ 变小）
• 减速降轨：$v \downarrow$ → 近心运动 → $r \downarrow$ → $v \uparrow$（最终 $v$ 变大）
• 变轨点：加速度 $a$ 相同（同一点 $r$ 相同），速度不同（外轨>内轨）

5.4 双星与多星系统

双星：$\omega_1 = \omega_2 = \omega$，$m_1r_1 = m_2r_2$，$r_1 + r_2 = L$

$G\frac{m_1m_2}{L^2} = m_1\omega^2r_1 = m_2\omega^2r_2$

周期：$T = 2\pi\sqrt{\frac{L^3}{G(m_1+m_2)}}$

🌟 双星问题解题要点：
1. 两星角速度（周期）相等
2. 向心力由彼此万有引力提供（大小相等）
3. 轨道半径与质量成反比：$\frac{r_1}{r_2} = \frac{m_2}{m_1}$
4. 总质量：$M = m_1 + m_2 = \frac{4\pi^2L^3}{GT^2}$
⚡ 模块六：机械能守恒与能量守恒

6.1 功与功率

功：$W = Fx\cos\theta$（$\theta$ 为 $F$ 与 $x$ 夹角）

功率：$P = \frac{W}{t} = Fv\cos\theta$

机车启动：$P = Fv$，$F - f = ma$

🚗 机车启动两种模式：
1. 恒定功率启动：$P$ 不变，$v \uparrow$ → $F \downarrow$ → $a \downarrow$ → 匀速（$F = f$）
2. 恒定加速度启动：$a$ 不变，$F$ 不变，$v \uparrow$ → $P \uparrow$ → 达 $P_{额}$ 后变功率启动
最大速度：$v_m = \frac{P}{f}$（两种模式最终相同）

6.2 动能定理（功能关系核心）

$W_{合} = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$

或：$W_G + W_N + W_f + W_{其他} = \Delta E_k$

✨ 动能定理应用五步法：
1. 选对象：明确研究对象（单体或系统）
2. 找过程：确定初末状态（速度、位置）
3. 算功：分析所有力做功（正功、负功、零功）
4. 列方程：$W_{合} = \Delta E_k$
5. 求解：注意矢量方向，功的正负
⚠️ 动能定理注意事项：
• 适用于直线、曲线、恒力、变力、单个、连续过程
• 功是过程量，能是状态量
• 保守力做功与路径无关（重力、弹力、万有引力）
• 摩擦力做功与路径有关（生热 $Q = f \cdot \Delta x_{相对}$）

6.3 机械能守恒

条件：只有重力或弹力做功（$W_{其他} = 0$）

表达式：$E_{k1} + E_{p1} = E_{k2} + E_{p2}$

或：$\Delta E_k = -\Delta E_p$（动能增加=势能减少）

或：$\Delta E_A = -\Delta E_B$（A增加=B减少，系统内）

🎯 机械能守恒判断三步法：
1. 看是否有非保守力做功（摩擦力、推力等）
2. 看是否有机械能与其他能转化（爆炸、燃烧等）
3. 看是否有系统外能量输入（电动机、人做功等）
注意：单物体机械能守恒条件严格；系统机械能守恒可含内弹力

6.4 能量守恒与功能关系

能量守恒：$E_{初} = E_{末}$ 或 $\Delta E_{增} = \Delta E_{减}$

摩擦生热：$Q = f \cdot x_{相对}$

功能原理：$W_{非保守力} = \Delta E_{机械}$

力做功 能量转化 关系式
重力做功 $W_G$ 重力势能 $E_p$ $W_G = -\Delta E_p$
弹力做功 $W_弹$ 弹性势能 $E_{弹}$ $W_弹 = -\Delta E_{弹}$
合外力做功 $W_{合}$ 动能 $E_k$ $W_{合} = \Delta E_k$
非保守力做功 $W_{非}$ 机械能 $E$ $W_{非} = \Delta E$
摩擦力做功 $W_f$ 内能 $Q$ $Q = |W_f|$（相对滑动）

🔗 力学模块融合总结

曲线运动+能量综合

1. 平抛+能量：机械能守恒 $\frac{1}{2}mv_0^2 + mgh_1 = \frac{1}{2}mv^2 + mgh_2$

2. 圆周+能量：临界条件 + 动能定理，如 $mg(2R) = \frac{1}{2}mv^2 - \frac{1}{2}mv_0^2$

3. 天体+能量：机械能 $E = -G\frac{Mm}{2a}$（椭圆轨道，$a$为半长轴）

4. 传送带+能量：$Q = f \cdot \Delta x$，电动机多消耗能量 $\Delta E = Q + \Delta E_k$
