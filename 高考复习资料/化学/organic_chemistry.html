🌿 有机化学基础

烃类 · 烃的衍生物 · 有机反应类型 · 有机推断
🎯 模块核心

有机化学是研究有机化合物的组成、结构、性质、制备方法与应用的科学。高考中主要考查有机物的结构、性质、反应类型以及有机推断与合成。

高考分值：约15-20分（选择题6分 + 有机推断题12-15分）
重点：官能团性质、有机反应类型、同分异构体、有机推断

第一章 烃类化合物

1.1 烷烃

🔸 通式：$C_nH_{2n+2}$（n≥1）
🔸 结构特点：碳碳单键，链状结构，饱和烃
🔸 代表物：甲烷（CH₄）
💡 甲烷的性质：
物理性质：无色无味气体，密度比空气小，难溶于水

化学性质（稳定，光照条件下与卤素反应）：
  • 氧化反应：$CH_4 + 2O_2 \stackrel{点燃}{\longrightarrow} CO_2 + 2H_2O$
  • 取代反应：$CH_4 + Cl_2 \stackrel{光照}{\longrightarrow} CH_3Cl + HCl$

1.2 烯烃

🔸 通式：$C_nH_{2n}$（n≥2）
🔸 结构特点：含碳碳双键，不饱和烃
🔸 代表物：乙烯（C₂H₄）
💡 乙烯的性质：
物理性质：无色稍有气味气体，密度略小于空气，难溶于水

化学性质（活泼，易发生加成、氧化反应）：
  • 氧化反应：$C_2H_4 + 3O_2 \stackrel{点燃}{\longrightarrow} 2CO_2 + 2H_2O$（火焰明亮，有黑烟）
    使酸性KMnO₄溶液褪色
  • 加成反应：
    与Br₂：$CH_2=CH_2 + Br_2 → CH_2BrCH_2Br$（使溴水褪色）
    与H₂：$CH_2=CH_2 + H_2 \stackrel{Ni,\Delta}{\longrightarrow} CH_3CH_3$
    与HCl：$CH_2=CH_2 + HCl \stackrel{催化剂}{\longrightarrow} CH_3CH_2Cl$
    与H₂O：$CH_2=CH_2 + H_2O \stackrel{催化剂}{\longrightarrow} CH_3CH_2OH$
  • 加聚反应：$nCH_2=CH_2 \stackrel{催化剂}{\longrightarrow} [CH_2-CH_2]_n$

1.3 炔烃

🔸 通式：$C_nH_{2n-2}$（n≥2）
🔸 结构特点：含碳碳三键，不饱和烃
🔸 代表物：乙炔（C₂H₂）
💡 乙炔的性质：
物理性质：无色无味气体，密度比空气小，微溶于水

化学性质：
  • 氧化反应：火焰明亮，有浓烟（含碳量高）
  • 加成反应：与烯烃类似，可发生1:1或1:2加成

1.4 芳香烃

🔸 代表物：苯（C₆H₆）
🔸 结构特点：平面正六边形，碳碳键是介于单键和双键之间的特殊键
💡 苯的性质：
物理性质：无色液体，有特殊气味，不溶于水，密度比水小，有毒

化学性质（稳定，难加成，易取代）：
  • 氧化反应：$2C_6H_6 + 15O_2 \stackrel{点燃}{\longrightarrow} 12CO_2 + 6H_2O$（火焰明亮，浓烟）
  • 取代反应：
    卤代：$C_6H_6 + Br_2 \stackrel{FeBr_3}{\longrightarrow} C_6H_5Br + HBr$
    硝化：$C_6H_6 + HNO_3 \stackrel{浓H_2SO_4,\Delta}{\longrightarrow} C_6H_5NO_2 + H_2O$
  • 加成反应：$C_6H_6 + 3H_2 \stackrel{Ni,\Delta}{\longrightarrow} C_6H_{12}$（环己烷）
💡 甲苯的性质：
甲苯比苯活泼，甲基使苯环邻、对位氢活泼，易发生取代反应。
硝化反应：生成2,4,6-三硝基甲苯（TNT，炸药）

第二章 烃的衍生物

2.1 卤代烃

🔸 官能团：-X（卤素原子）
🔸 代表物：溴乙烷（C₂H₅Br）
💡 溴乙烷的化学性质：
  • 水解反应（取代反应）：
    $C_2H_5Br + NaOH \stackrel{H_2O,\Delta}{\longrightarrow} C_2H_5OH + NaBr$
  • 消去反应：
    $C_2H_5Br + NaOH \stackrel{乙醇,\Delta}{\longrightarrow} CH_2=CH_2↑ + NaBr + H_2O$
⚠️ 消去反应条件：与卤素原子相连的碳原子的邻位碳原子上必须有氢原子。

2.2 醇

🔸 官能团：-OH（羟基）
🔸 代表物：乙醇（C₂H₅OH）
💡 乙醇的化学性质：
  • 与活泼金属反应：
    $2C_2H_5OH + 2Na → 2C_2H_5ONa + H_2↑$
  • 氧化反应：
    燃烧：$C_2H_5OH + 3O_2 \stackrel{点燃}{\longrightarrow} 2CO_2 + 3H_2O$
    催化氧化：$2C_2H_5OH + O_2 \stackrel{Cu/Ag,\Delta}{\longrightarrow} 2CH_3CHO + 2H_2O$
    使酸性KMnO₄溶液褪色
  • 消去反应：
    $C_2H_5OH \stackrel{浓H_2SO_4,170℃}{\longrightarrow} CH_2=CH_2↑ + H_2O$
  • 酯化反应（见羧酸部分）

2.3 酚

🔸 官能团：-OH（直接连在苯环上）
🔸 代表物：苯酚（C₆H₅OH）
💡 苯酚的性质：
物理性质：无色晶体，有特殊气味，常温下微溶于水，温度高于65℃时与水互溶，有毒，有腐蚀性

化学性质：
  • 弱酸性（俗称石炭酸）：
    $C_6H_5OH + NaOH → C_6H_5ONa + H_2O$
    酸性：H₂CO₃ > C₆H₅OH > HCO₃⁻
  • 取代反应（与浓溴水）：
    生成2,4,6-三溴苯酚白色沉淀（用于苯酚的定性检验和定量测定）
  • 显色反应：与FeCl₃溶液显紫色（用于检验酚羟基）

2.4 醛

🔸 官能团：-CHO（醛基）
🔸 代表物：乙醛（CH₃CHO）
💡 乙醛的化学性质：
  • 氧化反应：
    银镜反应：$CH_3CHO + 2Ag(NH_3)_2OH \stackrel{\Delta}{\longrightarrow} CH_3COONH_4 + 2Ag↓ + 3NH_3 + H_2O$
    与新制Cu(OH)₂：$CH_3CHO + 2Cu(OH)_2 + NaOH \stackrel{\Delta}{\longrightarrow} CH_3COONa + Cu_2O↓ + 3H_2O$
    催化氧化：$2CH_3CHO + O_2 \stackrel{催化剂,\Delta}{\longrightarrow} 2CH_3COOH$
  • 还原反应（加成反应）：
    $CH_3CHO + H_2 \stackrel{Ni,\Delta}{\longrightarrow} CH_3CH_2OH$

2.5 羧酸

🔸 官能团：-COOH（羧基）
🔸 代表物：乙酸（CH₃COOH）
💡 乙酸的化学性质：
  • 酸性：弱酸，但比碳酸强
    $2CH_3COOH + Na_2CO_3 → 2CH_3COONa + H_2O + CO_2↑$
  • 酯化反应：
    $CH_3COOH + C_2H_5OH \underset{\Delta}{\stackrel{浓H_2SO_4}{\rightleftharpoons}} CH_3COOC_2H_5 + H_2O$
⚠️ 酯化反应要点：
  • 酸脱羟基，醇脱氢
  • 浓H₂SO₄作催化剂和吸水剂
  • 可逆反应，加热使平衡正向移动
  • 饱和Na₂CO₃溶液作用：吸收乙醇、中和乙酸、降低酯的溶解度

2.6 酯

🔸 官能团：-COO-（酯基）
🔸 代表物：乙酸乙酯（CH₃COOCH₂CH₃）
💡 酯的性质：
物理性质：低级酯有芳香气味，密度一般比水小，难溶于水

化学性质（水解反应）：
  • 酸性条件下水解（可逆）：
    $CH_3COOC_2H_5 + H_2O \underset{\Delta}{\stackrel{稀H_2SO_4}{\rightleftharpoons}} CH_3COOH + C_2H_5OH$
  • 碱性条件下水解（皂化反应，不可逆）：
    $CH_3COOC_2H_5 + NaOH \stackrel{\Delta}{\longrightarrow} CH_3COONa + C_2H_5OH$

第三章 有机反应类型总结

3.1 基本反应类型

反应类型 定义 特点 实例
取代反应 有机物分子里的某些原子或原子团被其他原子或原子团所替代的反应 有进有出 烷烃卤代、苯的硝化、酯化、水解
加成反应 有机物分子中不饱和键两端的原子与其他原子或原子团直接结合生成新化合物的反应 只进不出 烯烃与H₂、Br₂、HCl、H₂O加成
消去反应 有机化合物在一定条件下，从一个分子中脱去一个或几个小分子（如H₂O、HX等），而生成含不饱和键化合物的反应 只出不进 醇消去、卤代烃消去
氧化反应 有机物分子中加氧或去氢的反应 - 燃烧、催化氧化、使KMnO₄褪色
还原反应 有机物分子中加氢或去氧的反应 - 醛与H₂加成、硝基苯还原
加聚反应 由不饱和的单体聚合生成高分子化合物的反应 无小分子生成 乙烯加聚、丙烯加聚
缩聚反应 单体间相互反应生成高分子，同时还生成小分子（如H₂O、HX等）的反应 有小分子生成 酯化缩聚、氨基酸缩聚

第四章 同分异构体

4.1 同分异构体的类型

💡 同分异构体：分子式相同，结构不同的化合物。
异构类型 形成原因 实例
碳链异构 碳骨架不同 正丁烷与异丁烷
位置异构 官能团位置不同 1-丙醇与2-丙醇
官能团异构 官能团不同 乙醇（醇）与二甲醚（醚）

4.2 同分异构体书写技巧

💡 烷烃同分异构体书写（减碳法）：
  1. 写出最长碳链
  2. 减少一个碳原子作甲基，移动位置
  3. 减少两个碳原子作乙基或两个甲基，移动位置
  4. 检查是否有重复
📝 己烷（C₆H₁₄）的同分异构体：
1. 直链：CH₃CH₂CH₂CH₂CH₂CH₃（正己烷）
2. 一个甲基：2-甲基戊烷、3-甲基戊烷
3. 两个甲基：2,2-二甲基丁烷、2,3-二甲基丁烷

第五章 有机化学拓展（整合补充）

5.1 系统命名法详解

💡 系统命名法步骤：
  1. 选主链：含官能团的最长碳链（官能团优先：-COOH > -CHO > -OH > -NH₂ > C=C > C≡C > 苯环 > 支链）
  2. 编号：从离官能团最近的一端开始，使官能团位次最小；若官能团在中间，使取代基位次之和最小
  3. 写名称：取代基位置-取代基名称-母体名称（官能团位置-母体）

注意：官能团位次为1时可省略；取代基按次序规则排列（甲基<乙基<丙基...）

结构表示方法：
分子式：C₂H₆O（仅表示原子种类和数目）
结构式：画出所有化学键
结构简式：CH₃CH₂OH（合并相同原子团）
键线式：拐点或端点为碳，省略氢，官能团保留（如：OH、CHO、COOH）

5.2 同分异构体类型详解

异构类型 定义/特点 典型实例
碳链异构 碳骨架不同 正丁烷与异丁烷
位置异构 官能团位置不同 1-丙醇与2-丙醇；邻、间、对二甲苯
官能团异构 官能团不同，分子式相同 CₙH₂ₙ：烯烃与环烷烃；CₙH₂ₙ₊₂O：醇与醚；CₙH₂ₙO：醛、酮、烯醇
立体异构 顺反异构、对映异构 顺-2-丁烯与反-2-丁烯；手性碳化合物
⚠️ 同分异构体书写技巧（减碳法）：

以C₆H₁₄（己烷）为例：

  1. 直链：CH₃CH₂CH₂CH₂CH₂CH₃
  2. 减一个碳作甲基：主链5碳，甲基可在2位或3位 → 2-甲基戊烷、3-甲基戊烷
  3. 减两个碳作乙基或两个甲基：主链4碳，两个甲基可在2,2位或2,3位

记忆：C₁-C₁₀烷烃同分异构体数：1,1,1,2,3,5,9,18,35,75

5.3 有机反应类型总结（完整版）

反应类型 定义/特点 典型实例
取代反应 有机物分子中某些原子或原子团被其他原子或原子团替代 烷烃卤代、苯的卤代/硝化/磺化、醇与HX、酯化、水解
加成反应 不饱和键两端的原子直接结合其他原子或原子团 C=C、C≡C与H₂、X₂、HX、H₂O；苯环加氢；C=O加氢
消去反应 从分子中脱去小分子（H₂O、HX等）形成不饱和键 醇分子内脱水（浓H₂SO₄/170℃）；卤代烃脱HX（NaOH/醇/△）
氧化反应 加氧或去氢（有机化学定义） 燃烧、催化氧化（醇→醛）、KMnO₄氧化、银镜反应
还原反应 加氢或去氧 C=C、C≡C、C=O、NO₂与H₂加成（催化氢化）
聚合反应 小分子生成高分子 加聚（C=C打开）；缩聚（生成高分子的同时脱去小分子）
显色反应 特征颜色变化 苯酚+FeCl₃→紫色；淀粉+I₂→蓝色；蛋白质+浓HNO₃→黄色

5.4 烯烃加成规律

💡 马氏规则与反马氏：
  • 马氏规则：不对称烯烃与HX加成，H加在含H较多的C上（如：丙烯+HBr → 2-溴丙烷）
  • 反马氏规则：在过氧化物存在下，HBr与不对称烯烃加成，Br加在含H较多的C上（自由基机理）
  • 硼氢化-氧化：反马氏加水，生成伯醇（如：R-CH=CH₂ → R-CH₂-CH₂OH）
💡 二烯烃（共轭二烯）：

1,3-丁二烯：CH₂=CH-CH=CH₂，可发生1,2-加成和1,4-加成

  • 1,2-加成：Br₂加在C1、C2上（低温有利）
  • 1,4-加成：Br₂加在C1、C4上，C2、C3形成双键（高温有利）

5.5 苯的同系物特性

💡 取代基定位效应：
  • 邻对位定位基（活化苯环）：-CH₃、-OH、-NH₂、-X（卤素弱钝化但邻对位）→ 新取代基进入邻对位
  • 间位定位基（钝化苯环）：-NO₂、-CHO、-COOH、-SO₃H → 新取代基进入间位
💡 侧链氧化：与苯环直接相连的C上有H时，可被KMnO₄氧化为-COOH（不论侧链多长，都变成苯甲酸）

5.6 醇的消去与催化氧化规律

醇的结构 消去反应 催化氧化产物
R-CH₂-OH（伯醇） 能（邻位有H） R-CHO（醛）
R₁-CH(R₂)-OH（仲醇） 能（邻位有H） R₁-CO-R₂（酮）
R₁-C(R₂)(R₃)-OH（叔醇） 能（邻位有H） 不能氧化（无α-H）
CH₃OH（甲醇） 不能（无邻位C） HCHO（甲醛）

5.7 酚的特殊反应

💡 苯酚的酸性：C₆H₅OH ⇌ C₆H₅O⁻ + H⁺（酸性：H₂CO₃ > C₆H₅OH > HCO₃⁻）
  • 与Na反应生成C₆H₅ONa + H₂
  • 与NaOH反应生成C₆H₅ONa + H₂O
  • 与Na₂CO₃反应生成C₆H₅ONa + NaHCO₃（不能生成CO₂）
  • 通入CO₂：C₆H₅ONa + CO₂ + H₂O → C₆H₅OH↓ + NaHCO₃（无论CO₂过量与否都生成NaHCO₃）
💡 酚的取代反应（邻对位）：与浓溴水生成2,4,6-三溴苯酚（白色沉淀，用于定性检验）

5.8 银镜反应实验要点

【实验步骤与注意事项】
  1. 银氨溶液配制：向AgNO₃溶液中滴加稀氨水，至沉淀恰好溶解
    Ag⁺ + NH₃·H₂O = AgOH↓ + NH₄⁺
    AgOH + 2NH₃·H₂O = [Ag(NH₃)₂]⁺ + OH⁻ + 2H₂O
  2. 反应条件：水浴加热（60-70℃），不能直接加热（会生成黑色Ag₂O沉淀）
  3. 试管要求：洁净（可用热NaOH洗），否则银镜不均匀
  4. 反应后试管洗涤：用稀HNO₃（Ag溶于HNO₃）
  5. 甲醛的特殊性：HCHO + 4[Ag(NH₃)₂]OH → (NH₄)₂CO₃ + 4Ag↓ + 6NH₃ + 2H₂O（1mol醛基生成4mol Ag）

5.9 卤代烃在合成中的桥梁作用

💡 卤代烃是引入官能团的重要中间体：
  • 引入-OH：卤代烃水解
  • 引入C=C：卤代烃消去
  • 增长碳链：与Na反应或格氏试剂反应
  • 引入-CN：与NaCN取代，再水解得-COOH

第六章 有机推断技巧

6.1 有机推断的突破口

💡 特征反应条件：
  • 光照：烷烃卤代
  • Ni/△、H₂：加成反应（还原反应）
  • 浓H₂SO₄/△：酯化、消去、硝化
  • NaOH水溶液/△：卤代烃水解、酯水解
  • NaOH醇溶液/△：卤代烃消去
  • Cu/Ag、△：醇的催化氧化
  • 银氨溶液/△：醛基的氧化
  • 新制Cu(OH)₂/△：醛基的氧化
💡 特征现象：
  • 使溴水褪色：含碳碳双键、三键或醛基
  • 使酸性KMnO₄褪色：含碳碳双键、三键、醛基或苯的同系物
  • 银镜反应：含醛基
  • 与FeCl₃显紫色：含酚羟基
  • 与浓溴水产生白色沉淀：含酚羟基

4.2 同分异构体书写技巧

💡 书写顺序：
  1. 碳链异构：主链由长到短，支链由整到散
  2. 位置异构：官能团位置依次移动
  3. 官能团异构：考虑不同类别物质
📝 C₄H₈O的同分异构体（醛和酮）：

醛类：CH₃CH₂CH₂CHO（丁醛）、(CH₃)₂CHCHO（异丁醛）

酮类：CH₃COCH₂CH₃（丁酮）

4.3 常见官能团异构关系

分子式通式 可能的官能团异构
CₙH₂ₙ 烯烃、环烷烃
CₙH₂ₙ₋₂ 炔烃、二烯烃、环烯烃

5.1 有机推断突破口

💡 特征反应条件：
  • 光照：烷烃卤代
  • Ni/△、H₂：加成反应（烯、炔、醛、酮、苯环）
  • 浓H₂SO₄/△：酯化、醇消去、硝化
  • NaOH/H₂O/△：卤代烃水解、酯水解
  • NaOH/醇/△：卤代烃消去
  • Cu/Ag/△：醇催化氧化
  • 银氨溶液/△：醛氧化
  • 新制Cu(OH)₂/△：醛氧化
💡 特征现象：
  • 使溴水褪色：烯烃、炔烃、醛、酚
  • 使酸性KMnO₄褪色：烯、炔、醇、醛、苯的同系物
  • 银镜反应：含醛基（-CHO）
  • 砖红色沉淀：含醛基（与新制Cu(OH)₂）
  • 显色反应（FeCl₃紫色）：含酚羟基
  • 白色沉淀（溴水）：含酚羟基

5.2 有机合成路线设计

💡 合成原则：
  1. 原料易得、成本低廉
  2. 步骤简单、副反应少
  3. 产物易分离、纯度高
  4. 原子利用率高、绿色环保
📝 由乙烯合成乙酸乙酯：

CH₂=CH₂ + H₂O →[催化剂] CH₃CH₂OH
2CH₃CH₂OH + O₂ →[Cu/△] 2CH₃CHO + 2H₂O
2CH₃CHO + O₂ →[催化剂] 2CH₃COOH
CH₃COOH + CH₃CH₂OH ⇌[浓H₂SO₄/△] CH₃COOCH₂CH₃ + H₂O

5.3 有机计算

燃烧通式：
CₓHᵧ + (x + y/4)O₂ → xCO₂ + (y/2)H₂O
CₓHᵧO₂ + (x + y/4 - z/2)O₂ → xCO₂ + (y/2)H₂O
💡 重要规律：
  • 等物质的量有机物燃烧，耗氧量看(x + y/4 - z/2)
  • 等质量烃燃烧，含氢量越高，耗氧量越多
  • 最简式相同的有机物，等质量燃烧产物相同

🏆 点击化学与生物正交化学

热点背景：2022年诺贝尔化学奖授予Carolyn Bertozzi、Morten Meldal和K. Barry Sharpless，表彰他们在点击化学和生物正交化学方面的贡献。2025年这些技术在药物递送和生物成像领域取得重大突破。

化学考点链接：
• 叠氮-炔环加成：$R-N_3 + R'-C≡CH \stackrel{Cu(I)}{\longrightarrow} R-N_3-C=C-R'$，区域选择性生成1,4-二取代三唑
• 应变促进反应：环辛炔（DBCO）无需铜催化，与叠氮快速反应，生物相容性好
• 生物正交性：反应在生理条件下进行，不干扰天然生化过程
• 应用：抗体药物偶联物（ADC）、活体细胞标记、靶向治疗
铜催化叠氮-炔环加成（CuAAC）：
机理：Cu(I)活化端炔形成铜乙炔化物，与叠氮配位后发生[3+2]环加成
特点：原子经济性100%，反应条件温和，产物纯化简单
