📐 物质的量与化学计算

化学计算的基石 · 从微观到宏观的桥梁
🎯 模块核心

物质的量是连接微观粒子与宏观质量的桥梁，是化学计算的核心工具。掌握物质的量的计算，就掌握了化学方程式计算、溶液浓度计算、气体计算等所有化学计算的基础。

高考分值：约12-18分（选择题6-9分 + 计算题6-9分）
难度系数：⭐⭐⭐（中等偏易，但必须熟练掌握）
核心公式：$n = \frac{N}{N_A} = \frac{m}{M} = \frac{V}{V_m} = c \cdot V$

第一章 物质的量的基本概念

1.1 物质的量（n）

💡 定义：物质的量是表示含有一定数目粒子集合体的物理量，单位是摩尔（mol）。
💡 理解要点：
  • 物质的量是国际单位制中7个基本物理量之一
  • 1mol任何粒子所含的粒子数均为阿伏伽德罗常数个
  • 使用物质的量时必须指明粒子种类（原子、分子、离子、电子等）
  • 粒子数可以是整数，也可以不是整数
基本公式：
$n = \frac{N}{N_A}$
物理量 符号 单位 数值/说明
物质的量 $n$ mol（摩尔） 表示粒子数目的物理量
阿伏伽德罗常数 $N_A$ $mol^{-1}$ $N_A = 6.02 \times 10^{23} mol^{-1}$
粒子数 $N$ 个 微观粒子的数目
📝 例题1：1mol H₂O中含有多少个水分子？多少个氢原子？多少个氧原子？多少个电子？

【分析思路】
1. 明确已知条件：物质的量 $n(H_2O) = 1mol$
2. 确定求解目标：水分子数、氢原子数、氧原子数、电子数
3. 选择公式：使用 $N = n \cdot N_A$ 计算粒子数
4. 分析分子组成：H₂O分子含2个H原子、1个O原子，共3个原子
5. 计算电子数：H有1个电子，O有8个电子，每个H₂O分子含 $1\times2 + 8 = 10$ 个电子

【规范解答】
解：已知 $n(H_2O) = 1mol$，$N_A = 6.02 \times 10^{23} mol^{-1}$

① 水分子数：
根据公式 $N = n \cdot N_A$
$N(H_2O) = 1mol \times 6.02 \times 10^{23} mol^{-1} = 6.02 \times 10^{23}$ 个

② 氢原子数：
每个H₂O分子含2个H原子
$N(H) = 2 \times N(H_2O) = 2 \times 6.02 \times 10^{23} = 1.204 \times 10^{24}$ 个

③ 氧原子数：
每个H₂O分子含1个O原子
$N(O) = 1 \times N(H_2O) = 6.02 \times 10^{23}$ 个

④ 电子数：
每个H原子含1个电子，每个O原子含8个电子
每个H₂O分子含电子数 = $1\times2 + 8 = 10$ 个
$N(e^-) = 10 \times N(H_2O) = 10 \times 6.02 \times 10^{23} = 6.02 \times 10^{24}$ 个

【方法总结】
• 粒子数计算的核心公式：$N = n \cdot N_A$
• 原子数 = 分子数 × 每个分子中所含该原子的个数
• 电子数计算：先确定每个分子的电子总数，再乘以分子数

1.2 摩尔质量（M）

💡 定义：单位物质的量的物质所具有的质量叫做摩尔质量。
计算公式：
$M = \frac{m}{n}$ 或 $m = n \cdot M$ 或 $n = \frac{m}{M}$
物理量 符号 单位 说明
摩尔质量 $M$ $g/mol$（或 $g·mol^{-1}$） 注意单位！不是g！
质量 $m$ g 宏观物质的质量
物质的量 $n$ mol 物质的量
💡 重要规律：
  • 摩尔质量以g/mol为单位时，数值上等于该物质的相对原子质量或相对分子质量
  • 相对原子质量：$M(H) = 1 g/mol$，$M(O) = 16 g/mol$，$M(Na) = 23 g/mol$
  • 相对分子质量：$M(H_2O) = 18 g/mol$，$M(CO_2) = 44 g/mol$，$M(H_2SO_4) = 98 g/mol$
⚠️ 常见错误：
  • ❌ $M = 18g$（错误单位）
  • ✅ $M = 18g/mol$（正确单位）
📝 例题2：9g H₂O的物质的量是多少？含有多少个水分子？

【分析思路】
1. 明确已知条件：质量 $m(H_2O) = 9g$
2. 确定求解目标：物质的量 $n$、水分子数 $N$
3. 确定摩尔质量：$H_2O$ 的相对分子质量 = $1\times2 + 16 = 18$，所以 $M(H_2O) = 18g/mol$
4. 第一步：用 $n = \frac{m}{M}$ 求物质的量
5. 第二步：用 $N = n \cdot N_A$ 求分子数

【规范解答】
解：已知 $m(H_2O) = 9g$，$M(H_2O) = 18g/mol$，$N_A = 6.02 \times 10^{23} mol^{-1}$

① 求物质的量：
根据公式 $n = \frac{m}{M}$
$n(H_2O) = \frac{9g}{18g/mol} = 0.5mol$

② 求水分子数：
根据公式 $N = n \cdot N_A$
$N(H_2O) = 0.5mol \times 6.02 \times 10^{23} mol^{-1} = 3.01 \times 10^{23}$ 个

【易错提醒】
• 注意区分质量 $m$（单位：g）和摩尔质量 $M$（单位：g/mol）
• 计算时注意单位约分：$\frac{g}{g/mol} = mol$
• 结果用科学计数法表示，注意有效数字

第二章 气体摩尔体积

2.1 气体摩尔体积（Vₘ）

💡 定义：单位物质的量的气体所占的体积叫做气体摩尔体积。
计算公式：
$V_m = \frac{V}{n}$ 或 $V = n \cdot V_m$ 或 $n = \frac{V}{V_m}$
物理量 符号 单位 说明
气体摩尔体积 $V_m$ $L/mol$ 与温度、压强有关
气体体积 $V$ L（或m³） 注意单位换算
物质的量 $n$ mol 气体的物质的量
🔄 标准状况（STP）：
  • 条件：0℃（273K），101kPa（1标准大气压）
  • 标准状况下气体摩尔体积：$V_m = 22.4 L/mol$
  • 适用范围：只适用于气体（包括混合气体）
⚠️ 重要提醒：
  • ❌ 标准状况下，水（H₂O）是液态，不能用22.4L/mol
  • ❌ 标准状况下，SO₃是固态，不能用22.4L/mol
  • ❌ 标准状况下，酒精（C₂H₅OH）是液态，不能用22.4L/mol
  • ❌ 标准状况下，苯（C₆H₆）是液态，不能用22.4L/mol
  • ✅ 标准状况下，H₂、O₂、N₂、CO₂、Cl₂、NH₃等都是气体，可以用22.4L/mol
📝 例题3：标准状况下，11.2L CO₂的物质的量是多少？质量是多少？

【分析思路】
1. 明确已知条件：标准状况下，$V(CO_2) = 11.2L$
2. 确定求解目标：物质的量 $n$、质量 $m$
3. 判断适用条件：CO₂在标准状况下是气体，可用 $V_m = 22.4L/mol$
4. 第一步：用 $n = \frac{V}{V_m}$ 求物质的量
5. 第二步：先求 $M(CO_2)$，再用 $m = n \cdot M$ 求质量

【规范解答】
解：已知标准状况下 $V(CO_2) = 11.2L$，$V_m = 22.4L/mol$

① 求物质的量：
根据公式 $n = \frac{V}{V_m}$
$n(CO_2) = \frac{11.2L}{22.4L/mol} = 0.5mol$

② 求质量：
$M(CO_2) = 12 + 16\times2 = 44g/mol$
根据公式 $m = n \cdot M$
$m(CO_2) = 0.5mol \times 44g/mol = 22g$

【答】CO₂的物质的量为0.5mol，质量为22g。

【方法总结】
• 标准状况下气体计算：$n = \frac{V}{22.4}$（V单位为L）
• 常见气体摩尔质量要熟记：$M(CO_2)=44$，$M(O_2)=32$，$M(N_2)=28$，$M(H_2)=2$
• 三步法：$V \rightarrow n \rightarrow m$ 或 $m \rightarrow n \rightarrow V$

2.2 阿伏伽德罗定律及其推论

💡 阿伏伽德罗定律：同温同压下，相同体积的任何气体含有相同数目的分子。
数学表达式：
同温同压下：$\frac{V_1}{V_2} = \frac{n_1}{n_2} = \frac{N_1}{N_2}$
📋 阿伏伽德罗定律的推论：
条件 推论 说明
同温同压 $\frac{V_1}{V_2} = \frac{n_1}{n_2}$ 体积比等于物质的量比
同温同容 $\frac{p_1}{p_2} = \frac{n_1}{n_2}$ 压强比等于物质的量比
同温同压 $\frac{\rho_1}{\rho_2} = \frac{M_1}{M_2}$ 密度比等于摩尔质量比（相对密度）
同温同压同体积 $\frac{m_1}{m_2} = \frac{M_1}{M_2}$ 质量比等于摩尔质量比
📝 例题4：同温同压下，等质量的CO和CO₂相比较，下列叙述正确的是（）
A. 体积比为7:11 B. 体积比为11:7 C. 密度比为7:11 D. 密度比为11:7

【分析思路】
1. 明确已知条件：同温同压下，CO和CO₂质量相等（设为m）
2. 确定求解目标：体积比和密度比
3. 确定相关公式：
• 同温同压下：$\frac{V_1}{V_2} = \frac{n_1}{n_2}$（阿伏伽德罗定律推论）
• 同温同压下：$\frac{\rho_1}{\rho_2} = \frac{M_1}{M_2}$（相对密度）
4. 计算摩尔质量：$M(CO) = 28g/mol$，$M(CO_2) = 44g/mol$
5. 计算物质的量：$n = \frac{m}{M}$，质量相同时，n与M成反比

【规范解答】
解：设CO和CO₂的质量都为m

① 计算物质的量：
$n(CO) = \frac{m}{M(CO)} = \frac{m}{28}mol$
$n(CO_2) = \frac{m}{M(CO_2)} = \frac{m}{44}mol$

② 计算体积比：
同温同压下，$\frac{V(CO)}{V(CO_2)} = \frac{n(CO)}{n(CO_2)} = \frac{m/28}{m/44} = \frac{44}{28} = \frac{11}{7}$
即体积比为 11:7，选项B正确

③ 计算密度比：
同温同压下，$\frac{\rho(CO)}{\rho(CO_2)} = \frac{M(CO)}{M(CO_2)} = \frac{28}{44} = \frac{7}{11}$
即密度比为 7:11，选项C正确

【答案】B、C

【方法总结】
• 口诀："同温同压，体积比=物质的量比=摩尔质量反比"
• 密度比：同温同压下，密度比等于摩尔质量比（相对密度）
• 等质量问题：质量相同时，n与M成反比，即摩尔质量越大，物质的量越小

第三章 物质的量浓度

3.1 物质的量浓度（c）

💡 定义：单位体积溶液中所含溶质的物质的量，叫做物质的量浓度。
计算公式：
$c = \frac{n}{V}$ 或 $n = c \cdot V$ 或 $V = \frac{n}{c}$
物理量 符号 单位 说明
物质的量浓度 $c$ $mol/L$（或 $mol·L^{-1}$） 注意溶液体积，不是溶剂体积
溶质的物质的量 $n$ mol 溶质的物质的量
溶液体积 $V$ L 注意单位换算：1L = 1000mL
⚠️ 重要提醒：
  • 溶液体积 ≠ 溶剂体积
  • 计算时注意单位换算：mL → L（除以1000）
  • 溶液体积一般通过溶液质量和密度计算：$V = \frac{m}{\rho}$
📝 例题5：将40g NaOH溶于水配成500mL溶液，求该溶液的物质的量浓度。

【分析思路】
1. 明确已知条件：溶质质量 $m(NaOH) = 40g$，溶液体积 $V = 500mL$
2. 确定求解目标：物质的量浓度 $c$
3. 确定公式：$c = \frac{n}{V}$，需要先求 $n$
4. 计算物质的量：用 $n = \frac{m}{M}$ 计算
5. 单位换算：将体积从mL换算为L

【规范解答】
解：已知 $m(NaOH) = 40g$，$V = 500mL = 0.5L$

① 计算NaOH的摩尔质量：
$M(NaOH) = 23 + 16 + 1 = 40g/mol$

② 计算NaOH的物质的量：
根据公式 $n = \frac{m}{M}$
$n(NaOH) = \frac{40g}{40g/mol} = 1mol$

③ 计算物质的量浓度：
根据公式 $c = \frac{n}{V}$
$c(NaOH) = \frac{1mol}{0.5L} = 2mol/L$

【答】该NaOH溶液的物质的量浓度为2mol/L。

【易错提醒】
• ⚠️ 溶液体积 ≠ 溶剂体积：题目说的是"配成500mL溶液"，不是"加入500mL水"
• ⚠️ 单位换算：计算前必须将mL换算为L，$500mL = 0.5L$
• ⚠️ 浓度单位：物质的量浓度单位是mol/L，不是mol

【方法总结】
• 基本公式：$c = \frac{n}{V} = \frac{m}{M \cdot V}$
• 计算步骤：$m \rightarrow n \rightarrow c$ 或直接用 $c = \frac{m}{M \cdot V}$

3.2 物质的量浓度与质量分数的换算

换算公式：
$c = \frac{1000 \cdot \rho \cdot w}{M}$
💡 公式说明：
  • $c$：物质的量浓度（mol/L）
  • $\rho$：溶液密度（g/cm³）
  • $w$：溶质质量分数（小数或百分数化成小数）
  • $M$：溶质摩尔质量（g/mol）
  • 1000：单位换算系数（cm³ → L）
📝 例题6：已知浓硫酸的密度为1.84g/cm³，质量分数为98%，求其物质的量浓度。

【分析思路】
1. 明确已知条件：密度 $\rho = 1.84g/cm³$，质量分数 $w = 98\%$
2. 确定求解目标：物质的量浓度 $c$
3. 选择公式：使用换算公式 $c = \frac{1000 \cdot \rho \cdot w}{M}$
4. 确定摩尔质量：$H_2SO_4$ 的相对分子质量 = $1\times2 + 32 + 16\times4 = 98$
5. 单位换算：质量分数化为小数：$98\% = 0.98$

【规范解答】
解：已知 $\rho = 1.84g/cm³$，$w = 98\% = 0.98$

① 计算H₂SO₄的摩尔质量：
$M(H_2SO_4) = 1\times2 + 32 + 16\times4 = 98g/mol$

② 代入公式计算：
根据公式 $c = \frac{1000 \cdot \rho \cdot w}{M}$
$c = \frac{1000 \times 1.84 \times 0.98}{98} = \frac{1803.2}{98} = 18.4mol/L$

【答】该浓硫酸的物质的量浓度为18.4mol/L。

【公式推导】
假设溶液体积为1L（1000mL）：
溶液质量 = $1000mL \times 1.84g/cm³ = 1840g$
溶质质量 = $1840g \times 98\% = 1803.2g$
溶质的物质的量 = $\frac{1803.2g}{98g/mol} = 18.4mol$
物质的量浓度 = $\frac{18.4mol}{1L} = 18.4mol/L$

【方法总结】
• 公式：$c = \frac{1000 \cdot \rho \cdot w}{M}$（$\rho$单位：g/cm³）
• 1000是单位换算系数：将cm³换算为L
• 注意：质量分数要化成小数代入计算

3.3 溶液的稀释与混合

💡 稀释定律：稀释前后溶质的物质的量不变
稀释公式：
$c_1 \cdot V_1 = c_2 \cdot V_2$
💡 说明：
  • $c_1$、$V_1$：稀释前溶液的浓度和体积
  • $c_2$、$V_2$：稀释后溶液的浓度和体积
  • 稀释前后溶质的物质的量不变：$n_1 = n_2$
📝 例题7：如何用18.4mol/L的浓硫酸配制500mL 1mol/L的稀硫酸？

【分析思路】
1. 明确已知条件：浓硫酸浓度 $c_1 = 18.4mol/L$，目标浓度 $c_2 = 1mol/L$，目标体积 $V_2 = 500mL$
2. 确定求解目标：需要浓硫酸的体积 $V_1$ 及配制步骤
3. 选择原理：稀释定律——稀释前后溶质的物质的量不变
4. 确定公式：$c_1 \cdot V_1 = c_2 \cdot V_2$
5. 计算所需体积：解方程求 $V_1$

【规范解答】
解：已知 $c_1 = 18.4mol/L$，$c_2 = 1mol/L$，$V_2 = 500mL$

① 计算所需浓硫酸体积：
根据稀释定律：$c_1 \cdot V_1 = c_2 \cdot V_2$
$18.4mol/L \times V_1 = 1mol/L \times 500mL$
$V_1 = \frac{1 \times 500}{18.4} = 27.2mL$

② 配制步骤：
  1. 计算：计算需要27.2mL浓硫酸
  2. 量取：用量筒量取27.2mL浓硫酸
  3. 稀释：将浓硫酸沿烧杯壁缓慢注入约200mL蒸馏水中，边加边搅拌（酸入水！）
  4. 冷却：冷却至室温（浓硫酸稀释放热）
  5. 转移：转移至500mL容量瓶
  6. 洗涤：用蒸馏水洗涤烧杯和玻璃棒2-3次，洗液转入容量瓶
  7. 定容：加蒸馏水至接近刻度线1-2cm处，改用胶头滴管滴加至刻度线
  8. 摇匀：塞好瓶塞，反复颠倒摇匀

【答】需要量取27.2mL浓硫酸，按上述步骤配制。

【核心要点】
• 稀释定律本质：$n_1 = n_2$（稀释前后溶质的物质的量不变）
• 安全操作：浓硫酸稀释必须"酸入水"，不能"水入酸"
• 温度控制：必须冷却至室温后再转移，防止容量瓶受热变形
• 洗涤转移：烧杯和玻璃棒必须洗涤2-3次，确保溶质全部转移
⚠️ 浓硫酸稀释注意事项：
  • 酸入水，沿器壁，慢慢倒，不断搅
  • 绝对禁止将水倒入浓硫酸中（会引起暴沸，液体飞溅）
  • 稀释过程中会放出大量热，需冷却至室温后再转移
  • 如果不慎将浓硫酸溅到皮肤上，应立即用大量水冲洗，再涂上3%-5%的NaHCO₃溶液

第四章 一定物质的量浓度溶液的配制

4.1 配制步骤

配制一定物质的量浓度溶液的步骤（七字诀）：
计称溶转洗定摇
  1. 计——计算：计算所需溶质的质量或浓溶液的体积
  2. 称——称量：用托盘天平称量固体溶质（左物右码）
  3. 溶——溶解：在烧杯中用适量蒸馏水溶解
  4. 转——转移：将溶液沿玻璃棒注入容量瓶
  5. 洗——洗涤：用蒸馏水洗涤烧杯和玻璃棒2-3次，洗液转入容量瓶
  6. 定——定容：加蒸馏水至接近刻度线1-2cm处，改用胶头滴管滴加至刻度线
  7. 摇——摇匀：塞好瓶塞，反复颠倒摇匀
💡 所需仪器：

  • 容量瓶（规格根据所需体积选择：50mL、100mL、250mL、500mL、1000mL）

第五章 化学方程式计算

5.1 化学方程式计算的基本步骤

计算步骤：
  1. 写——写出配平的化学方程式
  2. 标——标出相关物质的化学计量数
  3. 列——列出已知量和未知量
  4. 解——列比例式求解
  5. 答——简明写出答案
💡 计算技巧：
  • 守恒法：质量守恒、电荷守恒、电子守恒
  • 差量法：利用反应前后质量差、体积差等进行计算
  • 关系式法：找出已知物质与待求物质之间的物质的量关系
  • 极值法：用于混合物计算，假设全部为某一组分
  • 十字交叉法：用于混合物中各组分含量的计算
📝 例题8：将5.6g铁粉加入100mL 2mol/L的硫酸铜溶液中，充分反应后，求：
(1) 生成铜的质量
(2) 反应后溶液中Fe²⁺的物质的量浓度（假设体积不变）

【分析思路】
1. 明确已知条件：$m(Fe) = 5.6g$，$c(CuSO_4) = 2mol/L$，$V = 100mL = 0.1L$
2. 确定求解目标：(1) 生成Cu的质量；(2) $c(Fe^{2+})$
3. 写出化学方程式：$Fe + CuSO_4 = FeSO_4 + Cu$
4. 计算各物质的量：分别计算Fe和CuSO₄的物质的量
5. 判断过量：根据化学计量数（1:1）判断哪种物质过量
6. 按不足量计算：根据不足量的物质计算产物

【规范解答】
解：已知 $m(Fe) = 5.6g$，$c(CuSO_4) = 2mol/L$，$V = 0.1L$

① 写出反应方程式：
$Fe + CuSO_4 = FeSO_4 + Cu$

② 计算各反应物的物质的量：
$n(Fe) = \frac{m}{M} = \frac{5.6g}{56g/mol} = 0.1mol$
$n(CuSO_4) = c \cdot V = 2mol/L \times 0.1L = 0.2mol$

③ 判断过量：
根据方程式，Fe与CuSO₄按1:1反应
需要CuSO₄：0.1mol，实际有CuSO₄：0.2mol
∵ $0.1mol < 0.2mol$ ∴ Fe不足，CuSO₄过量
按Fe的量进行计算

④ 计算生成铜的质量：
根据方程式：$n(Cu) = n(Fe) = 0.1mol$
$m(Cu) = n \cdot M = 0.1mol \times 64g/mol = 6.4g$

⑤ 计算Fe²⁺的物质的量浓度：
根据方程式：$n(Fe^{2+}) = n(Fe) = 0.1mol$
$c(Fe^{2+}) = \frac{n}{V} = \frac{0.1mol}{0.1L} = 1mol/L$

【答】(1) 生成铜的质量为6.4g；(2) 反应后溶液中Fe²⁺的物质的量浓度为1mol/L。

【方法总结】
• 过量判断三步法：①计算各物质的量 ②根据计量数比较 ③确定不足量物质
• 计算原则：按不足量的物质进行计算
• 注意：溶液体积不变时，$c = \frac{n}{V_{溶液}}$，不是$V_{溶剂}}$

5.2 过量计算

💡 过量计算步骤：
  1. 写出配平的化学方程式
  2. 计算各反应物的物质的量
  3. 根据化学计量数判断哪种物质过量
  4. 按不足量的物质进行计算
📝 例题9：将10g NaOH与10g HCl混合反应，求生成的NaCl的质量。

【分析思路】
1. 明确已知条件：$m(NaOH) = 10g$，$m(HCl) = 10g$
2. 确定求解目标：生成NaCl的质量
3. 写出化学方程式：$NaOH + HCl = NaCl + H_2O$
4. 计算各物质的量：分别计算NaOH和HCl的物质的量
5. 判断过量：根据化学计量数（1:1）判断哪种物质过量
6. 按不足量计算：根据不足量的物质计算产物NaCl

【规范解答】
解：已知 $m(NaOH) = 10g$，$m(HCl) = 10g$

① 写出反应方程式：
$NaOH + HCl = NaCl + H_2O$

② 计算各反应物的物质的量：
$M(NaOH) = 23 + 16 + 1 = 40g/mol$
$n(NaOH) = \frac{m}{M} = \frac{10g}{40g/mol} = 0.25mol$

$M(HCl) = 1 + 35.5 = 36.5g/mol$
$n(HCl) = \frac{m}{M} = \frac{10g}{36.5g/mol} = 0.274mol$

③ 判断过量：
根据方程式，NaOH与HCl按1:1反应
∵ $0.25mol < 0.274mol$ ∴ NaOH不足，HCl过量
按NaOH的量进行计算

④ 计算生成NaCl的质量：
根据方程式：$n(NaCl) = n(NaOH) = 0.25mol$
$M(NaCl) = 23 + 35.5 = 58.5g/mol$
$m(NaCl) = n \cdot M = 0.25mol \times 58.5g/mol = 14.6g$

【答】生成NaCl的质量为14.6g。

【易错提醒】
• HCl的摩尔质量是36.5g/mol，不是35.5g/mol（Cl的原子量）
• 不能直接按质量比10:10=1:1判断，必须换算为物质的量比
• 等质量的NaOH和HCl，由于摩尔质量不同，物质的量也不同

第六章 综合计算技巧

6.1 守恒法

🔧 质量守恒：化学反应前后，参加反应的各物质的质量总和等于反应后生成的各物质的质量总和。
🔧 电荷守恒：溶液中阳离子所带正电荷总数等于阴离子所带负电荷总数，溶液呈电中性。
🔧 电子守恒：氧化还原反应中，氧化剂得到的电子总数等于还原剂失去的电子总数。
📝 例题10（电子守恒）：用0.1mol/L的Na₂SO₃溶液30mL恰好将2×10⁻³mol的XO₄⁻还原，则元素X在还原产物中的化合价是（）

【分析思路】
1. 明确已知条件：$c(Na_2SO_3) = 0.1mol/L$，$V = 30mL$，$n(XO_4^-) = 2\times10^{-3}mol$
2. 确定求解目标：X在还原产物中的化合价
3. 分析氧化还原过程：
• Na₂SO₃是还原剂，被氧化，S元素化合价升高
• XO₄⁻是氧化剂，被还原，X元素化合价降低
4. 计算电子转移：
• S从+4价升至+6价，每个S失去2个电子
• X从+7价降至+x价，每个X得到(7-x)个电子
5. 应用电子守恒：失去的电子总数 = 得到的电子总数

【规范解答】
解：已知 $c(Na_2SO_3) = 0.1mol/L$，$V = 30mL = 0.03L$，$n(XO_4^-) = 2\times10^{-3}mol$

① 分析氧化过程（Na₂SO₃被氧化）：
Na₂SO₃ → Na₂SO₄，S元素化合价：+4 → +6
每个S原子失去电子数 = $6 - 4 = 2$ 个
$n(Na_2SO_3) = c \cdot V = 0.1mol/L \times 0.03L = 0.003mol$
失去电子总数 = $0.003mol \times 2 = 0.006mol$

② 分析还原过程（XO₄⁻被还原）：
XO₄⁻中，设X化合价为+a，则 $a + (-2)\times4 = -1$，解得 $a = +7$
设还原产物中X化合价为+x
每个X原子得到电子数 = $(7-x)$ 个
得到电子总数 = $2\times10^{-3}mol \times (7-x)$

③ 应用电子守恒列方程：
失去的电子总数 = 得到的电子总数
$0.006 = 2\times10^{-3} \times (7-x)$
$3 = 7 - x$
$x = 4$

【答案】X在还原产物中的化合价为 +4，选项选对应+4价的选项。

【方法总结】
• 电子守恒法步骤：①标化合价 ②算电子转移 ③列守恒方程
• 氧化剂：得电子，化合价降低；还原剂：失电子，化合价升高
• 关键：准确计算每个原子得失电子数，注意化学计量数
• 技巧：离子中元素化合价计算：各元素化合价代数和 = 离子电荷数

6.2 差量法

🔧 差量法原理：利用反应前后质量差、体积差、物质的量差等进行计算。
📝 例题11：将足量铁粉放入100mL 1mol/L的CuSO₄溶液中，充分反应后，过滤、洗涤、干燥，得到固体多少克？固体质量增加了多少克？

【分析思路】
1. 明确已知条件：$c(CuSO_4) = 1mol/L$，$V = 100mL = 0.1L$，Fe粉足量
2. 确定求解目标：(1) 固体总质量；(2) 固体质量增加量
3. 写出反应方程式：$Fe + CuSO_4 = FeSO_4 + Cu$
4. 分析固体质量变化：
• 反应前固体：Fe粉（足量，部分反应）
• 反应后固体：剩余Fe粉 + 生成的Cu
• 每1mol Fe反应，溶解56g Fe，析出64g Cu
• 固体质量变化：$\Delta m = 64g - 56g = +8g$（增加8g）
5. 计算CuSO₄的物质的量：确定反应了多少mol
6. 应用差量法：根据反应的量计算质量变化

【规范解答】
解：已知 $c(CuSO_4) = 1mol/L$，$V = 0.1L$

① 写出反应方程式：
$Fe + CuSO_4 = FeSO_4 + Cu$

② 计算CuSO₄的物质的量：
$n(CuSO_4) = c \cdot V = 1mol/L \times 0.1L = 0.1mol$

③ 分析固体质量变化（差量法）：
根据方程式，每1mol Fe反应：
• 溶解（进入溶液）的Fe质量：$M(Fe) = 56g/mol$
• 析出（附着在固体上）的Cu质量：$M(Cu) = 64g/mol$
• 固体质量变化：$\Delta m = 64g - 56g = +8g$（增加8g）

④ 计算固体质量增加量：
反应了0.1mol CuSO₄，则固体质量增加：
$\Delta m = 0.1mol \times 8g/mol = 0.8g$

⑤ 计算生成Cu的质量：
$n(Cu) = n(CuSO_4) = 0.1mol$
$m(Cu) = n \cdot M = 0.1mol \times 64g/mol = 6.4g$

⑥ 计算固体总质量：
设加入Fe粉质量为 $m$
反应消耗的Fe质量：$0.1mol \times 56g/mol = 5.6g$
剩余Fe粉质量：$m - 5.6g$
固体总质量 = 剩余Fe粉 + Cu = $(m - 5.6g) + 6.4g = m + 0.8g$

【答】固体总质量为 $(m + 0.8)g$（其中 $m$ 为加入Fe粉的质量），固体质量增加了0.8g。

【方法总结】
• 差量法原理：利用反应前后固体（或气体、溶液）质量差进行计算
• 固体质量变化分析：
- 金属置换反应：$\Delta m = M(析出金属) - M(溶解金属)$
- 若 $\Delta m > 0$，固体质量增加；若 $\Delta m < 0$，固体质量减少
• 优点：不需要知道反应物的具体质量，只需知道变化的量

【易错提醒】
• ⚠️ "足量铁粉"意味着CuSO₄完全反应，按CuSO₄的量计算
• ⚠️ 固体质量增加 ≠ 生成Cu的质量，而是 Cu质量 - 消耗Fe质量
• ⚠️ 固体总质量包括剩余的铁和生成的铜

6.3 关系式法

🔧 关系式法：找出已知物质与待求物质之间的物质的量关系，一步计算。
📝 例题12：用含杂质10%的石灰石（主要成分为CaCO₃）与足量盐酸反应，制取4.48L（标准状况）CO₂，需要石灰石多少克？

【分析思路】
1. 明确已知条件：标准状况下 $V(CO_2) = 4.48L$，石灰石含杂质10%（即含CaCO₃ 90%）
2. 确定求解目标：需要石灰石的质量
3. 写出反应方程式：$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$
4. 找出关系式：从方程式看出 $CaCO_3 \sim CO_2$（1:1关系）
5. 计算CO₂的物质的量：用 $n = \frac{V}{V_m}$
6. 计算纯CaCO₃质量：根据关系式，$n(CaCO_3) = n(CO_2)$
7. 计算石灰石质量：考虑杂质含量，石灰石质量 = $\frac{m(CaCO_3)}{纯度}$

【规范解答】
解：已知标准状况下 $V(CO_2) = 4.48L$，石灰石纯度 = $1 - 10\% = 90\% = 0.9$

① 写出反应方程式：
$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$

② 找出关系式：
从方程式可知：$CaCO_3 \sim CO_2$（物质的量比为1:1）

③ 计算CO₂的物质的量：
$n(CO_2) = \frac{V}{V_m} = \frac{4.48L}{22.4L/mol} = 0.2mol$

④ 计算纯CaCO₃的物质的量和质量：
根据关系式：$n(CaCO_3) = n(CO_2) = 0.2mol$
$M(CaCO_3) = 40 + 12 + 16\times3 = 100g/mol$
$m(CaCO_3) = n \cdot M = 0.2mol \times 100g/mol = 20g$

⑤ 计算石灰石的质量：
石灰石含CaCO₃ 90%，则：
石灰石质量 = $\frac{m(CaCO_3)}{纯度} = \frac{20g}{0.9} = 22.2g$

【答】需要石灰石22.2克。

【方法总结】
• 关系式法步骤：①写方程式 ②找关系 ③算已知 ④推未知
• 杂质计算：纯物质质量 = 不纯物质质量 × 纯度
• 反推不纯物质：不纯物质质量 = $\frac{纯物质质量}{纯度}$
• 注意：纯度 = 1 - 杂质质量分数

【易错提醒】
• ⚠️ 注意是纯物质还是不纯物质，题目问的是石灰石（含杂质），不是纯CaCO₃
• ⚠️ 标准状况下气体才能用22.4L/mol
• ⚠️ 关系式中的化学计量数要配平后再确定

第七章 易错点总结与防范

7.1 常见易错点

易错点 错误示例 正确做法

溶液体积 将溶剂体积当作溶液体积 溶液体积 ≠ 溶剂体积

7.2 阿伏伽德罗常数题常见陷阱

⚠️ 常见陷阱：
  • ❌ 标准状况下，水、酒精、SO₃等为液态或固态，不能用22.4L/mol
  • ❌ 稀有气体是单原子分子（He、Ne等），不是双原子
  • ❌ 可逆反应不能进行到底，反应物转化率小于100%
  • ❌ 胶体粒子是多个分子的集合体，不是单个分子
  • ❌ 没有说明体积时，不能计算粒子数
  • ❌ 注意隐含反应（如NO₂与N₂O₄的平衡）

第八章 氧化还原反应（整合补充）

8.1 氧化还原反应基础

💡 基本概念：
  • 氧化反应：失去电子，化合价升高，被氧化
  • 还原反应：得到电子，化合价降低，被还原
  • 氧化剂：得电子，化合价降低，被还原，具有氧化性
  • 还原剂：失电子，化合价升高，被氧化，具有还原性

口诀：升失氧还（升价、失电子、被氧化、作还原剂），降得还氧（降价、得电子、被还原、作氧化剂）

电子守恒核心公式：
氧化剂得电子总数 = 还原剂失电子总数
$n_{(氧化剂)} \times 变价原子数 \times |化合价变化| = n_{(还原剂)} \times 变价原子数 \times |化合价变化|$

8.2 常见氧化剂还原剂及产物

氧化剂 还原产物（酸性） 典型反应
KMnO₄（酸性） Mn²⁺（无色） MnO₄⁻ + 5e⁻ + 8H⁺ → Mn²⁺ + 4H₂O
K₂Cr₂O₇（酸性） Cr³⁺（绿色） Cr₂O₇²⁻ + 6e⁻ + 14H⁺ → 2Cr³⁺ + 7H₂O
HNO₃（浓） NO₂ Cu + 4HNO₃(浓) = Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
HNO₃（稀） NO 3Cu + 8HNO₃(稀) = 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
浓H₂SO₄（加热） SO₂ Cu + 2H₂SO₄(浓) △ CuSO₄ + SO₂↑ + 2H₂O
Cl₂ Cl⁻ 歧化反应：Cl₂ + 2NaOH = NaCl + NaClO + H₂O
Fe³⁺ Fe²⁺ 2Fe³⁺ + Fe = 3Fe²⁺
H₂O₂（氧化剂） H₂O H₂O₂ + 2Fe²⁺ + 2H⁺ = 2Fe³⁺ + 2H₂O
💡 歧化反应与归中反应：
  • 歧化反应：同一元素既升又降。例：Cl₂ + 2NaOH = NaCl + NaClO + H₂O（Cl：0→-1和+1）
  • 归中反应：同一元素高低价态向中间价态靠拢。例：2H₂S + SO₂ = 3S↓ + 2H₂O（S：-2→0，+4→0）
  • 价态规律：相邻价态不反应，相间价态归中间，跳位转化是歧化

8.3 氧化性还原性强弱比较

💡 判断依据：
  1. 方程式判断：氧化性：氧化剂 > 氧化产物；还原性：还原剂 > 还原产物
  2. 活动性顺序：金属单质还原性：K > Ca > Na > Mg > Al > Zn > Fe > Sn > Pb > (H) > Cu > Hg > Ag > Pt > Au；对应阳离子氧化性相反
  3. 反应条件：反应条件越简单，氧化性/还原性越强
  4. 产物价态：相同还原剂被不同氧化剂氧化，氧化剂氧化性越强，还原剂升价越高

第九章 离子反应（整合补充）

9.1 离子方程式书写规则

⚠️ 拆写规则（关键）：
  • 必须拆：强酸、强碱、可溶性盐（查溶解性表）
  • 不能拆：单质、氧化物、难溶物、弱电解质（弱酸、弱碱、水）、非电解质、固体间的反应
  • 微溶物处理：澄清溶液拆（如澄清石灰水Ca²⁺ + 2OH⁻），悬浊液不拆（如石灰乳Ca(OH)₂）
  • 酸式盐：NaHSO₄在水中拆为Na⁺ + H⁺ + SO₄²⁻；NaHCO₃拆为Na⁺ + HCO₃⁻（HCO₃⁻弱酸酸式根不拆）

9.2 离子共存判断（高频考点）

💡 不能共存的情况：
  1. 生成沉淀：Ba²⁺与SO₄²⁻、CO₃²⁻；Ag⁺与Cl⁻、Br⁻、I⁻；Ca²⁺与CO₃²⁻、SO₄²⁻（微溶）；Mg²⁺、Fe³⁺、Al³⁺、Cu²⁺、Fe²⁺与OH⁻
  2. 生成气体：H⁺与CO₃²⁻、HCO₃⁻、SO₃²⁻、HSO₃⁻、S²⁻、HS⁻
  3. 生成弱电解质：H⁺与OH⁻、CH₃COO⁻、F⁻、ClO⁻、CN⁻等；OH⁻与NH₄⁺
  4. 氧化还原：Fe³⁺与I⁻、S²⁻；NO₃⁻(H⁺)与Fe²⁺、I⁻、SO₃²⁻、S²⁻；MnO₄⁻(H⁺)、Cr₂O₇²⁻(H⁺)、ClO⁻与还原性离子
  5. 双水解彻底：Al³⁺与CO₃²⁻、HCO₃⁻、S²⁻、HS⁻、AlO₂⁻；Fe³⁺与CO₃²⁻、HCO₃⁻、AlO₂⁻
  6. 络合反应：Fe³⁺与SCN⁻（血红色）、C₆H₅O⁻；Ag⁺与NH₃·H₂O（过量生成银氨络离子）
💡 隐含条件分析：
  • 无色溶液：排除Cu²⁺（蓝）、Fe³⁺（黄）、Fe²⁺（浅绿）、MnO₄⁻（紫红）、Cr₂O₇²⁻（橙）
  • 酸性溶液：pH < 7、使石蕊变红、与Zn反应放H₂ → 隐含H⁺，NO₃⁻在酸性下具强氧化性
  • 碱性溶液：pH > 7、使石蕊变蓝/酚酞变红 → 隐含OH⁻
  • 水电离c(H⁺) = 10⁻¹² mol/L：可能是强酸或强碱抑制了水的电离

第十章 晶体结构计算（整合补充）

10.1 晶胞计算基本公式

晶胞密度计算：

ρ = (Z × M) / (Nₐ × V)

Z：晶胞中粒子数（均摊法计算）
M：摩尔质量 (g/mol)
Nₐ：阿伏伽德罗常数 (6.02×10²³ mol⁻¹)
V：晶胞体积 (cm³)

单位换算：
1 pm = 10⁻¹⁰ cm = 10⁻¹² m
1 nm = 10⁻⁷ cm = 10⁻⁹ m
1 Å = 10⁻⁸ cm = 10⁻¹⁰ m

立方晶胞：V = a³（a为边长）

10.2 均摊法计算晶胞粒子数

位置 顶点 棱上 面心 体心
贡献 1/8 1/4 1/2 1
说明 8个晶胞共用 4个晶胞共用 2个晶胞共用 独占
📝 典型晶胞计算示例：

NaCl型（面心立方）：
Na⁺：12条棱×1/4 + 1个体心 = 4个
Cl⁻：8个顶点×1/8 + 6个面心×1/2 = 4个
Z = 4个NaCl单元

CsCl型（体心立方）：
Cs⁺：1个体心 = 1个
Cl⁻：8个顶点×1/8 = 1个
Z = 1个CsCl单元

金刚石（原子晶体）：
8个顶点×1/8 + 6个面心×1/2 + 4个体内 = 8个C原子
每个C连4个键，每键被2C共享，共16个C-C键/晶胞

第十一章 滴定计算与误差分析（整合补充）

11.1 中和滴定计算

计算公式：

c(待) = [c(标) × V(标) × n(待)] / [V(待) × n(标)]

其中n为酸碱的元数（如H₂SO₄为2元，Na₂CO₃与HCl反应到终点为2元或1元）

关键：准确判断终点，正确读取体积（保留2位小数），单位统一

11.2 氧化还原滴定

🔹 高锰酸钾法

MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O

指示剂：自身指示剂（紫色→无色）

应用：测Fe²⁺、H₂C₂O₄、H₂O₂等

注意：不用额外加指示剂，终点为微红色30s不褪

🔹 碘量法

I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻

指示剂：淀粉（蓝色→无色）

应用：测Cu²⁺、ClO⁻、H₂O₂等

注意：近终点时加淀粉，防止包裹I₂

11.3 滴定误差分析

操作 V(标)变化 c(待)结果 原因
滴定管未润洗（装标准液） 偏大 偏高 标准液被稀释，耗量增多
锥形瓶润洗 偏大 偏高 待测液增多
滴定前尖嘴有气泡，滴后无 偏大 偏高 读数包含气泡体积
滴后尖嘴有气泡 偏小 偏低 实际滴出液少
读数俯视初读数 偏大 偏高 V(标) = 末 - 初，初读偏小
读数仰视末读数 偏大 偏高 末读偏大
待测液溅出 偏小 偏低 溶质减少
标准液滴到锥形瓶外 偏大 偏高 无效消耗标准液

第十二章 有机化学计算（整合补充）

12.1 燃烧法确定分子式

燃烧通式：

CₓHᵧO₂ + (x + y/4 - z/2)O₂ → xCO₂ + y/2H₂O

计算方法：
n(C) = n(CO₂) = m(CO₂)/44
n(H) = 2n(H₂O) = 2m(H₂O)/18
n(O) = [m(有机物) - 12n(C) - n(H)]/16

最简式法：
求出C:H:O最简整数比，结合相对分子质量确定分子式

12.2 不饱和度计算

🔹 公式

Ω = (2C + 2 + N - H - X) / 2

其中：C碳数，N氮数，H氢数，X卤素数

氧不影响不饱和度

🔹 结构对应

Ω = 1：一个双键或一个环

Ω = 2：两个双键/一个三键/两个环/一个双键+一个环

Ω = 4：可能含苯环（苯环Ω=4，3个双键+1个环）

12.3 官能团定量计算

官能团 定量反应 计量关系 应用
-CHO 银镜反应 -CHO ~ 2Ag 醛基含量测定
-CHO 与新制Cu(OH)₂ -CHO ~ Cu₂O 醛基检验与测定
-OH（醇） 与Na反应 2-OH ~ H₂ 羟基数目测定
-COOH 与NaHCO₃ -COOH ~ CO₂ 羧基检验与测定
C=C 与Br₂加成 C=C ~ Br₂ 双键数目测定
苯环 与H₂加成 苯环 ~ 3H₂ 验证苯环结构
