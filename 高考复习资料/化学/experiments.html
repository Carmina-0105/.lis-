🧪 化学实验专题

基本操作 · 物质制备 · 性质检验 · 定量实验
🎯 模块核心

化学实验是化学学科的重要组成部分，高考中主要考查实验基本操作、物质的制备与性质检验、定量实验等内容。

高考分值：约15-20分（选择题6分 + 实验大题9-14分）
重点：实验基本操作、气体制备、离子检验、定量滴定

第一章 化学实验基本操作

1.1 常用仪器及使用

仪器类别 仪器名称 主要用途 注意事项
计量仪器 量筒 量取液体体积 不能加热，不能作反应容器，读数平视
容量瓶 配制一定物质的量浓度的溶液 不能加热，不能溶解固体，使用前要检漏
滴定管 滴定操作，精确量取液体 酸式/碱式要区分，使用前要润洗，读数精确到0.01mL
加热仪器 酒精灯 提供热源 用外焰加热，禁止对点，熄灭用灯帽
酒精喷灯 高温加热 温度可达1000℃以上
分离仪器 漏斗 过滤、向小口容器转移液体 过滤时"一贴二低三靠"
分液漏斗 分液、萃取、滴加液体 使用前要检漏，上层液体上口出，下层液体下口出

1.2 基本操作要点

💡 药品取用：
  • 固体药品：粉末用药匙，块状用镊子
  • 液体药品：少量用胶头滴管，多量直接倾倒
  • 原则：不能用手接触，不能尝，不能直闻（扇闻）
💡 加热操作：
  • 直接加热：试管、坩埚、蒸发皿、燃烧匙
  • 垫石棉网加热：烧杯、烧瓶、锥形瓶
  • 水浴加热：温度不超过100℃，受热均匀
💡 溶解与过滤：
  • 溶解：用玻璃棒搅拌加速溶解
  • 过滤：一贴（滤纸紧贴漏斗）、二低（滤纸低于漏斗边缘，液面低于滤纸边缘）、三靠（烧杯靠玻璃棒，玻璃棒靠三层滤纸，漏斗下端靠烧杯壁）
💡 蒸发与结晶：
  • 蒸发：用蒸发皿，玻璃棒不断搅拌，防止局部过热
  • 结晶：蒸发结晶（溶解度随温度变化小）、降温结晶（溶解度随温度变化大）

第二章 常见气体的制备

2.1 气体制备装置

🔸 固+固加热型：制O₂、NH₃等
装置：大试管+单孔塞+导管+酒精灯
注意：试管口略向下倾斜，防止冷凝水倒流
🔸 固+液不加热型：制H₂、CO₂、H₂S、SO₂、NO₂、C₂H₂等
装置：启普发生器或简易装置（锥形瓶+分液漏斗）
注意：启普发生器适用于块状固体和液体反应
🔸 固+液加热型：制Cl₂、HCl、NO等
装置：圆底烧瓶+分液漏斗+酒精灯
注意：需要加热，注意防倒吸

2.2 重要气体的制备

气体 反应原理 收集方法 检验方法 尾气处理
O₂ $2KMnO_4 \stackrel{\Delta}{\longrightarrow} K_2MnO_4 + MnO_2 + O_2↑$
或 $2H_2O_2 \stackrel{MnO_2}{\longrightarrow} 2H_2O + O_2↑$		 排水法或向上排空气法 带火星木条复燃 无需处理
H₂ $Zn + H_2SO_4 = ZnSO_4 + H_2↑$ 排水法或向下排空气法 点燃有淡蓝色火焰 点燃或收集
CO₂ $CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2↑$ 向上排空气法 通入澄清石灰水变浑浊 无需处理
Cl₂ $MnO_2 + 4HCl(浓) \stackrel{\Delta}{\longrightarrow} MnCl_2 + 2H_2O + Cl_2↑$ 向上排空气法或排饱和食盐水法 使湿润的淀粉-KI试纸变蓝 用NaOH溶液吸收
NH₃ $2NH_4Cl + Ca(OH)_2 \stackrel{\Delta}{\longrightarrow} CaCl_2 + 2NH_3↑ + 2H_2O$ 向下排空气法 使湿润的红色石蕊试纸变蓝 用水或稀硫酸吸收（防倒吸）
SO₂ $Na_2SO_3 + H_2SO_4 = Na_2SO_4 + H_2O + SO_2↑$ 向上排空气法 使品红溶液褪色，加热恢复红色 用NaOH溶液吸收
NO $3Cu + 8HNO_3(稀) = 3Cu(NO_3)_2 + 2NO↑ + 4H_2O$ 排水法（不能用排空气法） 遇空气变红棕色 与O₂混合通入NaOH溶液
NO₂ $Cu + 4HNO_3(浓) = Cu(NO_3)_2 + 2NO_2↑ + 2H_2O$ 向上排空气法 红棕色气体 用NaOH溶液吸收
C₂H₄ $C_2H_5OH \stackrel{浓H_2SO_4,170℃}{\longrightarrow} CH_2=CH_2↑ + H_2O$ 排水法 使溴水或酸性KMnO₄褪色 点燃
C₂H₂ $CaC_2 + 2H_2O → Ca(OH)_2 + C_2H_2↑$ 排水法 使溴水或酸性KMnO₄褪色 点燃

第四章 定量实验

4.1 一定物质的量浓度溶液的配制

🔸 主要仪器：容量瓶、烧杯、玻璃棒、胶头滴管、量筒（或托盘天平）
💡 操作步骤：
  1. 计算：计算所需溶质的质量或浓溶液的体积
  2. 称量/量取：用托盘天平称量固体，或用量筒量取液体
  3. 溶解/稀释：在烧杯中溶解或稀释，用玻璃棒搅拌
  4. 转移：冷却后转移至容量瓶，用玻璃棒引流
  5. 洗涤：洗涤烧杯和玻璃棒2-3次，洗液一并转入容量瓶
  6. 定容：加水至接近刻度线1-2cm处，改用胶头滴管滴加至刻度线
  7. 摇匀：塞好瓶塞，反复颠倒摇匀
⚠️ 误差分析：
  • 偏高：砝码生锈、量取浓溶液时仰视、定容时俯视
  • 偏低：砝码缺角、转移时溶液溅出、定容时仰视、未冷却就转移
  • 无影响：容量瓶中原有少量蒸馏水

4.2 酸碱中和滴定

🔸 主要仪器：酸式滴定管、碱式滴定管、锥形瓶、铁架台、滴定管夹
💡 操作步骤：
  1. 洗涤：用蒸馏水洗涤滴定管、锥形瓶
  2. 润洗：用待装液润洗滴定管2-3次
  3. 装液：装入待装液，排气泡，调零，记录初始读数
  4. 取液：用移液管或滴定管准确量取待测液于锥形瓶中
  5. 滴定：左手控制滴定管，右手摇动锥形瓶，眼睛观察溶液颜色变化
  6. 终点判断：滴入最后半滴标准液，溶液颜色突变且30s内不恢复
  7. 读数：记录终点读数
  8. 计算：重复2-3次，取平均值计算
计算公式：
$c_{待测} = \frac{c_{标准} \cdot V_{标准}}{V_{待测}}$
⚠️ 误差分析：
  • 偏高：滴定管未润洗、锥形瓶用待测液润洗、滴定前仰视/滴定后俯视
  • 偏低：待测液溅出、滴定前俯视/滴定后仰视、标准液浓度偏小

第五章 实验拓展（整合补充）

5.1 气体制备四步法则

💡 气体制备流程：

发生装置 → 净化装置 → 干燥装置 → 收集/性质验证装置 → 尾气处理装置

发生装置类型 适用条件 典型气体
固固加热型 固体+固体，需加热 O₂、NH₃、CH₄
固液加热型 固体+液体，需加热 Cl₂、HCl、SO₂（浓H₂SO₄+Cu）
固液不加热型（简易） 固体+液体，不加热 H₂、CO₂、H₂S
固液不加热型（分液） 固体+液体，不加热，需控制速率 C₂H₂、NO、NO₂、SO₂（Na₂SO₃+H₂SO₄）
液液加热型 液体+液体，需加热 C₂H₄

5.2 温度计位置（高频考点）

💡 温度计位置要点：
  • 蒸馏/分馏：温度计水银球位于支管口处（测蒸气温度）
  • 乙烯制备：温度计插入反应液中（控制170℃）
  • 硝基苯制备：温度计插入水浴中（控制50-60℃）
  • 银镜反应：水浴加热，温度计插入水浴中（控制60-70℃）

5.3 干燥剂选择

⚠️ 干燥剂选择（牢记不可干燥的气体）：
  • 浓H₂SO₄（酸性、强氧化性）：不能干燥NH₃（反应）、H₂S（氧化）、HI、HBr（氧化）
  • 无水CaCl₂（中性）：不能干燥NH₃（形成CaCl₂·8NH₃配合物）
  • 碱石灰（NaOH+CaO，碱性）：不能干燥酸性气体（CO₂、SO₂、HCl、H₂S、Cl₂、NO₂等）
  • P₂O₅（酸性）：不能干燥NH₃

5.4 重要气体制备详解

🔸 Cl₂的制备（重点）：
MnO₂ + 4HCl(浓) →(△) MnCl₂ + Cl₂↑ + 2H₂O

装置顺序：
  1. 发生装置：圆底烧瓶+分液漏斗（浓盐酸）+酒精灯加热
  2. 净化装置：饱和食盐水（洗气瓶，除HCl）
  3. 干燥装置：浓H₂SO₄（洗气瓶，除水）
  4. 收集装置：向上排空气法或排饱和食盐水法
  5. 尾气处理：NaOH溶液（注意防倒吸）
关键细节：
  • 必须用浓盐酸（稀盐酸不反应）
  • 饱和食盐水作用：Cl₂在饱和食盐水中溶解度小（同离子效应），HCl极易溶
  • 检验：湿润淀粉KI试纸（变蓝：Cl₂ + 2KI = 2KCl + I₂）
🔸 NH₃的制备（重点）：
2NH₄Cl + Ca(OH)₂ →(△) CaCl₂ + 2NH₃↑ + 2H₂O

注意：
  • 不能用NH₄NO₃（易爆）或NH₄HCO₃（分解产物复杂）
  • 不能用NaOH或KOH代替Ca(OH)₂（腐蚀玻璃，且易结块不利于NH₃逸出）
  • 收集：向下排空气法；试管口塞一团棉花（防对流，使收集更纯）
  • 干燥：碱石灰（不能用浓H₂SO₄、P₂O₅、无水CaCl₂）
🔸 C₂H₄（乙烯）的制备：
C₂H₅OH →(浓H₂SO₄/170℃) CH₂=CH₂↑ + H₂O

关键细节：
  • 浓H₂SO₄作用：催化剂、脱水剂、吸水剂
  • 体积比：V(乙醇):V(浓H₂SO₄) ≈ 1:3
  • 温度计插入反应液中（控制170℃）
  • 加入碎瓷片（或沸石）防暴沸
  • 温度控制：140℃时生成乙醚，需迅速升温至170℃
  • 净化：NaOH溶液除SO₂、CO₂；碱石灰干燥
🔸 C₂H₂（乙炔）的制备：
CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑

注意：
  • 不能用启普发生器（反应剧烈放热，易损坏；生成的Ca(OH)₂糊状易堵塞）
  • 用饱和食盐水代替水（减缓反应速率，获得平稳气流）
  • 净化：CuSO₄溶液除H₂S、PH₃杂质（电石中含S、P杂质）

5.5 尾气处理

⚠️ 有毒气体尾气处理：
  • Cl₂、SO₂、H₂S、NO₂、HCl：用NaOH溶液吸收
  • NO（不与水反应）：与O₂混合后通入NaOH溶液；或收集（气球）
  • NH₃：用水吸收，防倒吸（倒扣漏斗、安全瓶）
  • CO：点燃或收集（气球）
  • 防倒吸装置：倒扣漏斗、安全瓶（短进短出）、球形干燥管

5.6 气体检验方法总结

气体 检验方法 现象
O₂ 带火星的木条 木条复燃
H₂ 点燃 淡蓝色火焰，有爆鸣声
CO₂ 澄清石灰水 石灰水变浑浊
Cl₂ 湿润的淀粉-KI试纸 试纸变蓝
NH₃ 湿润的红色石蕊试纸 试纸变蓝
SO₂ 品红溶液 品红褪色，加热恢复红色
NO 暴露在空气中 无色变红棕色
NO₂ 观察颜色 红棕色气体
