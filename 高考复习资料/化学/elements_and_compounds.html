🧪 元素化合物

金属元素 · 非金属元素 · 元素周期律
🎯 模块核心

元素化合物是化学学习的基础，涵盖金属元素（钠、铝、铁等）和非金属元素（氯、硫、氮、硅等）的性质、制备和应用。这部分知识是化学实验、工业流程和综合推断题的重要基础。

高考分值：约20-25分（选择题6-12分 + 综合题12-15分）
重点：钠铝铁的性质、氯硫氮的化合物、元素周期律应用

第一章 金属元素及其化合物

1.1 钠及其化合物

🔸 钠（Na）的性质
  • 物理性质：银白色金属，质软（可用小刀切割），密度小于水（0.97g/cm³），熔点低（97.8℃）
  • 化学性质：活泼金属，易失去电子，强还原剂
钠的化学反应：
1. 与氧气反应：
常温：$4Na + O_2 = 2Na_2O$（白色固体）
加热：$2Na + O_2 \stackrel{\Delta}{\longrightarrow} Na_2O_2$（淡黄色固体）

2. 与水反应：
$2Na + 2H_2O = 2NaOH + H_2↑$
现象：浮、熔、游、响、红（加酚酞）

3. 与酸反应：
$2Na + 2HCl = 2NaCl + H_2↑$

4. 与盐溶液反应（先与水反应）：
$2Na + CuSO_4 + 2H_2O = Cu(OH)_2↓ + Na_2SO_4 + H_2↑$
🔸 氧化钠（Na₂O）与过氧化钠（Na₂O₂）
性质 Na₂O Na₂O₂
颜色状态 白色固体 淡黄色固体
氧元素化合价 -2 -1
与水反应 $Na_2O + H_2O = 2NaOH$ $2Na_2O_2 + 2H_2O = 4NaOH + O_2↑$
与CO₂反应 $Na_2O + CO_2 = Na_2CO_3$ $2Na_2O_2 + 2CO_2 = 2Na_2CO_3 + O_2$
用途 制NaOH 供氧剂、漂白剂
💡 Na₂O₂的重要计算技巧：
Na₂O₂与CO₂、H₂O反应时，固体增加的质量相当于CO或H₂的质量。
关系式：$Na_2O_2 \sim CO$，$Na_2O_2 \sim H_2$
🔸 碳酸钠（Na₂CO₃）与碳酸氢钠（NaHCO₃）
性质 Na₂CO₃（纯碱/苏打） NaHCO₃（小苏打）
溶解性 易溶于水 可溶于水（溶解度小于Na₂CO₃）
溶液碱性 较强（水解程度大） 较弱
热稳定性 稳定，高温不分解 不稳定：$2NaHCO_3 \stackrel{\Delta}{\longrightarrow} Na_2CO_3 + H_2O + CO_2↑$
与酸反应 $CO_3^{2-} + 2H^+ = CO_2↑ + H_2O$ $HCO_3^- + H^+ = CO_2↑ + H_2O$（反应快）
与CaCl₂反应 $Na_2CO_3 + CaCl_2 = CaCO_3↓ + 2NaCl$ 不反应
用途 玻璃、造纸、纺织、洗涤剂 发酵粉、治疗胃酸过多、灭火剂
⚠️ Na₂CO₃与NaHCO₃的鉴别：
  • 加热法：NaHCO₃受热分解产生CO₂，使澄清石灰水变浑浊
  • 加CaCl₂溶液：Na₂CO₃产生白色沉淀，NaHCO₃无现象
  • 逐滴加盐酸：Na₂CO₃开始无气泡，后产生气泡；NaHCO₃立即产生气泡

1.2 铝及其化合物

🔸 铝（Al）的性质
  • 物理性质：银白色金属，密度小（2.7g/cm³），导电性、导热性好
  • 化学性质：活泼金属，但表面易形成致密氧化膜，具有抗腐蚀性
铝的化学反应：
1. 与氧气反应：
$4Al + 3O_2 = 2Al_2O_3$（致密氧化膜，保护内部铝）

2. 与酸反应：
$2Al + 6HCl = 2AlCl_3 + 3H_2↑$

3. 与强碱反应：
$2Al + 2NaOH + 2H_2O = 2NaAlO_2 + 3H_2↑$
或：$2Al + 2NaOH + 6H_2O = 2Na[Al(OH)_4] + 3H_2↑$

4. 铝热反应：
$2Al + Fe_2O_3 \stackrel{高温}{\longrightarrow} 2Fe + Al_2O_3$（焊接铁轨）
🔸 氧化铝（Al₂O₃）与氢氧化铝（Al(OH)₃）
Al₂O₃和Al(OH)₃都是两性化合物，既能与酸反应，又能与强碱反应。
Al₂O₃的反应：
与酸：$Al_2O_3 + 6HCl = 2AlCl_3 + 3H_2O$
与碱：$Al_2O_3 + 2NaOH = 2NaAlO_2 + H_2O$

Al(OH)₃的反应：
与酸：$Al(OH)_3 + 3HCl = AlCl_3 + 3H_2O$
与碱：$Al(OH)_3 + NaOH = NaAlO_2 + 2H_2O$

Al(OH)₃的制备：
$Al^{3+} + 3NH_3·H_2O = Al(OH)_3↓ + 3NH_4^+$
注意：不能用强碱，因为Al(OH)₃会溶于过量强碱
Al³⁺与OH⁻反应的图像分析：
向AlCl₃溶液中逐滴加入NaOH溶液：
先沉淀：$Al^{3+} + 3OH^- = Al(OH)_3↓$
后溶解：$Al(OH)_3 + OH^- = AlO_2^- + 2H_2O$

沉淀最大值与溶解消耗NaOH的体积比为 3:1

1.3 铁及其化合物

🔸 铁（Fe）的性质
  • 物理性质：银白色金属，有磁性，密度7.86g/cm³，熔点1535℃
  • 化学性质：较活泼金属，常见化合价为+2、+3
铁的化学反应：
1. 与氧气反应：
常温：缓慢氧化（生锈）
点燃：$3Fe + 2O_2 \stackrel{点燃}{\longrightarrow} Fe_3O_4$

2. 与酸反应：
与非氧化性酸：$Fe + 2HCl = FeCl_2 + H_2↑$
与氧化性酸（稀硝酸）：$Fe + 4HNO_3 = Fe(NO_3)_3 + NO↑ + 2H_2O$（Fe少量）

3. 与盐溶液反应：
$Fe + CuSO_4 = FeSO_4 + Cu$（湿法炼铜）
$Fe + 2FeCl_3 = 3FeCl_2$

4. 与水蒸气反应：
$3Fe + 4H_2O(g) \stackrel{高温}{\longrightarrow} Fe_3O_4 + 4H_2$
🔸 铁的氧化物
氧化物 FeO Fe₂O₃ Fe₃O₄
颜色 黑色 红棕色 黑色
俗称 - 铁红 磁性氧化铁
铁化合价 +2 +3 +2、+3
与酸反应 $FeO + 2H^+ = Fe^{2+} + H_2O$ $Fe_2O_3 + 6H^+ = 2Fe^{3+} + 3H_2O$ $Fe_3O_4 + 8H^+ = Fe^{2+} + 2Fe^{3+} + 4H_2O$
🔸 Fe²⁺与Fe³⁺的转化
转化 反应方程式
Fe²⁺ → Fe³⁺（氧化） $2Fe^{2+} + Cl_2 = 2Fe^{3+} + 2Cl^-$
$4Fe^{2+} + O_2 + 4H^+ = 4Fe^{3+} + 2H_2O$
$2Fe^{2+} + H_2O_2 + 2H^+ = 2Fe^{3+} + 2H_2O$
Fe³⁺ → Fe²⁺（还原） $2Fe^{3+} + Fe = 3Fe^{2+}$
$2Fe^{3+} + Cu = 2Fe^{2+} + Cu^{2+}$
$2Fe^{3+} + 2I^- = 2Fe^{2+} + I_2$
💡 Fe²⁺与Fe³⁺的检验：
Fe³⁺的检验：
1. KSCN溶液：溶液变血红色（$Fe^{3+} + 3SCN^- = Fe(SCN)_3$）
2. NaOH溶液：产生红褐色沉淀

Fe²⁺的检验：
1. 先加KSCN溶液无现象，再加氯水变血红色
2. K₃[Fe(CN)₆]溶液：产生蓝色沉淀（滕氏蓝）

第二章 非金属元素及其化合物

2.1 氯及其化合物

🔸 氯气（Cl₂）的性质
  • 物理性质：黄绿色气体，有刺激性气味，有毒，密度比空气大，可溶于水（1:2），易液化
  • 化学性质：强氧化性，可与金属、非金属、水、碱等反应
氯气的化学反应：
1. 与金属反应：
$2Na + Cl_2 \stackrel{点燃}{\longrightarrow} 2NaCl$（白烟）
$2Fe + 3Cl_2 \stackrel{点燃}{\longrightarrow} 2FeCl_3$（棕褐色烟）
$Cu + Cl_2 \stackrel{点燃}{\longrightarrow} CuCl_2$（棕黄色烟）

2. 与非金属反应：
$H_2 + Cl_2 \stackrel{点燃或光照}{\longrightarrow} 2HCl$

3. 与水反应：
$Cl_2 + H_2O \rightleftharpoons HCl + HClO$

4. 与碱反应：
$Cl_2 + 2NaOH = NaCl + NaClO + H_2O$（制漂白液）
$2Cl_2 + 2Ca(OH)_2 = CaCl_2 + Ca(ClO)_2 + 2H_2O$（制漂白粉）
🔸 次氯酸（HClO）与漂白粉
  • HClO性质：弱酸、强氧化性、不稳定性（$2HClO \stackrel{光照}{\longrightarrow} 2HCl + O_2↑$）
  • 漂白粉有效成分：Ca(ClO)₂
  • 漂白原理：$Ca(ClO)_2 + CO_2 + H_2O = CaCO_3↓ + 2HClO$
💡 氯水的成分与性质：
成分：三分子（Cl₂、H₂O、HClO）、四离子（H⁺、Cl⁻、ClO⁻、OH⁻）

性质：
1. 酸性：使石蕊试液先变红（H⁺）后褪色（HClO漂白）
2. 氧化性：氧化Fe²⁺、I⁻、S²⁻等
3. 漂白性：使有色布条褪色
4. 与AgNO₃反应：产生白色沉淀（Cl⁻）

2.2 硫及其化合物

🔸 硫（S）的性质
  • 物理性质：淡黄色固体，质脆，不溶于水，微溶于酒精，易溶于CS₂
  • 化学性质：既有氧化性又有还原性
硫的化学反应：
1. 与金属反应（氧化性）：
$Fe + S \stackrel{\Delta}{\longrightarrow} FeS$
$2Cu + S \stackrel{\Delta}{\longrightarrow} Cu_2S$

2. 与非金属反应：
$S + O_2 \stackrel{点燃}{\longrightarrow} SO_2$（还原性）
$S + H_2 \stackrel{\Delta}{\longrightarrow} H_2S$（氧化性）
🔸 二氧化硫（SO₂）
  • 物理性质：无色有刺激性气味气体，有毒，密度比空气大，易溶于水（1:40）
  • 化学性质：酸性氧化物、还原性、氧化性、漂白性
SO₂的化学反应：
1. 酸性氧化物通性：
$SO_2 + H_2O \rightleftharpoons H_2SO_3$
$SO_2 + 2NaOH = Na_2SO_3 + H_2O$

2. 还原性：
$2SO_2 + O_2 \underset{\Delta}{\stackrel{催化剂}{\rightleftharpoons}} 2SO_3$
$SO_2 + Cl_2 + 2H_2O = H_2SO_4 + 2HCl$

3. 氧化性：
$SO_2 + 2H_2S = 3S↓ + 2H_2O$

4. 漂白性：
使品红溶液褪色（可逆，加热恢复红色）
⚠️ SO₂与CO₂的鉴别：
  • 品红溶液：SO₂使品红褪色，CO₂不能
  • 酸性KMnO₄溶液：SO₂使其褪色（还原性），CO₂不能
  • 溴水：SO₂使溴水褪色，CO₂不能
🔸 硫酸（H₂SO₄）
  • 物理性质：无色油状液体，高沸点（338℃），难挥发，易溶于水放出大量热
  • 稀硫酸：具有酸的通性
  • 浓硫酸：吸水性、脱水性、强氧化性
浓硫酸的特性：
1. 吸水性：作干燥剂（不能干燥NH₃、H₂S等碱性或还原性气体）

2. 脱水性：将有机物中的H、O按2:1脱去（如使蔗糖炭化）

3. 强氧化性：
$Cu + 2H_2SO_4(浓) \stackrel{\Delta}{\longrightarrow} CuSO_4 + SO_2↑ + 2H_2O$
$C + 2H_2SO_4(浓) \stackrel{\Delta}{\longrightarrow} CO_2↑ + 2SO_2↑ + 2H_2O$

注意：常温下，浓硫酸使Fe、Al钝化

2.3 氮及其化合物

🔸 氮气（N₂）的性质
  • 物理性质：无色无味气体，密度略小于空气，难溶于水
  • 化学性质：稳定（N≡N键能大），高温或放电条件下反应
氮气的化学反应：
1. 与氧气反应（放电）：
$N_2 + O_2 \stackrel{放电}{\longrightarrow} 2NO$

2. 与氢气反应（合成氨）：
$N_2 + 3H_2 \underset{高温高压}{\stackrel{催化剂}{\rightleftharpoons}} 2NH_3$

3. 与镁反应：
$3Mg + N_2 \stackrel{点燃}{\longrightarrow} Mg_3N_2$
🔸 氮的氧化物
氧化物 NO NO₂
颜色 无色 红棕色
毒性 有毒 有毒
与水反应 不反应 $3NO_2 + H_2O = 2HNO_3 + NO$
与氧气反应 $2NO + O_2 = 2NO_2$ -
🔸 氨（NH₃）与铵盐
  • 物理性质：无色有刺激性气味气体，极易溶于水（1:700），易液化
  • 化学性质：碱性气体、还原性
氨的化学反应：
1. 与水反应：
$NH_3 + H_2O \rightleftharpoons NH_3·H_2O \rightleftharpoons NH_4^+ + OH^-$

2. 与酸反应：
$NH_3 + HCl = NH_4Cl$（白烟）

3. 还原性：
$4NH_3 + 5O_2 \underset{\Delta}{\stackrel{催化剂}{\longrightarrow}} 4NO + 6H_2O$（氨的催化氧化）

4. 实验室制法：
$2NH_4Cl + Ca(OH)_2 \stackrel{\Delta}{\longrightarrow} CaCl_2 + 2NH_3↑ + 2H_2O$
💡 NH₄⁺的检验：
加入NaOH溶液，加热，产生能使湿润红色石蕊试纸变蓝的气体（NH₃）。
$NH_4^+ + OH^- \stackrel{\Delta}{\longrightarrow} NH_3↑ + H_2O$
🔸 硝酸（HNO₃）
  • 物理性质：无色液体，易挥发，有刺激性气味
  • 化学性质：强酸性、强氧化性、不稳定性
硝酸的化学反应：
1. 不稳定性：
$4HNO_3 \stackrel{光照或加热}{\longrightarrow} 4NO_2↑ + O_2↑ + 2H_2O$（浓硝酸呈黄色是因为溶解了NO₂）

2. 强氧化性：
$Cu + 4HNO_3(浓) = Cu(NO_3)_2 + 2NO_2↑ + 2H_2O$
$3Cu + 8HNO_3(稀) = 3Cu(NO_3)_2 + 2NO↑ + 4H_2O$

注意：常温下，浓硝酸使Fe、Al钝化

2.4 硅及其化合物

🔸 硅（Si）的性质
  • 物理性质：灰黑色固体，有金属光泽，硬而脆，半导体材料
  • 化学性质：常温下稳定，高温下与某些物质反应
硅的化学反应：
1. 与氟气反应：
$Si + 2F_2 = SiF_4$

2. 与氢氟酸反应：
$Si + 4HF = SiF_4↑ + 2H_2↑$

3. 与氢氧化钠溶液反应：
$Si + 2NaOH + H_2O = Na_2SiO_3 + 2H_2↑$

4. 与氧气反应（高温）：
$Si + O_2 \stackrel{高温}{\longrightarrow} SiO_2$
🔸 二氧化硅（SiO₂）
  • 物理性质：无色晶体，熔点高，硬度大，不溶于水
  • 化学性质：酸性氧化物，但能与HF反应
SiO₂的化学反应：
1. 与碱反应：
$SiO_2 + 2NaOH = Na_2SiO_3 + H_2O$

2. 与碱性氧化物反应：
$SiO_2 + CaO \stackrel{高温}{\longrightarrow} CaSiO_3$

3. 与氢氟酸反应：
$SiO_2 + 4HF = SiF_4↑ + 2H_2O$

4. 与碳反应（高温）：
$SiO_2 + 2C \stackrel{高温}{\longrightarrow} Si + 2CO↑$（制粗硅）
💡 硅酸盐材料：
1. 水泥：硅酸盐材料，原料为石灰石和黏土
2. 玻璃：硅酸盐材料，原料为纯碱、石灰石和石英
3. 陶瓷：硅酸盐材料，原料为黏土

第三章 元素周期律

3.1 元素周期表的结构

💡 元素周期表的结构：
  • 周期：横行，共7个周期（3个短周期、4个长周期）
  • 族：纵列，共16个族（7个主族、7个副族、1个0族、1个第Ⅷ族）
  • 周期序数 = 电子层数
  • 主族序数 = 最外层电子数 = 最高正化合价

3.2 元素周期律

💡 元素周期律：元素的性质随着原子序数的递增而呈周期性变化。
性质 同周期（左→右） 同主族（上→下）
原子半径 逐渐减小 逐渐增大
主要化合价 最高正价：+1→+7（O、F除外）
负价：-4→-1		 相同
金属性 逐渐减弱 逐渐增强
非金属性 逐渐增强 逐渐减弱
最高价氧化物对应水化物酸碱性 碱性减弱，酸性增强 碱性增强，酸性减弱
气态氢化物稳定性 逐渐增强 逐渐减弱
金属性强弱判断依据：
1. 单质与水或酸反应置换出氢气的难易程度
2. 最高价氧化物对应水化物的碱性强弱
3. 单质间的置换反应

非金属性强弱判断依据：
1. 单质与氢气化合的难易程度
2. 气态氢化物的稳定性
3. 最高价氧化物对应水化物的酸性强弱
4. 单质间的置换反应

3.3 元素推断技巧

💡 元素推断的突破口：
  • 最外层电子数是内层电子数2倍：C
  • 最外层电子数是次外层电子数一半：Li、Si
  • 电子层数等于最外层电子数：H、Be、Al
  • 最外层电子数等于电子层数2倍：He、C、S
  • 地壳中含量最多的元素：O
  • 地壳中含量最多的金属元素：Al
  • 短周期中原子半径最大的元素：Na
  • 形成化合物种类最多的元素：C
  • 最高价氧化物对应水化物酸性最强的元素：Cl

第四章 原子结构与化学键（整合补充）

4.1 原子结构与元素周期表

原子组成关系：
原子序数 = 核电荷数 = 质子数 = 核外电子数（原子）
质量数(A) = 质子数(Z) + 中子数(N)
阴离子：核外电子数 = 质子数 + 电荷数
阳离子：核外电子数 = 质子数 - 电荷数
💡 周期表结构规律：
  • 周期：1-7周期元素种类数：2、8、8、18、18、32、32（不完全）
  • 族：主族（IA-VIIA）、副族（IB-VIIB）、VIII族（3列）、0族
  • 分区：s区（IA、IIA）、p区（IIIA-VIIA、0）、d区（IIIB-VIII）、ds区（IB、IIB）、f区（镧系、锕系）
  • 对角线规则：Li与Mg、Be与Al、B与Si性质相似

4.2 第一电离能与电负性

性质 同周期（左→右） 同主族（上→下）
第一电离能(I₁) 总体增大（IIA、VA反常） 逐渐减小
电负性 逐渐增大（F最大4.0） 逐渐减小
⚠️ 电离能反常现象（必须记住）：
  • IIA族(ns²全满) > IIIA族(ns²np¹)：如Mg(全满) > Al
  • VA族(ns²np³半满) > VIA族(ns²np⁴)：如P(半满) > S、N > O
  • 解释：全满、半满电子构型更稳定，失电子更难，I₁更高

4.3 微粒半径比较（三看原则）

🎯 半径比较方法：
  1. 先看电子层数：层数越多，半径越大。如：r(Na) > r(Li)，r(S²⁻) > r(O²⁻)
  2. 层同看核电荷数：核电荷数越大，半径越小（引力强）。如：r(Na⁺) > r(Mg²⁺) > r(Al³⁺)（均为10电子）
  3. 层同核电荷同看最外层电子数：电子数越多，半径越大（斥力大）。如：r(Fe²⁺) > r(Fe³⁺)

综合实例：r(S²⁻) > r(Cl⁻) > r(K⁺) > r(Ca²⁺)（均为18电子，核电荷数越大半径越小）

4.4 化学键类型与判断

💡 化学键类型判断：
  • 离子键：活泼金属（IA、IIA）与活泼非金属（VIA、VIIA）之间；NH₄⁺与酸根之间
  • 共价键：非金属之间；AlCl₃、BeCl₂等共价化合物（注意：Al、Be的化合物常为共价）
  • 金属键：金属单质、合金中
  • 配位键：一方提供孤对电子，一方提供空轨道。如NH₄⁺、H₃O⁺、[Cu(NH₃)₄]²⁺、Fe(SCN)₃
共价键参数：
键能：气态分子中，断开1mol化学键所需的能量。键能越大，键越牢固，分子越稳定。
键长：成键原子核间距。键长越短，键能越大。
键角：决定分子空间构型。

4.5 分子间作用力与氢键

作用力类型 强度 存在范围 对性质影响
范德华力 弱（几kJ/mol） 所有分子间 熔沸点：组成结构相似时，M越大，范德华力越大，熔沸点越高
氢键 较强（10-40 kJ/mol） H与N、O、F相连时 熔沸点：分子间氢键使熔沸点升高（H₂O、HF、NH₃）；分子内氢键使熔沸点降低
化学键 强（100-600 kJ/mol） 分子内或晶体内 决定化学性质
💡 氢键对性质的影响：
  • 熔沸点：H₂O > H₂Te > H₂Se > H₂S（水分子间氢键）；HF > HI > HBr > HCl
  • 溶解性：与水分子形成氢键的物质易溶于水（如乙醇、乙酸、氨气）
  • 密度：冰密度小于水（氢键形成四面体结构，体积膨胀）
  • 酸性：邻羟基苯甲酸酸性弱于对位（分子内氢键稳定了酸分子，难解离）

第五章 晶体结构（整合补充）

5.1 四大晶体类型对比

晶体类型 构成微粒 微粒间作用 熔沸点 硬度 导电性 实例
离子晶体 阴、阳离子 离子键 较高 较大 固态不导电，熔融或溶解导电 NaCl、CsCl、CaF₂
分子晶体 分子 分子间作用力（氢键） 低 小 一般不导电（部分溶于水导电） 干冰、冰、I₂、大多数有机物
原子晶体 原子 共价键 很高 很大 一般不导电（Si、Ge半导体） 金刚石、SiO₂、SiC、Si₃N₄、BN
金属晶体 金属阳离子+自由电子 金属键 一般较高（Hg低） 一般较大 良好导电导热性 金属单质、合金

5.2 晶体熔沸点比较规律

🎯 晶体熔沸点比较规律：
  1. 不同类型：原子晶体 > 离子晶体 > 分子晶体（一般规律，需结合具体数据）
  2. 同类型原子晶体：键长越短，键能越大，熔沸点越高。如：金刚石(C-C) > SiC(C-Si) > 晶体硅(Si-Si)
  3. 同类型离子晶体：离子半径越小，所带电荷越多，晶格能越大，熔沸点越高。如：MgO > NaCl（Mg²⁺半径小、电荷高）
  4. 同类型分子晶体：组成结构相似，相对分子质量越大，范德华力越大，熔沸点越高；存在氢键的异常高
  5. 金属晶体：阳离子半径越小，所带电荷越多，金属键越强，熔沸点越高。如：Al > Mg > Na

5.3 晶胞计算（均摊法）

均摊法计算晶胞中原子个数：
顶点：1/8；棱上：1/4；面上：1/2；内部：1

常见晶胞：
NaCl型：Na⁺（体心+12棱×1/4）= 4；Cl⁻（8顶点×1/8 + 6面×1/2）= 4
CsCl型：Cs⁺（体心）= 1；Cl⁻（8顶点×1/8）= 1
金刚石：8个C（8顶点×1/8 + 6面×1/2 + 4体内）= 8

第六章 金属元素拓展（整合补充）

6.1 碱金属元素通性

💡 碱金属（Li、Na、K、Rb、Cs、Fr）通性：
  • 最外层均为1个电子，易失去，强还原性
  • 同主族从上到下：原子半径增大，金属性增强，单质还原性增强
  • 与水反应剧烈程度：Cs > Rb > K > Na > Li
  • 密度总体增大（K反常，密度小于Na）
  • 熔沸点逐渐降低
  • Li特殊：与Mg性质相似（对角线规则），在空气中生成Li₂O（不是过氧化物）

6.2 铁的重要反应拓展

铁与不同氧化剂反应产物：
1. Cl₂（强氧化剂）：2Fe + 3Cl₂ $\stackrel{\Delta}{\longrightarrow}$ 2FeCl₃（Fe→+3价）

2. S（弱氧化剂）：Fe + S $\stackrel{\Delta}{\longrightarrow}$ FeS（Fe→+2价）

3. HNO₃（过量）：Fe + 4HNO₃(稀) = Fe(NO₃)₃ + NO↑ + 2H₂O（Fe→+3价）

4. HNO₃（Fe过量）：3Fe + 8HNO₃(稀) = 3Fe(NO₃)₂ + 2NO↑ + 4H₂O（Fe→+2价，因过量Fe与Fe³⁺反应：2Fe³⁺ + Fe = 3Fe²⁺）
【例题】Fe(OH)₂制备实验（防氧化是关键）

【实验目的】制备白色Fe(OH)₂沉淀，观察其颜色及氧化过程

【实验原理】
反应方程式：$FeSO_4 + 2NaOH = Fe(OH)_2\downarrow + Na_2SO_4$
Fe(OH)₂极易被氧化：$4Fe(OH)_2 + O_2 + 2H_2O = 4Fe(OH)_3$

【分析思路】
1. 核心问题：Fe(OH)₂为什么容易被氧化？
Fe²⁺具有还原性，空气中O₂具有氧化性，二者容易发生氧化还原反应
2. 防氧化策略：
• 排除溶液中的溶解氧
• 隔绝空气（液面上加保护层）
• 防止Fe²⁺被氧化（加还原剂）
3. 关键操作分析：
• FeSO₄现配现用 + 加铁粉：防止Fe²⁺被氧化为Fe³⁺
• NaOH煮沸：除去溶解氧
• 滴管插入液面下：避免滴加过程中带入空气
• 液面上加苯/煤油：隔绝空气

【实验步骤与要点】
  1. 配制FeSO₄溶液：现配现用，加入少量铁粉（Fe + 2Fe³⁺ = 3Fe²⁺），防止Fe²⁺被氧化
  2. 处理NaOH溶液：煮沸除去溶解的氧气，冷却后使用
  3. 滴加操作：将NaOH溶液的滴管插入FeSO₄溶液液面以下再挤出，避免带入空气
  4. 隔绝空气：在液面上加一层苯或煤油，密度小且不溶于水，可隔绝空气
  5. 观察现象：白色絮状沉淀 → 灰绿色（部分氧化） → 红褐色（完全氧化为Fe(OH)₃）

【现象解释】
• 白色：纯净的Fe(OH)₂沉淀
• 灰绿色：Fe(OH)₂与Fe(OH)₃的混合物
• 红褐色：Fe(OH)₃，Fe²⁺被完全氧化为Fe³⁺

【方法总结】
• 防氧化三要素：除氧、隔氧、加还原剂
• 实验设计原则：根据物质性质（还原性）设计防氧化措施
• 现象观察：颜色变化反映氧化程度，可用于判断反应进程

第七章 非金属元素拓展（整合补充）

7.1 卤族元素通性

💡 卤族元素（F、Cl、Br、I、At）通性：
  • 最外层均为7个电子，易得1个电子，强氧化性
  • 同主族从上到下：原子半径增大，非金属性减弱，单质氧化性减弱
  • 氧化性：F₂ > Cl₂ > Br₂ > I₂
  • 还原性：F⁻ < Cl⁻ < Br⁻ < I⁻
  • 置换反应：Cl₂ + 2Br⁻ = 2Cl⁻ + Br₂；Cl₂ + 2I⁻ = 2Cl⁻ + I₂
  • F₂特殊：能与水剧烈反应：2F₂ + 2H₂O = 4HF + O₂
  • HF特殊：弱酸，能腐蚀玻璃（SiO₂ + 4HF = SiF₄↑ + 2H₂O）

7.2 硫的化合物拓展

💡 SO₂与CO₂的鉴别方法：
  • 品红溶液：SO₂使品红褪色（可逆），CO₂不能
  • 酸性KMnO₄溶液：SO₂使其褪色（还原性），CO₂不能
  • 溴水：SO₂使溴水褪色，CO₂不能
  • FeCl₃溶液：SO₂使其褪色（2Fe³⁺ + SO₂ + 2H₂O = 2Fe²⁺ + SO₄²⁻ + 4H⁺）
⚠️ 浓硫酸的重要性质：
  • 吸水性：作干燥剂（不能干燥NH₃、H₂S等碱性或还原性气体）
  • 脱水性：将有机物中的H、O按2:1脱去（如使蔗糖炭化）
  • 强氧化性：与Cu、C等反应
  • 常温下使Fe、Al钝化（生成致密氧化膜）

7.3 氮的化合物拓展

💡 硝酸的重要性质：
  • 浓硝酸：强氧化性，与Cu反应生成NO₂，使铁铝钝化
  • 稀硝酸：强氧化性，与Cu反应生成NO
  • 极稀硝酸：与活泼金属反应可能生成NH₄⁺
  • 王水：浓HNO₃与浓HCl按1:3混合，能溶解Au、Pt
NOₓ与O₂、H₂O反应的计算关系：
4NO + 3O₂ + 2H₂O = 4HNO₃
4NO₂ + O₂ + 2H₂O = 4HNO₃

技巧：当V(NO₂):V(O₂) = 4:1时恰好完全反应；当V(NO):V(O₂) = 4:3时恰好完全反应

7.4 无机推断题突破口

💡 特征颜色与现象：
特征 物质
淡黄色固体 S、Na₂O₂、AgBr
红棕色固体/液体 Fe₂O₃、NO₂、液溴
黑色固体 Fe₃O₄、CuO、MnO₂、C、FeS
蓝色溶液 含Cu²⁺溶液
浅绿色溶液 含Fe²⁺溶液
黄色/棕黄色溶液 含Fe³⁺溶液
血红色溶液 Fe(SCN)₃
黄绿色气体 Cl₂
红棕色气体 NO₂
无色刺激性气体 SO₂、NH₃、HCl
💡 特征反应现象：
  • 焰色反应：Na⁺（黄色）、K⁺（紫色，透过蓝色钴玻璃）
  • 使品红褪色：SO₂（可逆）、Cl₂（不可逆）
  • 使澄清石灰水变浑浊：CO₂、SO₂
  • 与碱反应生成白色沉淀，后溶解：Al³⁺
  • 与酸反应生成气体：CO₃²⁻（CO₂）、HCO₃⁻（CO₂）、S²⁻（H₂S）、SO₃²⁻（SO₂）

7.5 铜及其化合物

🔸 Cu的溶解方法：
  • Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O（快，红棕色气体）
  • 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O（慢，无色变红棕）
  • Cu + 2H₂SO₄(浓) → CuSO₄ + SO₂↑ + 2H₂O（需加热）
💡 Cu²⁺颜色变化：
[Cu(H₂O)₄]²⁺（蓝色）→ 加Cl⁻ → [CuCl₄]²⁻（黄绿色/绿色）
→ 加过量NH₃ → [Cu(NH₃)₄]²⁺（深蓝色）

7.6 硅及其化合物

🔸 Si的特殊性：
  • 与酸反应：Si + 4HF → SiF₄↑ + 2H₂↑（唯一能与Si反应的酸）
  • 与碱反应：Si + 2NaOH + H₂O → Na₂SiO₃ + 2H₂↑
💡 SiO₂的性质：
  • 酸性氧化物，但与水不反应（不能直接制硅酸）
  • 与HF反应：SiO₂ + 4HF → SiF₄↑ + 2H₂O（刻蚀玻璃）
  • 与碱反应：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O（碱溶液不能用玻璃塞）
硅酸制备：Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃（强酸制弱酸）
